Impor job

Impor dari csv atau xlsx

Anda juga dapat mengimpor job dalam format teks umum

  • Klik File → Open → Impor Pekerjaan, dialog Open Folder muncul. Pilih C:\Program Files\Metamation\Praxis\Samples\Jobs\C0\C0.csv.

  • Jika semua Part ada di lokasi csv atau di direktori folder lookup, jendela pekerjaan baru muncul dengan mencantumkan semua part yang cocok dengan field csv.

  • Part yang hilang diabaikan dan tidak muncul di jendela pekerjaan baru.

  • Awalnya status Part adalah Nascent.

  • Klik OK, sekarang part diimpor ke dalam Part Library. Validasi CAD, tooling untuk bend dan cut tooling instance dilakukan yang dapat dipantau di layar Pmonitor.

Buatlah komponen lembaran kosong untuk file CAD yang tidak ada

Saat mengimpor job menggunakan spreadsheet (.csv atau .xlsx), Anda terkadang mungkin memiliki baris yang merujuk file CAD yang tidak tersedia di sistem.

  • Untuk menangani ini, buka pabrik → Pengaturan → Pekerjaan, di bawah Impor spreadsheet aktifkan Buatlah komponen lembaran kosong untuk file CAD yang tidak ada

Ketika opsi ini dipilih:

  • Jika file CAD hilang untuk setiap baris selama impor job, sistem akan secara otomatis membuat part kosong sebagai pengganti file CAD yang hilang.

  • Di jendela pekerjaan baru, part yang hilang ini secara visual disorot dengan warna kuning untuk menunjukkan bahwa file CAD terkait hilang.

  • Part yang hilang masih akan memiliki field yang dipetakan (seperti nama part, material, ketebalan, jumlah, dll.) dari CSV. Namun, status part akan ditampilkan sebagai Nascent, yang berarti ini hanya placeholder kosong tanpa geometri aktual belum.

Ini memastikan bahwa impor job masih dapat dilanjutkan tanpa kesalahan karena file CAD yang hilang — Anda dapat memperbarui atau mengganti part kosong dengan file CAD yang benar ketika tersedia nanti.

Jangan tampilkan kotak dialog Pekerjaan Baru jika spreadsheet tidak memiliki kondisi kegagalan

pabrik → Pengaturan → Pekerjaan → Impor spreadsheet→ Jangan tampilkan kotak dialog Pekerjaan Baru jika spreadsheet tidak memiliki kondisi kegagalan

  • Ketika checkbox ini dipilih, sistem secara otomatis membuat job baru tanpa menampilkan dialog New Job jika spreadsheet tidak memiliki kesalahan (tidak ada part yang hilang, tanggal yang hilang, atau data yang tidak valid).

  • Ketika checkbox ini dikosongkan, dialog New Job selalu muncul untuk tinjauan pengguna dan konfirmasi, bahkan jika spreadsheet tidak memiliki kesalahan.

Prefer raw-material column over material + thickness

pabrik → Pengaturan → Pekerjaan→Prefer Raw-Material column over Material + Thickness

  • Ketika checkbox ini dipilih, sistem mengharapkan spreadsheet menggunakan kolom Raw-Material (misalnya, Steel-20). Ini berarti tipe material dan ketebalan harus digabungkan dalam satu nilai dalam satu kolom.

  • Ketika checkbox ini dikosongkan, sistem mengharapkan spreadsheet memiliki kolom terpisah untuk Material dan Thickness (misalnya, Steel dan 2.00). Sistem menggabungkannya secara internal selama impor.

Watch folder menggunakan CSV/XLSX

  • Di pabrik → Pengaturan→ Pantau centang Aktifkan 'Folder Langsung' dan Pantau juga subfolder.

  • Tetapkan path watch folder (UNC atau mapped drive).

  • Di Impor spreadsheet, centang Impor spreadsheet dari 'Folder Langsung'.

  • Pilih Impor Pekerjaan dari opsi Spreadsheet Import.

  • Salin C:\Program Files\Metamation\Praxis\Samples\Jobs\C0\C0.csv dan paste ke dalam Watch Folder.

  • Sistem memeriksa CSV dan mencoba membuat job secara otomatis.

  • Jika file CAD hilang, tidak ada job yang dibuat karena data CAD yang diperlukan tidak ditemukan.

CAD lookup directory & recursive CAD search - tidak dikonfigurasi

  • Di pabrik → Pengaturan → Pekerjaan → Impor spreadsheet, jika tidak ada CAD lookup directories yang diatur dan Gunakan pencarian CAD rekursif tidak digunakan, sistem tidak mencari file CAD secara otomatis.

  • Jika checkbox Buatlah komponen lembaran kosong untuk file CAD yang tidak ada dipilih, job masih akan dibuat bahkan ketika file CAD hilang.

  • Dalam hal ini, semua part dalam job akan dibuat sebagai part nascent (kosong) — yang berarti mereka tidak memiliki geometri karena file CAD tidak dapat ditemukan.

CAD lookup directory & recursive CAD search - dikonfigurasi

Ketika Direktori pencarian CAD diatur ke (C:\Program Files\Metamation\Praxis\Samples) dan Gunakan pencarian CAD rekursif diaktifkan di bawah pabrik → Pengaturan → Pekerjaan → Impor spreadsheet:

  • Sistem secara otomatis mencari folder yang ditentukan dan semua subfolder-nya untuk file CAD yang diperlukan yang disebutkan dalam spreadsheet.

  • Buatlah komponen lembaran kosong untuk file CAD yang tidak ada juga diaktifkan, jadi jika ada file CAD yang masih tidak ditemukan, sistem akan membuat part kosong sebagai placeholder.
