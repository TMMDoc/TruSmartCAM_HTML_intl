Impor part

Sebelum mengimpor part, pastikan format file kompatibel dengan TruSmartCAM. Lihat format file yang didukung untuk detail lebih lanjut.

Jika material dan ketebalan tidak dipetakan ke part setelah impor, tetapkan material menggunakan panel part.

Metode impor

Ada tiga metode untuk mengimpor part ke dalam Part Library:

  1. Impor melalui file

  2. Drag and drop

  3. Watch folder

Impor melalui file

File part 2D atau 3D dapat diimpor ke dalam Part Library menggunakan opsi Impor komponen.

  • Klik ikon buka dari command bar dan pilih opsi Impor komponen.

  • Dialog Open muncul — pilih file CAD apapun (2D atau 3D).

  • Pratinjau file CAD yang dipilih ditampilkan.

  • Pilih part yang diperlukan dan klik Open.

  • Sistem secara otomatis mem-tool part untuk semua mesin yang tersedia dengan status tooling yang diperbarui.

Drag and drop

  • Drag and drop file part langsung ke dalam TruSmartCAM Part Library.

Watch folder

Watch settings memungkinkan Anda memantau folder untuk file baru dan secara otomatis mengimpornya ke dalam Part Library.

Untuk konfigurasi, lihat halaman Watch settings.

  • Gunakan path UNC (Universal Naming Convention) \\192.168.1.11\<Folder path> atau mapped drive M:\<Folder Path>.

  • PC yang menjalankan Engine harus memiliki akses read/write ke folder ini.

  • Jangan gunakan path lokal seperti C:\<Folder Path> karena layanan Watch mencari file di PC Engine — bukan PC lokal.

Konfigurasi impor

Gunakan perbandingan geometri pada revisi CAD yang sama

Saat mengimpor part dengan revisi CAD yang sama, TruSmartCAM dapat membandingkan geometri file CAD dengan part yang ada.

  • Untuk mengaktifkan ini, konfigurasi Tombol pencarian revisi (misalnya, REV) di pabrik → pengaturan →Impor/Ekspor. Kemudian, aktifkan opsi Gunakan perbandingan geometri pada revisi CAD yang sama.

Perbandingan geometri pada revisi CAD yang sama hanya berfungsi ketika Tombol pencarian revisi dikonfigurasi.

  • Sekarang, impor part ke dalam part library dengan revisi tertentu (misalnya, Revision 1).

  • Part yang sama diimpor lagi dengan nilai revisi yang sama.

  • TruSmartCAM membandingkan geometri file CAD yang baru diimpor dengan part yang ada.

  • Jika perbedaan geometri terdeteksi, TruSmartCAM menampilkan prompt yang menanyakan apakah akan:

    • Ganti part yang ada dengan geometri baru, mempertahankan nilai revisi yang sama.

    • Sembunyikan impor, mempertahankan part yang ada dan geometrinya.

Ini memungkinkan pengguna memperbarui geometri part tanpa mengubah nilai revisi, sambil tetap mempertahankan kontrol apakah part yang ada diganti.

Part yang diimpor secara otomatis ditambahkan ke library untuk penggunaan lebih lanjut. Untuk informasi lebih lanjut, lihat Library.
