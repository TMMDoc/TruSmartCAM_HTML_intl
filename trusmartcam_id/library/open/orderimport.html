Impor order

Order dapat dibuat atau diimpor menggunakan metode berikut:

  1. Mengimpor order dari file spreadsheet.

  2. Mengimpor order secara otomatis melalui watch settings.

  3. Membuat order dari part yang ada di part library.

  4. Membuat order dari assembly static nests.

Impor order dari spreadsheet

  • Navigasi ke buka → Impor _Tugas…​.

  • Format spreadsheet yang didukung termasuk CSV, XLSX, dan XML.

  • Pilih file spreadsheet yang diperlukan dan klik buka.

  • Proses impor mirip dengan alur kerja impor Job/Assembly.

Anda juga dapat langsung drag and drop spreadsheet ke halaman Orders untuk mengimpor order.

Impor order menggunakan watch settings

  • Aktifkan opsi Impor spreadsheet dari 'Folder Langsung' di Pengaturan pemantauan.

  • Pilih Impor Tugas di Opsi impor spreadsheet.

  • Konfigurasi lokasi folder dari mana order harus diimpor.

  • Sistem terus memantau folder yang dikonfigurasi.

  • File spreadsheet yang baru ditambahkan secara otomatis diimpor ke Orders.

Untuk langkah konfigurasi terperinci, lihat dokumentasi Watch settings.

Membuat order dari part library

  • Pilih part yang diperlukan dari Part Library.

  • Klik kanan dan pilih Baru…​ → Tugas untuk membuka dialog Tugas Baru.

  • Masukkan detail yang diperlukan seperti jumlah, tanggal jatuh tempo, dan prioritas.

  • Klik OK untuk membuat order.

  • Sistem secara otomatis beralih ke halaman Orders.

  • Order yang dibuat ditampilkan di halaman Orders Ready untuk pembuatan job.

Membuat order dari assembly

  • Klik kanan Assembly yang diperlukan.

  • Pilih Baru…​ → Tugas untuk membuka dialog Tugas Baru.

  • Masukkan Jumlah modul yang diperlukan.

  • Klik OK untuk membuat order.
