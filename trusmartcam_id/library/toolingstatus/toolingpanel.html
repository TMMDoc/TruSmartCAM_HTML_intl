Ekspor Output

Opsi ini mengekspor output part ke tujuan yang dikonfigurasi di Pengaturan Pabrik. (Lihat Bend Settings dan Cut Settings)

Hanya tooling yang terkait dengan operasi tooling yang dipilih yang diekspor, bukan seluruh part.

File yang diekspor tergantung pada tipe operasi. Untuk operasi Bend dan Fold, file NC dan laporan dihasilkan. Untuk operasi Cut dan Punch, hanya file part yang diekspor.

Check Tooling

  • Jika ada perubahan dalam CAM Settings atau bend tool hilang, Anda tidak perlu melakukan Re-tool part.

  • Sebaliknya, gunakan Check Tooling untuk memvalidasi tooling yang ada tanpa membuat perubahan apapun.

  • Jika validasi gagal, status tooling dari instance tersebut diperbarui, dan file output terkait dihapus.

  • Check Tooling juga mendeteksi part di mana kesalahan sebelumnya secara manual ditimpa menggunakan Abaikan Kondisi Kegagalan, dan memperbarui statusnya menjadi error jika diperlukan.

Salin Pengaturan Alat…​

Copy tooling memungkinkan Anda dengan cepat menerapkan bend tooling dari satu solusi ke solusi lainnya, menghemat waktu dan usaha.

Ini menyalin punch, urutan, dan pengaturan back gauge dari tooling referensi yang dipilih (solusi) dan mencoba menerapkan pengaturan yang sama ke semua solusi lainnya.

  • Jika pengaturan yang disalin layak untuk mesin target, itu disimpan dan digunakan.

  • Jika pengaturan tidak layak, tooling asli untuk mesin tersebut tetap tidak berubah.

Misalnya, Anda mungkin memiliki tooling yang berbeda untuk mesin yang berbeda:

  • Di sini, Machine 5230SX 6A menggunakan Gooseneck punch, Machine 5085X 6A B23 menggunakan Pointed punch.

Ketika Anda menggunakan Salin Pengaturan Alat…​ pada Machine 5230SX 6A, itu mencoba menyalin pengaturan Gooseneck ke semua mesin lain.

Punch, urutan, dan pengaturan back gauge yang sama diterapkan di mana pun memungkinkan. Jika pengaturan yang disalin tidak dapat digunakan, tooling asli tetap tidak berubah.

Setelah menerapkan copy tooling, pengaturan Gooseneck telah berhasil diterapkan ke semua mesin lain di mana itu layak.

Atur sebagai Flat Dasar…​

Fitur Atur sebagai Flat Dasar…​ mendefinisikan bend atau fold tooling mana yang digunakan sebagai ukuran flat referensi untuk menghasilkan solusi pemotongan. Ketika tooling ditandai sebagai Base-Flat, TruSmartCAM memperbarui ukuran flat part berdasarkan mesin tersebut dan meregenerasi solusi pemotongan sesuai. Tooling Base-Flat ditampilkan dengan border biru di UI.

Mesin bend yang berbeda dapat menghasilkan variasi sedikit dalam ukuran flat untuk part yang sama. Ketika Anda menetapkan tooling sebagai Base-Flat, TruSmartCAM menggunakan ukuran flat mesin tersebut sebagai referensi utama dan secara otomatis menghitung ulang solusi pemotongan.

Pemilihan base flat otomatis

Ketika pengaturan factory Terapkan koreksi flat selama proses tekuk diaktifkan:

  • TruSmartCAM secara otomatis menetapkan bend tooling (yang berisi tabel max bend length) sebagai Base-Flat selama impor part.

  • Jika bend tooling yang dipilih mengalami kesalahan atau tidak layak, TruSmartCAM secara cerdas beralih dan menetapkan bend tooling valid lainnya sebagai Base-Flat. Ini memastikan referensi ukuran flat dan solusi pemotongan selalu tetap konsisten dan valid tanpa intervensi manual.

Manajemen base flat yang ditingkatkan

TruSmartCAM menyediakan opsi Base-Flat di dialog Simpan ke Praxis saat menyimpan pengeditan bend atau fold tooling. Opsi ini hanya muncul untuk tooling dalam status OK dan memastikan bahwa ukuran flat dan solusi pemotongan tetap akurat.

Tetapkan sebagai Flat Dasar dan hitung solusi pemotongan

Opsi ini digunakan untuk menandai tooling sebagai Base-Flat untuk pertama kalinya dan secara otomatis menghasilkan solusi pemotongan yang sesuai.

Muncul ketika:

  • Tooling saat ini tidak ditandai sebagai Base-Flat.

  • Pengguna telah check out tooling untuk pengeditan dan menyimpan perubahan.

Untuk menggunakan opsi ini, check out bend atau fold tooling dan buat modifikasi yang diperlukan, seperti perubahan pada parameter bend atau ukuran flat. Tekan Ctrl+S untuk menyimpan perubahan. Di dialog Simpan ke Praxis, pilih Tetapkan sebagai Flat Dasar dan hitung solusi pemotongan dan klik OK. TruSmartCAM akan menetapkan tooling sebagai Base-Flat dan menghasilkan solusi pemotongan menggunakan ukuran flat yang diperbarui.

Perbarui Flat Dasar dan hitung ulang solusi pemotongan

Opsi ini digunakan ketika tooling sudah ditandai sebagai Base-Flat dan telah diedit, memerlukan ukuran flat dan solusi pemotongan untuk diregenerasi.

Muncul ketika:

  • Tooling sudah ditandai sebagai Base-Flat.

  • Pengguna telah check out, mengeditnya, dan menyimpan perubahan.

Saat mengedit tooling Base-Flat yang ada, perbarui parameter bend atau geometri flat-size sesuai kebutuhan. Di dialog Simpan ke Praxis, pilih Perbarui Flat Dasar dan hitung ulang solusi pemotongan. TruSmartCAM kemudian akan memperbarui tooling Base-Flat dan meregenerasi semua solusi pemotongan terkait untuk mencerminkan perubahan terbaru.

Hanya satu bend tooling yang dapat ditetapkan sebagai Base-Flat pada satu waktu.

Ketika Base-Flat diubah, solusi pemotongan yang ada secara otomatis diproses ulang.

Fitur ini tersedia untuk bend dan fold tooling dalam TruSmartCAM.

Atur ulang alat…

Opsi ini memproses ulang tooling yang dipilih untuk part.

Klik kanan tooling yang diperlukan di panel Tooling dan klik Atur ulang alat…

Dialog konfirmasi ditampilkan sebelum melanjutkan.

Klik Ya untuk melanjutkan dengan re-tooling, atau Tidak untuk membatalkan operasi.

Hanya tooling yang dipilih yang di-re-tool. Tooling lain yang terkait dengan part tidak terpengaruh. Output yang ada terkait dengan tooling yang dipilih dihapus dan diregenerasi selama proses re-tooling.

Gunakan opsi ini ketika data tooling sudah usang, tidak benar, atau ketika perubahan dibuat pada konfigurasi mesin atau tooling.

Abaikan Kondisi Kegagalan

  • Jika instance yang di-tool menampilkan peringatan, dan Anda masih ingin melanjutkan dengan itu, Anda dapat menimpa peringatan dengan mengklik Abaikan Kondisi Kegagalan.

  • Ini akan memungkinkan sistem untuk mengekspor file output untuk instance tersebut, bahkan jika ada masalah tooling.

Gunakan Ignore Errors hanya ketika Anda yakin peringatan dapat dengan aman diabaikan. Menimpa kesalahan aktual dapat menyebabkan kerusakan mesin atau risiko keselamatan.

Suppress/Unsuppress tooling

  • Ketika mesin tidak aktif atau Anda tidak ingin part tertentu diproses oleh mesin tertentu, gunakan Sembunyikan Pengaturan Alat untuk mengecualikan part tersebut dari nesting pada mesin tersebut.

  • Setelah menekan, mengarahkan mouse ke atas instance yang di-tool menampilkan pesan seperti: "Tooling suppressed by <User name> on <Date Time>".

  • Suppress tooling menghapus file output untuk mesin dan part tersebut.

  • Gunakan Tampilkan Pengaturan Alat untuk mengembalikan status tooling untuk part pada mesin tersebut.

  • Ketika unsuppressed, file output yang bersangkutan dihasilkan lagi.
