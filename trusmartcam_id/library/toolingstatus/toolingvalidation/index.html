Validasi tooling

Bagian ini menjelaskan cara mengakses dan memodifikasi informasi tooling untuk part dalam TruSmartCAM.

  • Klik kanan pada tile part apapun untuk membuka panel part dan mengakses opsi tooling.

  • Klik Pengaturan alat untuk beralih ke instance yang di-tool pertama yang tersedia.

  • Atau, klik langsung pada instance yang di-tool untuk melihat detail mesin dan opsi tooling.

  • Gunakan untuk Buka dalam mode dilindungi dari penulisan untuk melihat.

  • Gunakan untuk Check-out dokumen untuk diedit.

  • Instance yang di-tool yang dipilih terbuka di TecZone atau MetaCAM.

  • Lakukan pemeriksaan tooling manual dan buat perubahan yang diperlukan.

  • Ketika Anda mencoba menyimpan, dialog Unsaved Changes meminta Anda mengonfirmasi edit Anda.

  • Klik Yes untuk melanjutkan dengan penyimpanan.

  • Sistem melakukan Validasi Tooling lagi setelah penyimpanan.

  • Jika tidak ada kesalahan, opsi untuk Save changes to TruSmartCAM muncul.

  • Jika kesalahan tooling ditemukan, pesan kesalahan ditampilkan:

  • Setelah menyimpan, status tooling part diperbarui di part library.

  • Warna hijau pada instance yang di-tool menunjukkan itu sudah checked-in.

  • Tooled Instance Author diperbarui ke orang terakhir yang mengeditnya.

  • Field Diedit oleh pengguna menunjukkan apakah instance yang di-tool telah dimodifikasi secara manual. Ketika pengguna check out instance dan melakukan perubahan apapun, field ini secara otomatis diatur ke Yes saat penyimpanan.

  • Jika part sedang diedit, ikon pengguna muncul pada tilenya.

  • Mengarahkan mouse ke atas tile menampilkan siapa yang check out dan waktu check out.

  • Saat part sedang checked out, opsi Check-out dokumen untuk diedit dinonaktifkan untuk instance yang di-tool yang sama, dan opsi Batalkan Check-out menjadi diaktifkan.
