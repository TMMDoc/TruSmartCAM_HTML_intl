Instalasi

Persyaratan wajib

Masuk ke TruSmartCAM desktop adalah wajib saat meluncurkan SolidWorks untuk pertama kali.

Pastikan bahwa monitor TruSmartCAM berjalan di area notifikasi windows.

Jika TruSmartCAM desktop tidak terbuka, verifikasi bahwa layanan server TruSmartCAM berjalan untuk memungkinkan Add-in SolidWorks berfungsi. Jika keduanya tidak aktif, SolidWorks akan menampilkan pesan error saat diluncurkan.

Ikuti langkah-langkah berikut untuk menginstal dan mengatur Add-in TruSmartCAM untuk SolidWorks:

  1. Jalankan Setup TruSmartCAM, dan pada halaman pemilihan Components, aktifkan SolidWorks Extension. (Lewati opsi Server jika SolidWorks diinstal pada klien TruSmartCAM.)

  2. Selesaikan instalasi, kemudian luncurkan TruSmartCAM desktop dan masuk dengan kredensial pengguna yang valid.

  3. Luncurkan SolidWorks.

  4. Di SolidWorks, buka Tools → TruSmartCAM → About TruSmartCAM, untuk memverifikasi versi dan detail add-in.

  5. Untuk mengaktifkan atau menonaktifkan add-in saat startup, buka Tools → Add-ins, dan temukan TruSmartCAM di bawah Other Add-ins.
