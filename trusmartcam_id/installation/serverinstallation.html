Server installation

Ikuti instruksi untuk menginstal instalasi TruSmartCAM Server bersama dengan SQL.

Administrator Rights

Untuk menjalankan instalasi ini, pengguna harus memiliki hak administratif.

  • Jalankan Setup.TruSmartCAM.<version>.exe untuk memulai instalasi.

Version adalah nomor.

  • Baca perjanjian lisensi dan pilih I accept the agreement dan kemudian Klik Berikutnya .

Step 1 : Install Location

  • Sebutkan direktori instalasi atau terima default instalasi C:\Program Files\Metamation\Praxis dan klik Berikutnya.

    • C:\Program Files\Metamation\Praxis - Folder Install akan memiliki folder berikut : Bin, MetaCAM, Samples dan Tracker.

    • C:\Program Data\Metamation - Folder Data akan memiliki folder berikut : TruSmartCAM, TecZone dan MetaCAM.

Step 2 : Components Selection

Installation Type Components

Full

Server + Desktop client + Engine

Client

Desktop Client

Server

Server + Desktop client

Station

Picks all Stations Terminal like TecZone, MetaCAM, RightAngle(Bend Controller), Vulcan(Laser Machine Controller)

Custom

Picks by user choice.

  • Pilih Full Installation. Pilih Code Senders dan Bend Code Senders dari TruSmartCAM Tools. Sekarang Installation type berubah menjadi Khusus. Klik Berikutnya.

Step 3 : Choose Install SQL Server

Instalasi SQL Express 2022 hanya diperlukan untuk TruSmartCAM Server. Opsi ini tidak tersedia untuk Instalasi Client dan Sync. Juga ini Dicentang ON dan diinstal untuk pertama kalinya dan dapat diabaikan setelahnya. Jika SQL sudah diinstal di server TruSmartCAM, pengguna dapat mengabaikan Step 3.

  • Dalam Select Additional Tasks : Klik Install SQL Express dan Klik Berikutnya.

  • Dalam Ready to Install : Rangkum semua fitur instalasi dan Klik Install.

Step 4 : SQL Server Installation

Setelah komponen TruSmartCAM diinstal, installer melanjutkan dengan instalasi server SQL. Terima License Agreement untuk melanjutkan dan menginstal SQL express dengan opsi default.

Setelah instalasi selesai Klik Close dan selesaikan instalasi.

Tergantung pada versi Windows, menutup instalasi SQL dapat meminta restart.

Step 5 : Windows Desktop shortcuts

  • Setelah instalasi SQL, jika Sistem tidak restart, maka pilih checkbox Launch TruSmartCAM dan Klik Finish.

  • Jika Sistem restart, maka Luncurkan ikon shortcut TruSmartCAM di Desktop atau ketik TruSmartCAM di Windows Start Program dan Run as administrator.

Server installation : Ikon shortcut TruSmartCAM dan TruSmartCAM License tersedia di Desktop. + Client/SYNC Installation : Hanya ikon shortcut TruSmartCAM tersedia di Desktop.

Step 6 : License Registration

  • Di field Serial Number, ketik TruSmartCAM Serial key yang disediakan oleh Trumpf Metamation Service Team. Ketik <Company Name> di field "Registered to" dan E-Mail. Klik OK.

Jika firewall Jaringan memblokir registrasi lisensi atau Tidak ada internet di PC Server TruSmartCAM, maka lisensi TruSmartCAM dilakukan dengan aktivasi Offline. Email Prax.request dari Folder C:\ProgramData\Metamation\Praxis ke Trumpf Metamation Service Team. Trumpf Metamation Team akan menyediakan Prax.lock yang perlu dijatuhkan di folder ini.

Step 7 : Initial one time Configuration

  • Ketik Nama Database dan Atur Lokasi Penyimpanan Data. Pilih checkbox Gunakan Server SQL dan pilih nama server SQL untuk terhubung ke Database. Klik OK.

  • TruSmartCAM Desktop Client membuka menampilkan Database Connection Succeeded di bagian bawah.

  • File konfigurasi TruSmartCAM, TecZone dan MetaCAM dibuat dengan mesin config default atau terhadap mesin lisensi TruSmartCAM. Config yang sama akan dibagikan di semua TruSmartCAM Clients.

Layar Welcome menampilkan dengan Windows User LoginName sebagai TruSmartCAM Login dan Name. Pengguna TruSmartCAM pertama dianggap sebagai Admin Situs. Berikan Email dan Password. Klik OK .

Pilih checkbox Ingat saya untuk menyimpan kredensial login untuk waktu berikutnya. Hapus centang checkbox Ingat saya akan meminta dialog login dan kata sandi setiap kali aplikasi TruSmartCAM diluncurkan.

Step 8 : TruSmartCAM sign-in

  • Tutup TruSmartCAM dan buka kembali. Sistem meminta Anda untuk memasukkan kredensial login TruSmartCAM.

  • Nama pengguna yang masuk ditampilkan di sebelah ikon pengguna di header aplikasi TruSmartCAM.

  • Ketika Anda mengarahkan kursor ke ikon pengguna, sistem menampilkan:

    • Peran pengguna (misalnya: Programmer, Admin).

    • Pintasan untuk keluar: Ctrl+L.

  • Mengklik ikon pengguna membuka menu yang memungkinkan Anda untuk:

    • Edit informasi…​ - Perbarui detail profil Anda.

    • Ubah kata sandi…​ - Ubah kata sandi akun Anda.

    • Keluar - Keluar dari aplikasi.

    • Batal - Tutup menu tanpa membuat perubahan apa pun.

Step 9 : Factory Machine Configuration

  • TruSmartCAM License [All machines enables license bit] : Default TruSmartCAM Machine dan Material dianggap sebagai Factory Configuration.

  • TruSmartCAM License [No demo and few machine bit enabled] : Beberapa Mesin tersebut dan Material default dianggap sebagai Factory Configuration.

  • (Open → Import) Part baru dari C:\Program Files\Metamation\Praxis\Samples\Parts dan verifikasi apakah Part diimpor dengan sukses.

Note 1 : TruSmartCAM Monitor

  • TruSmartCAM Monitor dengan Engine Installation (Site admin rights):

    • TruSmartCAM Monitor dengan Engine bersama dengan Site Admin TruSmartCAM Rights akan memiliki opsi berikut:

      1. Push konfigurasi TruSmartCAM, konfigurasi TecZone, MetaCAM di semua TruSmartCAM Clients. Juga memiliki hak untuk membuat Login Pengguna TruSmartCAM Baru.

      2. Show Agents akan menampilkan total engine yang tersedia untuk melakukan tugas otomatisasi yang terhadap TruSmartCAM License.

  • TruSmartCAM Monitor tanpa Engine Installation (Site admin rights):

  • TruSmartCAM Monitor tanpa Engine Installation (Non Site Admin rights):

PC yang menjalankan aplikasi TruSmartCAM harus memiliki komponen Engine TruSmartCAM terinstal [Server PC atau di instalasi Client mana pun]. PC yang terinstal komponen Engine TruSmartCAM harus berjalan 24x7 untuk melakukan semua tugas otomatisasi.

Note 2 : Lock Tracker

  • Klik shortcut TruSmartCAM License yang tersedia di Desktop Server TruSmartCAM. Atau, buka url http://localhost:7171/metamation/locktrack di browser web mana pun .

Ini akan membuka modul Lock Tracker yang memiliki informasi Lisensi, informasi sesi TruSmartCAM saat ini dan Log.

LockTracker HOME Page : Ini menampilkan Nomor Serial TruSmartCAM, informasi Kadaluwarsa Serial dan informasi bit Serial yang tersedia.

LockTracker SESSIONS Page : Informasi koneksi Server TruSmartCAM saat ini seperti PC client Desktop TruSmartCAM, PMonitor, Status PEngine.

LockTracker LOGs Page : Client TruSmartCAM mulai dan keluar | Pmonitor mulai dan status koneksi/pemutusan PEngine dll., dapat dilihat di halaman Logs ini.

Note 3 : OpenGL

  • Aplikasi TruSmartCAM saat membuka yang tertutup segera dan juga pmonitor tidak ditampilkan di task bar.

  • Ini merepresentasikan masalah OpenGl.

  • Cara lain untuk mengonfirmasi ini: Buka C:\Program Files\Metamation\Praxis\Bin\Flux.exe yang menampilkan dialog kesalahan tentang masalah OpenGL.

  • Perbarui C:\ProgramData\Metamation\Praxis\Prax.ini, bagian options dengan GLVersion=1 membuat aplikasi TruSmartCAM membuka dengan sukses.

[Options]
GLVersion=1.0
