Port Requirement

TruSmartCAM Server : 7371

Server menunggu permintaan client pada port TCP ini. Koneksi masuk harus diizinkan ke port ini di Komputer Server.

Floating License Server : 5776

Floating License Server menunggu permintaan lisensi pada port ini. Koneksi masuk harus diizinkan ke port ini di komputer floating license server (TruSmartCAM Server).

License connection to Cloud : 7567

Port ini digunakan oleh TecZone (Bending Software) dan Floating License Server untuk mengirim permintaan lisensi ke Metamation Cloud. Jadi koneksi keluar harus diizinkan pada port ini untuk aktivasi dan pembaruan lisensi.

Installation and Support Requirements

  • Hak administratif diperlukan untuk menginstal aplikasi TruSmartCAM.

  • Alat diagnostik jarak jauh melalui Contractor VPN atau koneksi yang aman.
