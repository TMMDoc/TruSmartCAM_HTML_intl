Pengaturan output

Simpan output pemotongan - Hasil tooling disimpan hanya jika tooling OK dan pengaturan ekspor ini diaktifkan dan dikonfigurasi. Aktifkan opsi ini agar Cut NC disimpan.

Pratinjau simulasi pengaturan alat output - Aktifkan opsi ini untuk menghasilkan preview simulasi tooling bersama dengan output lainnya. Ini adalah animasi GIF pendek untuk tooling laser/punch.

Tujuan output pemotongan - Ini adalah tujuan untuk NC dan laporan yang akan disimpan.

Pengaturan MetaCAM

Cari bahan MetaCAM berdasarkan nama bahan baku + (Aktifkan ini untuk integrasi seperti DMM) - Ketika gaya ini diaktifkan, TruSmartCAM menghasilkan halaman berdasarkan nama raw-material dan menetapkannya ke part pada waktu auto-tooling.

Folder sinkronisasi MetaCAM - Pengaturan Cut ini digunakan untuk menambahkan folder MetaCAM tambahan yang harus disinkronkan bersama dengan config.dat ketika perintah config-push Monitor digunakan. Config-push menyimpan file lokal ke repositori TruSmartCAM yang diunduh di komputer lain ketika MetaCAM (re)mulai. Komputer Engine harus menyinkronkannya secara otomatis ketika auto tooling selesai untuk mesin MetaCAM selain Tools, beberapa kandidat spesifik instalasi lainnya adalah folder Etc, Reports, dan Machines.

Tempat sinkronisasi konfigurasi DMM - Ini adalah lokasi untuk sinkronisasi config Dynamic Machine Management yang akan diatur.

Pengaturan Flux

Aktifkan nesting multi-pass - Ini akan mengaktifkan laser untuk memotong beberapa kali pada kontur yang sama (lebih disukai untuk material yang lebih tebal).
