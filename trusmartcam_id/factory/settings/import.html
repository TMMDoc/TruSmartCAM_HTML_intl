Import

Pengaturan output

Simpan pola flat - Aktifkan opsi ini agar pola flat dari part yang diimpor disimpan.

Format untuk pola flat - Opsi ini digunakan untuk memilih format file DXF atau GEO.

Tujuan pola datar - Ini adalah lokasi di mana pola flat akan disimpan.

Pengaturan impor

Satuan untuk file DXF - Ini adalah unit pengukuran file DXF yang akan diimpor, pengguna dapat memilih antara mm atau inci.

Tombol pencarian bahan - Beberapa entri pencarian dapat ditugaskan menggunakan pemisah koma di antaranya. Ketika part diimpor, TruSmartCAM menggunakan kunci ini untuk mencari nilai material. Nilai yang dicari kemudian digunakan untuk menyelesaikan raw material. Nilai asli disimpan di Misc Properties part.

Tombol pencarian revisi - Ketika part diimpor, nilai revisi dari file part mentah (file part yang diimpor) dibandingkan dengan nilai revisi yang ada di repositori. Impor ditinggalkan jika revisi sama. Pemetaan field kustom sekarang juga didukung untuk file DXF. Untuk file DXF, data field kustom diekstraksi dari layer teks di gambar atau lampiran PPI.

Gunakan perbandingan geometri pada revisi CAD yang sama - Membandingkan geometri saat mengimpor part dengan revisi CAD yang sama dan meminta pengguna untuk mengganti atau melewati impor jika perbedaan terdeteksi, Lihat Import configuration untuk detail lebih lanjut.

Tombol nama komponen - Opsi ini bekerja mirip dengan material dan Tombol pencarian revisi. Atur satu atau lebih Tombol nama komponen untuk menggunakan PPI Nama komponen daripada nama file sebagai nama part.

Tombol pencarian ketebalan - Beberapa entri pencarian dapat ditugaskan menggunakan pemisah koma di antaranya. Ketika part diimpor, TruSmartCAM menggunakan kunci ini untuk mencari nilai ketebalan. Nilai yang dicari kemudian digunakan untuk menyelesaikan ketebalan. Nilai asli disimpan di Misc Properties part.

Toleransi perbandingan ketebalan - Ini digunakan dalam mencari raw material ketika part diimpor. TruSmartCAM mencoba menyelesaikan raw material setelah nama material dan ketebalan diurai. Untuk informasi lebih lanjut, lihat topik Thickness fudge di bawah.

Mengidentifikasi bentuk setelah flat diimpor - Ketika part diimpor, bentuk dalam di dalam part dapat secara otomatis dikenali dan diidentifikasi.

Toleransi deteksi bentuk - Nilai fudge identifikasi bentuk dapat diatur di:

  1. Global level - Nilai ini digunakan sebagai default untuk semua bentuk.

  2. Individual Shape Level - Nilai default dapat ditimpa dengan mengatur nilai fudge di tingkat bentuk individual. Memilih bentuk dalam dialog Shape Library menampilkan nilai fudge. Secara default, ini diatur ke 0, dan nilai global default digunakan untuk bentuk tersebut.

Putar flat secara horizontal - Umumnya meja mesin pemotong berbentuk persegi panjang dan lebih lebar. Untuk meningkatkan kelayakan tooling, putar part sepanjang lebar part setelah part diimpor.

Nilai ambang batas putaran komponen - Atur ini sebagai nomor minimum untuk memungkinkan rotasi. Part apa pun yang lebih kecil dari nilai yang diatur tidak akan diputar.

Pengaturan lain-lain

Aktifkan praimpor alat eksternal - Aktifkan opsi impor alat eksternal pra-impor untuk mengaktifkan Creo Reader. TruSmartCAM mengunggah instance keluarga yang diekstraksi sebelumnya dari part yang dijatuhkan. Harap dicatat bahwa ekstraksi instance dilakukan selama pengunggahan part (sebagai operasi pra-impor ), sehingga perangkat lunak Creo harus diinstal di komputer dari mana part sedang diunggah.

Abaikan warna diagram pada thumbnail Flat demi kejelasan - Aktifkan opsi ini untuk menghasilkan thumbnail yang lebih tajam dalam skala abu-abu. Warna layer dan entitas teks diabaikan demi kejelasan gambar.

Gunakan hanya satu Engine untuk semua tugas impor - Ketika diaktifkan, hanya satu inti mesin yang digunakan untuk tugas penting.

Thickness fudge

Toleransi perbandingan ketebalan memilih nilai ketebalan lembaran terdekat ketika ketebalan lembaran yang diinginkan tidak tersedia dalam inventaris dengan membandingkan nilai ketebalan lembaran dalam inventaris.

Misalnya, mari kita asumsikan sebagai berikut:

  1. Kita memiliki part dengan ketebalan lembaran 1.6 mm dan material sebagai baja.

  2. Nilai Toleransi perbandingan ketebalan adalah 0.1 mm.

  3. Inventaris memiliki lembaran baja dengan ketebalan 1.5 mm, 1.55 mm, dan 1.7 mm.

TruSmartCAM memilih lembaran dengan nilai ketebalan terdekat setelah membandingkan nilai Toleransi perbandingan ketebalan (0.1 mm) dengan item yang diimpor (berukuran 1.6 mm). Dalam hal ini, 1.55 mm akan menjadi pilihan.
