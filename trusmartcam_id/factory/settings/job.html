Pengaturan output

Simpan NC rancangan - Aktifkan opsi ini agar layout NC dan laporan disimpan.

Simpan file rancangan dan laporan - Aktifkan opsi ini agar file layout disimpan.

Pratinjau simulasi output dalam laporan - Aktifkan opsi ini untuk menghasilkan preview simulasi tooling bersama dengan output lainnya. Ini adalah animasi GIF pendek untuk tooling laser/punch.

Jangan hapus output program setelah penyelesaian rancangan/pekerjaan - Mengaktifkan opsi ini akan melewati operasi penghapusan untuk output program ketika job/layout ditandai sebagai selesai.

Pengaturan nesting

Gabungkan komponen dari beberapa Pekerjaan - Mendukung mode nesting campuran. Part dari beberapa job harus di-nest bersama untuk efisiensi packing yang lebih baik.

Jangan melepaskan rancangan secara otomatis - Ini akan menghentikan layout agar tersedia untuk produksi hingga semua layout dalam status released.

Nesting ulang dengan pemadatan dan pemotongan sisa - Nilai ini digunakan untuk memutuskan apakah layout yang di-nest ditahan dalam antrian persetujuan dan di-nest ulang ketika part baru direncanakan atau dirilis untuk produksi dan output terkait diekspor ke folder output.

Awalan nama lembaran interaktif - Bagian ini digunakan untuk mendukung nesting interaktif di mana pengguna memiliki lebih banyak kontrol atas part yang di-nest dan hasil nest. Gunakan skema penamaan, berbeda dari auto-nesting, untuk layout yang di-nest secara interaktif.

Perbarui inventaris lembaran pada saat pelepasan (atau tidak dilepaskannya) nesting - Jika opsi ini diaktifkan, inventaris lembaran akan diperbarui saat mengirim nest ke mesin. Juga memanggil kembali nest dari mesin akan mengembalikan lembaran ke inventaris.

Otomatisasi manajemen inventaris sisa - Ketika switch ini dipilih, perangkat lunak mengotomatiskan manajemen inventaris remnant. Jika tidak dipilih, manajemen remnant menjadi interaktif.

Refresh otomatis Rancangan yang usang - Ketika diperiksa, perangkat lunak menyegarkan semua ui:EGTag.SelectOutOfDate[Out-of-date Layouts] yang terhubung ke part setelah solusi pemotongannya dihitung.

Otomatisasi nesting twin-line - Periksa opsi ini untuk mengaktifkan otomatisasi twin-line.

Pengaturan alur kerja tarik

Gunakan 'Alur kerja tarik' - Opsi ini berguna untuk pengguna yang memerlukan sistem perencanaan dan nesting yang sangat otonom. Ketika diaktifkan, TruSmartCAM merencanakan nesting sesuai permintaan.

Jumlah nesting per stasiun - Nilai ini untuk mengatur berapa banyak layout per stasiun yang harus dibuat.

Jumlah maksimum lembaran per nesting - Nilai ini digunakan untuk mengontrol jumlah lembaran maksimum terhadap nest yang unik. Jadi, misalnya, jika ada pesanan dengan part besar (yang mengisi seluruh lembaran), diperlukan dalam jumlah besar (katakanlah 500) - TruSmartCAM tidak lagi membuat nest tunggal dengan 500 salinan dalam mode pull. Sebaliknya, ia membuat 10 salinan per nest di mana 10 adalah maksimal.

Program tekuk per stasiun - Nilai ini untuk mengatur berapa banyak program bend per stasiun yang harus dibuat.

Tandai tugas tersebut sebagai selesai, setelah output program dihapus - Aktifkan opsi ini agar layout ditandai Done ketika output dihapus dari tujuan output target.

Pengaturan impor spreadsheet

Direktori pencarian CAD - TruSmartCAM akan mencoba memuat entri part yang disebutkan dalam CSV yang diimpor dari perpustakaan part. Namun, jika part tidak tersedia di perpustakaan part maka akan mencoba mengambil part dari direktori pencarian berdasarkan urutan direktori yang diatur di dialog ini.

Gunakan pencarian CAD rekursif - Aktifkan opsi ini agar pencarian diulang untuk hasil berturut-turut.

Buatlah komponen lembaran kosong untuk file CAD yang tidak ada - Jika part tidak ada di perpustakaan TruSmartCAM atau direktori pencarian, TruSmartCAM akan membuat part kosong untuk part yang tidak tersedia tersebut.

Jangan tampilkan kotak dialog Pekerjaan Baru jika spreadsheet tidak memiliki kondisi kegagalan - Jika entri CSV bersih yaitu tidak ada ketidaksesuaian data dari DB atau semua field terdaftar dengan benar, TruSmartCAM tidak akan menampilkan jendela overview job dan akan memproses job segera setelah CSV diimpor atau dijatuhkan.

Lebih baik menggunakan kolom Bahan Baku daripada kolom Bahan + Ketebalan - Jenis spreadsheet yang didukung:

  1. Pattern 1: PartName, Quantity, DueDate, Priority, RawMaterial

  2. Pattern 2: PartName, Quantity, DueDate, Priority, Material, Thickness

  3. Pattern 3: PartName, Quantity, DueDate, Priority, Material, Thickness, RawMaterial

Jika opsi ini tidak diperiksa, Pattern 1 akan dianggap sebagai jenis spreadsheet default. Kami juga mendukung Pattern 3 jika format CSV adalah tipe Pattern 3.

Jika opsi ini diperiksa dan CSV adalah tipe Pattern 2, maka TruSmartCAM akan menyelesaikan Raw Material dari field Material dan Thickness dari CSV.

Pengaturan output spreadsheet

Ekspor laporan penyelesaian Pekerjaan/Rancangan (.csv)* - Periksa opsi ini untuk mengaktifkan ekspor CSV ketika Layout atau Job selesai. Laporan yang selesai disimpan di tujuan Reports yang ditentukan di Factory → Settings → Reports settings.
