Petakan field dari CAD (lookup keys)

Petakan field dari CAD (lookup keys)

  • pabrik → Pengaturan → Impor, di bawah bagian Pengaturan impor. Klik Ambil…​ Tombol pencarian bahan. Halaman _Pilih Bidang muncul.

  • Klik Telusuri…​. Pilih file C:\Program Files\Metamation{APPNAME}\Samples\Parts\00.04.dxf.

  • Select Field terisi dengan informasi MAT, REV, THK.

  • Pilih MAT dan Klik OK.

Field Material lookup keys diperbarui dengan field MAT. Lakukan hal yang sama untuk Revision dan Thickness keys.

File → _Buka…​ → Impor komponen, pilih file - C:\Program Files\Metamation{APPNAME}\Samples\Parts\00.04.dxf. Part diimpor dengan material, ketebalan dan juga revision.

Klik dua kali Part 00.04 dan halaman detail Part menampilkan informasi field yang dipetakan tersebut dari file CAD di bawah Lain-lain Properti.
