Export/Import material

Menggunakan format teks

Untuk mengekspor material inventori, navigasi ke pabrik → lembaran, klik Ekspor/Impor pengaturan bahan baku, pilih Export, pilih .txt, dan simpan ke folder target.

Buka file .csv yang diekspor, tambahkan Raw Material dan Sheet baru dengan detail yang diperlukan, kemudian simpan file.

Untuk mengimpor file yang diperbarui, navigasi ke pabrik → lembaran → Ekspor/Impor pengaturan bahan baku, pilih Import, pilih file TXT yang dimodifikasi, dan material serta sheet yang baru ditambahkan akan muncul di TruSmartCAM.

Menggunakan format CSV

  • Klik pabrik → lembaran → Ekspor/Impor pengaturan bahan baku → Ekspor lembaran. Pilih file CSV dan simpan ke folder target.

  • Tambahkan sheet dan parameter sheet, simpan file .csv.

  • Menggunakan Impor lembaran, pilih file .csv yang dimodifikasi.

  • Material dan sheet yang baru ditambahkan akan muncul di TruSmartCAM.

Import Material harus berupa file berbasis teks yang dikonstruksi secara khusus yang mencakup format berikut.

Material-Name=[BendMaterial],[Density],[FName];
RawMaterial-Thickness,[Name]=[Film],[Treatment],[Surface];
Sheet-Width,Height,Grain,Desc,HeatNumber=Qty,[Factory],[Shelf],[Cost],[Comments]

Contoh: Aluminum0.5,Aluminum-051500,1500,None,=112,,,22,2500,1250,Horizontal,=105,,,7,2500,1500,None,=100

Untuk dengan mudah membuat file import, ekspor terlebih dahulu material dan kemudian lakukan modifikasi padanya atau lihat strukturnya.
