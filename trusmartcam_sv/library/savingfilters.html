Spara filter

Sparade filter låter er återanvända ofta tillämpade kriterier i delar, jobb och layouter, vilket effektiviserar ert arbetsflöde. Varje modul levereras med standardfilter, men ytterligare filterlistor kan skapas av sajadministratörer eller programmerare efter behov. Återanvändning av sparade filter sparar tid och hjälper till att upprätthålla konsekvens mellan sökningar.

  • Klicka på ikonen för att visa standardfiltren.

Steg för att spara filter

  • Klicka på ikonen , välj ett filterobjekt.

  • Från rullgardinsmenyn för det valda filterobjektet, välj ett alternativ. En bockmarkering visas bredvid det valda villkoret.

  • För att tillämpa villkoret, klicka på ikonen och välj sedan Lägg till villkor (Ctrl+PLUS)]. Ni kan upprepa detta steg för att lägga till flera villkor.

  • För att spara dessa valda filter med villkor, klicka på Spara filter…​ (Ctrl+S)].

  • Ange namnet och klicka sedan på _Spara.

  • Från dialogrutan Spara filter kan ni också ta bort alla tidigare sparade filter.

  • Dessa filter sparas i standardfilterlistan. Här har Material sparats till standardlistan.

Filter skapade av sajadministratör och admin delas mellan alla användare. Filter skapade av programmerare och operatör delas endast mellan användare med roller som programmerare och operatör.

Filterkommandon

Kommando Ikon Genvägar Beskrivning

Clear Selection

Ctrl+X

Avmarkerar alla valda val och uppdaterar vyn för filterobjekt.

Select Highlighted

Ctrl+A

Väljer alla markerade val. Detta kommando förblir inaktiverat om objektmarkering inte är tillämplig för det valda fältet.

Match All

&

Visar endast objekt som uppfyller alla valda villkor.

Pin In/Out Menu

Ctrl+Down/Up

När den är fäst förblir rullgardinsmenyn öppen även när det aktuella fokus flyttas till ett annat fönster.

Close Menu

Esc

Stänger rullgardinsmenyn.
