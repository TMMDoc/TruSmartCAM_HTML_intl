Skärverktygningsvalidering

Skärverktygningsvalidering verifierar att det valda skärverktyget är lämpligt för delens geometri, materialtjocklek och skäroperation. Den säkerställer att konturer kan skäras utan verktygskonflikter, ofullständiga skär eller brott mot maskinbegränsningar.

För fullständiga detaljer om skärverktyg och skärparametrar som stöds, se TecZone-dokumentationen om 2D Cleanup och LTT.
