Importera order

Order kan skapas eller importeras med hjälp av följande metoder:

  1. Importera order från kalkylbladsfiler.

  2. Importera order automatiskt via bevakningsinställningar.

  3. Skapa order från befintliga delar i delbiblioteket.

  4. Skapa order från monteringens statiska nestningar.

Importera order från kalkylblad

  • Navigera till Öppna → Importera _beställningar…​.

  • Format för kalkylblad som stöds inkluderar CSV, XLSX och XML.

  • Välj den önskade kalkylbladsfilen och klicka på Öppna.

  • Importprocessen liknar arbetsflödet för import av jobb/monteringar.

Ni kan också direkt dra och släppa kalkylbladet på ordersidan för att importera order.

Importera order med hjälp av bevakningsinställningar

  • Aktivera alternativet Importera kalkylblad från 'Livemapp' i Kontrollinställningar.

  • Välj Importera beställningar i Option för importera kalkylblad.

  • Konfigurera katalogplatsen från vilken order ska importeras.

  • Systemet övervakar kontinuerligt den konfigurerade katalogen.

  • Nyligen tillagda kalkylbladsfiler importeras automatiskt till order.

För detaljerade konfigurationssteg, se dokumentationen för Watch settings.

Skapa order från delbibliotek

  • Välj de önskade delarna från delbiblioteket.

  • Högerklicka och välj Ny…​ → Beställning för att öppna dialogrutan Nya beställningar.

  • Ange de nödvändiga detaljerna såsom kvantitet, förfallodatum och prioritet.

  • Klicka på OK för att skapa ordrarna.

  • Systemet växlar automatiskt till ordersidan.

  • Skapade order visas på sidan Order redo för att skapa jobb.

Skapa order från montering

  • Högerklicka på den önskade monteringen.

  • Välj Ny…​ → Beställning för att öppna dialogrutan Nya beställningar.

  • Ange det nödvändiga Antal moduler.

  • Klicka på OK för att skapa ordrarna.
