Delfilter

Delbibliotekets filter låter er snabbt söka efter och lokalisera delar i biblioteket genom att tillämpa specifika villkor. Genom att begränsa resultaten till endast de delar som uppfyller era kriterier gör dessa filter det enklare att hantera och komma åt relevanta delar effektivt.

Status

  • Använd filtret Status för att söka delar baserat på verktygsstatus.

  • Rullgardinsmenyn visar verktygstillstånd som färgade rutor och alla tillgängliga maskiner och deras tillämpliga fel.

  • De färgade rutorna representerar följande tillstånd:

    • Vit: Verktygning OK

    • Orange: Verktygning har fel

    • Gul: Verktygning har varning

    • Grå: Verktygning undertryckt

  • Om en del har smal fläns eller verktygning inte tilldelad fel, kommer att välja dessa att visa respektive delar.

  • Delar med undertryckt verktygning på den valda maskinen visas genom att klicka på den gråfärgade rutan.

Bockningsmonteringar

  • Använd filtret Bockningsmonteringar för att söka delar baserat på det böjverktyg som används.

  • Rullgardinsmenyn visar alla tillgängliga böjverktyg och kan filtreras efter verktygsnamn, typ (stans, gåsnacke, radie, etc.) eller beskrivning.

  • Välj flera verktyg och klicka på Matcha alla|&** för att sortera verktygen som använder båda de valda verktygen.

Använda verktyg

  • Filtrering efter Använda verktyg fungerar liknande filtrering efter Bockningsmonteringar.

  • Verktygsikoner visas i proportionerliga storlekar (inte exakta, men relativa).

  • När förhandsgranskningen av verktygning är öppen, markerar mushovring över ett verktyg i filterlistan det verktyget i förhandsgranskningsrutan.

  • Flera verktyg kan väljas och tillämpas med hjälp av Matcha alla|&**.

Tillstånd

  • Använd filtret Tillstånd för att söka delar baserat på deras aktuella tillstånd.

  • Rullgardinsmenyn visar alla tillämpliga tillstånd beroende på de tillgängliga biblioteksdelarna.

  • Listan över tillstånd ändras dynamiskt enligt de valda delarna och deras status.

  • Välj till exempel Material Missing för att visa delar i ett olöst materialtillstånd.
