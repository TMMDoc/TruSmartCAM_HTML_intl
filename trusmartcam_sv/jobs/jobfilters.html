Jobbfilter

Jobbsidans filter hjälper er att hitta jobb, aktiva delar och häckar genom att visa endast objekt som matchar valda villkor. De förenklar sökande, sortering och hantering av jobb eller delar, vilket förbättrar den övergripande effektiviteten i arbetsflödet.

Exempel på tillgängliga filter

Prioritet och Förfaller

  • Filtret Prioritet gör att ni kan söka och visa delar eller jobb baserat på deras tilldelade prioritet (Brådskande, Hög, Låg, Normal).

  • När ni väljer en viss prioritetsnivå kommer endast delarna med den specifika prioriteten att markeras eller visas i resultaten.

  • Om ni till exempel väljer Hög Priority kommer systemet att filtrera och markera endast de delar som är markerade som High Priority på häcken.

  • Filtret Förfaller gör att ni kan söka och visa delar eller jobb baserat på deras tilldelade datum.

  • När ni väljer ett visst förfallodatum kommer endast delarna med det specifika förfallodatumet att markeras eller visas i resultaten.

  • Förutom fördefinierade förfallodatumfilter som Idag, Igår, Imorgon etc., kan ni också mata in anpassade datumvärden/-intervall i yymmdd-format i söktextrutan.

  • Ni kan också använda uttryck med ojämlikhetsoperatorer (<, <=, >, >=, =) följda av datumvärdet. Exempel: >=250901 skulle söka alla objekt som har förfallodatum på eller efter 1 september 2025.

Om filterlistan döljer sökinnehållet kan ni göra den transparent genom att använda kommandot Koppla om genomskinlighet|*Högerklicka med musen* .

Detaljnamn/ Layout name

  • Filtret Detaljnamn kommer att vara tillgängligt på jobb- och häcksidan.

  • När ni skriver delnamnet visas en förslagslista för att hjälpa er att snabbt välja rätt del.

  • Genom att välja ett eller flera objekt från förslagslistan begränsas sökresultaten till endast de valda delarna.

  • På jobbsidan utökas detta beteende till att inkludera layoutnamn, vilket gör att användare kan söka efter antingen delnamn eller layoutnamn.

Rest

  • Använd filtret Rest för att söka layouter baserat på resttillgänglighet.

  • Rullgardinsmenyn visar följande alternativ:

    • Ingen rest - listar layouter där ingen rest skapas.

    • Finns - listar layouter där en rest redan finns.

    • Tillagd - listar layouter där en ny rest läggs till.

  • Hjälper till att identifiera och hantera återstående arkmaterial effektivt.
