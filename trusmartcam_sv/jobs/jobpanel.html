Jobbpanel

Detta avsnitt förklarar jobbrelaterade åtgärder såsom Export outputs, Återställ jobb…​, Markera slutförd…​, Exportera jobb…​ och andra alternativ som används för att hantera jobbinstanser.

Öppna i skrivskyddat läge för visning
Öppnar det valda jobbet i skrivskyddat läge.

Check-out the document for editing
Aktiverar redigering av det valda jobbet.

Radera
Tar bort det valda jobbet.

Visa rapport
Öppnar produktionsrapporten för det valda jobbet.

Exportera utdata
Exporterar maskinärt utdatafiler (NC-filer och rapporter) genererade för det valda jobbet.

Plan job
Genererar häckade layouter för delarna i det valda jobbet.

Återställ jobb…​
Tar bort alla genererade layouter för jobbet och återställer dess delar till planerbart tillstånd.

Markera slutförd…​
Markerar jobbet och alla dess häckar som färdiga efter produktion.

Exportera jobb…​
Exporterar det kompletta jobbet som ett paket som kan återanvändas eller importeras till ett annat system.

Föreg / Nästa / Layout
Navigerar till föregående eller nästa layout inom det valda jobbet.

Sort and filter
Använd dessa filter för att visa andra jobb som delar samma material, kund, förfallodatum eller prioritet.

Exportera utdata

  • Välj det önskade jobbet.

  • Klicka på Exportera utdata.

  • Detta exporterar de maskinärt utdatafiler som genererats för det valda jobbet.

  • Export Outputs kan sparas antingen till standardplatsen konfigurerad i Fabriksinställningar eller till en anpassad plats genom att använda Sök…​.

  • Det exporterar endast maskinutdatafiler (såsom NC och rapporter) för jobbet.

  • Dessa filer används direkt för produktion och kan skickas till maskinen.

Återställ jobb…​

Återställer det valda jobbet genom att ta bort alla genererade layouter och göra delarna tillgängliga för planering igen.

Så här återställer ni ett jobb:

  • Högerklicka på jobbet och välj Återställ jobb…​.

  • Detta tar bort alla befintliga häckade layouter för det valda jobbet och återställer alla dess delar till planerbart tillstånd.

  • Om layouterna innehåller delar från andra jobb återställs även dessa delar.

Exportera jobb…​

  • Välj det önskade jobbet.

  • Klicka på Exportera jobb…​.

  • Detta alternativ exporterar det kompletta jobbet som ett paket som kan återanvändas eller importeras till ett annat system.

  • Den exporterade datan inkluderar all information som krävs för att återskapa jobbet:

    • Deldetaljer och kvantiteter (CSV-filer).

    • Layout- och häckningsdata (CSV / FXYLT-filer).

  • De exporterade filerna sparas som en jobbmapp på den konfigurerade exportplatsen.

  • Detta alternativ används för att överföra, dela eller säkerhetskopiera hela jobbet.

Markera slutförd…​

  • När jobbet har bearbetats på maskinen öppnar ni jobbpanelen och väljer Markera slutförd…​.

  • När ett jobb markeras som färdigt:

    • Alla häckar under jobbet markeras som färdiga.

    • Jobbet gråmarkeras, flyttas till slutet av listan och visas med en grön bock.

    • Alla layouter under jobbet tas bort från listan över aktiva layouter.

    • Del- och jobbstatus uppdateras och bearbetade kvantiteter flyttas till färdigt tillstånd.

    • Utdatafiler för alla layouter i jobbet tas bort från maskinutdataplatsen för att undvika dubbelanvändning.
