Klientinstallation

Kör Förutsättningar Installation som Server innan ni påbörjar Klient-konfigurationen. Utför TruSmartCAM Klientinstallation med datoranvändaren som har administratörsbehörighet.

  • Använd samma installationsprogram som användes för TruSmartCAM Serverinstallation. I klientmaskinen, efter att ha kört installationsprogrammet, välj Klient-alternativet på komponentvalssidan. Välj de TruSmartCAM-verktyg som krävs och klicka Nästa.

  • Sidan Redo att installera sammanfattar installationslistan. Klicka Installera maskiner för att fortsätta med installationen i klientmaskinen.

  • När installationen är klar, välj kryssrutan Launch TruSmartCAM och klicka Finish. När TruSmartCAM-skärmen startar, ange servernamnet eller IP-adressen för TruSmartCAM Server-datorn. Klicka OK .

  • När anslutningen är upprättad visar TruSmartCAM registreringsdialogrutan med programmerarbehörighet. Ange e-postadress, lösenord om det behövs och klicka OK. TruSmartCAM Bibliotekssida laddas med välkomst Användarnamn längst ner till vänster i informationsområdet.
