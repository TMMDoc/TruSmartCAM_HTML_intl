Format som stöds

TruSmartCAM stöder direkt CAD-import. Tillgängligheten av importformat beror på licenskonfigurationen. De format som stöds klassificeras enligt följande.

Följande format kan importeras i TruSmartCAM utan att kräva avancerad CAD-importlicens.
