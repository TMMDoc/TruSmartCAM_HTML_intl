Import zadań

Import z pliku csv lub xlsx

Zadania można również importować w popularnych formatach tekstowych

  • Kliknij Plik → Otwórz → Importuj_ zadania, pojawi się okno dialogowe Otwórz folder. Wybierz C:\Program Files\Metamation\Praxis\Samples\Jobs\C0\C0.csv.

  • Jeśli wszystkie części istnieją w lokalizacji csv lub w katalogach folderów wyszukiwania, pojawia się okno nowe zadanie z wyświetleniem wszystkich części pasujących do pól csv.

  • Brakujące części są ignorowane i nie są wyświetlane w oknie nowe zadanie.

  • Początkowo status części to Nascent.

  • Kliknij OK, teraz części są importowane do biblioteki części. Walidacja CAD, narzędziowanie dla gięcia i cięcia są wykonywane, co można monitorować na ekranie Pmonitor.

Utwórz pusty detal dla brakujących plików CAD

Podczas importowania zadań za pomocą arkusza kalkulacyjnego (.csv lub .xlsx), mogą występować wiersze odwołujące się do plików CAD, które nie są dostępne w systemie.

  • Aby to obsłużyć, przejdź do fabryka → Ustawienia → Zadanie, w sekcji Import arkusza włącz Utwórz pusty detal dla brakujących plików CAD

Gdy ta opcja jest zaznaczona:

  • Jeśli plik CAD brakuje dla któregokolwiek wiersza podczas importu zadania, system automatycznie utworzy pustą część w miejsce brakującego pliku CAD.

  • W oknie nowe zadanie, te brakujące części są wizualnie wyróżnione na żółto, aby wskazać, że powiązany plik CAD brakuje.

  • Brakująca część będzie nadal miała swoje zmapowane pola (takie jak nazwa części, materiał, grubość, ilość itp.) z pliku CSV. Jednak jej status części będzie pokazany jako Nascent, co oznacza, że jest to tylko pusty symbol zastępczy bez rzeczywistej geometrii.

To zapewnia, że import zadania może nadal przebiegać bez błędów wynikających z brakujących plików CAD — możesz później zaktualizować lub zastąpić pustą część prawidłowym plikiem CAD, gdy stanie się dostępny.

Nie wyświetlaj okna dialogowego Nowe zadanie, jeśli arkusz nie zawiera błędów

fabryka → Ustawienia → Zadanie → Import arkusza→ Nie wyświetlaj okna dialogowego Nowe zadanie, jeśli arkusz nie zawiera błędów

  • Gdy to pole wyboru jest zaznaczone, system automatycznie tworzy nowe zadanie bez wyświetlania okna dialogowego Nowe zadanie, jeśli arkusz kalkulacyjny nie zawiera błędów (brak brakujących części, brakujących dat lub nieprawidłowych danych).

  • Gdy to pole wyboru jest wyczyszczone, okno dialogowe Nowe zadanie zawsze pojawia się do przeglądu i potwierdzenia przez użytkownika, nawet jeśli arkusz kalkulacyjny nie zawiera błędów.

Preferuj kolumnę surowca nad materiałem + grubością

fabryka → Ustawienia → Zadanie→Preferuj kolumnę Surowiec nad Materiał + Grubość

  • Gdy to pole wyboru jest zaznaczone, system oczekuje, że arkusz kalkulacyjny będzie używał kolumny Surowiec (na przykład Steel-20). Oznacza to, że rodzaj materiału i grubość powinny być połączone w jednej wartości w jednej kolumnie.

  • Gdy to pole wyboru jest wyczyszczone, system oczekuje, że arkusz kalkulacyjny będzie miał oddzielne kolumny dla Materiału i Grubości (na przykład Steel i 2.00). System łączy je wewnętrznie podczas importu.

Obserwuj folder za pomocą CSV/XLSX

  • W fabryka → Ustawienia→ Monitoruj zaznacz Włącz „Folder na żywo” i Monitoruj również foldery podrzędne.

  • Ustaw ścieżkę folderu obserwacji (UNC lub zamapowany dysk).

  • W Import arkusza, zaznacz Importuj arkusz z „Folderu na żywo”.

  • Wybierz Importuj_ zadania z opcji Import arkusza kalkulacyjnego.

  • Skopiuj C:\Program Files\Metamation\Praxis\Samples\Jobs\C0\C0.csv i wklej go do folderu obserwacji.

  • System sprawdza plik CSV i próbuje automatycznie utworzyć zadanie.

  • Jeśli plik CAD brakuje, żadne zadanie nie jest tworzone, ponieważ wymagane dane CAD nie zostały znalezione.

Katalog wyszukiwania CAD i rekursywne wyszukiwanie CAD - nie skonfigurowane

  • W fabryka → Ustawienia → Zadanie → Import arkusza, jeśli nie jest ustawiony żaden Katalog wyszukiwania CAD i nie jest używane Użyj rekurencyjnego wyszukiwania CAD, system nie wyszukuje plików CAD automatycznie.

  • Jeśli pole wyboru Utwórz pusty detal dla brakujących plików CAD jest zaznaczone, zadanie zostanie utworzone nawet wtedy, gdy pliki CAD brakują.

  • W tym przypadku wszystkie części w zadaniu zostaną utworzone jako części nascent (puste) — co oznacza, że nie mają geometrii, ponieważ plików CAD nie można było znaleźć.

Katalog wyszukiwania CAD i rekursywne wyszukiwanie CAD - skonfigurowane

Gdy Katalogi przeglądania CAD jest ustawiony na (C:\Program Files\Metamation\Praxis\Samples) i Użyj rekurencyjnego wyszukiwania CAD jest włączone w fabryka → Ustawienia → Zadanie → Import arkusza:

  • System automatycznie przeszukuje określony folder i wszystkie jego podfoldery w poszukiwaniu wymaganych plików CAD wymienionych w arkuszu kalkulacyjnym.

  • Utwórz pusty detal dla brakujących plików CAD jest również włączone, więc jeśli jakikolwiek plik CAD nadal nie zostanie znaleziony, system utworzy pustą część jako symbol zastępczy.
