Wymagania dotyczące portów

TruSmartCAM Server : 7371

Serwer oczekuje na żądania klientów na tym porcie TCP. Połączenia przychodzące muszą być dozwolone na tym porcie na komputerze serwera.

Floating License Server : 5776

Floating License Server oczekuje na żądania licencji na tym porcie. Połączenia przychodzące muszą być dozwolone na tym porcie na komputerze serwera licencji floating (TruSmartCAM Server).

Połączenie licencji do Cloud : 7567

Ten port jest używany przez TecZone (Bending Software) oraz Floating License Server do wysyłania żądań licencji do Metamation Cloud. Dlatego połączenie wychodzące musi być dozwolone na tym porcie dla aktywacji i odnowień licencji.

Wymagania dotyczące instalacji i wsparcia

  • Uprawnienia administratora wymagane do zainstalowania aplikacji TruSmartCAM.

  • Zdalne narzędzia diagnostyczne przez VPN kontraktora lub bezpieczne połączenie.
