其他增強功能

SolidWorks 增益集已進行多項增強，以改善整體使用者體驗、效能和穩定性。

  • 統一的登入體驗： 增益集現在重複使用 TruSmartCAM 登入介面。兩個應用程式都套用相同的登入和密碼驗證。「記住我」憑證功能在 TruSmartCAM 和 SolidWorks 增益集之間無縫運作，無需重複登入。

  • API 席位需求： 增益集現在需要可用的 API 席位才能運作。

    • 如果 API 席位不可用，將顯示授權錯誤。

  • 已驗證的名稱-配置分隔符號： 名稱-配置分隔符號文字現在已驗證，以確保僅使用有效的路徑字元。這可以防止在儲存多配置零件時檔案名稱中出現無效字元。

  • 改進的訊息系統： 系統訊息、警告和錯誤現在會顯示在 SolidWorks 工作區的左下角狀態區域中，類似於 TruSmartCAM 介面。
