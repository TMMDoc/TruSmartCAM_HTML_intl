工作流程

工作流程

TruSmartCAM 是一個管理和優化鈑金製造流程的製造執行系統。TruSmartCAM 中的工作流程包含一系列步驟，您需要遵循這些步驟來完成特定的任務或流程。您可以自訂工作流程以適應不同的製造流程，並根據您的特定需求進行調整。TruSmartCAM 中的工作流程通常包括以下步驟：

1. 配置系統

配置 TruSmartCAM 以符合您的製造環境。此基礎步驟包括定義機器及其功能和參數、新增可用材料和厚度範圍、配置包含沖頭和模具的工具資料庫、為不同的機器配置建立設定表，以及管理具有適當權限的使用者帳戶。正確配置的系統可確保精確處理並防止後續階段出現錯誤。此設定通常在初始部署期間執行一次，並隨著工廠發展定期更新。

詳細資訊請參閱 Add/Remove machine、Material、 Sheets、Setup tooling 和 Configure users。

2. 將零件新增到資料庫

使用支援的格式（例如 DXF、DWG 或 STEP 檔案）將 2D 或 3D CAD 零件匯入到零件資料庫。在匯入過程中，TruSmartCAM 會自動分析零件幾何形狀、驗證可製造性、指派所需的鈑金技術（雷射、沖壓或組合），並產生預覽縮圖。已匯入的零件在資料庫檢視中顯示為磚塊，其中顏色編碼的圖示表示零件狀態、技術類型、材料需求和任何檢測到的問題。此視覺系統允許一眼快速評估零件的就緒狀態。

詳細資訊請參閱 Library。

3. 驗證零件和工具

檢查每個已匯入的零件是否存在潛在的生產問題。驗證零件幾何形狀是否有效且可製造、工具指派是否正確且可用、材料和厚度選擇是否與庫存匹配，以及成形零件的折彎序列是否正確定義。如果檢測到問題，請在 TecZone 中開啟零件以修改幾何形狀、重新指派或優化工具、調整處理參數或修正材料規格。在此階段解決問題可防止生產延誤並減少製造過程中的廢料。

詳細資訊請參閱 Part state 和 Tooling status。

4. 將零件組織成工作

根據生產需求將已驗證的零件分組成工作。工作代表一個邏輯生產單位，可能包括來自單一客戶訂單的零件、共享相同材料和厚度的零件、具有相似交貨期限的零件，或根據您的生產策略進行的組合。工作提供了規劃的容器，允許對相關零件進行批次處理，透過套料實現高效的材料利用，並簡化追蹤和報告。可以透過選擇零件手動建立工作，也可以根據預定義的規則和篩選條件自動建立。

詳細資訊請參閱 Jobs。

5. 將工作規劃到生產機器

建立工作後，將其指派給適當的生產機器。考慮機器功能，包括支援的技術（雷射、沖壓、轉塔）、材料和厚度容量、可用工具和目前工作負載。如果存在多台合適的機器，請根據生產優先順序、機器可用性、設定需求或成本優化進行選擇。指派可以是手動以提供靈活性，或根據預配置的規則和機器排程演算法自動進行。

詳細資訊請參閱 Plan job。

6. 為工作發佈 NC

在此階段，會產生切割佈局。您仍然可以使用 TecZone 編輯佈局並將切割技術新增到佈局中。之後，匯出 NC 輸出並將其傳輸到機器控制器或指定的網路位置。然後鎖定工作以防止進一步修改，並將其狀態更新為「已發佈」以進行生產追蹤。

詳細資訊請參閱 Send/Recall NC。

7. 完成生產並結案工作

在指派的機器上執行已發佈的工作。在生產過程中，機器操作員遵循 NC 程式、在切割過程中驗證零件品質、管理材料的裝載和卸載，並處理任何機器警報或問題。生產完成後，在 TruSmartCAM 中將個別零件或整個工作標記為完成。此最終步驟會更新庫存記錄、記錄實際生產時間和材料使用情況、封存工作資料以供報告和分析，並完成工作流程循環。

詳細資訊請參閱 Mark complete。
