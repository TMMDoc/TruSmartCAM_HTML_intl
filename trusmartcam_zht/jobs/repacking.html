重新包裝

TruSmartCAM 沒有單獨的重新包裝模組。相反，重新包裝是 透過提供零件組合靈活性的工作層級選項來處理的。

重新包裝可以透過三種方式完成。

  • 使用 混合多個套裁的零件 進行重新包裝。

  • 使用 新增零件… 互動式精靈進行手動重新包裝。

  • 使用 TecZone 進行手動重新包裝。

使用 混合多個套裁的零件 進行重新包裝

  • 要啟用此選項，請前往 工廠 → 設定 → 套裁 → 排樣設置 → 混合多個套裁的零件。

  • 啟用後，此選項允許在自動排版或互動式排版期間將來自不同工作的零件組合在一起， 從而產生組合的包裝或排版。

  • 例如：

    • Job-1 有 1 個零件，Job-2 有 1 個零件。

    • 啟用 混合多個套裁的零件 後， TruSmartCAM 可以將這 2 個零件重新包裝成一個最佳化的排版。

  • 如果沒有此選項，每個工作的零件在 排版/包裝期間保持分離，使用不同的佈局名稱。

  • 也可以在不啟用此選項的情況下手動執行重新包裝。

使用新增零件互動式精靈進行手動重新包裝

  • 從佈局檢視中，選擇要重新包裝的零件。

  • 右鍵點擊零件，從 佈局面板中選擇 新增零件…。

  • 按照精靈中的提示完成重新包裝流程。

  • 選擇 顯示其他套裁零件 並將它們新增到 目前工作。

  • 檢閱變更並確認重新包裝。

欲了解更多詳細資訊，請參閱 Add Parts。

使用 TecZone 進行手動重新包裝

  • 透過點擊 開啟 TecZone。

  • 在 TecZone 介面中，點擊 Add。

  • 出現一個新視窗，您可以在其中選擇要重新包裝的零件。點擊 顯示… 然後選擇 其他套裁零件。 完成選擇後，點擊 _拾取。

  • 出現一個新視窗，您可以在其中定義零件需要重新包裝的位置。

  • 點擊 儲存對 Praxis 的更改， 重新包裝的佈局將在 TruSmartCAM 中可用。
