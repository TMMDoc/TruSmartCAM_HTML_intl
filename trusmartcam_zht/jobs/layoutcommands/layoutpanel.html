佈局面板

本節說明與佈局相關的動作，如 發送到機台…、 從機台召回…​、 匯出輸出、 標記完成…​、新增零件…、 適應/緊湊…、 分割展開圖…，以及其他用於管理 佈局實例的選項。

以唯讀模式打開進行查看
以唯讀模式開啟選定的佈局。

Check-out the document for editing
啟用選定佈局的編輯功能。

刪除
刪除選定的佈局。

查看報告
開啟選定佈局的標準佈局報告和標籤特定報告。

發送到機台…
將選定的佈局傳送到機器，並將輸出檔案 （fxlyt、NC、report、csv）匯出到工作輸出位置。

新增零件…
使用佈局重新包裝精靈將更多零件新增到包裝不佳的佈局。

適應/緊湊…
重新排版選定的佈局以壓縮零件並釋放可用區域。

更改 LTT…
變更分配給選定佈局的板材。

分割展開圖…
將選定的佈局拆分為兩個或多個單獨的佈局。

Rename layout…​
重新命名選定的佈局。

套裁 / 上一個 / 下一個 / 零件
導覽至父工作、選定工作中的上一個或下一個佈局，或向下導覽至 佈局中包含的零件。

Sort and filter
使用這些篩選器查看與選定佈局共用相同屬性的 其他佈局：Same Machine、Same Material、Same LTT、Same Priority、 Same State、Same Sheet、Same Part、Same Remnant State 或 Same Rolling direction。

發送到機台…

  • 當排版就緒（已填滿或緊急需要）時，您可以將其傳送到 機器。

  • 右鍵點擊排版並選擇 發送到機台…。

  • 會自動建立一個資料夾，以排版命名。

  • 輸出檔案（fxlyt、NC、report、csv）會匯出到 JOB 輸出位置。

  • 傳送到機器的排版會顯示藍色勾選標記。

一旦釋放，這些排版就不會再被重試進行重新包裝。

從機台召回…​

  • 如果輸出需要驗證或更新（例如，新增更多零件）：

    • 右鍵點擊已匯出的佈局。

    • 選擇 從機台召回…​。

  • 輸出檔案和資料夾（fxlyt、NC、report、csv）會被刪除。

  • 排版再次變為可編輯。

  • 您可以進行變更，然後再次將其傳送到機器。

匯出輸出

  • 右鍵點擊任何已傳送到機器的佈局（排版）並點擊 匯出輸出。

  • Export Outputs 選項會將 NC、Report 和 Layout 檔案匯出到 各自的機器輸出資料夾。

  • 此選項會使用佈局的目前單位設定重新產生 NC 檔案。

Export Outputs 不適用於已完成的佈局和 正在等待重新包裝佇列中的佈局。

Mark complete

  • 一旦在機器上處理完排版，右鍵點擊排版並 選擇 標記完成…​。

  • 您可以選擇多個排版並同時將它們標記為完成。

  • 當排版被標記為完成時：

    • 它會變灰、移至清單末尾，並顯示綠色勾選 標記。

    • 佈局會從活動佈局清單中移除。

    • 零件和工作狀態會更新，已處理的數量移至 完成狀態。

    • 該佈局的輸出檔案會從機器輸出 位置移除，以避免重複使用。

新增零件…

  • 使用此選項將更多零件新增到相對包裝不佳的佈局。像 互動式排版一樣，這是一個精靈驅動的選項，可以同時用於 單個或多個佈局。

  • 當使用此選項時，TruSmartCAM 會在主工作區域中 裝載佈局重新包裝精靈。按住 Ctrl 鍵並選擇一個或多個 要包裝到佈局上的零件，然後按 下一個。

  • 選定的零件會在下一步驟中顯示，帶有可編輯的 數量欄位。如需要，請在 Update Qty 欄位中更新零件數量。點擊 下一個 繼續進行排版。請注意， 數量只能增加，不能減少。

  • 所有佈局都會單獨排版，排版結果顯示新包裝的零件放置在其上。 選擇佈局會顯示所有現有的佈局零件。 如果在重新包裝期間沒有放置新零件，結果會自動棄置。具有多個副本的佈局在 重新包裝後可能會產生不同的模式。

適應/緊湊…

  • This wizard-based command can be used to adapt a layout from one machine to another when the original machine is down or when machine workload needs to be balanced.

  • It can be used when the sheet planned for is out of stock and the layout needs to be replanned on another available sheet.

  • You can merge layouts packed on smaller sheets onto larger sheets to use material more efficiently.

  • You can split layouts packed on larger sheets into smaller sheets if large sheet stocks are not available.

  • You can enhance the packing efficiency of multiple layouts (even from different machines) by merging them together.

  • To use it, select one or more layouts and click 適應/緊湊… to open the wizard in the work area.

  • The left pane shows all selected layouts and the Parts pane shows the layout parts.

  • Select the target machine and sheet material, then click 下一個 to perform the renesting.

  • The new nest results are shown in the results pane.

  • Click 下一個 to save the adapted or compacted layout.

拆分佈局

分割展開圖… 選項讓您透過修改數量 將現有佈局拆分為多個佈局， 而無需重做刀具或排版。

此功能在以下 情境中特別有用：

  1. 當您需要立即生產總數量的一部分，同時 將剩餘數量安排在後續生產中。

  2. 當生產需求需要分配到多個班次或 天數以最佳化工作流程和資源利用率。

  3. 當特定機器暫時不可用或被佔用，且您需要 調整佈局以適應目前的生產約束。

要拆分佈局，請執行以下步驟：

  • 在佈局檢視中，選擇所需的佈局。

  • 點擊 分割展開圖…。

  • 在 新副本 中輸入所需的值。

  • 點擊 確定。

選定的佈局現在被拆分為兩個佈局，數量已調整。
