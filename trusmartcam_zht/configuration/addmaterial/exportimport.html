匯出/匯入材料

使用文字格式

若要匯出庫存材料，請導覽至 工廠 → 板材，點選 匯出/匯入原材料設定，選擇 Export，選擇 .txt，並將其儲存至目標資料夾。

開啟匯出的 .csv 檔案，新增新的原材料和板材及必要的詳細資訊，然後儲存檔案。

若要匯入更新的檔案，請導覽至 工廠 → 板材 → 匯出/匯入原材料設定，選擇 Import，選擇修改的 TXT 檔案，新增的材料和板材將出現在 TruSmartCAM 中。

使用 CSV 格式

  • 點選 工廠 → 板材 → 匯出/匯入原材料設定 → 匯出板材。選擇 CSV 檔案並儲存至目標資料夾。

  • 新增板材和板材參數，儲存 .csv 檔案。

  • 使用 匯入板材，選擇修改的 .csv 檔案。

  • 新增的材料和板材將出現在 TruSmartCAM 中。

材料匯入需要是特殊建構的文字檔案，涵蓋以下格式。

Material-Name=[BendMaterial],[Density],[FName];
RawMaterial-Thickness,[Name]=[Film],[Treatment],[Surface];
Sheet-Width,Height,Grain,Desc,HeatNumber=Qty,[Factory],[Shelf],[Cost],[Comments]

範例： Aluminum0.5,Aluminum-051500,1500,None,=112,,,22,2500,1250,Horizontal,=105,,,7,2500,1500,None,=100

若要輕鬆建立匯入檔案，請先匯出材料，然後對其進行修改或檢視結構。
