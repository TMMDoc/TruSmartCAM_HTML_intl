工具面板

本節說明與工具相關的操作，例如 匯出輸出、Check Tooling、屏蔽模具、設定為基面…、複製模具… 以及其他用於管理工具實例的選項。

以唯讀模式打開進行查看
以唯讀模式開啟選定的工具。

Check-out the document for editing
啟用選定工具的編輯。

匯出輸出
將選定的工具輸出匯出到配置的目的地。

Check Tooling
驗證現有工具而不進行任何變更，如果驗證失敗則更新其狀態。

複製模具…
從選定的參考工具複製沖頭、序列和後擋料設定，並在可行的情況下套用到其他解決方案。

設定為基面…
將選定的折彎或折邊工具標記為產生切割解決方案的參考平面尺寸。

Re-tool…​
重新處理零件的選定工具而不影響其他工具。

屏蔽模具
抑制選定的工具，使其從進一步處理和輸出產生中排除。

上一個 / 下一個 / 零件
導覽至上一個或下一個工具，或返回到父零件。

排序和篩選
顯示與選定工具具有類似特徵的其他工具以供快速參考。

匯出輸出

此選項將零件輸出匯出到 工廠設定 中配置的目的地。（請參閱 Bend Settings 和 Cut Settings）

僅匯出與選定工具操作關聯的工具，而不是整個零件。

匯出的檔案取決於操作類型。對於 Bend 和 Fold 操作，會產生 NC 檔案和報告。對於 Cut 和 Punch 操作，僅匯出零件檔案。

Check Tooling

  • 如果 CAM 設定發生變更或折彎工具遺失，您不需要重新處理零件工具。

  • 相反，使用檢查工具來驗證現有工具而不進行任何變更。

  • 如果驗證失敗，該實例的工具狀態會更新，並移除相關的輸出檔案。

  • 檢查工具還會檢測先前使用 忽略錯誤 手動覆蓋錯誤的零件，並在需要時將其狀態更新為錯誤。

複製模具…

複製工具允許您快速將折彎工具從一個解決方案套用到其他解決方案，節省時間和精力。

這會從選定的參考工具（解決方案）複製沖頭、序列和後擋料設定，並嘗試將相同的設定套用到所有其他解決方案。

  • 如果複製的設定對目標機器可行，則會儲存並使用。

  • 如果設定不可行，則該機器的原始工具保持不變。

例如，您可能對不同的機器有不同的工具：

  • 這裡，Machine 5230SX 6A 使用鵝頸沖頭，Machine 5085X 6A B23 使用尖頭沖頭。

當您在 Machine 5230SX 6A 上使用 複製模具… 時，它會嘗試將鵝頸設定複製到所有其他機器。

相同的沖頭、序列和後擋料設定會在可能的情況下套用。如果無法使用複製的設定，則原始工具保持不變。

套用複製工具後，鵝頸設定已成功套用到所有其他可行的機器。

設定為基面…

設定為基面… 功能定義哪個折彎或折邊工具用作產生切割解決方案的參考平面尺寸。當工具標記為 Base-Flat 時，TruSmartCAM 會根據該機器更新零件的平面尺寸並相應地重新產生切割解決方案。Base-Flat 工具在 UI 中以藍色邊框顯示。

不同的折彎機器可能會對同一零件產生略微不同的平面尺寸。當您將工具設定為 Base-Flat 時，TruSmartCAM 會使用該機器的平面尺寸作為主參考並自動重新計算切割解決方案。

自動基準平面選擇

當啟用工廠設定 折彎過程中應用平面校正 時：

  • TruSmartCAM 會在零件匯入期間自動將折彎工具（包含最大折彎長度表）設定為 Base-Flat。

  • 如果選定的折彎工具遇到錯誤或不可行，TruSmartCAM 會智能切換並將另一個有效的折彎工具設定為 Base-Flat。這確保平面尺寸參考和切割解決方案始終保持一致和有效，無需手動介入。

增強的基準平面管理

TruSmartCAM 在儲存折彎或折邊工具編輯時，在 儲存到 Praxis 對話方塊中提供 Base-Flat 選項。這些選項僅針對處於正常狀態的工具顯示，並確保平面尺寸和切割解決方案保持準確。

設定為基面並計算切割程式

此選項用於首次將工具標記為 Base-Flat 並自動產生相應的切割解決方案。

出現時機：

  • 工具目前未標記為 Base-Flat。

  • 使用者已簽出工具進行編輯並正在儲存變更。

要使用此選項，請簽出折彎或折邊工具並進行所需的修改，例如對折彎參數或平面尺寸的變更。按 Ctrl+S 儲存變更。在 儲存到 Praxis 對話方塊中，選擇 設定為基面並計算切割程式 並按一下 確定。TruSmartCAM 將設定工具為 Base-Flat 並使用更新的平面尺寸產生切割解決方案。

更新基面並重新計算切割程式

此選項用於工具已標記為 Base-Flat 並已編輯，需要重新產生平面尺寸和切割解決方案。

出現時機：

  • 工具已標記為 Base-Flat。

  • 使用者已簽出、編輯並正在儲存變更。

編輯現有的 Base-Flat 工具時，根據需要更新折彎參數或平面尺寸幾何形狀。在 儲存到 Praxis 對話方塊中，選擇 更新基面並重新計算切割程式。TruSmartCAM 然後會更新 Base-Flat 工具並重新產生所有相關的切割解決方案以反映最新的變更。

一次只能將一個折彎工具設定為 Base-Flat。

當 Base-Flat 變更時，現有的切割解決方案會自動重新處理。

此功能可用於 TruSmartCAM 中的折彎和折邊工具。

重新裝備…​

此選項重新處理零件的選定工具。

在工具面板中的所需工具上按一下右鍵並按一下 重新裝備…​

在繼續之前會顯示確認對話方塊。

按一下 是 以繼續重新工具處理，或按一下 否 以取消操作。

僅重新處理選定的工具。與零件關聯的其他工具不受影響。與選定工具相關的現有輸出會在重新工具處理過程中移除並重新產生。

當工具資料過時、不正確或對機器或工具配置進行變更時，請使用此選項。

忽略錯誤

  • 如果已處理工具的實例顯示警告，而您仍想繼續，可以透過按一下 忽略錯誤 來覆蓋警告。

  • 這將允許系統為該實例匯出輸出檔案，即使存在工具問題。

僅在您確定可以安全忽略警告時才使用 忽略錯誤。覆蓋實際錯誤可能導致機器損壞或安全風險。

抑制/取消抑制工具

  • 當機器不活動或您不希望特定零件由特定機器處理時，使用 屏蔽模具 將該零件從該機器的排版中排除。

  • 抑制後，將滑鼠懸停在已處理工具實例上會顯示訊息，例如："Tooling suppressed by <User name> on <Date Time>"。

  • 抑制工具會移除該機器和零件的輸出檔案。

  • 使用 解除屏蔽模具 以恢復該機器上零件的工具狀態。

  • 取消抑制後，會再次產生相應的輸出檔案。
