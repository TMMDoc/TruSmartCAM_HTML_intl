零件篩選器

零件資料庫篩選器讓您可以透過套用特定條件快速搜尋和定位資料庫中的零件。透過將結果縮小到僅符合您的條件的零件，這些篩選器使您更容易有效地管理和存取相關零件。

狀態

  • 使用 狀態 篩選器根據工具狀態搜尋零件。

  • 下拉式選單將工具狀態顯示為彩色方塊，以及所有可用的機器及其適用的錯誤。

  • 彩色方塊代表以下狀態：

    • 白色：工具正常

    • 橙色：工具有錯誤

    • 黃色：工具有警告

    • 灰色：工具已抑制

  • 如果零件有窄法蘭或未指派工具錯誤，選擇這些將顯示相應的零件。

  • 透過按一下灰色方塊，將顯示在所選機器上抑制工具的零件。

折彎安裝

  • 使用 折彎安裝 篩選器根據使用的折彎工具搜尋零件。

  • 下拉式選單顯示所有可用的折彎工具，可以按工具名稱、類型（沖頭、鵝頸、半徑等）或說明進行篩選。

  • 選擇多個工具並按一下 匹配全部|&** 以排序使用兩個所選工具的工具。

使用的模具

  • 按 使用的模具 篩選的工作方式類似於按 折彎安裝 篩選。

  • 工具圖示以成比例的大小顯示（不精確，但相對）。

  • 當工具預覽開啟時，將滑鼠懸停在篩選器清單中的工具上會在預覽窗格中反白顯示該工具。

  • 可以選擇多個工具並使用 匹配全部|&** 套用。

狀態

  • 使用 狀態 篩選器根據零件的當前狀態搜尋零件。

  • 下拉式選單根據可用的資料庫零件顯示所有適用的狀態。

  • 狀態清單會根據所選零件及其狀態動態變更。

  • 例如，選擇 Material Missing 以檢視處於未解決材料狀態的零件。
