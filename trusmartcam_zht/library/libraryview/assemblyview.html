組件

從組件建立靜態排版

可以使用互動式排版精靈為組件產生靜態排版。這些排版是僅與所選組件關聯的固定配置。

  • 切換到 組件 檢視。

  • 在所需的組件上按一下右鍵並選擇 建立排樣…。

  • 互動式排版精靈開啟。

  • 預設情況下，互動式排版精靈切換到 選擇機台和材料 頁面。

  • 機器顯示有顏色條，指示可用的生產負荷：

    • 綠色條：已排程的小時數

    • 灰色條：可用小時數（8小時輪班中）

  • 在每台機器下，選定的零件會與其原材料分組（零件/總數量）。

  • 如果零件沒有適用於機器的有效工具解決方案，則該零件和材料將被忽略，不會對給定機器進行處理。

  • 如果為一台機器選擇了原材料，則該原材料不可用於其他機器。

  • 一旦設定了機器排版計劃，選擇首選板材並覆蓋板材參數，如 優先級、數量、板材邊距 等。

  • 您可以篩選板材以顯示 全部板材、標準板材 或 剩餘板材。按一下 下一個。

此階段無法增加數量。您可以保持與板材庫存數量相同，或減少數量並檢查排版結果。

  • 更新數量 出現在互動式排版計劃視窗中。

  • 預設情況下，拾取全部 (F5) 包含所有零件進行排版，或者您可以手動選擇優先零件並調整零件數量。按一下 下一個。

  • 排版開始，排版到板材中的零件會以藍色狀態更新。

  • 機器的生產負荷根據排版結果更新。

  • 如果某些零件仍未排版，將顯示訊息 部分零件仍然未排樣 - 這些零件保留在原始工作中。

    • 如果沒有合適的板材可用，零件可能會保持未排版狀態。

  • 靜態排版可以編輯以包含未排版的零件：

    • 選擇要插入未排版零件的配置並簽出以進行編輯。

  • TecZone 開啟。按一下 _拾取 或 拾取_全部 從父組件中選擇未排版的零件。

  • 根據需要在配置中排列零件。

  • 產生的排版會自動以父組件命名，以便於識別。

靜態排版不能包含來自其他組件的零件。允許混合組件排版會導致無法在不發布其他組件的情況下發布一個組件，因此設計上不允許混合。

非鈑金元件

  • 當匯入組件時，鈑金零件會被提取並新增到零件資料庫中的組件中。

  • 非鈑金元件會被分離並新增到 部件 中。

  • 組件會新增到組件資料庫，其 BOM（物料清單）包含零件和元件。

  • 如果零件已經作為鈑金零件匯入，系統會更新現有條目並從零件資料庫中刪除重複項。

  • 選擇組件會顯示零件和元件，元件數量顯示在元件影像下方（所選組件元件計數和總元件計數）。

  • 元件工具提示顯示名稱、材料和大小。元件在生產過程中被排除 - 它們不會被排版，並且不會為元件產生板材。

  • 範例組件和元件可在以下位置取得：C:\Program Files\Metamation\Praxis\Samples\Assys\Components，可用於測試元件匯入和組件工作流程。

從組件建立工作

  • 在組件上按一下右鍵，按一下 新…​，然後按一下 套裁。

  • 在新工作對話方塊中，設定組件計數，按一下 確定，然後根據需要更新工作詳細資訊（例如到期日和優先順序）。

  • 系統從組件建立工作並將靜態排版複製為工作排版，更新板材計數。複製的排版會加上工作 ID 作為前綴。

  • 工作排版可以編輯或刪除，而不會影響原始靜態排版。

  • 工作排版可以使用 適應/緊湊… 或 新增零件… 重新打包。

  • 詳細步驟請參閱 Adapt/Compact 和 Add parts。

如果零件沒有出現在排版中，請檢查排版編輯對話方塊 - 您可以手動插入未排版的零件。

使用儲存的搜尋和篩選器快速尋找多個組件中的零件。

使用清楚、描述性的組件和排版名稱，以避免將排版複製到工作時產生混淆。
