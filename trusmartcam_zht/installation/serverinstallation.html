伺服器安裝

按照說明安裝 TruSmartCAM 伺服器以及 SQL。

管理員權限

若要執行此安裝，使用者必須具有管理員權限。

  • 執行 Setup.TruSmartCAM.<version>.exe 以開始安裝。

版本是一個數字。

  • 閱讀授權協議，選擇 I accept the agreement，然後點擊 下一個 。

步驟 1：安裝位置

  • 提及安裝目錄或接受安裝預設值 C:\Program Files\Metamation\Praxis，然後點擊 下一個。

    • C:\Program Files\Metamation\Praxis - 安裝資料夾將包含以下資料夾：Bin、MetaCAM、Samples 和 Tracker。

    • C:\Program Data\Metamation - 資料資料夾將包含以下資料夾：TruSmartCAM、TecZone 和 MetaCAM。

步驟 2：元件選擇

安裝類型 元件

Full

Server + Desktop client + Engine

Client

Desktop Client

Server

Server + Desktop client

Station

選擇所有工作站終端機，如 TecZone、MetaCAM、RightAngle（折彎控制器）、Vulcan（雷射機器控制器）

Custom

由使用者選擇。

  • 選擇 Full Installation。從 TruSmartCAM Tools 中選擇 Code Senders 和 Bend Code Senders。現在安裝類型變更為 定制。點擊 下一個。

步驟 3：選擇安裝 SQL Server

SQL Express 2022 安裝僅適用於 TruSmartCAM 伺服器。此選項不適用於 Client 和 Sync 安裝。此外，這在首次安裝時會勾選並安裝，之後可以忽略。如果 TruSmartCAM 伺服器中已安裝 SQL，使用者可以忽略步驟 3。

  • 在 Select Additional Tasks：點擊 Install SQL Express，然後點擊 下一個。

  • 在 Ready to Install：總結所有安裝功能，然後點擊 Install。

步驟 4：SQL Server 安裝

安裝完 TruSmartCAM 元件後，安裝程式會繼續進行 SQL 伺服器安裝。接受授權協議以繼續，並使用預設選項安裝 SQL Express。

安裝完成後，點擊 Close 並完成安裝。

根據 Windows 版本，關閉 SQL 安裝可能會提示重新啟動。

步驟 5：Windows 桌面捷徑

  • SQL 安裝後，如果系統未重新啟動，則選擇 Launch TruSmartCAM 核取方塊，然後點擊 Finish。

  • 如果系統重新啟動，則啟動桌面上的 TruSmartCAM 捷徑圖示，或在 Windows 開始程式中輸入 TruSmartCAM，然後 以系統管理員身分執行。

伺服器安裝：桌面上有 TruSmartCAM 和 TruSmartCAM License 捷徑圖示。 + 用戶端/SYNC 安裝：桌面上僅有 TruSmartCAM 捷徑圖示。

步驟 6：授權註冊

  • 在 Serial Number 欄位中，輸入 Trumpf Metamation Service Team 提供的 TruSmartCAM Serial key。在「Registered to」欄位中輸入 <Company Name> 和電子郵件。點擊 OK。

如果網路防火牆阻止授權註冊或 TruSmartCAM 伺服器電腦沒有網際網路，則透過離線啟動完成 TruSmartCAM 授權。將 C:\ProgramData\Metamation\Praxis 資料夾中的 Prax.request 以電子郵件傳送給 Trumpf Metamation Service Team。Trumpf Metamation Team 將提供 Prax.lock，需要將其放到此資料夾中。

步驟 7：初始一次性配置

  • 輸入資料庫名稱並設定資料儲存位置。選擇 使用 SQL 伺服器 核取方塊，並選擇要連接到資料庫的 SQL 伺服器名稱。點擊 OK。

  • TruSmartCAM 桌面用戶端開啟，在左下方顯示 Database Connection Succeeded。

  • TruSmartCAM、TecZone 和 MetaCAM 配置檔案會使用預設配置機器或根據 TruSmartCAM 授權機器建立。所有 TruSmartCAM 用戶端都將共用相同的配置。

歡迎畫面顯示 Windows 使用者 LoginName 作為 TruSmartCAM 登入和名稱。第一個 TruSmartCAM 使用者被視為 現場管理員。提供電子郵件和密碼。點擊 確定 。

選擇 記住我 核取方塊以儲存下次使用的登入憑證。 清除 記住我 核取方塊將在每次啟動 TruSmartCAM 應用程式時提示登入和密碼對話方塊。

步驟 8：TruSmartCAM 登入

  • 關閉 TruSmartCAM 並重新開啟。系統會提示您輸入 TruSmartCAM 登入憑證。

  • 登入的使用者名稱顯示在 TruSmartCAM 應用程式標題中使用者圖示旁邊。

  • 當您將滑鼠游標停留在使用者圖示上時，系統會顯示：

    • 使用者角色（例如：Programmer、Admin）。

    • 登出的快捷鍵：Ctrl+L。

  • 點擊使用者圖示會開啟一個選單，讓您可以：

    • 編輯資訊…​ - 更新您的個人資料詳細資訊。

    • 更改密碼… - 變更您的帳戶密碼。

    • 登出 - 登出應用程式。

    • 取消 - 關閉選單而不做任何變更。

步驟 9：工廠機器配置

  • TruSmartCAM 授權 [所有機器啟用授權位元]：預設的 TruSmartCAM 機器和材料被視為工廠配置。

  • TruSmartCAM 授權 [無演示且啟用少數機器位元]：這些少數機器和預設材料被視為工廠配置。

  • （Open → Import）從 C:\Program Files\Metamation\Praxis\Samples\Parts 匯入新零件，並驗證零件是否成功匯入。

注意 1：TruSmartCAM Monitor

  • 安裝 Engine 的 TruSmartCAM Monitor（Site admin 權限）：

    • 具有 Engine 和 Site Admin TruSmartCAM 權限的 TruSmartCAM Monitor 將具有以下選項：

      1. 將 TruSmartCAM 配置、TecZone、MetaCAM 配置推送到所有 TruSmartCAM 用戶端。還有權限建立新的 TruSmartCAM 使用者登入。

      2. Show Agents 將顯示可用於執行自動化任務的總引擎數量，這與 TruSmartCAM 授權相關。

  • 未安裝 Engine 的 TruSmartCAM Monitor（Site admin 權限）：

  • 未安裝 Engine 的 TruSmartCAM Monitor（非 Site Admin 權限）：

執行 TruSmartCAM 應用程式的電腦必須安裝 Engine TruSmartCAM 元件 [伺服器電腦或任何用戶端安裝]。安裝 Engine TruSmartCAM 元件的電腦應全天候執行以執行所有自動化任務。

注意 2：Lock Tracker

  • 點擊 TruSmartCAM 伺服器桌面上可用的 TruSmartCAM License 捷徑。或者，在任何網頁瀏覽器中開啟 http://localhost:7171/metamation/locktrack 網址。

這將開啟 Lock Tracker 模組，其中包含授權資訊、TruSmartCAM 目前會話資訊和記錄。

LockTracker HOME 頁面：這顯示 TruSmartCAM 序號、序號到期資訊和可用的序號位元資訊。

LockTracker SESSIONS 頁面：目前 TruSmartCAM 伺服器連接資訊，如 TruSmartCAM 桌面用戶端電腦、PMonitor、PEngine 狀態。

LockTracker LOGs 頁面：TruSmartCAM 用戶端啟動和登出 | Pmonitor 啟動和 PEngine 連接/斷開狀態等，可以在此記錄頁面中查看。

注意 3：OpenGL

  • TruSmartCAM 應用程式在開啟時立即關閉，並且 pmonitor 未在工作列中顯示。

  • 這表示 OpenGl 問題。

  • 另一種確認方法：開啟 C:\Program Files\Metamation\Praxis\Bin\Flux.exe，會顯示有關 OpenGL 問題的錯誤對話方塊。

  • 更新 C:\ProgramData\Metamation\Praxis\Prax.ini，options 區段使用 GLVersion=1，使 TruSmartCAM 應用程式成功開啟。

[Options]
GLVersion=1.0
