Watch

Watch Settings 允許 TruSmartCAM 監視指定的資料夾並自動將檔案匯入至系統。這有助於自動匯入 Parts、Jobs、Assemblies 和 Orders，而無需手動介入。

監視設定

啟用「即時資料夾」 - 啟用對指定資料夾的自動監視。

監視此資料夾 - 指定要監視傳入檔案的資料夾。

也監視子資料夾 - 在監視處理中包括所有子資料夾。

檔案匯入後

匯入後 - 匯入檔案之後，使用者有兩個選項。可以從位置刪除檔案或將其移至不同的位置。

  • Delete imported file - 在成功匯入之後移除來源檔案。

  • Move imported file - 在成功匯入之後將來源檔案移至在 Move files to this folder 中指定的資料夾。

將檔案移至此資料夾 - 指定當 After import 設定為 Move imported file 時應將成功匯入的檔案移至的目的地資料夾。

將有錯誤的電子表格移至此資料夾 - 將失敗的匯入移至指定的錯誤資料夾。

電子表格匯入設定

從「即時資料夾」匯入電子表格 - 自動匯入在監視資料夾中偵測到的試算表檔案。

請勿匯入包含錯誤的電子表格 - 防止匯入包含驗證錯誤的檔案。

電子表格匯入選項 - 從下拉式清單中選取要匯入的資料類型。支援的試算表格式可能包括 CSV、XLSX 或 XML。可用的選項包括：

  • 匯入套裁 - 從試算表匯入工作資料。

  • 匯入訂單 - 從試算表匯入訂單資料。

  • 匯入組件 - 從試算表匯入組件資料。

  • 匯入零件 - 從試算表匯入零件資料。

其他資訊

  • 設定監視資料夾時，使用 UNC 路徑 (\\ServerName\FolderPath) 或對應的磁碟機 (M:\FolderPath)。請勿使用本機路徑，例如 C:\FolderPath。

  • 執行 TruSmartCAM Engine 服務的電腦必須具有對設定的監視資料夾的讀取和寫入存取權。如果沒有足夠的權限，可能無法成功偵測、匯入、移動或刪除檔案。

  • 必須在啟用 Watch Settings 之前設定試算表對應。否則，TruSmartCAM 將無法處理在監視資料夾中偵測到的試算表檔案。
