Enregistrer les filtres

Les filtres enregistrés vous permettent de réutiliser les critères fréquemment appliqués dans Pièces, Travaux et Mises en page, rationalisant ainsi votre flux de travail. Chaque module est livré avec des filtres par défaut, mais des listes de filtres supplémentaires peuvent être créées par les administrateurs de site ou les programmeurs selon les besoins. La réutilisation des filtres enregistrés permet de gagner du temps et aide à maintenir la cohérence des recherches.

  • Cliquez sur l’icône pour afficher les filtres par défaut.

Étapes pour enregistrer les filtres

  • Cliquez sur l’icône , sélectionnez un élément de filtre.

  • Dans le menu déroulant de l’élément de filtre sélectionné, choisissez une option. Une coche apparaîtra à côté de la condition sélectionnée.

  • Pour appliquer la condition, cliquez sur l’icône puis sélectionnez Ajouter condition (Ctrl+PLUS)]. Vous pouvez répéter cette étape pour ajouter plusieurs conditions.

  • Pour enregistrer ces filtres sélectionnés avec des conditions, cliquez sur Enregistrer filtre…​ (Ctrl+S)].

  • Entrez le nom puis cliquez sur _Enregistrer.

  • À partir de la boîte de dialogue Enregistrer le filtre, vous pouvez également supprimer tous les filtres précédemment enregistrés.

  • Ces filtres seront enregistrés dans la liste de filtres par défaut, ici Matériau est enregistré dans la liste par défaut.

Les filtres créés par SiteAdmin et Admin sont partagés entre tous les utilisateurs. Les filtres créés par Programmer et Operator sont partagés uniquement avec les rôles d’utilisateur Programmeurs et Opérateurs.

Commandes de filtrage

Commande Icône Raccourcis Description

Clear Selection

Ctrl+X

Désélectionne tous les choix sélectionnés et actualise la vue des éléments de filtre.

Select Highlighted

Ctrl+A

Sélectionne tous les choix mis en évidence. Cette commande reste désactivée si la mise en évidence des éléments n’est pas applicable pour le champ sélectionné.

Match All

&

Affiche uniquement les éléments qui répondent à toutes les conditions sélectionnées.

Pin In/Out Menu

Ctrl+Down/Up

Lorsqu’il est épinglé, le menu déroulant reste ouvert même lorsque le focus actuel se déplace vers une autre fenêtre.

Close Menu

Esc

Ferme le menu déroulant.
