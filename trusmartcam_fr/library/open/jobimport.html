Importer des tâches

Importer depuis csv ou xlsx

Vous pouvez également importer des tâches dans des formats texte courants

  • Cliquez sur File → Open → Importer Tâches, la boîte de dialogue Open Folder apparaît. Sélectionnez C:\Program Files\Metamation\Praxis\Samples\Jobs\C0\C0.csv.

  • Si toutes les pièces existent dans l’emplacement csv ou dans les répertoires de dossiers de recherche, la fenêtre nouvelle tâche apparaît en listant toutes les pièces correspondant aux champs csv.

  • Les pièces manquantes sont ignorées et n’apparaissent pas dans la fenêtre nouvelle tâche.

  • Initialement, l’état de la pièce est Nascent.

  • Cliquez sur OK, les pièces sont maintenant importées dans la bibliothèque de pièces. La validation CAD, l’outillage pour les instances d’outillage de pliage et de coupe sont effectués, ce qui peut être surveillé dans l’écran Pmonitor.

Créer une pièce vierge pour les fichiers CAO manquants

Lors de l’importation de tâches à l’aide d’une feuille de calcul (.csv ou .xlsx), vous pouvez parfois avoir des lignes faisant référence à des fichiers CAD qui ne sont pas disponibles dans le système.

  • Pour gérer cela, allez dans usine → Réglages → Tâche, sous Importation feuille de calcul activez Créer une pièce vierge pour les fichiers CAO manquants

Lorsque cette option est sélectionnée :

  • Si un fichier CAD est manquant pour une ligne lors de l’importation de la tâche, le système va automatiquement créer une pièce vierge à la place du fichier CAD manquant.

  • Dans la fenêtre nouvelle tâche, ces pièces manquantes sont visuellement mises en surbrillance en jaune pour indiquer que le fichier CAD associé est manquant.

  • La pièce manquante aura toujours ses champs mappés (comme le nom de la pièce, le matériau, l’épaisseur, la quantité, etc.) du CSV. Cependant, son état de pièce sera affiché comme Nascent, ce qui signifie qu’il s’agit simplement d’un emplacement vide sans géométrie réelle encore.

Cela garantit que l’importation de la tâche peut toujours se poursuivre sans erreurs dues aux fichiers CAD manquants — vous pouvez ultérieurement mettre à jour ou remplacer la pièce vierge par le fichier CAD correct lorsqu’il devient disponible.

Ne pas afficher la fenêtre de dialogue Nouvelle tâche si la feuille de calcul ne contient pas d’erreur

usine → Réglages → Tâche → Importation feuille de calcul→ Ne pas afficher la fenêtre de dialogue Nouvelle tâche si la feuille de calcul ne contient pas d’erreur

  • Lorsque cette case est cochée, le système crée automatiquement la nouvelle tâche sans afficher la boîte de dialogue New Job si la feuille de calcul ne contient pas d’erreurs (pas de pièces manquantes, de dates manquantes ou de données invalides).

  • Lorsque cette case est décochée, la boîte de dialogue New Job apparaît toujours pour révision par l’utilisateur et confirmation, même si la feuille de calcul ne contient pas d’erreurs.

Prefer raw-material column over material + thickness

usine → Réglages → Tâche→Prefer Raw-Material column over Material + Thickness

  • Lorsque cette case est cochée, le système attend que la feuille de calcul utilise la colonne Raw-Material (par exemple, Steel-20). Cela signifie que le type de matériau et l’épaisseur doivent être combinés en une seule valeur dans une seule colonne.

  • Lorsque cette case est décochée, le système attend que la feuille de calcul ait des colonnes séparées pour Material et Thickness (par exemple, Steel et 2.00). Le système les combine en interne lors de l’importation.

Surveiller un dossier avec CSV/XLSX

  • Dans usine → Réglages→ Voir cochez Activer 'Dossier live' et Voir également les sous-dossiers.

  • Définissez le chemin du dossier surveillé (UNC ou lecteur mappé).

  • Dans Importation feuille de calcul, cochez Importer la feuille de calcul depuis le 'Dossier live'.

  • Sélectionnez Importer Tâches dans les options Spreadsheet Import.

  • Copiez C:\Program Files\Metamation\Praxis\Samples\Jobs\C0\C0.csv et collez-le dans le Watch Folder.

  • Le système vérifie le CSV et tente de créer la tâche automatiquement.

  • Si le fichier CAD est manquant, aucune tâche n’est créée car les données CAD requises ne sont pas trouvées.

Répertoire de recherche CAD et recherche CAD récursive - non configurés

  • Dans usine → Réglages → Tâche → Importation feuille de calcul, si aucun CAD lookup directories n’est défini et Utiliser la recherche CAO récursive n’est pas utilisé, le système ne recherche pas automatiquement les fichiers CAD.

  • Si la case Créer une pièce vierge pour les fichiers CAO manquants est cochée, la tâche sera toujours créée même lorsque les fichiers CAD sont manquants.

  • Dans ce cas, toutes les pièces de la tâche seront créées en tant que pièces nascent (vierges) — ce qui signifie qu’elles n’ont pas de géométrie car les fichiers CAD n’ont pas pu être trouvés.

Répertoire de recherche CAD et recherche CAD récursive - configurés

Lorsque Répertoires de recherche CAO est défini à (C:\Program Files\Metamation\Praxis\Samples) et Utiliser la recherche CAO récursive est activé sous usine → Réglages → Tâche → Importation feuille de calcul:

  • Le système recherche automatiquement le dossier spécifié et tous ses sous-dossiers pour les fichiers CAD requis mentionnés dans la feuille de calcul.

  • Créer une pièce vierge pour les fichiers CAO manquants est également activé, donc si un fichier CAD n’est toujours pas trouvé, le système créera une pièce vierge comme emplacement réservé.
