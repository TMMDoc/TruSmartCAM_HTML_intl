Importer des commandes

Les commandes peuvent être créées ou importées en utilisant les méthodes suivantes :

  1. Importer des commandes à partir de fichiers de feuille de calcul.

  2. Importer des commandes automatiquement via les paramètres de surveillance.

  3. Créer des commandes à partir de pièces existantes dans la bibliothèque de pièces.

  4. Créer des commandes à partir d’imbrications statiques d’assemblage.

Importer des commandes à partir d’une feuille de calcul

  • Naviguez vers ouvrir → Importer _Ordres…​.

  • Les formats de feuille de calcul pris en charge incluent CSV, XLSX et XML.

  • Sélectionnez le fichier de feuille de calcul requis et cliquez sur ouvrir.

  • Le processus d’importation est similaire au flux de travail d’importation de travail/assemblage.

Vous pouvez également glisser-déposer directement la feuille de calcul dans la page Commandes pour importer des commandes.

Importer des commandes en utilisant les paramètres de surveillance

  • Activez l’option Importer la feuille de calcul depuis le 'Dossier live' dans Voir réglages.

  • Sélectionnez Importer Ordres dans Option d’importation de feuille de calcul.

  • Configurez l’emplacement du dossier à partir duquel les commandes doivent être importées.

  • Le système surveille en continu le dossier configuré.

  • Les nouveaux fichiers de feuille de calcul ajoutés sont automatiquement importés dans Commandes.

Pour des étapes de configuration détaillées, consultez la documentation Watch settings.

Créer des commandes à partir de la bibliothèque de pièces

  • Sélectionnez les pièces requises dans la bibliothèque de pièces.

  • Faites un clic droit et choisissez Nouveau…​ → Ordres pour ouvrir la boîte de dialogue Nouveaux ordres.

  • Entrez les détails requis tels que la quantité, la date d’échéance et la priorité.

  • Cliquez sur OK pour créer les commandes.

  • Le système bascule automatiquement vers la page Commandes.

  • Les commandes créées sont affichées sur la page Commandes prêtes pour la création de travail.

Créer des commandes à partir d’un assemblage

  • Faites un clic droit sur l’assemblage requis.

  • Sélectionnez Nouveau…​ → Ordres pour ouvrir la boîte de dialogue Nouveaux ordres.

  • Entrez le Nombre de modules requis.

  • Cliquez sur OK pour créer les commandes.
