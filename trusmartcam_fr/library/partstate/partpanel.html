Panneau de pièces

Cette section explique les actions liées aux pièces telles que Réimporter…​, Réoutillage…​, Renommer…​, Exporter sorties, etc., utilisées pour gérer les instances de pièces.

Ouvrir en mode lecture seule pour la visualisation
Opens the selected part in read-only mode.

Check-out the document for editing
Enables editing of the selected part.

Supprimer
Deletes the selected part.

Nouveau…​
Selected part can be converted to a new order, job or assembly.

Exporter sorties
Exports the part output to the destination configured.

Réimporter…​
Allows to import the selected part again.

Réoutillage…​
Allows to reprocess the selected part.

Renommer…​
To change the name of selected part.

Attribuer matériau…​
Specifies the material associated with the selected part.

Assigner le client…​
Specifies the customer associated with the selected part.

Sens de laminage
Defines the orientation (horizontal or vertical) along which the material is rolled.

Préc / Suivant / Outillages
Navigates to the previous or next part or tooling.

Sort and filter
Use these filters to view other parts that share the same attribute as the selected part: Same Material, Same Customer, Same Bend count, or Same State.

Réimporter…​

  • Cliquez avec le bouton droit sur n’importe quelle pièce et cliquez sur Réimporter…​.

  • Toutes les instances d’outillage de pièces sélectionnées sur toutes les machines sont effacées, y compris les instances outillées automatiquement et manuellement.

  • Les fichiers de sortie existants (CAD, Bend et Cut) sont supprimés.

  • Les pièces sont réimportées et une nouvelle validation CAD et un nouvel outillage sont appliqués pour toutes les machines.

  • De nouveaux fichiers de sortie sont créés en fonction du résultat de l’outillage mis à jour.

Réoutillage…​

  • Cliquez avec le bouton droit sur n’importe quelle pièce et cliquez sur Réoutillage…​.

  • Toutes les machines sont cochées par défaut et groupées par Laser, Punch, Panel, et Press brake.

  • Cliquez sur OK pour continuer. Cela supprime les instances outillées automatiquement et manuellement pour les machines sélectionnées, et supprime les fichiers de sortie associés.

  • Lorsque la case Conserver les sorties existantes est cochée, seules les pièces dont les sorties ont été supprimées précédemment seront réoutillées, les autres sorties restent intactes. Si elle est décochée, tous les fichiers de sortie pour les machines sélectionnées sont supprimés et regénérés lors du réoutillage.

  • Si les paramètres CAM ont changé pour une machine, sélectionnez cette machine et réoutillez les pièces.

  • Si un outil (LTT, punch ou bend) est modifié ou manquant, mettez à jour la machine et réoutillez uniquement les pièces affectées.

Renommer…​

Cette section explique comment renommer une ou plusieurs pièces dans TruSmartCAM.

  • Cliquez avec le bouton droit sur la pièce et cliquez sur Renommer…​.

  • Une boîte de dialogue de confirmation de renommage apparaît affichant le nom actuel de la pièce.

  • Entrez le nouveau nom et cliquez sur l’icône coche ✓ pour confirmer, ou cliquez sur l’icône croix ✗ pour annuler l’opération de renommage.

  • Pour renommer plusieurs pièces, sélectionnez toutes les pièces requises, puis cliquez avec le bouton droit sur l’une d’entre elles et cliquez sur Renommer…​.

  • Une boîte de dialogue de confirmation apparaît avec le nombre total de pièces sélectionnées.

  • Pour chaque pièce, entrez le nouveau nom et cliquez sur l’icône coche ✓ pour confirmer une par une.

  • Si une pièce a été sélectionnée par erreur, cliquez sur l’icône croix ✗ pour annuler. Alternativement, vous pouvez cliquer avec le bouton droit sur la pièce et cliquer sur Annuler Renommer pour ignorer le renommage.

Exporter sorties

Cette option exporte la sortie de la pièce vers la destination configurée dans Réglages usine. (Voir Paramètres de pliage et Paramètres de coupe)

L’emplacement de sortie est défini séparément pour Bend, Fold et Cut dans leurs paramètres de sortie respectifs. Tous les outillages associés aux opérations de pliage, de pliure et de coupe sont exportés pour la pièce entière.

Comme indiqué ci-dessous, les fichiers exportés sont enregistrés dans le dossier spécifié (par exemple, C:\PraxData\Out\Reports).

Les fichiers exportés dépendent du type d’opération. Pour les opérations Bend et Fold, des fichiers NC et des rapports sont générés. Pour les opérations Cut et Punch, seuls les fichiers de pièces sont exportés.

Attribution de matériau à la pièce

Les pièces avec le statut Matériau non attribué doivent être mises à jour avant tout traitement ultérieur. Les étapes ci-dessous expliquent comment attribuer un matériau.

  • Cliquez avec le bouton droit sur la pièce qui affiche le statut Matériau non attribué et sélectionnez Attribuer matériau.

  • La fenêtre Attribuer matériau s’ouvre.

  • La boîte de dialogue comporte quatre sections :

    • Suggestions - Suggère automatiquement des matériaux bruts bruts en fonction de l’épaisseur de la pièce importée.

    • Tous les matériaux - Liste tous les types de matériaux définis dans TruSmartCAM.

    • Toutes les matières brutes - Affiche tous les matériaux bruts disponibles dans le système.

    • Matières brutes utilisées - Affiche les matériaux bruts les plus récemment utilisés pour un accès rapide.

  • Vous pouvez utiliser le champ Rechercher…​ (Ctrl+F) en haut pour trouver rapidement des matériaux. Par exemple, tapez "steel" pour afficher tous les matériaux correspondants dans les trois sections.

  • Dans la liste filtrée, sélectionnez le matériau requis (par exemple, Steel-25) dans la liste Raw-Material.

  • Le matériau sélectionné sera attribué à la pièce.

Sens de laminage

Sens de laminage fait référence à l’orientation du grain dans une pièce, ce qui peut affecter la façon dont la pièce est coupée et traitée. Cette option vous permet de spécifier la Sens de laminage pour une pièce dans la bibliothèque de pièces. En fonction de la direction sélectionnée, la pièce sera placée dans la bonne orientation pour la coupe sur les machines disponibles.

  • Cliquez avec le bouton droit sur n’importe quelle vignette de pièce dans la bibliothèque de pièces ou la fenêtre Job.

  • Sélectionnez la direction de laminage préférée :

    • Aucun

    • Horizontal

    • Vertical

  • La pièce est réoutillée automatiquement pour toutes les machines de coupe disponibles en fonction de la direction de laminage sélectionnée.
