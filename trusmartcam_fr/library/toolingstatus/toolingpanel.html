Panneau d'outillage

Cette section explique les actions liées à l’outillage telles que Exporter sorties, Check Tooling, Masquer outillage, Configurer comme méplat de base…​, Copier montage…​, et d’autres options utilisées pour gérer les instances d’outillage.

Ouvrir en mode lecture seule pour la visualisation
Opens the selected tooling in read-only mode.

Check-out the document for editing
Enables editing of the selected tooling.

Exporter sorties
Exports the selected tooling output to the destination configured.

Check Tooling
Validates the existing tooling without making any changes and updates its status if the validation fails.

Copier montage…​
Copies the punch, sequence, and back gauge setup from the selected reference tooling and applies it to other solutions wherever feasible.

Configurer comme méplat de base…​
Marks the selected bend or fold tooling as the reference flat size for generating cutting solutions.

Re-tool…​
Reprocesses the selected tooling for the part without affecting other toolings.

Masquer outillage
Suppresses the selected tooling so that it is excluded from further processing and output generation.

Préc / Suivant / Pièce
Navigates to the previous or next tooling, or back to the parent part.

Sort and filter
Displays other toolings that share similar characteristics with the selected tooling for quick reference.

Exporter sorties

Cette option exporte la sortie de la pièce vers la destination configurée dans Réglages usine. (Voir Paramètres de pliage et Paramètres de coupe)

Seuls les outillages associés à l’opération d’outillage sélectionnée sont exportés, pas la pièce entière.

Les fichiers exportés dépendent du type d’opération. Pour les opérations Bend et Fold, des fichiers NC et des rapports sont générés. Pour les opérations Cut et Punch, seuls les fichiers de pièces sont exportés.

Check Tooling

  • S’il y a un changement dans les paramètres CAM ou si un outil de pliage est perdu, vous n’avez pas besoin de réoutiller la pièce.

  • Utilisez plutôt Check Tooling pour valider l’outillage existant sans effectuer de modifications.

  • Si la validation échoue, le statut d’outillage de cette instance est mis à jour, et les fichiers de sortie associés sont supprimés.

  • Check Tooling détecte également les pièces où les erreurs ont été précédemment remplacées manuellement utilisant Ignorer les erreurs, et met à jour leur statut en erreur si nécessaire.

Copier montage…​

La copie d’outillage vous permet d’appliquer rapidement l’outillage de pliage d’une solution à d’autres, ce qui permet de gagner du temps et des efforts.

Cela copie le punch, la séquence et la configuration de la butée arrière d’un outillage de référence sélectionné (solution) et tente d’appliquer les mêmes paramètres à toutes les autres solutions.

  • Si la configuration copiée est réalisable pour une machine cible, elle est enregistrée et utilisée.

  • Si la configuration n’est pas réalisable, l’outillage d’origine pour cette machine reste inchangé.

Par exemple, vous pouvez avoir différents outillages pour différentes machines :

  • Ici, Machine 5230SX 6A utilise un punch Gooseneck, Machine 5085X 6A B23 utilise un punch Pointed.

Lorsque vous utilisez Copier montage…​ sur Machine 5230SX 6A, cela tente de copier la configuration Gooseneck vers toutes les autres machines.

Le même punch, la même séquence et les mêmes paramètres de butée arrière sont appliqués partout où c’est possible. Si la configuration copiée ne peut pas être utilisée, l’outillage d’origine reste inchangé.

Après l’application de la copie d’outillage, la configuration Gooseneck a été appliquée avec succès à toutes les autres machines où elle est réalisable.

Configurer comme méplat de base…​

La fonctionnalité Configurer comme méplat de base…​ définit quel outillage de pliage ou de pliure est utilisé comme taille de développé de référence pour générer les solutions de découpe. Lorsqu’un outillage est marqué comme Base-Flat, TruSmartCAM met à jour la taille de développé de la pièce en fonction de cette machine et regénère les solutions de découpe en conséquence. Les outillages Base-Flat sont affichés avec une bordure bleue dans l’interface utilisateur.

Différentes machines de pliage peuvent produire de légères variations de taille de développé pour la même pièce. Lorsque vous définissez un outillage comme Base-Flat, TruSmartCAM utilise la taille de développé de cette machine comme référence principale et recalcule automatiquement les solutions de découpe.

Sélection automatique de base flat

Lorsque le paramètre d’usine Appliquer une correction à plat pendant le pliage est activé :

  • TruSmartCAM définit automatiquement l’outillage de pliage (qui contient le tableau de longueur de pli max) comme Base-Flat lors de l’importation de la pièce.

  • Si l’outillage de pliage sélectionné rencontre une erreur ou n’est pas réalisable, TruSmartCAM bascule intelligemment et définit un autre outillage de pliage valide comme Base-Flat. Cela garantit que la référence de taille de développé et les solutions de découpe restent toujours cohérentes et valides sans intervention manuelle.

Gestion améliorée de base flat

TruSmartCAM fournit des options Base-Flat dans la boîte de dialogue Enregistrer vers Praxis lors de l’enregistrement des modifications d’outillage de pliage ou de pliure. Ces options n’apparaissent que pour les outillages dans l’état OK et garantissent que la taille de développé et les solutions de découpe restent précises.

Configurer comme méplat de base et calculer le programme de coupe

Cette option est utilisée pour marquer un outillage comme Base-Flat pour la première fois et générer automatiquement la solution de découpe correspondante.

Apparaît lorsque :

  • L’outillage n’est actuellement pas marqué comme Base-Flat.

  • L’utilisateur a extrait l’outillage pour édition et enregistre les modifications.

Pour utiliser cette option, extrayez l’outillage de pliage ou de pliure et effectuez les modifications requises, telles que des modifications aux paramètres de pliage ou à la taille de développé. Appuyez sur Ctrl+S pour enregistrer les modifications. Dans la boîte de dialogue Enregistrer vers Praxis, choisissez Configurer comme méplat de base et calculer le programme de coupe et cliquez sur OK. TruSmartCAM définira l’outillage comme Base-Flat et générera la solution de découpe en utilisant la taille de développé mise à jour.

Mettre à jour le méplat de base et recalculer le programme de coupe

Cette option est utilisée lorsque l’outillage est déjà marqué comme Base-Flat et a été édité, nécessitant que la taille de développé et les solutions de découpe soient regénérées.

Apparaît lorsque :

  • L’outillage est déjà marqué comme Base-Flat.

  • L’utilisateur l’a extrait, édité et enregistre les modifications.

Lors de l’édition de l’outillage Base-Flat existant, mettez à jour les paramètres de pliage ou la géométrie de taille de développé selon les besoins. Dans la boîte de dialogue Enregistrer vers Praxis, choisissez Mettre à jour le méplat de base et recalculer le programme de coupe. TruSmartCAM mettra alors à jour l’outillage Base-Flat et regénérera toutes les solutions de découpe associées pour refléter les dernières modifications.

Un seul outillage de pliage peut être défini comme Base-Flat à la fois.

Lorsque le Base-Flat est modifié, les solutions de découpe existantes sont automatiquement retraitees.

Cette fonctionnalité est disponible pour les outillages de pliage et de pliure dans TruSmartCAM.

Réoutillage…​

Cette option retraite l’outillage sélectionné pour la pièce.

Cliquez avec le bouton droit sur l’outillage requis dans le panneau d’outillage et cliquez sur Réoutillage…​

Une boîte de dialogue de confirmation est affichée avant de continuer.

Cliquez sur Oui pour continuer le réoutillage, ou sur Non pour annuler l’opération.

Seul l’outillage sélectionné est réoutillé. Les autres outillages associés à la pièce ne sont pas affectés. Les sorties existantes liées à l’outillage sélectionné sont supprimées et regénérées lors du processus de réoutillage.

Utilisez cette option lorsque les données d’outillage sont obsolètes, incorrectes ou lorsque des modifications sont apportées aux configurations de machine ou d’outillage.

Ignorer les erreurs

  • Si une instance outillée affiche un avertissement, et que vous souhaitez quand même continuer avec elle, vous pouvez remplacer l’avertissement en cliquant sur Ignorer les erreurs.

  • Cela permettra au système d’exporter les fichiers de sortie pour cette instance, même s’il y avait des problèmes d’outillage.

Utilisez Ignore Errors uniquement lorsque vous êtes sûr que l’avertissement peut être ignoré en toute sécurité. Remplacer des erreurs réelles peut entraîner des dommages à la machine ou des risques de sécurité.

Supprimer/Rétablir l’outillage

  • Lorsqu’une machine est inactive ou que vous ne souhaitez pas qu’une pièce spécifique soit traitée par une machine particulière, utilisez Masquer outillage pour exclure cette pièce de l’imbrication sur cette machine.

  • Après la suppression, survoler l’instance outillée affiche un message tel que : "Tooling suppressed by <User name> on <Date Time>".

  • La suppression de l’outillage supprime le fichier de sortie pour cette machine et cette pièce.

  • Utilisez Afficher outillage pour restaurer le statut d’outillage pour la pièce sur cette machine.

  • Lorsqu’il est rétabli, le fichier de sortie respectif est à nouveau généré.
