Statut de l'outillage

Dans TruSmartCAM, chaque mosaïque de pièce indique visuellement le statut d’outillage de cette pièce pour toutes les machines disponibles. Cela aide les utilisateurs à voir rapidement pour quelles machines une pièce est outillée, et si l’outillage est prêt, nécessite une révision ou est manquant.

  • Le statut de l’outillage est affiché directement sur la mosaïque de pièce à l’aide d’icônes et d’indicateurs de couleur.

  • Chaque icône représente une technologie de machine (par exemple, Poinçonnage, Laser).

  • La couleur de l’icône indique le statut d’outillage actuel pour cette machine.

Le tableau ci-dessous explique la signification de chaque icône et couleur, aidant les utilisateurs à interpréter le statut d’outillage avec précision.

Tooling errors

A CAM (Computer-Aided Manufacturing) error occurs during the tooling ors manufacturing preparation stage. These errors arise due to issues like tool collisions, missing tooling, or invalid cutting conditions.

Bend tooling errors

  • Collisions détectées and Erreur de préhenseur à pince : The bending setup causes tool or gripper collisions during simulation.

  • Erreur de surintensité and Bride trop étroite ou trou trop proche : The bend tools are overloaded, or holes are located too close to the bend line, which may damage the part or tools.

  • Examen nécessaire and Courte plage de poinçonnage/vé : The tooling setup needs manual inspection, or the selected die/punch span is smaller than required for the part.

  • Outillage non attribué, Mauvaise butée arrière, or Part too big for this machine : Required bending tools are not available, backgauge is not configured properly, or the part is too large for the selected machine.

Cut tooling errors

  • Condition de coupe manquante : The cutting condition is not mapped for the raw material, so cutting cannot be performed.

  • Contours sans outils ou outils partiels : Some shapes or edges are missed or only partly covered during the tooling process, making the part incomplete for cutting.
