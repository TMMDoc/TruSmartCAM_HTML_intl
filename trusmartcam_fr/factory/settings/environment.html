Environnement

Réglages environnement

Unités de réglage par défaut du programme - L’utilisateur peut choisir si l’unité par défaut est mm ou pouce.

Nombre de décimales à afficher - Cette valeur sert à définir combien de décimales doivent être affichées.

Précision des décimales dans l’exportation/importation - Cette valeur sert à définir combien de décimales doivent être exactes lors de l’importation ou de l’exportation.

Afficher les outillages de pièce en groupes - Utilisez cette option pour sélectionner le groupe d’outillage pour afficher l’état d’outillage dans la bibliothèque de pièces.

Légendes des icônes de la barre de commande - L’activation de cette option donnera une légende sous chaque icône pour référencer ce que la page représentera. Lorsque cette option est désactivée, toutes les légendes des commutateurs sont supprimées et seule l’icône de chaque page différente est affichée.

Afficher les étiquettes d’état de tâche - L’activation de cette option affichera le nombre de pièces qui sont dans une tâche à la fin de la barre d’état de la tâche. Lorsque cette option est désactivée, le nombre de pièces est supprimé et la couleur pour afficher le processus restera.

Afficher les pièces imbriquées en fonction de la couleur de l’ordre de tâche - Cette option affichera chaque pièce différente dans une tâche avec une couleur différente. En désactivant cette option, toutes les pièces seront affichées dans la même couleur grise.

Afficher la durée dans le format de période (HH:MM:SS) - Cochez cette option pour afficher la durée au format HH:MM:SS. Lorsqu’elle est activée, la durée dans les détails de mise en page et les rapports de tâche/mise en page est affichée dans le format sélectionné. En raison des contraintes d’espace, la durée est affichée dans un format de secondes plus compact, sur les vignettes de mise en page.
