Surveillance

Les paramètres de surveillance permettent à TruSmartCAM de surveiller un dossier désigné et d’importer automatiquement des fichiers dans le système. Cela aide à automatiser l’importation de pièces, tâches, assemblages et commandes sans intervention manuelle.

Voir réglages

Activer 'Dossier live' - Active la surveillance automatique du dossier spécifié.

Voir ce dossier - Spécifie le dossier à surveiller pour les fichiers entrants.

Voir également les sous-dossiers - Inclut tous les sous-dossiers dans le processus de surveillance.

Après importation du fichier

Après importation - Après l’importation du fichier, l’utilisateur a deux options. Le fichier peut être supprimé de l’emplacement ou déplacé vers un emplacement différent.

  • Delete imported file - Supprime le fichier source après une importation réussie.

  • Move imported file - Déplace le fichier source vers le dossier spécifié dans Move files to this folder après une importation réussie.

Déplacer les fichiers dans ce dossier - Spécifie le dossier de destination où les fichiers importés avec succès doivent être déplacés lorsque After import est défini sur Move imported file.

Déplacer les feuilles de calcul contenant des erreurs dans ce dossier - Déplace les importations échouées vers le dossier d’erreur spécifié.

Réglages d’importation de feuille de calcul

Importer la feuille de calcul depuis le 'Dossier live' - Importe automatiquement les fichiers de feuille de calcul détectés dans le dossier de surveillance.

Ne pas importer la feuille de calcul contenant des erreurs - Empêche les fichiers contenant des erreurs de validation d’être importés.

Option d’importation de feuille de calcul - Sélectionne le type de données à importer depuis la liste déroulante. Les formats de feuille de calcul pris en charge peuvent inclure CSV, XLSX ou XML. Les options disponibles sont :

  • Importer Tâches - Importe les données de tâche depuis la feuille de calcul.

  • Importer Ordres - Importe les données de commande depuis la feuille de calcul.

  • Importer Sous-groupes - Importe les données d’assemblage depuis la feuille de calcul.

  • Importer Pièces - Importe les données de pièce depuis la feuille de calcul.

Informations supplémentaires

  • Utilisez un chemin UNC (\\ServerName\FolderPath) ou un lecteur mappé (M:\FolderPath) lors de la configuration du dossier de surveillance. N’utilisez pas de chemins locaux tels que C:\FolderPath.

  • L’ordinateur exécutant le service TruSmartCAM Engine doit avoir un accès en lecture et écriture au dossier de surveillance configuré. Sans autorisations suffisantes, les fichiers peuvent ne pas être détectés, importés, déplacés ou supprimés avec succès.

  • Le mappage de feuille de calcul doit être configuré avant d’activer les paramètres de surveillance. Sinon, TruSmartCAM ne pourra pas traiter les fichiers de feuille de calcul détectés dans le dossier de surveillance.
