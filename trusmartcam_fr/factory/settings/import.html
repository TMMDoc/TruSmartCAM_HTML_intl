Importation

Importation

Réglages sortie

Sauvegarder le modèle plat - Activez cette option pour que le motif à plat d’une pièce importée soit enregistré.

Format pour modèle plat - Cette option est utilisée pour choisir entre les formats de fichier DXF ou GEO.

Lieu de destination du modèle plat - Il s’agit de l’emplacement où le motif à plat sera enregistré.

Réglages importation

Unités pour fichiers DXF - Il s’agit de l’unité de mesure avec laquelle un fichier DXF sera importé, l’utilisateur peut choisir entre mm ou pouce.

Touches de recherche des matériaux - Plusieurs entrées de recherche peuvent être affectées en utilisant un séparateur virgule entre elles. Lorsqu’une pièce est importée, TruSmartCAM utilise ces clés pour rechercher les valeurs de matériau. Les valeurs recherchées sont ensuite utilisées pour résoudre la matière première. Les valeurs d’origine sont stockées dans les Propriétés diverses de la pièce.

Touches de recherche de révision - Lorsqu’une pièce est importée, la valeur de révision du fichier de pièce brut (le fichier de pièce en cours d’importation) est comparée à la valeur de révision existante dans le référentiel. L’importation est abandonnée si la révision est la même. Le mappage de champ personnalisé est maintenant également pris en charge pour les fichiers DXF. Pour les fichiers DXF, les données de champ personnalisé sont extraites du calque de texte dans le dessin ou d’une pièce jointe PPI.

Utiliser la comparaison de géométrie sur la même révision CAO - Compare la géométrie lors de l’importation d’une pièce avec la même révision CAD et invite l’utilisateur à remplacer ou à ignorer l’importation si des différences sont détectées, Voir Configuration d’importation pour plus de détails.

Touches de nom de pièce - Cette option fonctionne de manière similaire au matériau et aux Touches de recherche de révision. Définissez un ou plusieurs Touches de nom de pièce pour utiliser le PPI Nom de pièce au lieu du nom de fichier comme nom de pièce.

Touches de recherche d’épaisseur - Plusieurs entrées de recherche peuvent être affectées en utilisant un séparateur virgule entre elles. Lorsqu’une pièce est importée, TruSmartCAM utilise ces clés pour rechercher les valeurs d’épaisseur. Les valeurs recherchées sont ensuite utilisées pour résoudre l’épaisseur. Les valeurs d’origine sont stockées dans les Propriétés diverses de la pièce.

Fudge de comparaison d’épaisseur - Ceci est utilisé pour rechercher la matière première lorsqu’une pièce est importée. TruSmartCAM essaie de résoudre la matière première après que le nom du matériau et l’épaisseur soient analysés. Pour plus d’informations, consultez le sujet Thickness fudge ci-dessous.

Identifier les formes après l’importation d’un méplat - Lorsqu’une pièce est importée, les formes intérieures dans la pièce peuvent être automatiquement reconnues et identifiées.

Fudge d’identification de forme - La valeur de tolérance d’identification de forme peut être définie à :

  1. Niveau global - Cette valeur est utilisée par défaut pour toutes les formes.

  2. Niveau de forme individuelle - La valeur par défaut peut être remplacée en définissant la valeur de tolérance au niveau de la forme individuelle. La sélection de la forme dans la boîte de dialogue Bibliothèque de formes affiche la valeur de tolérance. Par défaut, elle est définie à 0, et la valeur globale par défaut est utilisée pour ces formes.

Tourner horizontalement à plat - Généralement, les tables de machines de découpe sont rectangulaires et plus larges. Pour améliorer la faisabilité d’outillage, faire pivoter la pièce le long de la largeur de la pièce après l’importation de la pièce.

Ne pas tourner une pièce plus courte que celle-ci - Définissez ceci comme le nombre minimum pour permettre la rotation. Toute pièce plus petite que la valeur définie ne sera pas pivotée.

Autres réglages

Activer la pré-importation des outils d’autres fabricants - Activez l’option d’importation des outils externes pré-importation pour activer Creo Reader. TruSmartCAM télécharge les instances de famille pré-extraites de la pièce déposée. Veuillez noter que l’extraction d’instance est effectuée pendant le téléchargement de la pièce (en tant qu’opération de pré-importation), donc le logiciel Creo doit être installé sur l’ordinateur à partir duquel la pièce est téléchargée.

Ignorer les couleurs du dessin dans les vignettes Flat pour plus de clarté - Activez cette option pour générer les miniatures nettes en niveaux de gris. Les couleurs de calque et les entités de texte sont ignorées pour plus de clarté d’image.

Utiliser uniquement un moteur pour toutes les tâches d’importation - Lorsqu’elle est activée, un seul cœur de moteur est utilisé pour les tâches importantes.

Thickness fudge

Le Fudge de comparaison d’épaisseur choisit la valeur d’épaisseur de feuille la plus proche lorsqu’une épaisseur de feuille souhaitée n’est pas disponible dans l’inventaire en comparant la valeur d’épaisseur des feuilles dans l’inventaire.

Par exemple, supposons ce qui suit :

  1. Nous avons une pièce avec une épaisseur de feuille de 1,6 mm et le matériau est l’acier.

  2. La valeur Fudge de comparaison d’épaisseur est de 0,1 mm.

  3. L’inventaire contient des feuilles d’acier avec des épaisseurs de 1,5 mm, 1,55 mm et 1,7 mm.

TruSmartCAM choisit une feuille avec la valeur d’épaisseur la plus proche après avoir comparé la valeur Fudge de comparaison d’épaisseur (0,1 mm) avec l’élément importé (mesurant 1,6 mm). Dans ce cas, 1,55 mm sera la sélection.
