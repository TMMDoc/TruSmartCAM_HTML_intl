Rapports

Réglages du rapport

Lieu de destination du rapport - Utilisez cette option pour spécifier l’emplacement pour enregistrer les rapports.

Format de sortie du rapport - Spécifie le format du rapport lors de l’exportation (PDF ou HTML).

Réglages du rapport de tâche

Inclure les images de configuration dans le rapport - Activez ce paramètre pour inclure des images des mises en page dans un rapport.

Inclure les images de pièces dans le rapport - Activez ce paramètre pour inclure des images des pièces dans un rapport.

Inclure les rapports de configuration dans le rapport de la tâche - Activez ce paramètre pour lier aux rapports de mise en page dans un rapport de tâche.

Réglages du rapport de marque

Activer les rapports de marque - Activez ce paramètre pour permettre la génération d’étiquettes soit depuis la tâche soit depuis une mise en page où un clic droit et la sélection de Export Layouts ou Send to Machine présentera respectivement une option pour afficher le rapport d’étiquette.

Taille de la marque (l x h) - Spécifie la largeur et la hauteur de l’étiquette en millimètres. Ces valeurs définissent les dimensions physiques de l’étiquette générée.

Champ Code QR - Définit le champ de données à encoder dans le code QR sur l’étiquette. La liste déroulante comprend Part Name, Part Order et None. La valeur de champ sélectionnée est affichée sous forme de code QR.

Champ Code-barres - Définit le champ de données à encoder dans le code-barres sur l’étiquette. La liste déroulante comprend Part Name, Part Order et None. La valeur de champ sélectionnée est affichée sous forme de code-barres.

Inclure les images de pièces dans le rapport - Lorsqu’elle est activée, inclut l’image de la pièce dans le rapport d’étiquette généré.
