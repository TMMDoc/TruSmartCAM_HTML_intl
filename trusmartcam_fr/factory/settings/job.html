Réglages sortie

Enregistrer les CN et rapports de configuration - Activez cette option pour que les NC et rapports de mise en page soient enregistrés.

Enregistrer le fichier de configuration - Activez cette option pour que le fichier de mise en page soit enregistré.

Prévisualisation de la simulation de l’outillage dans les rapports - Activez cette option pour générer l’aperçu de simulation d’outillage avec les autres sorties. Il s’agit d’une courte animation GIF pour l’outillage laser/poinçon.

Ne pas supprimer les sorties de programme à la fin de l’agencement/tâche - L’activation de cette option ignorera l’opération de suppression pour les sorties de programme lorsqu’une tâche/mise en page est marquée comme terminée.

Réglages d’imbrication

Pièces mélangées de tâches multiples - Prend en charge le mode d’imbrication mixte. Les pièces de plusieurs tâches doivent être imbriquées ensemble pour une meilleure efficacité d’emballage.

Ne pas relâcher automatiquement les agencements - Cela empêchera les mises en page d’être disponibles pour la production jusqu’à ce que toutes les mises en page soient dans un état libéré.

Séparation de joint pour déblocage automatique - Cette valeur est utilisée pour décider si une mise en page imbriquée est conservée dans la file d’attente d’approbation et est imbriquée à nouveau lorsque de nouvelles pièces sont planifiées ou si elles sont libérées pour la production et que les sorties associées sont exportées vers le dossier de sortie.

Préfixe du nom de tôle interactif - Cette section est utilisée pour prendre en charge l’imbrication interactive où l’utilisateur a plus de contrôle sur les pièces imbriquées et le résultat d’imbrication. Utilisez un schéma de nommage, différent de l’imbrication automatique, pour les mises en page imbriquées de manière interactive.

Mise à jour du stock magasin de tôles concernant le (dé)blocage d’imbrication - Si cette option est activée, l’inventaire de feuilles sera mis à jour lors de l’envoi d’un nest à une machine. Aussi, le rappel du nest de la machine retournera la feuille à l’inventaire.

Automatiser la gestion du stock restant - Lorsque ce commutateur est sélectionné, le logiciel automatise la gestion de l’inventaire des chutes. S’il n’est pas sélectionné, la gestion des chutes devient interactive.

Actualisation automatique des configurations obsolètes - Lorsqu’elle est cochée, le logiciel actualise toutes les ui:EGTag.SelectOutOfDate[Out-of-date Layouts] liées à une pièce après le calcul de ses solutions de découpe.

Automatiser imbrication TwinLine - Cochez cette option pour activer l’automatisation twin-line.

Réglages de Pull workflow

Utiliser 'Pull workflow' - Cette option est utile pour les utilisateurs qui ont besoin d’un système de planification et d’imbrication hautement autonome. Lorsqu’elle est activée, TruSmartCAM planifie l’imbrication sur demande.

Imbrications par poste - Cette valeur sert à définir combien de mises en page par station doivent être créées.

Tôles maximum par imbrication - Cette valeur est utilisée pour contrôler le nombre maximum de feuilles par rapport au nest unique. Ainsi, par exemple, s’il y a une commande avec une grande pièce (qui remplit toute la feuille), nécessaire en grandes quantités (par exemple 500) - TruSmartCAM ne crée plus un seul nest avec 500 copies en mode pull. Au lieu de cela, il crée 10 copies par nest où 10 est le max.

Programmes de pliage par poste - Cette valeur sert à définir combien de programmes de pliage par station doivent être créés.

Marquer la tâche comme étant complète lorsque les sorties du programmes sont retirées - Activez cette option pour qu’une mise en page soit marquée Terminée lorsque les sorties sont supprimées de la destination de sortie cible.

Réglages d’importation de feuille de calcul

Répertoires de recherche CAO - TruSmartCAM essaiera de charger l’entrée de pièce mentionnée dans le CSV importé depuis la bibliothèque de pièces. Cependant, si la pièce n’est pas disponible dans la bibliothèque de pièces, elle essaiera de récupérer les pièces depuis les répertoires de recherche en fonction de l’ordre des répertoires défini dans cette boîte de dialogue.

Utiliser la recherche CAO récursive - Activez cette option pour que la recherche soit répétée pour les résultats successifs.

Créer une pièce vierge pour les fichiers CAO manquants - Si les pièces n’existent pas dans la bibliothèque TruSmartCAM ou les répertoires de recherche, TruSmartCAM créera une pièce vierge pour ces pièces non disponibles.

Ne pas afficher la fenêtre de dialogue Nouvelle tâche si la feuille de calcul ne contient pas d’erreur - Si les entrées CSV sont propres, c’est-à-dire aucune non-concordance de données de la base de données ou tous les champs sont listés correctement, TruSmartCAM n’affichera pas la fenêtre d’aperçu de la tâche et traitera la tâche dès que le CSV est importé ou déposé.

Privilégier la colonne Matière brute plutôt que Matériau + épaisseur - Types de feuilles de calcul pris en charge :

  1. Modèle 1 : PartName, Quantity, DueDate, Priority, RawMaterial

  2. Modèle 2 : PartName, Quantity, DueDate, Priority, Material, Thickness

  3. Modèle 3 : PartName, Quantity, DueDate, Priority, Material, Thickness, RawMaterial

Si cette option n’est pas cochée, le Modèle 1 sera considéré comme le type de feuille de calcul par défaut. Nous prenons également en charge le Modèle 3 au cas où le format CSV serait de type Modèle 3.

Si cette option est cochée et que CSV est de type Modèle 2, alors TruSmartCAM résoudra Raw Material à partir des champs Material et Thickness du CSV.

Réglages pour l’édition de la feuille de calcul

Exporter le rapport d’exécution de la tâche/configuration (.csv)* - Cochez cette option pour activer l’exportation CSV lorsqu’une mise en page ou une tâche est terminée. Les rapports terminés sont enregistrés à la destination Rapports spécifiée dans Factory → Settings → Reports settings.
