Exporter/Importer le matériau

Utilisation du format texte

Pour exporter les matériaux d’inventaire, naviguez vers usine → tôles, cliquez sur Exporter/importer les réglages CAM de matière brute , sélectionnez Export, choisissez .txt et enregistrez-le dans le dossier cible.

Ouvrez le fichier .csv exporté, ajoutez une nouvelle matière première et une feuille avec les détails nécessaires, puis enregistrez le fichier.

Pour importer le fichier mis à jour, naviguez vers usine → tôles → Exporter/importer les réglages CAM de matière brute, sélectionnez Import, choisissez le fichier TXT modifié et les matériaux et feuilles nouvellement ajoutés apparaîtront dans TruSmartCAM.

Utilisation du format CSV

  • Cliquez sur usine → tôles → Exporter/importer les réglages CAM de matière brute → Exporter tôles. Choisissez le fichier CSV et enregistrez-le dans le dossier cible.

  • Ajoutez des feuilles et des paramètres de feuille, enregistrez le fichier .csv.

  • En utilisant Importer tôles, choisissez le fichier .csv modifié.

  • Les matériaux et feuilles nouvellement ajoutés apparaîtront dans TruSmartCAM.

L’importation de matériau doit être un fichier texte spécialement construit qui couvre le format suivant.

Material-Name=[BendMaterial],[Density],[FName];
RawMaterial-Thickness,[Name]=[Film],[Treatment],[Surface];
Sheet-Width,Height,Grain,Desc,HeatNumber=Qty,[Factory],[Shelf],[Cost],[Comments]

Exemple : Aluminum0.5,Aluminum-051500,1500,None,=112,,,22,2500,1250,Horizontal,=105,,,7,2500,1500,None,=100

Pour créer facilement un fichier d’importation, exportez d’abord les matériaux, puis apportez-y des modifications ou visualisez la structure.
