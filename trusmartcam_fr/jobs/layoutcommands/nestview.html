Vue de mise en page

  • La sélection de l’option nests affiche une liste de toutes les imbrications dans le travail actuellement sélectionné.

  • Double-cliquez sur une imbrication pour voir une répartition détaillée de cette imbrication.

  • La répartition de l’imbrication affiche :

    • Les pièces utilisées dans l’imbrication.

    • Des informations détaillées telles que le matériau utilisé, le temps de cycle et d’autres métriques.

  • Vous pouvez faire défiler différentes imbrications et cliquer sur n’importe quelle pièce pour effectuer les mêmes actions disponibles dans la zone Pièces.

  • Un clic droit sur une imbrication fera apparaître le panneau Layout.

    • Envoi vers machine…​ libérera l’imbrication pour la production sur les machines, à ce moment l’option d’imbrication changera en Recall from Machine.

    • Ajout de pièces…​ permettra d’ajouter des pièces supplémentaires à l’imbrication.

    • Adaptation/compactage…​ permettra de modifier par lot les matériaux ou les machines utilisés dans l’imbrication.

    • Modifier LTT…​ permettra d’appliquer une technologie laser différente pour le matériau et l’épaisseur sélectionnés dans l’imbrication.

    • Diviser les configurations…​ permettra de prendre une imbrication qui a plusieurs répétitions dans une nouvelle imbrication.

    • Renommer…​ permettra de renommer l’imbrication.
