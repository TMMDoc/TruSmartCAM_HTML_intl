Améliorations diverses

Plusieurs améliorations ont été apportées à l’extension SolidWorks pour améliorer l’expérience utilisateur globale, les performances et la stabilité.

  • Expérience de connexion unifiée : L’extension réutilise désormais l’interface de connexion TruSmartCAM. La même connexion et validation de mot de passe sont appliquées pour les deux applications. Les identifiants "Se souvenir de moi" fonctionnent de manière transparente entre TruSmartCAM et l’extension SolidWorks, éliminant le besoin de connexions répétées.

  • Exigence de licence API : L’extension nécessite désormais une licence API disponible pour fonctionner.

    • Une erreur de licence sera affichée si une licence API n’est pas disponible.

  • Séparateur nom-configuration validé : Le texte de séparation nom-config est maintenant validé pour garantir que seuls des caractères de chemin valides sont utilisés. Cela évite les caractères non valides dans les noms de fichiers lors de l’enregistrement de pièces multi-configurations.

  • Système de messagerie amélioré : Les messages système, avertissements et erreurs apparaissent maintenant dans la zone d’état en bas à gauche de l’espace de travail SolidWorks, similairement à l’interface TruSmartCAM.
