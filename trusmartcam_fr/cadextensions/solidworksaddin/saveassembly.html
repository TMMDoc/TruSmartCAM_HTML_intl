Enregistrer l'assemblage

La fonction Enregistrer l’assemblage permet aux utilisateurs d’exporter les assemblages SolidWorks et leurs pièces composantes directement dans la bibliothèque de pièces TruSmartCAM. Cela garantit que toutes les pièces de tôlerie, les informations sur la matière première et les quantités de pièces sont transférées avec précision pour la planification en aval, l’outillage et la création de travaux.

Cette fonctionnalité est particulièrement utile pour gérer les assemblages complets tels que les armoires, les boîtiers ou les fabrications structurelles qui impliquent plusieurs composants de tôlerie.

Étapes pour enregistrer un assemblage

  • Lancez SolidWorks et ouvrez le fichier d’assemblage requis.

  • Dans la barre d’outils TruSmartCAM, cliquez sur Save Assembly. La boîte de dialogue Enregistrer le sous-groupe dans Praxis apparaît, résumant les détails de l’assemblage.

  • La boîte de dialogue d’assemblage fournit les informations suivantes :

    • Assembly Name - Affiche le nom de l’assemblage SolidWorks actuellement actif. Ce nom est utilisé comme référence par défaut pour l’assemblage dans TruSmartCAM.

    • Total de pièces - Indique le nombre total d’instances de composants présentes dans l’assemblage SolidWorks. Cela inclut à la fois les pièces de tôlerie et celles qui ne le sont pas.

    • Pièces en tôle - Affiche le nombre total de composants de tôlerie dans l’assemblage, ainsi que le nombre de pièces de tôlerie uniques.

    • Pièces hors tôle - Liste le nombre total et le nombre unique de composants non métalliques détectés dans l’assemblage.

    • Unités à produire - Spécifie le nombre d’assemblages (produits finis) à produire. La quantité saisie est utilisée pour calculer les besoins totaux en pièces dans le travail ou l’assemblage résultant dans TruSmartCAM.

  • Après avoir vérifié les détails, cliquez sur OK.

  • TruSmartCAM enregistre toutes les pièces répertoriées dans le tableau dans la bibliothèque de pièces, ainsi que leurs informations de matériau associées. (Pour les détails de correspondance de matériaux, reportez-vous à la section Résolution automatique de la matière)

Options supplémentaires

Créer un travail

  • Activez cette option pour créer un nouveau travail TruSmartCAM directement à partir de l’assemblage sélectionné, permettant des opérations immédiates de planification de travail et d’imbrication.

  • Lorsqu’elle est activée, la boîte de dialogue Nouveau travail apparaît, vous permettant de spécifier :

    • Le nom de Job Reference (par défaut : nom de l’assemblage).

    • Customer et Comments (facultatif).

    • Due Date, Priority et Quantity pour chaque pièce.

    • Des paramètres supplémentaires tels que des paramètres spécifiques au client ou des valeurs de rotation pour chaque pièce.

    • Cliquez sur OK pour créer et ouvrir le nouveau travail dans TruSmartCAM.

Créer un assemblage

Activez cette option pour enregistrer l’assemblage et ses quantités de pièces correctes dans la page d’assemblage TruSmartCAM. Contrairement à l’option Créer un travail, cela n’ouvre pas la boîte de dialogue Nouveau travail pendant le processus.

  • Seules ces pièces de tôlerie sont traitées et importées dans la bibliothèque de pièces TruSmartCAM.

  • Les pièces non métalliques sont affichées à titre de référence mais ne sont pas importées dans TruSmartCAM, car seules les pièces de tôlerie fabricables sont prises en compte pour la planification de production.
