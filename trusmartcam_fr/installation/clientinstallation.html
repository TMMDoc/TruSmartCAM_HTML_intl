Installation du client

Exécutez l’installation Prérequis comme pour le serveur avant de commencer la configuration du client. Exécutez l’installation du client TruSmartCAM avec l’utilisateur de connexion de l’ordinateur ayant des privilèges administratifs.

  • Utilisez le même installateur qui a été utilisé pour l’installation du serveur TruSmartCAM. Dans la machine cliente, après l’exécution de l’installateur, sélectionnez l’option Client sur la page de sélection des composants. Choisissez les outils TruSmartCAM requis et cliquez sur Suivant.

  • La page Ready to Install résume la liste d’installation. Cliquez sur Installer les machines pour procéder à l’installation dans la machine cliente.

  • Une fois l’installation terminée, cochez la case Launch TruSmartCAM et cliquez sur Finish. Lorsque l’écran TruSmartCAM est lancé, fournissez le nom du serveur ou l’adresse IP du PC serveur TruSmartCAM. Cliquez sur OK .

  • Une fois connecté, TruSmartCAM affiche l’invite d’inscription avec l’autorisation de programmeur. Définissez l’ID e-mail, le mot de passe si nécessaire et cliquez sur OK. La page Bibliothèque TruSmartCAM se charge avec un message de bienvenue UserName dans la zone d’information en bas à gauche.
