Configuration des ports requise

Serveur TruSmartCAM : 7371

Le serveur attend les requêtes des clients sur ce port TCP. Les connexions entrantes doivent être autorisées sur ce port sur l’ordinateur serveur.

Serveur de licence flottante : 5776

Le serveur de licence flottante attend les requêtes de licence sur ce port. Les connexions entrantes doivent être autorisées sur ce port sur l’ordinateur serveur de licence flottante (Serveur TruSmartCAM).

Connexion de licence au Cloud : 7567

Ce port est utilisé par TecZone (logiciel de pliage) et le serveur de licence flottante pour envoyer des requêtes de licence à Metamation Cloud. Ainsi, la connexion sortante doit être autorisée sur ce port pour les activations et renouvellements de licence.
