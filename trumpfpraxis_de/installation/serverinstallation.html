Serverinstallation

Befolgen Sie die Anweisungen, um die TruSmartCAM Serverinstallation zusammen mit SQL zu installieren.

Administratorrechte

Zur Ausführung dieser Installation muss der Benutzer über Administratorrechte verfügen.

  • Führen Sie Setup.TruSmartCAM.<version>.exe aus, um die Installation zu starten.

Version ist eine Nummer.

  • Lesen Sie die Lizenzvereinbarung, wählen Sie I accept the agreement und klicken Sie auf Next.

Schritt 1 : Installationsverzeichnis

  • Geben Sie das Installationsverzeichnis an oder übernehmen Sie den Standardpfad C:\Program Files\Metamation\Praxis und klicken Sie auf Next.

    • C:\Program Files\Metamation\Praxis - Der Installationsordner enthält folgende Ordner: Bin, MetaCAM, Samples und Tracker.

    • C:\Program Data\Metamation - Der Datenordner enthält folgende Ordner: TruSmartCAM, TecZone und MetaCAM.

Schritt 2 : Komponentenauswahl

Installationstyp Komponenten

Full

Server + Desktop Client + Engine

Client

Desktop Client

Server

Server + Desktop Client

Station

Wählt alle Stations-Terminals wie TecZone, MetaCAM, RightAngle(Bend Controller), Vulcan(Laser Machine Controller)

Custom

Auswahl durch den Benutzer.

  • Wählen Sie Full Installation. Wählen Sie Code Senders und Bend Code Senders aus TruSmartCAM Tools. Der Installationstyp ändert sich nun zu Custom. Klicken Sie auf Next.

Schritt 3 : SQL Server Installation auswählen

Die Installation von SQL Express 2022 ist nur für den TruSmartCAM Server erforderlich. Diese Option ist für Client- und Sync-Installationen nicht verfügbar. Diese Option ist beim ersten Mal aktiviert und wird installiert; danach kann sie ignoriert werden. Wenn SQL bereits auf dem TruSmartCAM Server installiert ist, kann der Benutzer Schritt 3 überspringen.

  • Unter Select Additional Tasks: Klicken Sie auf Install SQL Express und klicken Sie auf Next.

  • Unter Ready to Install: Alle Installationsmerkmale werden zusammengefasst. Klicken Sie auf Install.

Schritt 4 : SQL Server Installation

Sobald die TruSmartCAM-Komponenten installiert sind, fährt das Installationsprogramm mit der SQL Server-Installation fort. Akzeptieren Sie die Lizenzvereinbarung und installieren Sie SQL Express mit den Standardoptionen.

Nach Abschluss der Installation klicken Sie auf Close, um die Installation abzuschließen.

Je nach Windows-Version kann das Schließen der SQL-Installation einen Neustart erfordern.

Schritt 5 : Windows Desktop-Verknüpfungen

  • Wenn das System nach der SQL-Installation nicht neu startet, aktivieren Sie das Kontrollkästchen Launch TruSmartCAM und klicken Sie auf Finish.

  • Wenn das System neu startet, starten Sie TruSmartCAM über das Desktop-Verknüpfungssymbol oder geben Sie TruSmartCAM im Windows-Startmenü ein und wählen Sie Als Administrator ausführen.

Serverinstallation: TruSmartCAM und TruSmartCAM License Verknüpfungssymbole sind auf dem Desktop verfügbar. + Client/SYNC-Installation: Nur das TruSmartCAM-Verknüpfungssymbol ist auf dem Desktop verfügbar.

Schritt 6 : Lizenzregistrierung

  • Geben Sie im Feld Serial Number den TruSmartCAM Serial key ein, der vom Trumpf Metamation Service Team bereitgestellt wurde. Geben Sie <Firmenname> im Feld „Registered to" und die E-Mail-Adresse ein. Klicken Sie auf OK.

Falls eine Netzwerk-Firewall die Lizenzregistrierung blockiert oder kein Internet auf dem TruSmartCAM Server-PC verfügbar ist, erfolgt die TruSmartCAM-Lizenzierung über Offline-Aktivierung. Senden Sie die Datei Prax.request aus dem Ordner C:\ProgramData\Metamation\Praxis per E-Mail an das Trumpf Metamation Service Team. Das Trumpf Metamation Team stellt die Datei Prax.lock bereit, die in diesem Ordner abgelegt werden muss.

Schritt 7 : Einmalige Erstkonfiguration

  • Geben Sie den Datenbanknamen ein und legen Sie den Datenspeicherort fest. Aktivieren Sie das Kontrollkästchen Use SQL Server und wählen Sie den SQL Server-Namen aus, um eine Verbindung zur Datenbank herzustellen. Klicken Sie auf OK.

  • Der TruSmartCAM Desktop Client öffnet sich und zeigt unten Database Connection Succeeded an.

  • TruSmartCAM-, TecZone- und MetaCAM-Konfigurationsdateien wurden mit Standard-Maschinenkonfigurationen oder gemäß den TruSmartCAM-Lizenzmaschinen erstellt. Dieselbe Konfiguration wird für alle TruSmartCAM Clients freigegeben.

Der Willkommensbildschirm zeigt den Windows-Benutzeranmeldenamen als TruSmartCAM-Login und -Namen. Der erste TruSmartCAM-Benutzer wird als Site Admin betrachtet. Geben Sie E-Mail und Passwort ein. Klicken Sie auf OK.

Aktivieren Sie das Kontrollkästchen Remember me, um die Anmeldedaten für das nächste Mal zu speichern. Wenn Sie das Kontrollkästchen Remember me deaktivieren, wird bei jedem Start der TruSmartCAM-Anwendung ein Anmeldedialog angezeigt.

Schritt 8 : TruSmartCAM Anmeldung

  • Schließen Sie TruSmartCAM und öffnen Sie es erneut. Das System fordert Sie auf, die TruSmartCAM-Anmeldedaten einzugeben.

  • Der angemeldete Benutzername wird neben dem Benutzersymbol in der TruSmartCAM-Anwendungskopfzeile angezeigt.

  • Wenn Sie den Mauszeiger über das Benutzersymbol bewegen, zeigt das System Folgendes an:

    • Die Benutzerrolle (zum Beispiel: Programmer, Admin).

    • Eine Tastenkombination zum Abmelden: Ctrl+L.

  • Durch Klicken auf das Benutzersymbol öffnet sich ein Menü, das Ihnen Folgendes ermöglicht:

    • Edit information…​ - Aktualisieren Sie Ihre Profildetails.

    • Change password…​ - Ändern Sie Ihr Kontopasswort.

    • Sign out - Melden Sie sich von der Anwendung ab.

    • Cancel - Schließen Sie das Menü ohne Änderungen.

Schritt 9 : Werksmaschinenkonfiguration

  • TruSmartCAM License [Alle Maschinen aktivieren Lizenzbit]: Die Standard-TruSmartCAM-Maschine und das Standardmaterial werden als Werkskonfiguration betrachtet.

  • TruSmartCAM License [Kein Demo und wenige Maschinenbits aktiviert]: Diese wenigen Maschinen und das Standardmaterial werden als Werkskonfiguration betrachtet.

  • (Open → Import) Importieren Sie ein neues Teil aus C:\Program Files\Metamation\Praxis\Samples\Parts und überprüfen Sie, ob das Teil erfolgreich importiert wurde.

Hinweis 1 : TruSmartCAM Monitor

  • TruSmartCAM Monitor mit Engine-Installation (Site Admin-Rechte):

    • TruSmartCAM Monitor mit Engine zusammen mit Site Admin TruSmartCAM-Rechten bietet folgende Optionen:

      1. TruSmartCAM-Konfiguration, TecZone-, MetaCAM-Konfiguration an alle TruSmartCAM Clients verteilen. Außerdem Rechte zur Erstellung neuer TruSmartCAM-Benutzeranmeldungen.

      2. Show Agents zeigt die insgesamt verfügbaren Engines für Automatisierungsaufgaben gemäß der TruSmartCAM-Lizenz an.

  • TruSmartCAM Monitor ohne Engine-Installation (Site Admin-Rechte):

  • TruSmartCAM Monitor ohne Engine-Installation (ohne Site Admin-Rechte):

Der PC, auf dem die TruSmartCAM-Anwendung ausgeführt wird, muss die Engine TruSmartCAM-Komponente installiert haben [Server-PC oder auf einer beliebigen Client-Installation]. Der PC mit installierter Engine TruSmartCAM-Komponente sollte rund um die Uhr laufen, um alle Automatisierungsaufgaben auszuführen.

Hinweis 2 : Lock Tracker

  • Klicken Sie auf die Verknüpfung TruSmartCAM License auf dem TruSmartCAM Server-Desktop. Alternativ öffnen Sie die URL http://localhost:7171/metamation/locktrack in einem beliebigen Webbrowser.

Dies öffnet das Lock Tracker-Modul, das Lizenzinformationen, aktuelle TruSmartCAM-Sitzungsinformationen und Protokolle enthält.

LockTracker HOME-Seite: Zeigt die TruSmartCAM-Seriennummer, Seriennummer-Ablaufinformationen und verfügbare Seriennummer-Bits an.

LockTracker SESSIONS-Seite: Aktuelle TruSmartCAM Server-Verbindungsinformationen wie TruSmartCAM Desktop Client-PC, PMonitor, PEngine-Status.

LockTracker LOGs-Seite: TruSmartCAM Client-Start und -Abmeldung | PMonitor-Start und PEngine-Verbindungs-/Trennungsstatus usw. können auf dieser Protokollseite eingesehen werden.

Hinweis 3 : OpenGL

  • Die TruSmartCAM-Anwendung schließt sich beim Öffnen sofort wieder und PMonitor wird nicht in der Taskleiste angezeigt.

  • Dies deutet auf ein OpenGL-Problem hin.

  • Eine weitere Möglichkeit zur Bestätigung: Öffnen Sie C:\Program Files\Metamation\Praxis\Bin\Flux.exe – es wird ein Fehlerdialog zum OpenGL-Problem angezeigt.

  • Aktualisieren Sie C:\ProgramData\Metamation\Praxis\Prax.ini, Abschnitt Options, mit GLVersion=1, damit die TruSmartCAM-Anwendung erfolgreich geöffnet wird.

[Options]
GLVersion=1.0
