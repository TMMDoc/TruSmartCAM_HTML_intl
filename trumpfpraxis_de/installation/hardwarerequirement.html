Portanforderung

TruSmartCAM-Server: 7371

Der Server wartet auf die Client-Anfragen an diesem TCP-Port. Eingehende Verbindungen sollten für diesen Port auf dem Server-Computer zugelassen werden.

Floating-Lizenzserver: 5776

Der Floating-Lizenzserver wartet auf Lizenzanfragen an diesem Port. Eingehende Verbindungen sollten für diesen Port auf dem Floating-Lizenzserver-Computer (TruSmartCAM-Server) zugelassen werden.

Lizenzverbindung zur Cloud: 7567

Dieser Port wird von Flux (Biege-Software) und dem Floating-Lizenzserver verwendet, um Lizenzanfragen an die Metamation-Cloud zu senden. Die ausgehende Verbindung sollte daher an diesem Port für Lizenzaktivierungen und -verlängerungen zugelassen werden.

Installations- und Supportanforderungen

  • Für die Installation der TruSmartCAM-App sind Administratorrechte erforderlich.

  • Ferndiagnose-Tools über VPN des Auftragnehmers oder eine sichere Verbindung.
