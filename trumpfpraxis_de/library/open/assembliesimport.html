Baugruppen importieren

Baugruppen sind Gruppen verwandter Teile, die zu einer einzelnen Einheit zusammengefasst sind, sodass Sie mehrere Teile gemeinsam als eine Entität verwalten können.

Sie können Baugruppen mit folgenden Methoden importieren:

  1. Import über Datei

  2. Drag and Drop

  3. Import aus Tabelle

Import über Datei

Eine Baugruppe kann mit der Option Import Assemblies…​ in die Teilebibliothek importiert werden.

  • Klicken Sie auf das Öffnen-Symbol in der Befehlsleiste und wählen Sie Import Assemblies…​, oder verwenden Sie File → Open → Import Assemblies…​.

  • Der Öffnen-Dialog erscheint. Wählen Sie die gewünschte(n) Baugruppendatei(en).

  • Eine Vorschau der ausgewählten Baugruppe wird angezeigt.

  • Klicken Sie auf Öffnen, um die Baugruppe zu importieren.

  • Importierte Baugruppen werden in der Baugruppenansicht in der Teilebibliothek aufgelistet.

Drag and Drop

  • Ziehen Sie eine Baugruppe direkt in die TruSmartCAM-Teilebibliothek.

Import aus Tabelle

  • Verwenden Sie File → Open → Import Assemblies…​ und wählen Sie eine CSV-, XLSX- oder XML-Datei, oder ziehen Sie die Datei per Drag and Drop.

  • Überprüfen und aktualisieren Sie Baugruppendetails und Mengen vor dem Import. Verwenden Sie Add…​ oder Remove…​ zum Ändern.

  • Nach dem Import können Baugruppen vorbereitet und statischen Verschachtelungen zugewiesen werden.
