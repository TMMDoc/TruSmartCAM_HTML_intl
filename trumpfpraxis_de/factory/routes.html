Routen

Basierend auf Ihrer Lizenz verfügt TruSmartCAM über einen Standardsatz von CAM-Maschinen und Routingkonfigurationen. Sie können diese Konfigurationen an Ihre spezifischen Fertigungsabläufe anpassen.

Auf der Routenseite werden Maschinen als Bearbeitungsstationen dargestellt, die jeweils durch ein Bild und einen Namen gekennzeichnet sind.

Routen verbinden die Stationen und zeigen die möglichen Arbeitsablaufpfade, denen ein Teil vom Anfang bis zum Ende folgen kann. Sie können Stationseigenschaften anzeigen, indem Sie mit der Maus über eine Station fahren.

Ein Klick auf eine Station öffnet ein Panel mit detaillierten Informationen über diese Station, einschließlich Name, Typ und Verfügbarkeitsstatus.

  • Um die Stationsverfügbarkeit zu bearbeiten, klicken Sie auf die Station und aktivieren/deaktivieren Sie das Kontrollkästchen Enabled im Panel.

Export/Import von Routingkonfigurationen

Die Funktion Export/Import ermöglicht es Benutzern, ihre aktuellen Routingkonfigurationen zu speichern und auf andere Projekte oder Instanzen anzuwenden.

Export CAD routes

Klicken Sie auf Export CAD routes…​, um alle aktuellen Routenbedingungsregeln in eine CSV-Datei zu speichern.

Die CSV listet Materialien und ihre Routingbedingungen auf. Spalten repräsentieren Maschinennamen und Bedingungen (z. B. Material, SpecialTool, HasBend, MaxBendLength). Ein Wert von 1 bedeutet Routing zu einer Maschine; eine leere Zelle bedeutet kein Routing.

Sie können diese CSV-Datei ändern und erneut in das System importieren.

Import CAD routes

Verwenden Sie Import CAD routes…​, um eine geänderte CSV-Datei anzuwenden.

Führen Sie vor dem Import Reset CAD routes aus, um bestehende Konfigurationen zu löschen.

Das System liest die Datei und aktualisiert die Routenbedingungen. Regeln können Bedingungen wie HasBend und MaxBendLength enthalten.

Wenn HasBend = 1, gilt die Regel für Teile mit Biegungen.
Wenn MaxBendLength =1, schränkt die Regel zusätzlich basierend auf der längsten Biegelänge des Teils ein.

Beispiel 1

Teil 00.13 hat eine längste Biegung von 500 mm. Es entspricht der ersten Regel, daher wird TBC-5030-V2 ausgeschlossen.

Beispiel 2

Teil 00.20 hat eine längste Biegung von 600 mm. Es entspricht nicht der ersten Regel und folgt der zweiten Regel, wobei es zu allen Maschinen geroutet wird.

Reset CAD routes

Verwenden Sie Reset CAD routes, um alle bestehenden Konfigurationen vor dem Import einer neuen CSV zu löschen.

Wenn Sie ohne Zurücksetzen importieren, können alte Regeln mit neuen in Konflikt geraten.
