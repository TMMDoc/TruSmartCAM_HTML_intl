Output settings

Save layout NC and reports - Aktivieren Sie diese Option, damit Layout-NC und Berichte gespeichert werden.

Save layout file - Aktivieren Sie diese Option, damit die Layout-Datei gespeichert wird.

Output simulation preview in reports - Aktivieren Sie diese Option, um die Bearbeitungssimulationsvorschau zusammen mit den anderen Ausgaben zu generieren. Dies ist eine kurze GIF-Animation für die Laser-/Stanzbearbeitung.

Do not delete program outputs on layout/job completion - Durch Aktivierung dieser Option wird der Löschvorgang für die Programmausgaben übersprungen, wenn ein Auftrag/Layout als abgeschlossen markiert wird.

Nest settings

Mix parts from multiple Jobs - Unterstützung für gemischtes Verschachteln. Teile aus mehreren Aufträgen werden zusammen verschachtelt, um eine bessere Packungseffizienz zu erzielen.

Do not auto-release layouts - Dies verhindert, dass Layouts für die Produktion verfügbar sind, bis alle Layouts in einem freigegebenen Zustand sind.

Re-nesting packing cut-off - Dieser Wert wird verwendet, um zu entscheiden, ob ein verschachteltes Layout in der Genehmigungswarteschlange gehalten und erneut verschachtelt wird, wenn neue Teile geplant werden, oder ob sie für die Produktion freigegeben und zugehörige Ausgaben in den Ausgabeordner exportiert werden.

Interactive sheet-name prefix - Dieser Abschnitt wird verwendet, um interaktives Verschachteln zu unterstützen, bei dem der Benutzer mehr Kontrolle über die zu verschachtelnden Teile und das Verschachtelungsergebnis hat. Verwenden Sie ein Namensschema, das sich von der automatischen Verschachtelung unterscheidet, für die interaktiv verschachtelten Layouts.

Update sheet inventory on nest (un)release - Wenn diese Option aktiviert ist, wird der Blechbestand aktualisiert, wenn eine Verschachtelung an eine Maschine gesendet wird. Das Zurückrufen der Verschachtelung von der Maschine gibt das Blech an den Bestand zurück.

Automate remnant inventory management - Wenn dieser Schalter ausgewählt ist, automatisiert die Software die Restblechbestandsverwaltung. Wenn nicht ausgewählt, wird die Restblechverwaltung interaktiv.

Auto-refresh Out-of-date layouts - Wenn aktiviert, aktualisiert die Software alle Out-of-date Layouts, die mit einem Teil verknüpft sind, nachdem dessen Schneidlösungen berechnet wurden.

Automate twin-line nesting - Aktivieren Sie diese Option, um die Twin-Line-Automatisierung zu aktivieren.

Pull workflow settings

Use 'Pull workflow' - Diese Option ist nützlich für Benutzer, die ein hochgradig autonomes Planungs- und Verschachtelungssystem benötigen. Wenn aktiviert, plant TruSmartCAM die Verschachtelung bei Bedarf.

Nests per station - Dieser Wert legt fest, wie viele Layouts pro Station erstellt werden sollen.

Maximum sheets per nest - Dieser Wert wird verwendet, um die maximale Blechanzahl pro eindeutiger Verschachtelung zu steuern. Wenn es beispielsweise einen Auftrag mit einem großen Teil (das das gesamte Blech ausfüllt) gibt, das in großer Menge benötigt wird (z. B. 500) – erstellt TruSmartCAM im Pull-Modus nicht mehr eine einzelne Verschachtelung mit 500 Kopien. Stattdessen werden 10 Kopien pro Verschachtelung erstellt, wobei 10 das Maximum ist.

Bend programs per station - Dieser Wert legt fest, wie viele Biegeprogramme pro Station erstellt werden sollen.

Mark the task completed, when the program outputs are removed - Aktivieren Sie diese Option, damit ein Layout als erledigt markiert wird, wenn die Ausgaben vom Zielausgabeverzeichnis entfernt werden.

Spreadsheet import settings

CAD lookup directories - TruSmartCAM versucht, den in der importierten CSV genannten Teileeintrag aus der Teilebibliothek zu laden. Wenn das Teil jedoch nicht in der Teilebibliothek verfügbar ist, wird versucht, die Teile aus Suchverzeichnissen basierend auf der Reihenfolge der in diesem Dialog festgelegten Verzeichnisse abzurufen.

Use recursive CAD search - Aktivieren Sie diese Option, damit die Suche für aufeinanderfolgende Ergebnisse wiederholt wird.

Create a blank part for the missing CAD files - Wenn Teile nicht in der TruSmartCAM-Bibliothek oder den Suchverzeichnissen vorhanden sind, erstellt TruSmartCAM ein leeres Teil für diese nicht verfügbaren Teile.

Do not show New Job dialog if the spreadsheet has no error - Wenn CSV-Einträge sauber sind, d. h. keine Datenabweichungen von der DB oder alle Felder korrekt aufgelistet sind, zeigt TruSmartCAM das Auftragsübersichtsfenster nicht an und verarbeitet den Auftrag, sobald die CSV importiert oder abgelegt wird.

Prefer Raw-Material column over Material + Thickness - Unterstützte Tabellentypen:

  1. Pattern 1: PartName, Quantity, DueDate, Priority, RawMaterial

  2. Pattern 2: PartName, Quantity, DueDate, Priority, Material, Thickness

  3. Pattern 3: PartName, Quantity, DueDate, Priority, Material, Thickness, RawMaterial

Wenn diese Option nicht aktiviert ist, wird Pattern 1 als Standard-Tabellentyp betrachtet. Wir unterstützen auch Pattern 3, falls das CSV-Format vom Typ Pattern 3 ist.

Wenn diese Option aktiviert ist und die CSV vom Typ Pattern 2 ist, löst TruSmartCAM das Rohmaterial aus den Feldern Material und Thickness der CSV auf.

Spreadsheet output settings

Export Job/Layout completion report (.csv) - Aktivieren Sie diese Option, um den CSV-Export zu aktivieren, wenn ein Layout oder Auftrag abgeschlossen ist. Die abgeschlossenen Berichte werden am Berichtsziel gespeichert, das unter Factory → Settings → Reports-Einstellungen angegeben ist.
