The user interface

The user interface

The TruSmartCAM user interface provides access to all major functions through the command bar and menu bar, enabling efficient navigation and operation.

In the image:

1 - Command bar

2 - Menu bar

3 - Subcommands

Command bar

Open - Imports parts, jobs, orders, and assemblies into the library. For details, see Open.

Language - Changes the user interface language. For details, see Language.

Library - Opens the library home page, where parts and assemblies are shown separately. For details, see Library.

Jobs - Displays imported or created jobs. For details, see Jobs.

Factory - Opens the central location for configuring and managing routes, materials, users, and customer data for production setup. For details, see Factory.

Info - Displays information about the software.

Menu bar

File - Provides options to open parts, jobs, orders, and assemblies, and to exit the application.

View - Allows switching between screens and accessing different views.

Help - Provides access to the help manual and software update options.

Admin - Contains administrative settings and configuration options.

Subcommands

Parts - Displays the Part Library items.

Assys - Switches to assembly items in the Library view.

Export - Exports the current part or tooling report as CSV.
