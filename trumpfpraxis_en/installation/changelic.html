Change license

If a license needs to be updated or extended, then this can be performed by following the below steps:

Close the software and services on the server (Pserver and Pmonitor).

Start a DOS command prompt in admin mode.

Browse to C:\Program Files\Metamation\Praxis\Bin by typing in cd\ and press Enter.

Type in PSERVER and press Enter

Type in LIC and press Enter and you will see the below dialog appear.

Clicking on Actions you have the following options:

  • Renew which will attempt to renew the license.

  • Release which will unregister the license and make it available for registration on a new computer.

  • Close which will close the current dialog box.

Clicking on View you can select Details and the information about your current license will be displayed.

In order for a Renewed license change to take effect then you must restart the Pserver applications / services after renewing.
