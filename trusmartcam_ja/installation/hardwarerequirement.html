ポート要件

TruSmartCAM Server : 7371

サーバーはこのTCPポートでクライアントリクエストを待ち受けます。サーバーコンピューターでは、このポートへの着信接続を許可する必要があります。

Floating License Server : 5776

Floating License Serverはこのポートでライセンスリクエストを待ち受けます。フローティングライセンスサーバーコンピューター（TruSmartCAM Server）では、このポートへの着信接続を許可する必要があります。

License connection to Cloud : 7567

このポートは、TecZone（曲げ加工ソフトウェア）とFloating License ServerがMetamation Cloudへライセンスリクエストを送信するために使用されます。したがって、ライセンスのアクティベーションと更新のために、このポートでの発信接続を許可する必要があります。

インストールおよびサポート要件

  • TruSmartCAMアプリケーションをインストールするには、管理者権限が必要です。

  • コントラクターVPNまたはセキュアな接続を介したリモート診断ツール。
