レポート送信者

  • デフォルトでは、ネストが機械に送信されると、すべての出力ファイル（NC、レポート、CSV）がネストにちなんで名前が付けられたフォルダーに保存されます。

  • すべての出力をネストフォルダーに送信する代わりに、レポート送信者を使用して、NC、レポート、CSVファイルを特定のフォルダーにエクスポートできます。

  • これを設定するには、ファクトリー → ルート → Code sender に移動し、ReportSender を選択します。

  • NCファイル、レポートファイル、CSVファイルの宛先フォルダーを設定します。

  • ネストに対して マシンに送信… が実行されると、NC、レポート、CSVファイルは、デフォルトのネストフォルダーではなく、設定されたフォルダーにエクスポートされます。

  • ネストに対して マシンから呼び戻す… が実行されると、対応するNC、レポート、CSVファイルが設定されたレポート送信者の宛先フォルダーから削除されます。

  • これにより、有効なファイルのみが出力フォルダーに残り、エクスポートされたファイルの管理が容易になります。
