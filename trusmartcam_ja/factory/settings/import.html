Import

アウトプット設定

平面パターンを保存する - インポートされたパーツのフラットパターンを保存するには、このオプションを有効にします。

平面パターン形式 - このオプションは、DXFまたはGEOファイル形式のいずれかを選択するために使用されます。

平面パターンの送信先 - これは、フラットパターンが保存される場所です。

インポート設定

DXFファイル用の単位 - これは、DXFファイルがインポートされる測定単位であり、ユーザーはmmまたはinchを選択できます。

材料検索キー - 複数のルックアップエントリは、それらの間にコンマ区切り記号を使用して割り当てることができます。パーツがインポートされると、TruSmartCAMはこれらのキーを使用して材料値を検索します。検索された値は、原材料を解決するために使用されます。元の値は、パーツのMiscプロパティに保存されます。

リビジョン検索キー - パーツがインポートされると、rawパーツファイル（インポートされるパーツファイル）のリビジョン値がリポジトリの既存のリビジョン値と比較されます。リビジョンが同じ場合、インポートは中止されます。カスタムフィールドマッピングは、DXFファイルでもサポートされるようになりました。DXFファイルの場合、カスタムフィールドデータは、図面のテキストレイヤーまたはPPIアタッチメントから抽出されます。

同じCADリビジョンで形状比較を使用する - 同じCADリビジョンを持つパーツをインポートするときにジオメトリを比較し、差異が検出された場合にユーザーに置換またはインポートのスキップを促します。詳細については、 Import configuration を参照してください。

パーツ名キー - このオプションは、材料およびリビジョン検索キーと同様に機能します。ファイル名の代わりにPPI パーツ名をパーツ名として使用するには、1つ以上のパーツ名キーを設定します。

板厚検索キー - 複数のルックアップエントリは、それらの間にコンマ区切り記号を使用して割り当てることができます。パーツがインポートされると、TruSmartCAMはこれらのキーを使用して厚さ値を検索します。検索された値は、厚さを解決するために使用されます。元の値は、パーツのMiscプロパティに保存されます。

板厚比較ファッジ - これは、パーツがインポートされるときに原材料を検索するために使用されます。TruSmartCAMは、材料名と厚さが解析された後、原材料を解決しようとします。詳細については、以下のThickness fudgeトピックを参照してください。

平面のインポート後に形状を識別する - パーツがインポートされると、パーツ内の内部形状を自動的に認識して識別できます。

形状識別ファッジ - 形状識別ファッジ値は以下で設定できます：

  1. グローバルレベル - この値は、すべての形状のデフォルトとして使用されます。

  2. 個別形状レベル - 個別の形状レベルでファッジ値を設定することで、デフォルト値をオーバーライドできます。Shape Libraryダイアログで形状を選択すると、ファッジ値が表示されます。デフォルトでは、これは0に設定されており、このような形状にはデフォルトのグローバル値が使用されます。

平面を水平に回転 - 一般的に、切断機のテーブルは長方形で幅が広いです。ツーリングの実行可能性を高めるために、パーツがインポートされた後、パーツ幅に沿ってパーツを回転させます。

パーツ回転閾値 - これを回転を許可する最小数として設定します。設定された値より小さいパーツは回転されません。

種々の設定

外部ツールのプレインポートを有効にする - Creo Readerを有効にするには、事前インポート外部ツールのインポートオプションを有効にします。TruSmartCAMは、ドロップされたパーツの事前抽出されたファミリーインスタンスをアップロードします。インスタンス抽出はパーツアップロード中に（事前インポート操作として）実行されるため、パーツがアップロードされるコンピューターにCreoソフトウェアがインストールされている必要があることに注意してください。

明確にするために平面サムネイルで描画色を無視する - このオプションを有効にすると、グレースケールのシャープなサムネイルが生成されます。 画像の明瞭性のために、レイヤーの色とテキストエンティティは無視されます。

すべてのインポートタスクで一つのエンジンしか使用しない - 有効にすると、重要なタスクには1つのエンジンコアのみが使用されます。

Thickness fudge

*板厚比較ファッジ*は、在庫内のシートの厚さ値を比較することにより、希望するシート厚さが在庫にない場合に最も近いシート厚さ値を選択します。

たとえば、以下を仮定しましょう：

  1. シート厚さが1.6 mmで材料が鋼のパーツがあります。

  2. *板厚比較ファッジ*値は0.1 mmです。

  3. 在庫には、厚さ1.5 mm、1.55 mm、1.7 mmの鋼シートがあります。

TruSmartCAMは、*板厚比較ファッジ*値（0.1 mm）をインポートされたアイテム（1.6 mm）と比較した後、最も近い厚さ値のシートを選択します。この場合、1.55 mmが選択されます。
