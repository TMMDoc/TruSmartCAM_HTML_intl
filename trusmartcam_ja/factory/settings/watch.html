Watch

Watch Settingsを使用すると、TruSmartCAMは指定されたフォルダーを監視し、ファイルをシステムに自動的にインポートできます。これにより、手動操作なしでParts、Jobs、Assemblies、Ordersのインポートを自動化できます。

設定を見る

「ライブフォルダ」を有効にする - 指定されたフォルダーの自動監視を有効にします。

このフォルダを見る - 受信ファイルを監視するフォルダーを指定します。

サブフォルダも見る - 監視プロセスにすべてのサブフォルダーを含めます。

ファイルインポート後

インポート後 - ファイルがインポートされた後、ユーザーには2つのオプションがあります。ファイルを場所から削除するか、別の場所に移動することができます。

  • Delete imported file - インポートが成功した後、ソースファイルを削除します。

  • Move imported file - インポートが成功した後、ソースファイルを*Move files to this folder*で指定されたフォルダーに移動します。

ファイルをこのフォルダに移動する - *After import*が*Move imported file*に設定されている場合に、正常にインポートされたファイルを移動する宛先フォルダーを指定します。

エラーを含むプレッドシートをこのフォルダに移動する - インポートに失敗したファイルを指定されたエラーフォルダーに移動します。

スプレッドシートのインポート設定

スプレッドシートを「ライブフォルダ」からインポートする - ウォッチフォルダーで検出されたスプレッドシートファイルを自動的にインポートします。

エラーを含むスプレッドシートはインポートしない - 検証エラーを含むファイルがインポートされないようにします。

スプレッドシートのインポートオプション - ドロップダウンからインポートするデータのタイプを選択します。サポートされているスプレッドシート形式には、CSV、XLSX、XMLなどがあります。利用可能なオプションは次のとおりです：

  • ジョブのインポート - スプレッドシートからjobデータをインポートします。

  • オーダーのインポート - スプレッドシートから注文データをインポートします。

  • アセンブリーのインポート - スプレッドシートからアセンブリデータをインポートします。

  • パーツのインポート - スプレッドシートからパーツデータをインポートします。

Additional information

  • ウォッチフォルダーを設定する際は、UNCパス`(\\ServerName\FolderPath)またはマップされたドライブ(M:\FolderPath)`を使用してください。`C:\FolderPath`のようなローカルパスは使用しないでください。

  • TruSmartCAM Engineサービスを実行しているコンピューターには、設定されたウォッチフォルダーへの読み取りおよび書き込みアクセス権が必要です。十分な権限がないと、ファイルが検出、インポート、移動、または削除されない場合があります。

  • Watch Settingsを有効にする前に、スプレッドシートマッピングを設定する必要があります。そうしないと、TruSmartCAMはウォッチフォルダーで検出されたスプレッドシートファイルを処理できません。
