Saving filters

Saving filters

保存されたフィルターを使用すると、パーツ、ジョブ、レイアウトで頻繁に適用される条件を再利用でき、ワークフローを効率化できます。各モジュールにはデフォルトのフィルターが付属していますが、必要に応じてサイト管理者またはプログラマーが追加のフィルターリストを作成できます。保存されたフィルターを再利用することで時間を節約し、検索全体で一貫性を維持できます。

  • アイコンをクリックして、デフォルトのフィルターを表示します。

フィルターを保存する手順

  • アイコンをクリックし、任意のフィルター項目を選択します。

  • 選択したフィルター項目のドロップダウンメニューからオプションを選択します。選択した条件の横にチェックマークが表示されます。

  • 条件を適用するには、アイコンをクリックし、*条件の追加 (Ctrl+プラス)]*を選択します。この手順を繰り返して複数の条件を追加できます。

  • これらの選択したフィルターを条件付きで保存するには、*フィルターの保存… (Ctrl+S)]*をクリックします。

  • 名前を入力し、*保存*をクリックします。

  • *フィルターの保存*ダイアログから、以前に保存したフィルターを削除することもできます。

  • これらのフィルターはデフォルトのフィルターリストに保存されます。ここでは*材料*がデフォルトリストに保存されています。

SiteAdminおよびAdminによって作成されたフィルターは、すべてのユーザー間で共有されます。ProgrammerおよびOperatorによって作成されたフィルターは、ProgrammerおよびOperatorユーザーロール間でのみ共有されます。

Filter commands

コマンド アイコン ショートカット 説明

Clear Selection

Ctrl+X

すべての選択を解除し、フィルター項目ビューを更新します。

Select Highlighted

Ctrl+A

ハイライトされたすべての選択肢を選択します。選択したフィールドに項目のハイライト表示が適用されない場合、このコマンドは無効のままになります。

Match All

&

選択されたすべての条件を満たす項目のみを表示します。

Pin In/Out Menu

Ctrl+Down/Up

ピン留めすると、現在のフォーカスが別のウィンドウに移動してもドロップダウンメニューが開いたままになります。

Close Menu

Esc

ドロップダウンメニューを閉じます。
