Part status

パーツがTruSmartCAMにインポートされると、関連する板金技術が自動的にパーツに適用されます。このプロセスの一環として、TruSmartCAMは検証を実行します。

パーツの状態は、パーツタイル上のパーツ名の下に直接表示され、パーツの現在の検証状態を示します。

パーツ状態タイプは、パーツレベルの検証チェックの結果としてパーツが存在する可能性のあるさまざまな状態を示します。これらの状態は、パーツの設計、材料、または形状に関する問題を示し、ツーリングを進める前に解決する必要があります。

Part errors

A CAD (Computer-Aided Design) error occurs when there is an issue in the parts design or geometry, such as open contours, missing geometry, or unassigned material. These errors originate from the CAD model and prevent the part from being processed further.

  • 曲げ加工が実行不可能 : TruSmartCAM automatically tries to find a valid bending method for the part. If it cannot find any suitable bend solution, the part is marked as bend infeasible.

  • Cut infeasible : If bend warnings are ignored, TruSmartCAM then attempts to find a cutting method. If no valid cutting solution is found, the part is marked as cut infeasible.

Material errors

  • 材料が存在しません : The part’s thickness is read from the CAD model and matched with materials in the TruSmartCAM Raw Material database. If no matching material is found, the part is marked with a 材料が存在しません error.

  • 材料が割り当てられていません : If the CAD model does not have a material mapping field or if the field is empty, TruSmartCAM cannot assign a material. In this case, the part is marked with a 材料が割り当てられていません error.

Geometry errors

  • 開いたコンターが検出されました : The CAD file has one or more open (incomplete) shapes, which prevents proper tooling.

  • 外面形状が複数検出されました : The CAD file has more than one outer closed loop, which creates confusion during processing.

  • Nascent error : When a part is loaded from a spreadsheet (CSV or XLSX), but the actual CAD file is missing, it is marked as a nascent part.

  • 図形データがありません : The CAD model has no valid geometry or the geometry is unreadable, making it unusable for tooling.
