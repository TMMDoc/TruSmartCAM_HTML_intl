Import orders

オーダーは、次の方法を使用して作成またはインポートできます：

  1. スプレッドシートファイルからオーダーをインポートする。

  2. 監視設定を通じてオーダーを自動的にインポートする。

  3. パートライブラリの既存のパーツからオーダーを作成する。

  4. アセンブリ静的ネストからオーダーを作成する。

スプレッドシートからオーダーをインポート

  • 開く → *オーダーのインポート…*に移動します。

  • サポートされているスプレッドシート形式には、CSV、XLSX、XMLが含まれます。

  • 必要なスプレッドシートファイルを選択し、*開く*をクリックします。

  • インポートプロセスは、Job/Assemblyインポートワークフローと同様です。

スプレッドシートをOrdersページに直接ドラッグアンドドロップしてオーダーをインポートすることもできます。

監視設定を使用してオーダーをインポート

  • *設定を見る*で*スプレッドシートを「ライブフォルダ」からインポートする*オプションを有効にします。

  • *スプレッドシートのインポートオプション*で*オーダーのインポート*を選択します。

  • オーダーをインポートするフォルダーの場所を設定します。

  • システムは設定されたフォルダーを継続的に監視します。

  • 新しく追加されたスプレッドシートファイルは、自動的にOrdersにインポートされます。

詳細な設定手順については、Watch settingsドキュメントを参照してください。

パートライブラリからオーダーを作成

  • パートライブラリから必要なパーツを選択します。

  • 右クリックして*新規…* → *オーダー*を選択し、*新規オーダー*ダイアログを開きます。

  • 数量、期限、優先度などの必要な詳細を入力します。

  • *OK*をクリックしてオーダーを作成します。

  • システムは自動的にOrdersページに切り替わります。

  • 作成されたオーダーは、ジョブ作成用のOrders Readyページに表示されます。

アセンブリからオーダーを作成

  • 必要なアセンブリを右クリックします。

  • 新規… → *オーダー*を選択して、*新規オーダー*ダイアログを開きます。

  • 必要な*アセンブリー数*を入力します。

  • *OK*をクリックしてオーダーを作成します。
