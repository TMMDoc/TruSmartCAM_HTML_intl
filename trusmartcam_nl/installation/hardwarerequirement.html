Poortvereisten

TruSmartCAM Server : 7371

Server wacht op clientverzoeken op deze TCP-poort. Inkomende verbindingen moeten worden toegestaan naar deze poort op de servercomputer.

Floating License Server : 5776

Floating License Server wacht op licentieaanvragen op deze poort. Inkomende verbindingen moeten worden toegestaan naar deze poort op de floating license server- computer (TruSmartCAM Server).

Licentieverbinding naar Cloud : 7567

Deze poort wordt gebruikt door TecZone (Bending Software) en Floating License Server om licentieaanvragen naar Metamation Cloud te verzenden. Daarom moet uitgaande verbinding worden toegestaan op deze poort voor licentie-activeringen en -verlengingen.

Installatie- en ondersteuningsvereisten

  • Administratieve rechten vereist om de TruSmartCAM-applicatie te installeren.

  • Externe diagnostische tools via Contractor VPN of een beveiligde verbinding.
