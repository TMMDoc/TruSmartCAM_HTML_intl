Serverinstallatie

Volg de instructies om de TruSmartCAM Server-installatie samen met SQL te installeren.

Administratorrechten

Om deze installatie uit te voeren, moet de gebruiker administratieve rechten hebben.

  • Voer Setup.TruSmartCAM.<version>.exe uit om de installatie te starten.

Version is een nummer.

  • Lees de licentieovereenkomst en kies I accept the agreement en klik vervolgens op Volgende .

Stap 1 : Installatielocatie

  • Geef de installatiemap op of accepteer de standaardinstallatie C:\Program Files\Metamation\Praxis en klik op Volgende.

    • C:\Program Files\Metamation\Praxis - De installatiemap bevat de volgende mappen : Bin, MetaCAM, Samples en Tracker.

    • C:\Program Data\Metamation - De gegevensmap bevat de volgende mappen : TruSmartCAM, TecZone en MetaCAM.

Stap 2 : Componentenselectie

Installatietype Componenten

Full

Server + Desktop client + Engine

Client

Desktop Client

Server

Server + Desktop client

Station

Kiest alle Stations Terminal zoals TecZone, MetaCAM, RightAngle(Bend Controller), Vulcan(Laser Machine Controller)

Custom

Kiest naar gebruikerskeuze.

  • Kies Full Installation. Kies Code Senders en Bend Code Senders uit TruSmartCAM Tools. Nu verandert het installatietype in Aangepast. Klik op Volgende.

Stap 3 : SQL Server installeren kiezen

SQL Express 2022-installatie is alleen nodig voor de TruSmartCAM Server. Deze optie is niet beschikbaar voor Client- en Sync-installaties. Ook is deze standaard aangevinkt en wordt deze voor de eerste keer geïnstalleerd en kan daarna worden genegeerd. Als SQL al is geïnstalleerd op de TruSmartCAM-server, kan de gebruiker Stap 3 negeren.

  • In Select Additional Tasks : Klik op Install SQL Express en klik op Volgende.

  • In Ready to Install : Vat alle installatiekenmerken samen en klik op Install.

Stap 4 : SQL Server-installatie

Zodra TruSmartCAM-componenten zijn geïnstalleerd, gaat het installatieprogramma verder met de SQL Server-installatie. Accepteer de licentieovereenkomst om door te gaan en installeer SQL Express met de standaardopties.

Zodra de installatie is voltooid, klikt u op Close en voltooit u de installatie.

Afhankelijk van de Windows-versie kan het sluiten van de SQL-installatie om een herstart vragen.

Stap 5 : Windows Desktop-snelkoppelingen

  • Na de SQL-installatie, als het systeem niet opnieuw opstart, selecteert u het selectievakje Launch TruSmartCAM en klikt u op Finish.

  • Als het systeem opnieuw opstart, start u de TruSmartCAM-snelkoppeling op het bureaublad of typ TruSmartCAM in Windows Start Programma en Uitvoeren als administrator.

Serverinstallatie : TruSmartCAM en TruSmartCAM License snelkoppelingspictogrammen zijn beschikbaar op het bureaublad. + Client/SYNC-installatie : Alleen TruSmartCAM-snelkoppeling pictogram beschikbaar op het bureaublad.

Stap 6 : Licentieregistratie

  • Typ in het veld Serial Number de TruSmartCAM Serial key die is verstrekt door Trumpf Metamation Service Team. Typ <Company Name> in het veld "Registered to" en E-Mail. Klik op OK.

In het geval dat de netwerkfirewall de licentieregistratie blokkeert of er geen internet is op de TruSmartCAM Server-PC, wordt de TruSmartCAM-licentie uitgevoerd via offline activering. E-mail de Prax.request uit de map C:\ProgramData\Metamation\Praxis naar het Trumpf Metamation Service Team. Het Trumpf Metamation Team zal de Prax.lock verstrekken die in deze map moet worden geplaatst.

Stap 7 : Initiële eenmalige configuratie

  • Typ de databasenaam en stel de gegevensopslaglocatie in. Selecteer het selectievakje SQL-server gebruiken en selecteer de SQL Server-naam om verbinding te maken met de database. Klik op OK.

  • TruSmartCAM Desktop Client opent en toont Database Connection Succeeded linksonder aan de onderkant.

  • TruSmartCAM-, TecZone- en MetaCAM-configuratiebestanden zijn aangemaakt met standaardconfiguratiebestanden machines of tegen TruSmartCAM-licentiemachines. Dezelfde configuratie wordt gedeeld tussen alle TruSmartCAM-clients.

Het welkomstscherm toont Windows-gebruiker LoginName als TruSmartCAM-aanmelding en -naam. De eerste TruSmartCAM-gebruiker wordt beschouwd als Sitebeheerder. Geef e-mail en wachtwoord op. Klik op OK .

Selecteer het selectievakje Onthoud mij om de aanmeldingsgegevens voor de volgende keer op te slaan. Het uitvinken van het selectievakje Onthoud mij zal vragen om het aanmeldings- en wachtwoorddialoogvenster telkens wanneer de TruSmartCAM-applicatie wordt gestart.

Stap 8 : TruSmartCAM-aanmelding

  • Sluit TruSmartCAM en open het opnieuw. Het systeem vraagt u om de TruSmartCAM- aanmeldingsgegevens in te voeren.

  • De aangemelde gebruikersnaam wordt weergegeven naast het gebruikerspictogram in de TruSmartCAM- applicatiekoptekst.

  • Wanneer u over het gebruikerspictogram beweegt, toont het systeem:

    • De gebruikersrol (bijvoorbeeld: Programmer, Admin).

    • Een snelkoppeling om af te melden: Ctrl+L.

  • Door op het gebruikerspictogram te klikken, opent u een menu waarmee u kunt:

    • Informatie bewerken… - Werk uw profielgegevens bij.

    • Wachtwoord wijzigen… - Wijzig uw accountwachtwoord.

    • Uitloggen - Meld u af bij de applicatie.

    • Annuleren - Sluit het menu zonder wijzigingen aan te brengen.

Stap 9 : Configuratie van fabrieksmachines

  • TruSmartCAM License [Alle machines met ingeschakelde licentiebit] : Standaard TruSmartCAM-machine en -materiaal worden beschouwd als fabrieksconfiguratie.

  • TruSmartCAM License [Geen demo en enkele machinebit ingeschakeld] : Die enkele machines en standaardmateriaal worden beschouwd als fabrieksconfiguratie.

  • (Open → Import) een nieuw onderdeel uit C:\Program Files\Metamation\Praxis\Samples\Parts en controleer of het onderdeel succesvol is geïmporteerd.

Opmerking 1 : TruSmartCAM Monitor

  • TruSmartCAM Monitor met Engine-installatie (Site admin-rechten):

    • TruSmartCAM Monitor met Engine samen met Site Admin TruSmartCAM-rechten heeft de volgende opties:

      1. Push TruSmartCAM-configuratie, TecZone-, MetaCAM-configuratie naar alle TruSmartCAM- clients. Heeft ook rechten om een nieuwe TruSmartCAM-gebruikersaanmelding te maken.

      2. Show Agents toont het totale aantal beschikbare engines om automatiseringstaken uit te voeren die overeenkomen met de TruSmartCAM-licentie.

  • TruSmartCAM Monitor zonder Engine-installatie (Site admin-rechten):

  • TruSmartCAM Monitor zonder Engine-installatie (Niet-site admin-rechten):

PC waarop de TruSmartCAM-applicatie draait, moet de Engine TruSmartCAM-component hebben geïnstalleerd [Server-PC of in elke clientinstallatie]. De PC met geïnstalleerde Engine TruSmartCAM-component moet 24/7 draaien om alle automatiseringstaken uit te voeren.

Opmerking 2 : Lock Tracker

  • Klik op de snelkoppeling TruSmartCAM License die beschikbaar is op het TruSmartCAM Server-bureaublad. U kunt ook http://localhost:7171/metamation/locktrack openen in een webbrowser.

Dit opent de Lock Tracker-module met licentie-informatie, TruSmartCAM- huidige sessie-informatie en Log.

LockTracker HOME Page : Dit toont het TruSmartCAM-serienummer, serienummer vervaldag- informatie en beschikbare serienummer bit-informatie.

LockTracker SESSIONS Page : Huidige verbonden TruSmartCAM Server-informatie zoals TruSmartCAM Desktop-client-PC, PMonitor, PEngine-status.

LockTracker LOGs Page : TruSmartCAM-client start en log off | Pmonitor start en PEngine connect/disconnect-status etc., kunnen worden bekeken op deze Logs-pagina.

Opmerking 3 : OpenGL

  • TruSmartCAM-applicatie die tijdens het openen onmiddellijk wordt gesloten en ook pmonitor wordt niet weergegeven in de taakbalk.

  • Dit vertegenwoordigt een OpenGl-probleem.

  • Een andere manier om dit te bevestigen: Open C:\Program Files\Metamation\Praxis\Bin\Flux.exe die een foutdialoog toont over het OpenGL-probleem.

  • Update C:\ProgramData\Metamation\Praxis\Prax.ini, optiesectie met GLVersion=1 zorgt ervoor dat de TruSmartCAM-applicatie succesvol wordt geopend.

[Options]
GLVersion=1.0
