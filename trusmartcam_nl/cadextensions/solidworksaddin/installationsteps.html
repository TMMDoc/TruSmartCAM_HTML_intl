Installatie

Verplichte vereisten

Aanmelden bij TruSmartCAM desktop is verplicht bij het voor de eerste keer starten van SolidWorks.

Zorg ervoor dat de TruSmartCAM monitor actief is in het Windows meldingsgebied.

Als TruSmartCAM desktop niet geopend is, controleer dan of de TruSmartCAM server service actief is om de SolidWorks Add-in te laten functioneren. Als beide inactief zijn, toont SolidWorks een foutmelding bij het opstarten.

Volg deze stappen om de TruSmartCAM Add-in voor SolidWorks te installeren en in te stellen:

  1. Voer de TruSmartCAM Setup uit en schakel op de pagina Componentenselectie SolidWorks Extension in. (Sla de Server optie over als SolidWorks is geïnstalleerd op een TruSmartCAM client.)

  2. Voltooi de installatie, start vervolgens TruSmartCAM desktop en meld u aan met geldige gebruikersreferenties.

  3. Start SolidWorks.

  4. Ga in SolidWorks naar Tools → TruSmartCAM → About TruSmartCAM, om de add-in versie en details te verifiëren.

  5. Om de add-in bij het opstarten in of uit te schakelen, gaat u naar Tools → Add-ins, en zoekt u TruSmartCAM onder Other Add-ins.
