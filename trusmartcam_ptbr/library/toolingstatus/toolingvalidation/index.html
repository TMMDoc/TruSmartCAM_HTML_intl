Validação de ferramental

Esta seção descreve como acessar e modificar as informações de ferramental para peças no TruSmartCAM.

  • Clique com o botão direito em qualquer tile de peça para abrir o painel da peça e acessar as opções de ferramental.

  • Clique em Ferramentais para alternar para a primeira instância de ferramental disponível.

  • Alternativamente, clique diretamente em uma instância de ferramental para visualizar os detalhes da máquina e as opções de ferramental.

  • Use para Abrir em modo somente leitura para visualização.

  • Use para Fazer check-out do documento para edição.

  • A instância de ferramental selecionada é aberta no TecZone ou MetaCAM.

  • Realize verificações manuais de ferramental e faça as alterações necessárias.

  • Quando você tentar salvar, um diálogo Unsaved Changes solicita que confirme suas edições.

  • Clique em Yes para prosseguir com o salvamento.

  • O sistema realiza a Validação de Ferramental novamente após o salvamento.

  • Se não houver erros, a opção Save changes to TruSmartCAM é exibida.

  • Se erros de ferramental forem encontrados, uma mensagem de erro é exibida:

  • Após salvar, o status de ferramental da peça é atualizado na biblioteca de peças.

  • Uma cor verde na instância de ferramental indica que está checked-in.

  • O Autor da Instância de Ferramental é atualizado para a última pessoa que a editou.

  • O campo Editado pelo usuário indica se a instância de ferramental foi modificada manualmente. Quando um usuário faz check-out da instância e realiza quaisquer alterações, este campo é automaticamente definido como Yes ao salvar.

  • Se uma peça está sendo editada atualmente, um ícone de usuário aparece em seu tile.

  • Passar o mouse sobre o tile mostra quem fez o check-out e o horário em que foi realizado.

  • Enquanto a peça está em check-out, a opção Fazer check-out do documento para edição fica desabilitada para a mesma instância de ferramental, e a opção Cancelar check-out fica habilitada.
