Montagem

Criar nests estáticos a partir de uma montagem

Nests estáticos podem ser gerados para uma montagem usando o assistente de nesting Interativo. Estes nests são layouts fixos associados apenas à montagem selecionada.

  • Mude para a visualização assys.

  • Clique com o botão direito na montagem desejada e selecione Create Nests…​.

  • O assistente de Nesting Interativo é aberto.

  • Por padrão, o assistente de nesting interativo muda para a página Select Machine & Material.

  • As máquinas são mostradas com barras coloridas indicando carga de produção disponível:

    • Barra verde: Horas programadas

    • Barra cinza: Horas disponíveis (de um turno de 8 horas)

  • Sob cada máquina, as Peças selecionadas são agrupadas com seu Material Bruto (Peças/Quantidade Total).

  • Se uma peça não tiver uma solução de ferramentas válida para uma máquina, essa peça e material são ignorados para processamento naquela máquina.

  • Se um material bruto for selecionado para uma máquina, ele não estará disponível para outras máquinas.

  • Uma vez que o plano de nest da máquina esteja definido, selecione a chapa preferida e substitua os parâmetros da chapa como Priority, Quantity, Margins, etc.

  • Você pode filtrar chapas para mostrar All Sheets, Standard Sheets ou Remnant Sheets. Clique em next.

Não é possível aumentar as quantidades nesta etapa. Você pode manter a mesma quantidade do inventário de chapas ou reduzi-la e verificar os resultados do nesting.

  • Update Qty aparece na janela do plano de nesting Interativo.

  • Por padrão, Pick All (F5) inclui todas as peças para nesting, ou você pode selecionar manualmente peças prioritárias e ajustar quantidades de peças. Clique em next.

  • O nesting inicia e as peças aninhadas na chapa são atualizadas com status de cor azul.

  • A carga de produção da máquina é atualizada com base no resultado do nest.

  • Se algumas peças permanecerem sem aninhar, a mensagem Some parts remain unnested aparece - essas peças permanecem no trabalho original.

    • Uma peça pode permanecer sem aninhar se nenhuma chapa adequada estiver disponível.

  • Nests Estáticos podem ser editados para incluir peças não aninhadas:

    • Selecione o layout onde você deseja inserir peças não aninhadas e faça o check-out para edição.

  • O TecZone é aberto. Clique em Pick ou Pick All para selecionar peças não aninhadas da montagem pai.

  • Posicione a peça no layout conforme necessário.

  • Nests gerados são automaticamente nomeados após a montagem pai para fácil identificação.

Nests estáticos não podem conter peças de outras montagens. Permitir nests de montagens mistas impediria a liberação de uma montagem sem liberar as outras, portanto a mistura é proibida por design.

Componentes não metálicos

  • Quando uma montagem é importada, peças de chapa metálica são extraídas e adicionadas às montagens na Biblioteca de Peças.

  • Componentes não metálicos são separados e adicionados aos Components.

  • A montagem é adicionada à Biblioteca de Montagens e sua BOM (Bill of Materials) contém tanto peças quanto componentes.

  • Se uma peça já foi importada como peça de chapa metálica, o sistema atualiza a entrada existente e remove duplicatas da Biblioteca de Peças.

  • Selecionar uma montagem exibe tanto peças quanto componentes, com quantidades de componentes mostradas sob a imagem do componente (contagem de componentes da montagem selecionada e contagem total de componentes).

  • Dicas de ferramentas dos componentes exibem o nome, material e tamanho. Componentes são excluídos durante a produção - eles não são aninhados, e nenhuma chapa é gerada para componentes.

  • Montagens e componentes de exemplo estão disponíveis em: C:\Program Files\Metamation\Praxis\Samples\Assys\Components, que podem ser usados para testar importação de componentes e fluxos de trabalho de montagem.

Criando trabalho a partir de montagem

  • Clique com o botão direito na montagem, clique em New…​ e depois Job.

  • No diálogo New Job, defina a contagem de montagem, clique em OK e atualize os detalhes do trabalho (como data de vencimento e prioridade) conforme necessário.

  • O sistema cria um trabalho a partir da montagem e copia nests estáticos como nests de trabalho, atualizando contagens de chapas. Nests copiados são prefixados com o ID do trabalho.

  • Nests de trabalho podem ser editados ou excluídos sem afetar os nests estáticos originais.

  • Nests de trabalho podem ser reempacotados usando Adapt/Compact…​ ou Add parts…​.

  • Para etapas detalhadas, consulte Adapt/Compact e Adicionar peças.

Se uma peça não aparecer em um nest, verifique o diálogo de edição do nest - você pode inserir peças não aninhadas manualmente.

Use pesquisas e filtros salvos para encontrar rapidamente peças em múltiplas montagens.

Use nomes descritivos e claros para montagens e nests para evitar confusão ao copiar nests em trabalhos.
