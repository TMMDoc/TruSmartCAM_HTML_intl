Importar montagens

Montagens são grupos de peças relacionadas combinadas em uma única unidade, permitindo que você gerencie múltiplas peças juntas como uma entidade.

Você pode importar montagens usando os seguintes métodos:

  1. Importar via arquivo

  2. Arrastar e soltar

  3. Importar de planilha

Importar via arquivo

Uma montagem pode ser importada para a Biblioteca de Peças usando a opção Import Assemblies…​.

  • Clique no ícone Open na barra de comandos e selecione Import Assemblies…​, ou use File → Open → Import Assemblies…​.

  • O diálogo Open é exibido. Selecione o(s) arquivo(s) de montagem desejado(s).

  • Uma pré-visualização da montagem selecionada é exibida.

  • Clique em Open para importar a montagem.

  • As montagens importadas são listadas na visualização Assemblies na Biblioteca de Peças.

Arrastar e soltar

  • Arraste e solte uma montagem diretamente na Biblioteca de Peças do TruSmartCAM.

Importar de planilha

  • Use File → Open → Import Assemblies…​ e selecione um arquivo CSV, XLSX ou XML, ou arraste e solte o arquivo.

  • Revise e atualize os detalhes e quantidades da montagem antes da importação. Use Add…​ ou Remove…​ para modificar.

  • Após a importação, as montagens podem ser preparadas e ter Nestings Estáticos atribuídos.
