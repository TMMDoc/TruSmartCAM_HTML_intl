Nesting twin-line

Nesting twin-line

O nesting twin-line permite que peças com bordas comuns sejam cortadas usando uma única linha em vez de duas. Isso economiza material, reduz o tempo de corte e melhora a eficiência geral de produção.

  • Para habilitar este recurso, clique em fábrica → Tarefa → Ajustes de nesting→ Automatizar nesting de linha dupla.

  • Quando esta opção está habilitada, o sistema verifica automaticamente cada peça importada quanto à viabilidade de twin-line.

  • Peças viáveis são automaticamente marcadas como Pode usar TwinLine é Sim durante a importação.

  • No diálogo de job, você pode habilitar Habilitar nesting de linhas duplas se o job requer nesting twin-line.

Esta opção aparece apenas se Automatizar nesting de linha dupla estiver habilitado nas configurações de fábrica.

  • Após o planejamento, quaisquer nestings contendo peças que compartilham bordas comuns são automaticamente processados em twin-line.

  • Um layout é considerado twin-lined quando o campo

Está em TwinLine do layout está marcado como Sim.
