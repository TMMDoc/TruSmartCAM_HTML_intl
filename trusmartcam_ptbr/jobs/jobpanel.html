Painel de tarefa

Esta seção explica ações relacionadas a tarefa como Export outputs, Redefinir tarefa…​, Marcar como concluído…​, Exportar tarefa…​ e outras opções usadas para gerenciar instâncias de tarefa.

Abrir em modo somente leitura para visualização
Abre a tarefa selecionada em modo somente leitura.

Check-out the document for editing
Habilita a edição da tarefa selecionada.

Excluir
Exclui a tarefa selecionada.

Ver relatório
Abre o relatório de produção para a tarefa selecionada.

Exportar saídas
Exporta os arquivos de saída prontos para máquina (arquivos NC e relatórios) gerados para a tarefa selecionada.

Plan job
Gera layouts aninhados para as peças na tarefa selecionada.

Redefinir tarefa…​
Remove todos os layouts gerados para a tarefa e redefine suas peças para o estado planejável.

Marcar como concluído…​
Marca a tarefa e todos os seus nestings como completos após a produção.

Exportar tarefa…​
Exporta a tarefa completa como um pacote que pode ser reutilizado ou importado em outro sistema.

Anterior / Próximo / Layout
Navega para o layout anterior ou próximo dentro da tarefa selecionada.

Classificar e filtrar
Use estes filtros para visualizar outras tarefas que compartilham o mesmo material, cliente, data de entrega ou prioridade.

Exportar saídas

  • Selecione a tarefa desejada.

  • Clique em Exportar saídas.

  • Isso exporta os arquivos de saída prontos para máquina gerados para a tarefa selecionada.

  • Export Outputs pode ser salvo no local padrão configurado em Ajustes de fábrica ou em um local personalizado usando Navegar…​.

  • Exporta apenas arquivos de saída de máquina (como NC e relatórios) para a tarefa.

  • Esses arquivos são usados diretamente para produção e podem ser enviados para a máquina.

Redefinir tarefa…​

Redefine a tarefa selecionada removendo todos os layouts gerados e tornando as peças disponíveis para planejamento novamente.

Para redefinir uma tarefa:

  • Clique com o botão direito na tarefa e selecione Redefinir tarefa…​.

  • Isso exclui todos os layouts aninhados existentes para a tarefa selecionada e redefine todas as suas peças para o estado planejável.

  • Se os layouts contiverem peças de outras tarefas, essas peças também serão redefinidas.

Exportar tarefa…​

  • Selecione a tarefa desejada.

  • Clique em Exportar tarefa…​.

  • Esta opção exporta a tarefa completa como um pacote que pode ser reutilizado ou importado em outro sistema.

  • Os dados exportados incluem todas as informações necessárias para recriar a tarefa:

    • Detalhes de peças e quantidades (arquivos CSV).

    • Dados de layout e nesting (arquivos CSV / FXYLT).

  • Os arquivos exportados são salvos como uma pasta de tarefa no local de exportação configurado.

  • Esta opção é usada para transferir, compartilhar ou fazer backup de toda a tarefa.

Marcar como concluído…​

  • Após a tarefa ser processada na máquina, abra o painel Tarefa e selecione Marcar como concluído…​.

  • Quando uma tarefa é marcada como completa:

    • Todos os nestings sob a tarefa são marcados como completos.

    • A tarefa fica acinzentada, movida para o final da lista e exibida com uma marca de verificação verde.

    • Todos os layouts sob a tarefa são removidos da lista de layouts ativos.

    • Os status de peça e tarefa são atualizados, e as quantidades processadas passam para o estado concluído.

    • Os arquivos de saída para todos os layouts na tarefa são removidos do local de saída da máquina para evitar uso duplicado.
