Painel do trabalho

Esta seção explica ações relacionadas ao trabalho como Export outputs, Redefinir tarefa…​, Marcar como concluído…​, Exportar tarefa…​ e outras opções usadas para gerenciar instâncias de trabalho.

Abrir em modo somente leitura para visualização
Abre o trabalho selecionado em modo somente leitura.

Check-out the document for editing
Habilita a edição do trabalho selecionado.

Excluir
Exclui o trabalho selecionado.

Ver relatório
Abre o relatório de produção do trabalho selecionado.

Exportar saídas
Exporta os arquivos de saída prontos para máquina (arquivos NC e relatórios) gerados para o trabalho selecionado.

Plan job
Gera layouts aninhados para as peças no trabalho selecionado.

Redefinir tarefa…​
Remove todos os layouts gerados para o trabalho e redefine suas peças para o estado planejável.

Marcar como concluído…​
Marca o trabalho e todos os seus nests como concluídos após a produção.

Exportar tarefa…​
Exporta o trabalho completo como um pacote que pode ser reutilizado ou importado em outro sistema.

Anterior / Próximo / Layout
Navega para o layout anterior ou próximo dentro do trabalho selecionado.

Classificar e filtrar
Use estes filtros para visualizar outros trabalhos que compartilham o mesmo material, cliente, data de vencimento ou prioridade.
