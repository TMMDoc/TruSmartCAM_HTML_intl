Painel de layout

Esta seção explica ações relacionadas a layout como Enviar para a máquina…​, Recuperar da máquina…​, Exportar saídas, Marcar como concluído…​, Adicionar peças…​, Adaptar/compactar…​, Dividir layouts…​ e outras opções usadas para gerenciar instâncias de layout.

Abrir em modo somente leitura para visualização
Abre o layout selecionado em modo somente leitura.

Check-out the document for editing
Habilita a edição do layout selecionado.

Excluir
Exclui o layout selecionado.

Ver relatório
Abre o relatório de layout padrão e o relatório específico de etiquetas para o layout selecionado.

Enviar para a máquina…​
Envia o layout selecionado para a máquina e exporta os arquivos de saída (fxlyt, NC, report, csv) para o local de saída do job.

Adicionar peças…​
Adiciona mais peças a um layout com baixo aproveitamento usando o assistente de reempacotamento de layout.

Adaptar/compactar…​
Re-aninha o layout selecionado para compactar as peças e liberar área utilizável.

Alterar TTL…​
Altera a chapa atribuída ao layout selecionado.

Dividir layouts…​
Divide o layout selecionado em dois ou mais layouts separados.

Rename layout…​
Renomeia o layout selecionado.

Tarefa / Anterior / Próximo / Peças
Navega para o job pai, para o layout anterior ou próximo dentro do job selecionado, ou para as peças contidas no layout.

Classificar e filtrar
Use estes filtros para visualizar outros layouts que compartilham o mesmo atributo do layout selecionado: Same Machine, Same Material, Same LTT, Same Priority, Same State, Same Sheet, Same Part, Same Remnant State ou Same Rolling direction.

Enviar para a máquina…​

  • Quando um nesting está pronto (preenchido ou urgentemente necessário), você pode enviá-lo para a máquina.

  • Clique com o botão direito no nesting e selecione Enviar para a máquina…​.

  • Uma pasta é criada automaticamente, nomeada conforme o nesting.

  • Os arquivos de saída (fxlyt, NC, report, csv) são exportados para o local de saída do JOB.

  • Nestings enviados para a máquina são exibidos com uma marca de verificação azul.

Uma vez liberados, esses nestings não são mais tentados para reempacotamento.

Recuperar da máquina…​

  • Se a saída precisa de validação ou uma atualização (por exemplo, adicionar mais peças):

    • Clique com o botão direito no layout exportado.

    • Selecione Recuperar da máquina…​.

  • Os arquivos de saída e a pasta (fxlyt, NC, report, csv) são excluídos.

  • O nesting se torna editável novamente.

  • Você pode fazer alterações e enviá-lo para a máquina novamente.

Exportar saídas

  • Clique com o botão direito em qualquer layout (nesting) que tenha sido enviado para a máquina e clique em Exportar saídas.

  • A opção Export Outputs exporta os arquivos NC, Report e Layout para as respectivas pastas de saída da máquina.

  • Esta opção regenera o arquivo NC com as configurações de unidade atuais para o layout.

Export Outputs não está disponível para Layouts que estão Completos e Layouts que estão aguardando na fila para reempacotamento.

Marcar como completo

  • Após o nesting ser processado na máquina, clique com o botão direito no nesting e selecione Marcar como concluído…​.

  • Você pode selecionar múltiplos nestings e marcá-los como completos ao mesmo tempo.

  • Quando um nesting é marcado como completo:

    • Ele fica acinzentado, movido para o final da lista e exibido com uma marca de verificação verde.

    • O layout é removido da lista de layouts ativos.

    • Os status de peça e job são atualizados, e as quantidades processadas passam para o estado concluído.

    • Os arquivos de saída para esse layout são removidos do local de saída da máquina para evitar uso duplicado.

Adicionar peças…​

  • Use esta opção para adicionar mais peças a um layout com aproveitamento relativamente baixo. Assim como o Nesting Interativo, esta é uma opção guiada por assistente e pode ser usada em um ou múltiplos layouts ao mesmo tempo.

  • O TruSmartCAM monta o assistente de reempacotamento de Layout na área de trabalho principal quando esta opção é utilizada. Mantenha a tecla Ctrl pressionada e selecione uma ou mais peças para serem empacotadas no(s) layout(s) e pressione Próximo.

  • As peças selecionadas são exibidas na próxima etapa com uma coluna de quantidade editável. Atualize as quantidades de peças se necessário no campo Update Qty. Clique em Próximo para prosseguir com o nesting. Observe que as quantidades só podem ser aumentadas, não diminuídas.

  • Todos os layouts são aninhados separadamente e os resultados do nesting são exibidos com as peças recém-empacotadas colocadas neles. Selecionar o layout exibe todas as peças existentes do layout. O resultado é descartado automaticamente se nenhuma peça nova for colocada durante o reempacotamento. Um layout com múltiplas cópias pode resultar em padrões diferentes após o reempacotamento.

Adaptar/compactar…​

  • Este comando baseado em assistente pode ser usado para adaptar um layout de uma máquina para outra quando a máquina original está inativa ou quando a carga de trabalho da máquina precisa ser balanceada.

  • Pode ser usado quando a chapa planejada está fora de estoque e o layout precisa ser replanejado em outra chapa disponível.

  • Você pode mesclar layouts empacotados em chapas menores em chapas maiores para usar o material de forma mais eficiente.

  • Você pode dividir layouts empacotados em chapas maiores em chapas menores se estoques de chapas grandes não estiverem disponíveis.

  • Você pode melhorar a eficiência de empacotamento de múltiplos layouts (mesmo de máquinas diferentes) mesclando-os.

  • Para utilizá-lo, selecione um ou mais layouts e clique em Adaptar/compactar…​ para abrir o assistente na área de trabalho.

  • O painel esquerdo mostra todos os layouts selecionados e o painel Parts mostra as peças do layout.

  • Selecione a máquina de destino e o material da chapa, depois clique em Próximo para realizar o re-nesting.

  • Os novos resultados de nesting são exibidos no painel de resultados.

  • Clique em Próximo para salvar o layout adaptado ou compactado.

Dividindo um layout

A opção Dividir layouts…​ permite dividir um layout existente em múltiplos layouts modificando as quantidades, sem a necessidade de refazer o ferramental ou nesting.

Este recurso é particularmente útil nos seguintes cenários:

  1. Quando você precisa produzir uma parte da quantidade total imediatamente enquanto agenda a quantidade restante para produção posterior.

  2. Quando os requisitos de produção precisam ser distribuídos entre múltiplos turnos ou dias para otimizar o fluxo de trabalho e a utilização de recursos.

  3. Quando uma máquina específica está temporariamente indisponível ou ocupada e você precisa ajustar o layout para acomodar as restrições de produção atuais.

Para dividir um layout, execute as seguintes etapas:

  • Na visualização de Layouts, selecione o layout desejado.

  • Clique em Dividir layouts…​.

  • Insira o valor desejado em Novas cópias.

  • Clique em OK.

O layout selecionado agora está dividido em dois layouts com quantidades ajustadas.
