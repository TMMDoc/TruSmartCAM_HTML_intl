Clientes

A página Customers permite gerenciar todos os registros de clientes dentro do sistema. Você pode adicionar novos clientes, atualizar detalhes existentes e remover clientes que não são mais necessários.

  • Clique em factory → Customers.

  • Para criar um novo cliente, clique em Add New…​ e forneça as informações necessárias. O campo Name é obrigatório e deve ser único, o que significa que dois clientes não podem ter o mesmo nome. Os demais campos são opcionais.

  • Para atualizar os detalhes de um cliente, clique com o botão direito no cliente e selecione Edit information. Todos os campos, exceto o Name, podem ser modificados.

  • Para remover um cliente, clique com o botão direito no cliente e selecione Delete…​.

A exclusão só é possível se o cliente não estiver vinculado a nenhuma Peça ou Tarefa. Caso contrário, a opção de exclusão estará desabilitada.
