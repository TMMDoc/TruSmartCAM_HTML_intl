Relatórios

Ajustes do relatório

Destino do relatório - Use esta opção para especificar o local para salvar os relatórios.

Formato de saída do relatório - Especifica o formato do relatório durante a exportação (PDF ou HTML).

Ajustes do relatório da tarefa

Incluir imagens do layout no relatório - Habilite esta configuração para incluir imagens dos layouts em um relatório.

Incluir imagens da peça no relatório - Habilite esta configuração para incluir imagens das peças em um relatório.

Incluir relatórios de layout no relatório da tarefa - Habilite esta configuração para vincular aos relatórios de layout dentro de um relatório de job.

Ajustes do relatório de etiquetas

Habilitar relatórios de etiquetas - Habilite esta configuração para permitir a geração de etiquetas a partir do job ou de um layout onde clicar com o botão direito e selecionar Export Layouts ou Send to Machine respectivamente apresentará uma opção para mostrar o relatório de etiquetas.

Tamanho da etiqueta (L x A) - Especifica a largura e altura da etiqueta em milímetros. Estes valores definem as dimensões físicas da etiqueta gerada.

Campo do código QR - Define o campo de dados a ser codificado no código QR na etiqueta. A lista suspensa inclui Part Name, Part Order e None. O valor do campo selecionado é exibido como um código QR.

Campo do código de barras - Define o campo de dados a ser codificado no código de barras na etiqueta. A lista suspensa inclui Part Name, Part Order e None. O valor do campo selecionado é exibido como um código de barras.

Incluir imagens da peça no relatório - Quando habilitado, inclui a imagem da peça no relatório de etiqueta gerado.
