Ambiente

Ajustes do ambiente

Unidades padrão do programa - O usuário pode escolher se a unidade padrão é mm ou polegada.

Casas decimais a exibir - Este valor define quantas casas decimais devem ser exibidas.

Precisão decimal na exportação/importação - Este valor define quantas casas decimais devem ser exatas ao importar ou exportar.

Exibir ferramentais das peças em grupos - Use esta opção para selecionar o grupo de ferramentas para exibir o status de ferramentaria na biblioteca de peças.

Subtítulos para ícones da barra de comando - Habilitar esta opção adicionará uma legenda abaixo de cada ícone para referenciar a que página se refere. Quando esta opção está desabilitada, todas as legendas dos botões são removidas e apenas o ícone de cada página diferente é exibido.

Exibir etiquetas de status da tarefa - Habilitar esta opção exibirá o número de peças dentro de um job no final da barra de status do Job. Quando esta opção está desabilitada, a contagem de peças é removida, e a cor para exibir o processo permanecerá.

Exibir peças aninhadas pela cor do pedido - Esta opção exibirá cada peça diferente dentro de um Job como uma cor diferente. Desabilitar esta opção fará com que todas as peças sejam exibidas na mesma cor cinza.

Exibir a duração no formato de intervalo de tempo (HH:MM:SS) - Marque esta opção para exibir a duração no formato HH:MM:SS. Quando habilitada, a duração nos detalhes do layout e nos relatórios de job/layout são exibidos no formato selecionado. Devido às restrições de espaço, a duração é exibida em um formato mais compacto de segundos, nos blocos de layout.
