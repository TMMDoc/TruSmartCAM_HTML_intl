Ajustes de saída

Salvar layout NC - Habilite esta opção para que o NC do layout e os relatórios sejam salvos.

Salvar arquivo de layout e relatórios - Habilite esta opção para que o arquivo de layout seja salvo.

Pré-visualização de simulação de saída em relatórios - Habilite esta opção para gerar a pré-visualização da simulação de ferramentaria junto com as outras saídas. Esta é uma animação GIF curta para a ferramentaria laser/punção.

Não excluir saídas do programa ao concluir layout/tarefa - Habilitar esta opção ignorará a operação de exclusão das saídas do programa quando um job/layout for marcado como concluído.

Ajustes de nesting

Misturar peças de múltiplas tarefas - Suporta nesting de modo misto. Peças de múltiplos jobs devem ser aninhadas juntas para melhor eficiência de empacotamento.

Não liberar layouts automaticamente - Isso impedirá que os layouts fiquem disponíveis para produção até que todos os layouts estejam em estado liberado.

Limite de corte para re-nesting do agrupamento - Este valor é usado para decidir se um layout aninhado é mantido na fila de aprovação e é re-aninhado novamente quando novas peças são planejadas ou são liberadas para produção e as saídas relacionadas são exportadas para a pasta de saída.

Prefixo interativo para o nome da chapa - Esta seção é usada para suportar nesting interativo onde o usuário tem mais controle sobre as peças sendo aninhadas e o resultado do nesting. Use um esquema de nomenclatura, diferente do auto-nesting, para os layouts aninhados interativamente.

Atualizar o inventário de chapas ao liberar ou cancelar um nesting - Se esta opção estiver habilitada, o inventário de chapas será atualizado ao enviar um nesting para uma máquina. Também, recuperar o nesting da máquina retornará a chapa ao inventário.

Automatizar a gestão de remanescentes no inventário - Quando esta opção é selecionada, o software automatiza o gerenciamento de inventário de retalhos. Se não selecionada, o gerenciamento de retalhos se torna interativo.

Atualizar automaticamente layouts desatualizados - Quando marcado, o software atualiza todos os Layouts desatualizados vinculados a uma peça após suas soluções de corte serem calculadas.

Automatizar nesting de linha dupla - Marque esta opção para habilitar a automação twin-line.

Ajustes do fluxograma por demanda

Usar "Fluxograma por demanda" - Esta opção é útil para usuários que precisam de um sistema de planejamento e nesting altamente autônomo. Quando habilitada, o TruSmartCAM planeja o nesting sob demanda.

Nestings por estação - Este valor define quantos layouts por estação devem ser criados.

Máximo de chapas por nesting - Este valor é usado para controlar a contagem máxima de chapas contra o nesting único. Assim, por exemplo, se houver um pedido com uma peça grande (que preenche toda a chapa), necessária em grandes quantidades (digamos 500) - o TruSmartCAM não mais cria um único nesting com 500 cópias no modo pull. Em vez disso, cria 10 cópias por nesting onde 10 é o máximo.

Programas de dobra por estação - Este valor define quantos programas de dobra por estação devem ser criados.

Marque a tarefa como concluída quando as saídas do programa forem removidas - Habilite esta opção para que um layout seja marcado como Concluído quando as saídas são removidas do destino de saída alvo.

Ajustes de importação da planilha

Diretórios de pesquisa CAD - O TruSmartCAM tentará carregar a entrada de peça mencionada no CSV importado da biblioteca de peças. No entanto, se a peça não estiver disponível na biblioteca de peças, então tentará buscar as peças dos diretórios de consulta com base na ordem dos diretórios definidos neste diálogo.

Usar pesquisa recursiva em CAD - Habilite esta opção para que a busca seja repetida para resultados sucessivos.

Criar uma peça em branco para os arquivos CAD em falta - Se as peças não existirem na biblioteca do TruSmartCAM ou nos diretórios de consulta, o TruSmartCAM criará uma peça em branco para aquelas peças indisponíveis.

Não mostrar a caixa de diálogo Nova tarefa se a planilha não tiver erros - Se as entradas CSV estiverem limpas, ou seja, sem incompatibilidade de dados do DB ou todos os campos listados corretamente, o TruSmartCAM não mostrará a janela de visão geral do job e processará o job assim que o CSV for importado ou solto.
