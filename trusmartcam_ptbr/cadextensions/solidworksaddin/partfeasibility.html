Verificar viabilidade

O recurso de viabilidade de peça no Add-in TruSmartCAM para SolidWorks permite que os usuários verifiquem se uma peça de chapa metálica pode ser produzida com sucesso usando as máquinas de fábrica e configurações de ferramentas disponíveis no TruSmartCAM.

Esta funcionalidade replica as mesmas etapas de análise que o motor do TruSmartCAM executa durante o upload real da peça, mas é executada localmente — sem modificar ou substituir peças existentes na biblioteca de peças do TruSmartCAM.

  • O diálogo de viabilidade exibe os resultados da análise para todas as máquinas de fábrica disponíveis configuradas no TruSmartCAM.

  • O status de cada máquina é mostrado como:

    • Feasible - A peça pode ser processada com sucesso.

    • Warning - A peça pode ser processada com limitações (por exemplo, restrições de dobra ou corte).

    • Error - A peça falhou na viabilidade devido a um problema bloqueante.

  • Clique no ícone para abrir o arquivo de tecnologia avaliado (TecZone ou MetaCAM) para inspeção detalhada.

Para montagens, a mesma funcionalidade realiza uma análise de viabilidade abrangente para cada componente individual de chapa metálica.

  • Os resultados são exibidos no diálogo Assembly Feasibility, listando o material, configuração e status de viabilidade de cada peça.

  • Selecionar uma peça exibe suas informações detalhadas no painel direito, incluindo material, tamanho, peso e resultado de viabilidade.

  • A análise de viabilidade para montagens é realizada peça por peça, usando a mesma lógica das verificações de viabilidade de peças individuais.

  • O processo para automaticamente no primeiro erro bloqueante onde não é possível continuar.

  • Apenas peças de chapa metálica com materiais e parâmetros válidos são analisadas.

  • Componentes que não são de chapa metálica são ignorados, mas ainda listados como referência.

  • Como a execução de viabilidade não salva ou faz upload de dados para o TruSmartCAM, pode ser usada com segurança para validação de projeto em estágio inicial e cenários de análise "what-if".

Você deve ter uma licença válida do TecZone ou MetaCAM para abrir e revisar os arquivos de tecnologia gerados vinculados nos resultados de viabilidade.
