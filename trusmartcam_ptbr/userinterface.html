A interface do usuário

A interface do usuário

A interface do usuário do TruSmartCAM fornece acesso a todas as funções principais através da barra de comandos e barra de menus, permitindo navegação e operação eficientes.

Na imagem:

1 - Barra de comandos

2 - Barra de menus

3 - Subcomandos

Barra de comandos

abrir - Importa peças, tarefas, pedidos e montagens para a biblioteca. Para detalhes, consulte Abrir.

idioma - Altera o idioma da interface do usuário. Para detalhes, consulte Idioma.

biblioteca - Abre a página inicial da biblioteca, onde peças e montagens são exibidas separadamente. Para detalhes, consulte Biblioteca.

tarefas - Exibe tarefas importadas ou criadas. Para detalhes, consulte Tarefas.

fábrica - Abre o local central para configurar e gerenciar rotas, materiais, usuários e dados de clientes para configuração de produção. Para detalhes, consulte Factory.

informação - Exibe informações sobre o software.

Barra de menus

_Arquivo - Fornece opções para abrir peças, tarefas, pedidos e montagens, e para sair do aplicativo.

_Vista - Permite alternar entre telas e acessar diferentes visualizações.

_Ajuda - Fornece acesso ao manual de ajuda e opções de atualização do software.

Administrador - Contém configurações administrativas e opções de configuração.

Subcomandos

Parts - Exibe os itens da Biblioteca de Peças.

Assys - Alterna para itens de montagem na visualização da Biblioteca.

Export - Exporta o relatório de peça ou ferramental atual como CSV.
