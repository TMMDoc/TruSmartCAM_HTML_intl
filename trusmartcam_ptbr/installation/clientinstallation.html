Instalação do cliente

Execute a instalação de Pré-requisitos como no Servidor antes de iniciar a instalação do Cliente. Execute a instalação do Cliente TruSmartCAM com o Usuário de Login do Computador tendo privilégios administrativos.

  • Use o mesmo instalador utilizado para a instalação do Servidor TruSmartCAM. Na máquina cliente, após executar o instalador, selecione a opção Client na página de seleção de componentes. Selecione as ferramentas TruSmartCAM necessárias e clique em Próximo.

  • A página Ready to Install resume a lista de instalação. Clique em Instalar máquinas para prosseguir com a Instalação na Máquina Cliente.

  • Uma vez que a instalação estiver concluída, selecione a caixa de seleção Launch TruSmartCAM e clique em Finish. Quando a tela do TruSmartCAM for iniciada, forneça o nome do Servidor ou endereço IP do PC do Servidor TruSmartCAM. Clique em OK .

  • Uma vez conectado, o TruSmartCAM exibe o prompt de cadastro com permissão de programador. Defina o ID de E-mail, senha se necessário e clique em OK. A página Library do TruSmartCAM carrega com a mensagem de boas-vindas UserName na área de informações inferior esquerda.
