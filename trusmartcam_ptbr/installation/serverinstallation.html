Instalação do servidor

Siga as instruções para instalar o Servidor TruSmartCAM juntamente com o SQL.

Direitos de Administrador

Para executar esta instalação, o usuário deve possuir privilégios administrativos.

  • Execute Setup.TruSmartCAM.<version>.exe para iniciar a instalação.

Version é um número.

  • Leia o contrato de licença e escolha I accept the agreement e clique em Próximo.

Etapa 1 : Local de instalação

  • Informe o diretório de instalação ou aceite o padrão de instalação C:\Program Files\Metamation\Praxis e clique em Próximo.

    • C:\Program Files\Metamation\Praxis - A pasta de instalação terá as seguintes pastas: Bin, MetaCAM, Samples e Tracker.

    • C:\Program Data\Metamation - A pasta de dados terá as seguintes pastas: TruSmartCAM, TecZone e MetaCAM.

Etapa 2 : Seleção de componentes

Tipo de Instalação Componentes

Full

Server + Desktop client + Engine

Client

Desktop Client

Server

Server + Desktop client

Station

Seleciona todos os terminais de estação como TecZone, MetaCAM, RightAngle(Bend Controller), Vulcan(Laser Machine Controller)

Custom

Seleção pelo usuário.

  • Escolha Full Installation. Selecione Code Senders e Bend Code Senders em TruSmartCAM Tools. Agora o tipo de instalação muda para Personalizado. Clique em Próximo.

Etapa 3 : Escolha instalar SQL Server

A instalação do SQL Express 2022 é necessária apenas para o Servidor TruSmartCAM. Esta opção não está disponível para instalações Client e Sync. Além disso, esta opção é marcada como ON e instalada na primeira vez e pode ser ignorada posteriormente. Se o SQL já estiver instalado no servidor TruSmartCAM, o usuário pode ignorar a Etapa 3.

  • Em Select Additional Tasks: Clique em Install SQL Express e clique em Próximo.

  • Em Ready to Install: Resume todos os recursos de instalação e clique em Install.

Etapa 4 : Instalação do SQL Server

Após os componentes do TruSmartCAM serem instalados, o instalador prossegue com a instalação do SQL Server. Aceite o Contrato de Licença para prosseguir e instale o SQL Express com as opções padrão.

Após a conclusão da instalação, clique em Close e finalize a instalação.

Dependendo da versão do Windows, fechar a instalação do SQL pode solicitar uma reinicialização.

Etapa 5 : Atalhos da área de trabalho do Windows

  • Após a instalação do SQL, se o sistema não reiniciar, marque a caixa de seleção Launch TruSmartCAM e clique em Finish.

  • Se o sistema reiniciar, inicie o TruSmartCAM pelo ícone de atalho na área de trabalho ou digite TruSmartCAM no Windows Start Program e Execute como administrador.

Instalação do servidor: Os ícones de atalho TruSmartCAM e TruSmartCAM License estão disponíveis na área de trabalho. + Instalação Client/SYNC: Apenas o ícone de atalho TruSmartCAM está disponível na área de trabalho.

Etapa 6 : Registro de licença

  • No campo Serial Number, digite a chave serial do TruSmartCAM fornecida pela equipe de serviço Trumpf Metamation. Digite <Company Name> no campo "Registered to" e E-Mail. Clique em OK.

Caso o firewall de rede bloqueie o registro de licença ou não haja internet no PC Servidor do TruSmartCAM, então a licença do TruSmartCAM é feita por ativação offline. Envie por e-mail o Prax.request da pasta C:\ProgramData\Metamation\Praxis para a equipe de serviço Trumpf Metamation. A equipe Trumpf Metamation fornecerá o Prax.lock que precisa ser colocado nesta pasta.

Etapa 7 : Configuração inicial única

  • Digite o nome do banco de dados e defina o local de armazenamento de dados. Marque a caixa de seleção Usar Servidor SQL e selecione o nome do SQL Server para conectar ao banco de dados. Clique em OK.

  • O Desktop Client do TruSmartCAM abre mostrando Database Connection Succeeded na parte inferior.

  • Os arquivos de configuração do TruSmartCAM, TecZone e MetaCAM foram criados com máquinas de configuração padrão ou contra as máquinas da licença do TruSmartCAM. A mesma configuração será compartilhada entre todos os Clientes TruSmartCAM.

A tela de boas-vindas é exibida com o LoginName do Usuário Windows como Login e Nome do TruSmartCAM. O primeiro usuário do TruSmartCAM é considerado como Administrador do site. Forneça Email e Senha. Clique em OK.

Marque a caixa de seleção Lembrar de mim para salvar as credenciais de login para a próxima vez. Desmarcar a caixa de seleção Lembrar de mim solicitará o diálogo de login e senha sempre que o aplicativo TruSmartCAM for iniciado.

Etapa 8 : Login no TruSmartCAM

  • Feche o TruSmartCAM e reabra-o. O sistema solicita que você insira as credenciais de login do TruSmartCAM.

  • O nome de usuário logado é exibido ao lado do ícone de usuário no cabeçalho do aplicativo TruSmartCAM.

  • Ao passar o mouse sobre o ícone de usuário, o sistema exibe:

    • A função do usuário (por exemplo: Programmer, Admin).

    • Um atalho para sair: Ctrl+L.

  • Clicar no ícone de usuário abre um menu que permite:

    • Editar informações…​ - Atualizar os detalhes do seu perfil.

    • Alterar senha…​ - Alterar a senha da sua conta.

    • Fazer logout - Sair do aplicativo.

    • Cancelar - Fechar o menu sem fazer alterações.

Etapa 9 : Configuração de máquinas da fábrica

  • Licença TruSmartCAM [Todas as máquinas habilitam bit de licença]: A máquina e o material padrão do TruSmartCAM são considerados como configuração de fábrica.

  • Licença TruSmartCAM [Sem demo e poucos bits de máquina habilitados]: Essas poucas máquinas e o material padrão são considerados como configuração de fábrica.

  • (Open → Import) uma nova peça de C:\Program Files\Metamation\Praxis\Samples\Parts e verifique se a peça foi importada com sucesso.

Nota 1 : Monitor do TruSmartCAM

  • Monitor do TruSmartCAM com instalação de Engine (direitos de Site admin):

    • O Monitor do TruSmartCAM com Engine juntamente com direitos de Site Admin do TruSmartCAM terá as seguintes opções:

      1. Enviar configuração do TruSmartCAM, TecZone, MetaCAM para todos os Clientes TruSmartCAM. Também possui direitos para criar novo Login de Usuário do TruSmartCAM.

      2. Show Agents mostrará o total de engines disponíveis para realizar tarefas de automação que está vinculado à licença do TruSmartCAM.

  • Monitor do TruSmartCAM sem instalação de Engine (direitos de Site admin):

  • Monitor do TruSmartCAM sem instalação de Engine (direitos de não Site Admin):

O PC executando o aplicativo TruSmartCAM deve ter o componente Engine do TruSmartCAM instalado [PC Servidor ou em qualquer instalação Client]. O PC com o componente Engine do TruSmartCAM instalado deve estar funcionando 24x7 para realizar todas as tarefas de automação.

Nota 2 : Lock Tracker

  • Clique no atalho TruSmartCAM License disponível na área de trabalho do Servidor TruSmartCAM. Alternativamente, abra a url http://localhost:7171/metamation/locktrack em qualquer navegador web.

Isso abrirá o módulo Lock Tracker que possui informações de licença, informações da sessão atual do TruSmartCAM e Log.

Página HOME do LockTracker: Mostra o número serial do TruSmartCAM, informações de expiração do serial e informações de bits seriais disponíveis.

Página SESSIONS do LockTracker: Informações de conexão atual do Servidor TruSmartCAM como PC do Desktop Client TruSmartCAM, status do PMonitor e PEngine.

Página LOGs do LockTracker: Status de início e logoff do cliente TruSmartCAM | início do Pmonitor e status de conexão/desconexão do PEngine etc., podem ser visualizados nesta página de Logs.

Nota 3 : OpenGL

  • O aplicativo TruSmartCAM ao ser aberto fecha imediatamente e também o pmonitor não é exibido na barra de tarefas.

  • Isso representa um problema de OpenGL.

  • Outra forma de confirmar: Abra C:\Program Files\Metamation\Praxis\Bin\Flux.exe que exibe um diálogo de erro sobre o problema de OpenGL.

  • Atualize C:\ProgramData\Metamation\Praxis\Prax.ini, seção options com GLVersion=1 para que o aplicativo TruSmartCAM abra com sucesso.

[Options]
GLVersion=1.0
