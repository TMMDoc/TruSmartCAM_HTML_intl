Watch

Watch Settings -asetukset mahdollistavat TruSmartCAM:n valvoa määritettyä kansiota ja automaattisesti tuoda tiedostoja järjestelmään. Tämä auttaa automatisoimaan osien, töiden, kokoonpanojen ja tilausten tuonnin ilman manuaalista väliintuloa.

Valvontatoiminnon asetukset

Aktivoi 'Live-kansio' - Ottaa käyttöön automaattisen määritetyn kansion valvonnan.

Valvo tätä kansiota - Määrittää kansion, jota valvotaan saapuvien tiedostojen varalta.

Valvo myös alikansioita - Sisällyttää kaikki alikansiot valvontaprosessiin.

Tiedoston tuonnin jälkeen

Tuonnin jälkeen - Kun tiedosto on tuotu, käyttäjällä on kaksi vaihtoehtoa. Tiedosto voidaan poistaa sijainnista tai siirtää eri sijaintiin.

  • Delete imported file - Poistaa lähdetiedoston onnistuneen tuonnin jälkeen.

  • Move imported file - Siirtää lähdetiedoston kansioon, joka on määritetty Move files to this folder -kohdassa onnistuneen tuonnin jälkeen.

Siirrä tiedostot tähän kansioon - Määrittää kohdekansioon, johon onnistuneesti tuodut tiedostot tulisi siirtää, kun After import on asetettu tilaan Move imported file.

Siirrä virheitä sisältävät laskentataulukot tähän kansioon - Siirtää epäonnistuneet tuonnit määritettyyn virhekansioon.

Laskentataulukoiden tuonnin asetukset

Tuo laskentataulukko 'Live-kansiosta' - Tuo automaattisesti laskentataulukotiedostot, jotka havaitaan watch-kansiossa.

Älä tuo virheitä sisältäviä laskentataulukoita - Estää tiedostoja, jotka sisältävät validointivirheitä, tuomasta.

Laskentataulukoiden tuonnin valinnainen asetus - Valitsee tuotavan datan tyypin pudotusvalikosta. Tuettuja laskentataulukkomuotoja voivat olla CSV, XLSX tai XML. Saatavilla olevat vaihtoehdot ovat:

  • Tuo työt - Tuo työtietoja laskentataulukosta.

  • Tuo työtehtävät - Tuo tilausdataa laskentataulukosta.

  • Tuo rakenneryhmät - Tuo kokoonpanodataa laskentataulukosta.

  • Tuo osat - Tuo osadataa laskentataulukosta.

Lisätiedot

  • Käytä UNC-polkua (\\ServerName\FolderPath) tai kartoitettua asemaa (M:\FolderPath), kun määrität watch-kansiota. Älä käytä paikallisia polkuja kuten C:\FolderPath.

  • Tietokoneella, joka ajaa TruSmartCAM Engine -palvelua, on oltava luku- ja kirjoitusoikeudet määritettyyn watch-kansioon. Ilman riittäviä oikeuksia, tiedostoja ei välttämättä havaita, tuoda, siirtää tai poistaa onnistuneesti.

  • Laskentataulukon kartoitus on määritettävä ennen Watch Settings -asetusten käyttöönottoa. Muussa tapauksessa TruSmartCAM ei pysty käsittelemään watch-kansiossa havaittuja laskentataulukotiedostoja.
