ข้อกำหนดเบื้องต้น

การติดตั้ง TruSmartCAM จะแสดงข้อความเตือนที่กล่าวถึงด้านล่างที่ จุดเริ่มต้นของการติดตั้ง หากการติดตั้ง TruSmartCAM ถูกรันโดยไม่ได้รัน Prerequisites ก่อน การติดตั้ง Pre-Requisite จำเป็นเฉพาะครั้งแรก ใน TruSmartCAM Server/Client และสามารถละเว้นได้หลังจากนั้น

  • รัน PreSetup.TruSmartCAM.<Version>.exe เพื่อติดตั้ง Pre-Requisites ทั้งหมดที่จำเป็นในการรันแอปพลิเคชัน TruSmartCAM

Version คือปีพร้อมคำต่อท้ายตัวอักษร

  • คลิก Install เพื่อเริ่มการติดตั้ง

  • การติดตั้งนี้จะติดตั้ง native redistributable, .Net Frameworks และไดรเวอร์ อื่นๆ ทั้งหมดที่ใช้โดย TruSmartCAM

  • คลิก Finish เพื่อทำการติดตั้งให้เสร็จสมบูรณ์และรีสตาร์ทคอมพิวเตอร์
