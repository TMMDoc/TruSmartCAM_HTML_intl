การติดตั้ง Client

เรียกใช้การติดตั้ง Prerequisites เหมือนกับ Server ก่อน เริ่มการตั้งค่า Client ดำเนินการติดตั้ง TruSmartCAM Client ด้วย ผู้ใช้ล็อกอินคอมพิวเตอร์ที่มีสิทธิ์ผู้ดูแลระบบ

  • ใช้ตัวติดตั้งเดียวกันที่ใช้สำหรับการติดตั้ง TruSmartCAM Server ใน เครื่อง client หลังจากรันตัวติดตั้ง เลือกตัวเลือก Client บนหน้า การเลือกคอมโพเนนต์ เลือกเครื่องมือ TruSmartCAM ที่ต้องการและคลิก ถัดไป

  • หน้า Ready to Install จะสรุปรายการติดตั้ง คลิก ติดตั้งเครื่องจักร เพื่อดำเนินการติดตั้งในเครื่อง Client

  • เมื่อการติดตั้งเสร็จสมบูรณ์ เลือกช่องทำเครื่องหมาย Launch TruSmartCAM และคลิก Finish เมื่อหน้าจอ TruSmartCAM เปิดขึ้น ให้ระบุชื่อ Server หรือที่อยู่ IP ของ TruSmartCAM Server PC คลิก ตกลง

  • เมื่อเชื่อมต่อแล้ว TruSmartCAM จะแสดงข้อความแจ้งให้ลงทะเบียนด้วยสิทธิ์ programmer ตั้งค่า Email ID, รหัสผ่านหากจำเป็น และคลิก ตกลง หน้า Library ของ TruSmartCAM จะโหลดพร้อมข้อความต้อนรับ UserName ที่พื้นที่ข้อมูลด้านล่างซ้าย
