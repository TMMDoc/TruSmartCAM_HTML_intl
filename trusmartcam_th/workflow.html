ขั้นตอนการทำงาน

TruSmartCAM เป็นระบบบริหารการผลิต (Manufacturing Execution System) ที่จัดการและปรับปรุงกระบวนการผลิตแผ่นโลหะ ขั้นตอนการทำงานใน TruSmartCAM ประกอบด้วยชุดขั้นตอนที่คุณปฏิบัติตามเพื่อทำงานหรือกระบวนการเฉพาะให้เสร็จสมบูรณ์ คุณสามารถปรับแต่งขั้นตอนการทำงานให้เหมาะสมกับกระบวนการผลิตต่างๆ และดัดแปลงให้ตรงกับความต้องการเฉพาะของคุณ ขั้นตอนการทำงานใน TruSmartCAM โดยทั่วไปประกอบด้วยขั้นตอนต่อไปนี้:

1. กำหนดค่าระบบ

กำหนดค่า TruSmartCAM ให้ตรงกับสภาพแวดล้อมการผลิตของคุณ ขั้นตอนพื้นฐานนี้รวมถึงการกำหนดเครื่องจักรพร้อมความสามารถและพารามิเตอร์ การเพิ่มวัสดุที่มีอยู่และช่วงความหนา การกำหนดค่าไลบรารีเครื่องมือด้วยหัวเจาะและแม่พิมพ์ การสร้างแผ่นงานตั้งค่าสำหรับการกำหนดค่าเครื่องจักรต่างๆ และการจัดการบัญชีผู้ใช้งานด้วยสิทธิ์ที่เหมาะสม ระบบที่กำหนดค่าอย่างถูกต้องจะช่วยให้มั่นใจในการประมวลผลที่แม่นยำและป้องกันข้อผิดพลาดในขั้นตอนต่อไป การตั้งค่านี้โดยทั่วไปจะดำเนินการครั้งเดียวในระหว่างการติดตั้งเริ่มต้น พร้อมการอัปเดตเป็นระยะเมื่อโรงงานของคุณมีการพัฒนา

สำหรับข้อมูลเพิ่มเติม ดูที่ Add/Remove machine, Material, Sheets, Setup tooling และ Configure users

2. เพิ่มชิ้นงานเข้าสู่ไลบรารี

นำเข้าชิ้นงาน CAD แบบ 2D หรือ 3D เข้าสู่ไลบรารีชิ้นงานโดยใช้รูปแบบที่รองรับเช่น DXF, DWG หรือไฟล์ STEP ในระหว่างการนำเข้า TruSmartCAM จะวิเคราะห์รูปทรงเรขาคณิตของชิ้นงานโดยอัตโนมัติ ตรวจสอบความสามารถในการผลิต กำหนดเทคโนโลยีแผ่นโลหะที่ต้องการ (เลเซอร์ หัวเจาะ หรือการผสมกัน) และสร้างภาพขนาดย่อตัวอย่าง ชิ้นงานที่นำเข้าจะแสดงเป็นไทล์ในมุมมองไลบรารี โดยไอคอนที่แสดงสีจะบ่งบอกสถานะชิ้นงาน ประเภทเทคโนโลยี ความต้องการวัสดุ และปัญหาที่ตรวจพบ ระบบภาพนี้ช่วยให้สามารถประเมินความพร้อมของชิ้นงานได้อย่างรวดเร็วในแวบเดียว

สำหรับข้อมูลเพิ่มเติม ดูที่ Library

3. ตรวจสอบชิ้นงานและการติดตั้งเครื่องมือ

ตรวจสอบชิ้นงานที่นำเข้าแต่ละชิ้นเพื่อหาปัญหาที่อาจเกิดขึ้นในการผลิต ตรวจสอบว่ารูปทรงเรขาคณิตของชิ้นงานถูกต้องและสามารถผลิตได้ การกำหนดเครื่องมือถูกต้องและพร้อมใช้งาน การเลือกวัสดุและความหนาตรงกับสต็อก และลำดับการดัดโลหะได้รับการกำหนดอย่างถูกต้องสำหรับชิ้นงานที่มีการขึ้นรูป หากตรวจพบปัญหา ให้เปิดชิ้นงานใน TecZone เพื่อแก้ไขรูปทรงเรขาคณิต กำหนดใหม่หรือปรับปรุงเครื่องมือ ปรับพารามิเตอร์การประมวลผล หรือแก้ไขข้อมูลจำเพาะของวัสดุ การแก้ไขปัญหาในขั้นตอนนี้จะช่วยป้องกันความล่าช้าในการผลิตและลดของเสียในระหว่างการผลิต

สำหรับข้อมูลเพิ่มเติม ดูที่ Part state และ Tooling status

4. จัดระเบียบชิ้นงานเป็นงาน

จัดกลุ่มชิ้นงานที่ตรวจสอบแล้วเป็นงานตามข้อกำหนดการผลิต งานหมายถึงหน่วยการผลิตเชิงตรรกะที่อาจรวมชิ้นงานจากคำสั่งซื้อของลูกค้าเดียว ชิ้นงานที่ใช้วัสดุและความหนาร่วมกัน ชิ้นงานที่มีกำหนดส่งมอบใกล้เคียงกัน หรือการผสมกันตามกลยุทธ์การผลิตของคุณ งานให้คอนเทนเนอร์สำหรับการวางแผน อนุญาตให้ประมวลผลชิ้นงานที่เกี่ยวข้องเป็นกลุ่ม ช่วยให้การใช้ประโยชน์จากวัสดุมีประสิทธิภาพผ่านการซ้อน และทำให้การติดตามและรายงานง่ายขึ้น สามารถสร้างงานด้วยตนเองโดยการเลือกชิ้นงาน หรือสร้างอัตโนมัติตามกฎและตัวกรองที่กำหนดไว้ล่วงหน้า

สำหรับข้อมูลเพิ่มเติม ดูที่ Jobs

5. วางแผนงานไปยังเครื่องจักรผลิต

เมื่อสร้างงานแล้ว ให้กำหนดไปยังเครื่องจักรผลิตที่เหมาะสม พิจารณาความสามารถของเครื่องจักรรวมถึงเทคโนโลยีที่รองรับ (เลเซอร์ หัวเจาะ หอหมุน) ความจุวัสดุและความหนา เครื่องมือที่มีอยู่ และภาระงานปัจจุบัน หากมีเครื่องจักรที่เหมาะสมหลายเครื่อง ให้เลือกตามลำดับความสำคัญของการผลิต ความพร้อมของเครื่องจักร ข้อกำหนดการตั้งค่า หรือการปรับปรุงต้นทุน การกำหนดสามารถทำด้วยตนเองเพื่อความยืดหยุ่นหรือแบบอัตโนมัติตามกฎที่กำหนดค่าไว้ล่วงหน้าและอัลกอริธึมการกำหนดเวลาเครื่องจักร

สำหรับข้อมูลเพิ่มเติม ดูที่ Plan job

6. ปล่อย NC สำหรับงาน

ในขั้นตอนนี้ จะมีการสร้างเลย์เอาต์การตัด คุณยังสามารถแก้ไขเลย์เอาต์และเพิ่มเทคโนโลยีการตัดในเลย์เอาต์โดยใช้ TecZone หลังจากนั้น ส่งออกเอาต์พุต NC และถ่ายโอนไปยังตัวควบคุมเครื่องจักรหรือตำแหน่งเครือข่ายที่กำหนด งานจะถูกล็อกเพื่อป้องกันการแก้ไขเพิ่มเติม และสถานะจะถูกอัปเดตเป็น "Released" สำหรับการติดตามการผลิต

สำหรับข้อมูลเพิ่มเติม ดูที่ Send/Recall NC

7. ทำการผลิตให้เสร็จสมบูรณ์และสรุปงาน

ดำเนินการผลิตงานที่ปล่อยแล้วบนเครื่องจักรที่กำหนด ในระหว่างการผลิต ผู้ควบคุมเครื่องจักรจะปฏิบัติตามโปรแกรม NC ตรวจสอบคุณภาพชิ้นงานในระหว่างการตัด จัดการการโหลดและขนถ่ายวัสดุ และแก้ไขปัญหาหรือแจ้งเตือนของเครื่องจักรใดๆ หลังจากการผลิตเสร็จสิ้น ให้ทำเครื่องหมายชิ้นงานแต่ละชิ้นหรืองานทั้งหมดว่าเสร็จสมบูรณ์ใน TruSmartCAM ขั้นตอนสุดท้ายนี้จะอัปเดตบันทึกสต็อก บันทึกเวลาการผลิตจริงและการใช้วัสดุ เก็บข้อมูลงานสำหรับการรายงานและวิเคราะห์ และทำวงจรขั้นตอนการทำงานให้เสร็จสมบูรณ์

สำหรับข้อมูลเพิ่มเติม ดูที่ Mark complete
