เฝ้าดู

การตั้งค่าเฝ้าดูช่วยให้ TruSmartCAM ตรวจสอบโฟลเดอร์ที่กำหนดและนำเข้า ไฟล์เข้าสู่ระบบโดยอัตโนมัติ นี่ช่วยทำให้การนำเข้าชิ้นงาน งาน แอสเซมบลี และคำสั่งซื้อเป็นอัตโนมัติโดยไม่ต้องแทรกแซงด้วยตนเอง

การตั้งค่าเฝ้าติดตาม

เปิดใช้งาน 'Live Folder' - เปิดใช้งานการตรวจสอบอัตโนมัติของ โฟลเดอร์ที่ระบุ

เฝ้าติดตามโฟลเดอร์นี้ - ระบุโฟลเดอร์ที่จะถูกตรวจสอบ สำหรับไฟล์ที่เข้ามา

เฝ้าติดตามโฟลเดอร์ย่อยด้วย - รวมโฟลเดอร์ย่อยทั้งหมดใน กระบวนการตรวจสอบ

หลังการนำเข้าไฟล์

หลังการนำเข้า - หลังจากไฟล์ถูกนำเข้า ผู้ใช้มีสอง ตัวเลือก ไฟล์สามารถถูกลบออกจากตำแหน่งหรือย้ายไปยัง ตำแหน่งอื่น

  • Delete imported file - ลบไฟล์ต้นฉบับหลังจาก การนำเข้าสำเร็จ

  • Move imported file - ย้ายไฟล์ต้นฉบับไปยัง โฟลเดอร์ที่ระบุใน Move files to this folder หลังจากการนำเข้าสำเร็จ

ย้ายไฟล์ไปยังโฟลเดอร์นี้ - ระบุโฟลเดอร์ปลายทาง ที่ไฟล์ที่นำเข้าสำเร็จควรถูกย้ายไปเมื่อ After import ถูกตั้งค่าเป็น Move imported file

ย้ายสเปรดชีตที่มีข้อผิดพลาดไปยังโฟลเดอร์นี้ - ย้าย การนำเข้าที่ล้มเหลวไปยังโฟลเดอร์ข้อผิดพลาดที่ระบุ

การตั้งค่านำเข้าสเปรดชีต

นำเข้าสเปรดชีตจาก 'Live Folder' - นำเข้าไฟล์สเปรดชีตที่ตรวจพบในโฟลเดอร์เฝ้าดูโดยอัตโนมัติ

ไม่นำเข้าสเปรดชีตที่มีข้อผิดพลาด - ป้องกัน ไฟล์ที่มีข้อผิดพลาดในการตรวจสอบไม่ให้ถูกนำเข้า

ตัวเลือกการนำเข้าสเปรดชีต - เลือกประเภทของ ข้อมูลที่จะนำเข้าจากดรอปดาวน์ รูปแบบสเปรดชีตที่รองรับอาจรวมถึง CSV, XLSX หรือ XML ตัวเลือกที่มีคือ:

  • นำเข้างาน - นำเข้าข้อมูลงานจากสเปรดชีต

  • นำเข้าคำสั่งงาน - นำเข้าข้อมูลคำสั่งซื้อจาก สเปรดชีต

  • นำเข้าแอสเซมบลี - นำเข้าข้อมูลแอสเซมบลีจาก สเปรดชีต

  • นำเข้าชิ้นส่วน - นำเข้าข้อมูลชิ้นงานจาก สเปรดชีต

ข้อมูลเพิ่มเติม

  • ใช้เส้นทาง UNC (\\ServerName\FolderPath) หรือไดรฟ์ที่แมป (M:\FolderPath) เมื่อกำหนดค่าโฟลเดอร์เฝ้าดู ห้ามใช้เส้นทางท้องถิ่นเช่น C:\FolderPath

  • คอมพิวเตอร์ที่รันบริการ TruSmartCAM Engine ต้องมีสิทธิ์การอ่านและเขียน ไปยังโฟลเดอร์เฝ้าดูที่กำหนดค่าไว้ หากไม่มีสิทธิ์เพียงพอ ไฟล์อาจไม่ถูก ตรวจพบ นำเข้า ย้าย หรือลบได้สำเร็จ

  • การแมปสเปรดชีตต้องถูกกำหนดค่าก่อนการเปิดใช้งานการตั้งค่าเฝ้าดู มิฉะนั้น TruSmartCAM จะไม่สามารถประมวลผลไฟล์สเปรดชีตที่ตรวจพบใน โฟลเดอร์เฝ้าดู
