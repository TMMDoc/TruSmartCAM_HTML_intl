การติดตั้ง

ข้อกำหนดบังคับ

การลงชื่อเข้าใช้ TruSmartCAM desktop เป็นสิ่งจำเป็นเมื่อเปิด SolidWorks ครั้งแรก

ตรวจสอบให้แน่ใจว่ามอนิเตอร์ TruSmartCAM กำลังทำงานอยู่ในพื้นที่การแจ้งเตือนของ Windows

หาก TruSmartCAM desktop ไม่ได้เปิดอยู่ ให้ตรวจสอบว่าบริการเซิร์ฟเวอร์ TruSmartCAM กำลังทำงานอยู่เพื่อให้ SolidWorks Add-in ทำงานได้ หากทั้งสองไม่ทำงาน SolidWorks จะแสดงข้อความแสดงข้อผิดพลาดเมื่อเปิดใช้งาน

ปฏิบัติตามขั้นตอนเหล่านี้เพื่อติดตั้งและตั้งค่า TruSmartCAM Add-in สำหรับ SolidWorks:

  1. เรียกใช้การติดตั้ง TruSmartCAM และในหน้าการเลือกคอมโพเนนต์ ให้เปิดใช้งาน SolidWorks Extension (ข้ามตัวเลือก Server หาก SolidWorks ถูกติดตั้งบนไคลเอนต์ TruSmartCAM)

  2. ติดตั้งให้เสร็จสมบูรณ์ จากนั้นเปิด TruSmartCAM desktop และลงชื่อเข้าใช้ด้วย ข้อมูลรับรองผู้ใช้ที่ถูกต้อง

  3. เปิด SolidWorks

  4. ใน SolidWorks ไปที่ Tools → TruSmartCAM → About TruSmartCAM เพื่อตรวจสอบเวอร์ชัน และรายละเอียดของ add-in

  5. เพื่อเปิดหรือปิดการใช้งาน add-in เมื่อเริ่มระบบ ไปที่ Tools → Add-ins และ ค้นหา TruSmartCAM ภายใต้ Other Add-ins
