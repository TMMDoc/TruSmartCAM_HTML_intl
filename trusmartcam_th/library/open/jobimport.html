Import jobs

นำเข้าจาก csv หรือ xlsx

คุณยังสามารถนำเข้า jobs ในรูปแบบข้อความทั่วไปได้

  • คลิก File → Open → นำเข้างาน กล่องโต้ตอบ Open Folder จะปรากฏขึ้น เลือก C:\Program Files\Metamation\Praxis\Samples\Jobs\C0\C0.csv

  • หากชิ้นงานทั้งหมดมีอยู่ในตำแหน่ง csv หรือในไดเรกทอรีโฟลเดอร์ค้นหา หน้าต่าง งานใหม่ จะปรากฏโดยแสดงรายการชิ้นงานทั้งหมดที่ตรงกับ ฟิลด์ csv

  • ชิ้นงานที่ขาดหายไปจะถูกละเว้นและไม่ปรากฏในหน้าต่าง งานใหม่

  • เริ่มต้นสถานะของชิ้นงานคือ Nascent

  • คลิก OK ตอนนี้ชิ้นงานจะถูกนำเข้าใน Part Library การตรวจสอบ CAD การสร้างทูลลิงสำหรับการดัดและการตัดจะเสร็จสิ้นซึ่งสามารถ ตรวจสอบได้ในหน้าจอ Pmonitor

สร้างชิ้นส่วนเปล่าสำหรับไฟล์ CAD ที่ขาดหาย

เมื่อนำเข้า jobs โดยใช้สเปรดชีต (.csv หรือ .xlsx) คุณอาจมี แถวที่อ้างอิงถึงไฟล์ CAD ที่ไม่มีในระบบบางครั้ง

  • เพื่อจัดการกับสิ่งนี้ ให้ไปที่ โรงงาน → การตั้งค่า → งาน ภายใต้ การนำเข้าสเปรดชีต เปิดใช้งาน สร้างชิ้นส่วนเปล่าสำหรับไฟล์ CAD ที่ขาดหาย

เมื่อเลือกตัวเลือกนี้:

  • หากไฟล์ CAD หายไปสำหรับแถวใดๆ ระหว่างการนำเข้า job ระบบจะ สร้างชิ้นงานเปล่าโดยอัตโนมัติแทนที่ไฟล์ CAD ที่ขาดหายไป

  • ในหน้าต่าง งานใหม่ ชิ้นงานที่ขาดหายไปเหล่านี้จะถูก ไฮไลต์ด้วยสีเหลืองเพื่อระบุว่าไฟล์ CAD ที่เกี่ยวข้องหายไป

  • ชิ้นงานที่ขาดหายไปจะยังคงมีฟิลด์ที่แมป (เช่น ชื่อชิ้นงาน วัสดุ ความหนา ปริมาณ ฯลฯ) จาก CSV อย่างไรก็ตาม สถานะชิ้นงานจะแสดง เป็น Nascent หมายความว่าเป็นเพียงตัวยึดตำแหน่งเปล่าที่ไม่มีรูปทรงเรขาคณิตจริง ยัง

สิ่งนี้ทำให้มั่นใจได้ว่าการนำเข้า job ยังสามารถดำเนินต่อไปได้โดยไม่มีข้อผิดพลาดเนื่องจาก ไฟล์ CAD ที่ขาดหายไป — คุณสามารถอัปเดตหรือแทนที่ชิ้นงานเปล่าด้วย ไฟล์ CAD ที่ถูกต้องเมื่อมีให้ใช้งาน

ไม่ต้องแสดงไดอะล็อกงานใหม่หากสเปรดชีตไม่มีข้อผิดพลาด

โรงงาน → การตั้งค่า → งาน → การนำเข้าสเปรดชีต→ ไม่ต้องแสดงไดอะล็อกงานใหม่หากสเปรดชีตไม่มีข้อผิดพลาด

  • เมื่อเลือกช่องทำเครื่องหมายนี้ ระบบจะสร้าง job ใหม่โดยอัตโนมัติโดยไม่ แสดงกล่องโต้ตอบ New Job หากสเปรดชีตไม่มีข้อผิดพลาด (ไม่มีชิ้นงานที่ขาดหายไป วันที่ที่ขาดหายไป หรือข้อมูลที่ไม่ถูกต้อง)

  • เมื่อล้างช่องทำเครื่องหมายนี้ กล่องโต้ตอบ New Job จะปรากฏเสมอเพื่อให้ผู้ใช้ตรวจสอบ และยืนยัน แม้ว่าสเปรดชีตจะไม่มีข้อผิดพลาด

Prefer raw-material column over material + thickness

โรงงาน → การตั้งค่า → งาน→Prefer Raw-Material column over Material + Thickness

  • เมื่อเลือกช่องทำเครื่องหมายนี้ ระบบจะคาดหวังว่าสเปรดชีตจะใช้ คอลัมน์ Raw-Material (ตัวอย่างเช่น Steel-20) ซึ่งหมายความว่าประเภทวัสดุและ ความหนาควรรวมกันเป็นค่าเดียวในคอลัมน์เดียว

  • เมื่อล้างช่องทำเครื่องหมายนี้ ระบบจะคาดหวังว่าสเปรดชีตจะมี คอลัมน์แยกสำหรับ Material และ Thickness (ตัวอย่างเช่น Steel และ 2.00) ระบบจะรวมมันภายในระหว่างการนำเข้า

โฟลเดอร์ watch โดยใช้ CSV/XLSX

  • ใน โรงงาน → การตั้งค่า→ เฝ้าติดตาม เลือก เปิดใช้งาน 'Live Folder' และ เฝ้าติดตามโฟลเดอร์ย่อยด้วย

  • ตั้งค่าเส้นทางโฟลเดอร์ watch (UNC หรือไดรฟ์ที่แมป)

  • ใน การนำเข้าสเปรดชีต เลือก นำเข้าสเปรดชีตจาก 'Live Folder'

  • เลือก นำเข้างาน จากตัวเลือก Spreadsheet Import

  • คัดลอก C:\Program Files\Metamation\Praxis\Samples\Jobs\C0\C0.csv และวางลงใน โฟลเดอร์ Watch

  • ระบบจะตรวจสอบ CSV และพยายามสร้าง job โดยอัตโนมัติ

  • หากไฟล์ CAD หายไป จะไม่มี job ถูกสร้างขึ้นเพราะไม่พบข้อมูล CAD ที่ต้องการ

CAD lookup directory และ recursive CAD search - ไม่ได้กำหนดค่า

  • ใน โรงงาน → การตั้งค่า → งาน → การนำเข้าสเปรดชีต หากไม่มี CAD lookup directories ที่ตั้งค่าและ ใช้การค้นหา CAD แบบวนซ้ำ ไม่ได้ใช้ ระบบ จะไม่ค้นหาไฟล์ CAD โดยอัตโนมัติ

  • หากเลือกช่องทำเครื่องหมาย สร้างชิ้นส่วนเปล่าสำหรับไฟล์ CAD ที่ขาดหาย job จะยังคงถูกสร้างขึ้นแม้ว่าไฟล์ CAD จะหายไป

  • ในกรณีนี้ ชิ้นงานทั้งหมดใน job จะถูกสร้างเป็นชิ้นงาน nascent (เปล่า) — หมายความว่าไม่มีรูปทรงเรขาคณิตเพราะไม่พบไฟล์ CAD

CAD lookup directory และ recursive CAD search - กำหนดค่าแล้ว

เมื่อ ไดเรกทอรีค้นหา CAD ถูกตั้งค่าเป็น (C:\Program Files\Metamation\Praxis\Samples) และ ใช้การค้นหา CAD แบบวนซ้ำ เปิดใช้งานภายใต้ โรงงาน → การตั้งค่า → งาน → การนำเข้าสเปรดชีต:

  • ระบบจะค้นหาโฟลเดอร์ที่ระบุและโฟลเดอร์ย่อยทั้งหมดโดยอัตโนมัติ สำหรับไฟล์ CAD ที่ต้องการที่กล่าวถึงในสเปรดชีต

  • สร้างชิ้นส่วนเปล่าสำหรับไฟล์ CAD ที่ขาดหาย ยัง เปิดใช้งาน ดังนั้นหากยังไม่พบไฟล์ CAD ใดๆ ระบบจะสร้า ชิ้นงานเปล่าเป็นตัวยึดตำแหน่ง
