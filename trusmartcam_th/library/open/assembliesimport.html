Import assemblies

Assemblies คือกลุ่มของชิ้นงานที่เกี่ยวข้องซึ่งรวมกันเป็นหน่วยเดียว ช่วยให้คุณสามารถจัดการชิ้นงานหลายชิ้นร่วมกันเป็นหนึ่งเดียว

คุณสามารถนำเข้า assemblies โดยใช้วิธีต่อไปนี้:

  1. นำเข้าผ่านไฟล์

  2. ลากและวาง

  3. นำเข้าจากสเปรดชีต

นำเข้าผ่านไฟล์

Assembly สามารถนำเข้าสู่ Part Library โดยใช้ ตัวเลือก นำเข้าแอสเซมบลี

  • คลิกไอคอน Open ในแถบคำสั่งและเลือก นำเข้าแอสเซมบลี หรือใช้ _ไฟล์ → เปิด → นำเข้าแอสเซมบลี

  • กล่องโต้ตอบ Open จะปรากฏขึ้น เลือกไฟล์ assembly ที่ต้องการ

  • ตัวอย่างของ assembly ที่เลือกจะแสดงขึ้น

  • คลิก Open เพื่อนำเข้า assembly

  • Assemblies ที่นำเข้าจะแสดงอยู่ภายใต้มุมมอง Assemblies ใน Part Library

ลากและวาง

  • ลากและวาง assembly โดยตรงเข้าสู่ Part Library ของ TruSmartCAM

นำเข้าจากสเปรดชีต

  • ใช้ _ไฟล์ → เปิด → นำเข้าแอสเซมบลี และเลือกไฟล์ CSV, XLSX หรือ XML หรือลากและวางไฟล์

  • ตรวจสอบและอัปเดตรายละเอียดและปริมาณ assembly ก่อนนำเข้า ใช้ _เพิ่ม…​ หรือ _ลบ…​ เพื่อแก้ไข

  • หลังจากนำเข้า assemblies สามารถเตรียมและกำหนด Static Nests ได้
