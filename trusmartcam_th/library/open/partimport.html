Import parts

ก่อนนำเข้าชิ้นงาน ตรวจสอบให้แน่ใจว่ารูปแบบไฟล์เข้ากันได้กับ TruSmartCAM ดูที่ supported file formats สำหรับ รายละเอียดเพิ่มเติม

หากวัสดุและความหนาไม่ได้แมปกับชิ้นงานหลังจากนำเข้า ให้กำหนด วัสดุโดยใช้แผงชิ้นงาน

วิธีการนำเข้า

มีสามวิธีในการนำเข้าชิ้นงานเข้าสู่ Part Library:

  1. นำเข้าผ่านไฟล์

  2. ลากและวาง

  3. โฟลเดอร์ watch

นำเข้าผ่านไฟล์

ไฟล์ชิ้นงาน 2D หรือ 3D สามารถนำเข้าสู่ Part Library โดยใช้ ตัวเลือก นำเข้าชิ้นส่วน

  • คลิกไอคอน เปิด จากแถบคำสั่งและเลือก ตัวเลือก นำเข้าชิ้นส่วน

  • กล่องโต้ตอบ Open จะปรากฏขึ้น — เลือกไฟล์ CAD ใดก็ได้ (2D หรือ 3D)

  • ตัวอย่างของไฟล์ CAD ที่เลือกจะแสดงขึ้น

  • เลือกชิ้นงานที่ต้องการและคลิก Open

  • ระบบจะสร้างทูลลิงอัตโนมัติสำหรับชิ้นงานสำหรับเครื่องจักรที่มีทั้งหมดพร้อมสถานะทูลลิงที่อัปเดต

ลากและวาง

  • ลากและวางไฟล์ชิ้นงานโดยตรงเข้าสู่ Part Library ของ TruSmartCAM

โฟลเดอร์ watch

การตั้งค่า Watch ช่วยให้คุณสามารถตรวจสอบโฟลเดอร์สำหรับไฟล์ใหม่และนำเข้าโดยอัตโนมัติ เข้าสู่ Part Library

เพื่อกำหนดค่า ดูที่หน้า Watch settings

  • ใช้เส้นทาง UNC (Universal Naming Convention) \\192.168.1.11\<Folder path> หรือ ไดรฟ์ที่แมป M:\<Folder Path>

  • PC ที่รัน Engine ต้องมีสิทธิ์การอ่าน/เขียนไปยังโฟลเดอร์นี้

  • อย่าใช้เส้นทางภายในเช่น C:\<Folder Path> เพราะบริการ Watch มองหาไฟล์บน Engine PC — ไม่ใช่ PC ภายใน

การกำหนดค่าการนำเข้า

ใช้การเปรียบเทียบเรขาคณิตกับเวอร์ชัน CAD เดียวกัน

เมื่อนำเข้าชิ้นงานที่มีรีวิชัน CAD เดียวกัน TruSmartCAM สามารถเปรียบเทียบ รูปทรงเรขาคณิตของไฟล์ CAD กับชิ้นงานที่มีอยู่

  • เพื่อเปิดใช้งาน ให้กำหนดค่า คีย์ค้นหาเวอร์ชัน (ตัวอย่าง เช่น REV) ใน โรงงาน → การตั้งค่า →นำเข้า/ส่งออก จากนั้น เปิดใช้งาน ตัวเลือก ใช้การเปรียบเทียบเรขาคณิตกับเวอร์ชัน CAD เดียวกัน

การเปรียบเทียบรูปทรงเรขาคณิตในรีวิชัน CAD เดียวกันใช้งานได้เฉพาะเมื่อ คีย์ค้นหาเวอร์ชัน ถูกกำหนดค่า

  • ตอนนี้ นำเข้าชิ้นงานเข้าสู่ part library ด้วยรีวิชันเฉพาะ (ตัวอย่าง เช่น Revision 1)

  • ชิ้นงานเดียวกันถูกนำเข้าอีกครั้งด้วยค่ารีวิชันเดียวกัน

  • TruSmartCAM เปรียบเทียบรูปทรงเรขาคณิตของไฟล์ CAD ที่นำเข้าใหม่กับ ชิ้นงานที่มีอยู่

  • หากตรวจพบความแตกต่างของรูปทรงเรขาคณิต TruSmartCAM จะแสดงข้อความแจ้งเพื่อถามว่าจะ:

    • แทนที่ ชิ้นงานที่มีอยู่ด้วยรูปทรงเรขาคณิตใหม่ โดยเก็บค่ารีวิชัน เดียวกัน

    • ข้าม การนำเข้า รักษาชิ้นงานและรูปทรงเรขาคณิตที่มีอยู่

สิ่งนี้ช่วยให้ผู้ใช้สามารถอัปเดตรูปทรงเรขาคณิตของชิ้นงานโดยไม่ต้องเปลี่ยนค่ารีวิชัน ในขณะเดียวกันก็ยังคงควบคุมว่าจะแทนที่ชิ้นงานที่มีอยู่หรือไม่

ชิ้นงานที่นำเข้าจะถูกเพิ่มลงใน library โดยอัตโนมัติเพื่อใช้งานต่อไป สำหรับข้อมูลเพิ่มเติม ดูที่ Library
