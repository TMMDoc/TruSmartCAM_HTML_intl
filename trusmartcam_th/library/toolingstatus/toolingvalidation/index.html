Tooling validation

ส่วนนี้อธิบายวิธีการเข้าถึงและแก้ไขข้อมูลทูลลิงสำหรับ ชิ้นงานใน TruSmartCAM

  • คลิกขวาที่ไทล์ชิ้นงานใดก็ได้เพื่อเปิดแผงชิ้นงานและเข้าถึงตัวเลือกทูลลิง

  • คลิก เครื่องมือ เพื่อสลับไปยังอินสแตนซ์ทูลลิงแรกที่มี

  • หรือคลิกโดยตรงที่อินสแตนซ์ทูลลิงเพื่อดูรายละเอียดเครื่องจักร และตัวเลือกทูลลิง

  • ใช้ เพื่อ เปิดในโหมดอ่านอย่างเดียวสำหรับดู

  • ใช้ เพื่อ เช็คเอาท์เอกสารเพื่อแก้ไข

  • อินสแตนซ์ทูลลิงที่เลือกจะเปิดใน TecZone หรือ MetaCAM

  • ดำเนินการตรวจสอบทูลลิงด้วยตนเองและทำการเปลี่ยนแปลงที่จำเป็น

  • เมื่อคุณพยายามบันทึก กล่องโต้ตอบ Unsaved Changes จะแจ้งให้คุณยืนยัน การแก้ไขของคุณ

  • คลิก Yes เพื่อดำเนินการบันทึกต่อ

  • ระบบจะดำเนินการตรวจสอบทูลลิงอีกครั้งหลังจากบันทึก

  • หากไม่มีข้อผิดพลาด ตัวเลือกบันทึกการเปลี่ยนแปลงไปยัง TruSmartCAM จะปรากฏขึ้น

  • หากพบข้อผิดพลาดของทูลลิง ข้อความแสดงข้อผิดพลาดจะแสดงขึ้น:

  • หลังจากบันทึก สถานะทูลลิงของชิ้นงานจะถูกอัปเดตใน part library

  • สีเขียวบนอินสแตนซ์ทูลลิงบ่งชี้ว่าถูกเช็คอิน

  • ผู้เขียนอินสแตนซ์ทูลลิงจะถูกอัปเดตเป็นบุคคลสุดท้ายที่แก้ไข

  • ฟิลด์ แก้ไขโดยผู้ใช้ บ่งชี้ว่าอินสแตนซ์ทูลลิง ได้รับการแก้ไขด้วยตนเองหรือไม่ เมื่อผู้ใช้เช็คเอาต์อินสแตนซ์และ ทำการเปลี่ยนแปลง ฟิลด์นี้จะถูกตั้งค่าเป็น Yes โดยอัตโนมัติเมื่อบันทึก

  • หากชิ้นงานกำลังถูกแก้ไขอยู่ ไอคอนผู้ใช้จะปรากฏบนไทล์

  • การนำเมาส์ไปชี้ที่ไทล์จะแสดงว่าใครเช็คเอาต์และเวลาที่เช็คเอาต์

  • ในขณะที่ชิ้นงานถูกเช็คเอาต์ ตัวเลือก เช็คเอาท์เอกสารเพื่อแก้ไข จะถูกปิดใช้งาน สำหรับอินสแตนซ์ทูลลิงเดียวกัน และตัวเลือก ยกเลิกการเช็คเอาต์ จะเปิดใช้งาน
