Part status

เมื่อนำชิ้นงานเข้าสู่ TruSmartCAM เทคโนโลยีแผ่นโลหะที่เกี่ยวข้องจะถูก ประมวลผลไปยังชิ้นงานโดยอัตโนมัติ เป็นส่วนหนึ่งของกระบวนการนี้ TruSmartCAM จะดำเนินการ

สถานะของชิ้นงานจะแสดงโดยตรงบนไทล์ชิ้นงาน ใต้ชื่อชิ้นงาน ซึ่งระบุสภาพการตรวจสอบปัจจุบันของชิ้นงาน

ประเภทสถานะของชิ้นงานอธิบายสภาพต่างๆ ที่ชิ้นงานสามารถมีได้อันเป็นผลมาจาก การตรวจสอบในระดับชิ้นงาน สถานะเหล่านี้บ่งชี้ปัญหาที่เกี่ยวข้องกับ การออกแบบ วัสดุ หรือรูปทรงเรขาคณิตของชิ้นงาน และต้องได้รับการแก้ไขก่อนที่การสร้างทูลลิงจะ ดำเนินต่อได้

ข้อผิดพลาดของชิ้นงาน

ข้อผิดพลาด CAD (Computer-Aided Design) เกิดขึ้นเมื่อมีปัญหาในการ ออกแบบหรือรูปทรงเรขาคณิตของชิ้นงาน เช่น เส้นโครงร่างที่เปิด รูปทรงเรขาคณิตที่หายไป หรือ วัสดุที่ยังไม่ได้กำหนด ข้อผิดพลาดเหล่านี้มาจากโมเดล CAD และป้องกันไม่ให้ชิ้นงาน ถูกประมวลผลต่อไป

  • การดัดไม่สามารถดำเนินการได้ : TruSmartCAM พยายาม หาวิธีการดัดที่ถูกต้องสำหรับชิ้นงานโดยอัตโนมัติ หากไม่พบวิธีการดัดที่เหมาะสม ชิ้นงานจะถูกทำเครื่องหมายว่าดัดไม่ได้

  • Cut infeasible : หากละเว้นคำเตือนการดัด TruSmartCAM จะพยายามหาวิธีการตัด หากไม่พบวิธีการตัดที่ถูกต้อง ชิ้นงานจะถูกทำเครื่องหมายว่าตัดไม่ได้

ข้อผิดพลาดของวัสดุ

  • ไม่พบวัสดุ : ความหนาของชิ้นงานถูกอ่าน จากโมเดล CAD และจับคู่กับวัสดุในฐานข้อมูล Raw Material ของ TruSmartCAM หากไม่พบวัสดุที่ตรงกัน ชิ้นงานจะถูกทำเครื่องหมายด้วย ข้อผิดพลาด ไม่พบวัสดุ

  • ยังไม่ได้กำหนดวัสดุ : หากโมเดล CAD ไม่ มีฟิลด์การแมปวัสดุหรือฟิลด์นั้นว่างเปล่า TruSmartCAM ไม่สามารถกำหนด วัสดุได้ ในกรณีนี้ ชิ้นงานจะถูกทำเครื่องหมายด้วย ข้อผิดพลาด ยังไม่ได้กำหนดวัสดุ

ข้อผิดพลาดทางเรขาคณิต

  • ตรวจพบเส้นขอบเปิด : ไฟล์ CAD มีรูปร่างที่ เปิด (ไม่สมบูรณ์) หนึ่งรูปหรือมากกว่า ซึ่งป้องกันไม่ให้สร้างทูลลิงได้อย่างเหมาะสม

  • ตรวจพบเส้นขอบด้านนอกหลายรายการ : ไฟล์ CAD มี has more than one outer closed loop, which creates confusion during processing.

  • Nascent error : When a part is loaded from a spreadsheet (CSV or XLSX), but the actual CAD file is missing, it is marked as a nascent part.

  • ไม่มีข้อมูลเรขาคณิต : The CAD model has no valid geometry or the geometry is unreadable, making it unusable for tooling.
