Orders

Orders

An order defines what parts need to be manufactured, in what quantity, and by when. It drives production planning and execution in TruSmartCAM. Orders can be created or imported using various methods, and they can be converted into jobs for production planning.

Order panel

Open in read-only mode for viewing
Opens the selected order in read-only mode.

Check-out the document for editing
Enables editing of the selected order.

Delete
Deletes the selected order.

New…​
Converts the order into a job, assembly, or another order.

Assign Customer…​
Assigns a customer to the order.

Set Due Date…​
Sets the due date for the order.

Priority
Specifies the priority level of the order.

Part Order
Specifies the reference ID assigned to a part within the order.

Quantity
Specifies the number of parts or instances in the order.

Prev / Next / Part
Navigates to the previous or next part within the order.

Sort and filter
Use these filters to view other orders that share the same material, customer, due date, or priority.

Edit orders

Select one or more orders and right-click to modify common order details such as quantity, due date, priority, and customer information.

If multiple selected orders require different values, use the Check-out for Editing option to edit each order individually based on the requirement.

Any updates made to the orders are applied immediately to the database. Use the Delete command to remove orders from the order library.

The Delete command removes only the orders. Parts linked to the respective orders are not deleted from the Part library.

Planning orders

Select one or more orders, right-click, and click New…​ → Job to convert the selected orders into jobs. Use the Sort and Filter options to group relevant orders together before job creation.

Once the orders are converted into a job:

  • The job is moved to the Jobs page

  • The orders are removed from the Orders list

  • The standard job planning workflow can then be used for production planning and nesting

For more information about job planning workflow, refer to the Auto Plan

Additional information

  • If a job containing orders is deleted, the associated orders are automatically moved back to the Orders Ready page, unless the job or part is marked as Completed.

  • Orders can be added to or removed from an existing job through the Edit Job window based on production requirements.
