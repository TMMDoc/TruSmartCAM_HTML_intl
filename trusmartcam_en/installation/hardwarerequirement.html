Port Requirement

TruSmartCAM Server : 7371

Server waits for the client requests on this TCP port. Incoming connections should be allowed to this port on the Server Computer.

Floating License Server : 5776

Floating License Server waits for licensing requests on this port. Incoming connections should be allowed to this port on the floating license server computer (TruSmartCAM Server).

License connection to Cloud : 7567

This port is used by TecZone (Bending Software) and Floating License Server to send licensing requests to Metamation Cloud. So outgoing connection should be allowed on this port for license activations and renewals.

Installation and Support Requirements

  • Remote diagnostic tools via Contractor VPN or a secure connection.
