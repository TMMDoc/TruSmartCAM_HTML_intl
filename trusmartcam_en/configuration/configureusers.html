Configure users

  • TruSmartCAM clients with different roles are used to control the factory configuration.

    • Adding new machines, materials and sheets.

    • Editing machine configurations.

    • Eding parts, jobs and assemblies.

    • Edit / Update nest sheets, Plan Job.

    • Add/Edit the users.

  • Pserver install PC consider as first TruSmartCAM user.

  • TruSmartCAM uses the windows credential as TruSmartCAM user name which is Site Admin [Superior user rights].

  • Next install ie., TruSmartCAM Client install computer username are considered as Programmer.

Only Site Admin can edit other user information and change the password.
Role Permission Denied

All Permissions

Admin

Programmer

Upload and edit Parts/Jobs/Assemblies, Edit Part tooling, Sheet Inventory, Plan a Job

Edit Factory configuration, Edit users

Layout/Job complete, Add new sheet, Layout commands

Everything else denied.
