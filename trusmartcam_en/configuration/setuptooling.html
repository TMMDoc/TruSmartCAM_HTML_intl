Setup tooling

All calculations of needed geometry adaption as well as nesting and bending technology of TruSmartCAM is based on the state of the art technology modules TecZone Fold and TecZone Laser. In order to get the highest level of precision all details of your physical machines need to be configured.

Especially in Bend Technology all existing tools need to be known to the offline system. All Tooling for the machines will be configured in the TruSmartCAM Factory dialog but is analog to the dialog of the technology modules TecZone Fold and TecZone Laser.

Please refer to the TecZone manuals to configure your system.

Catalog options

Clicking on the Catalog drop down will firstly list out any catalogs of tools that are already active along with a listing for any custom tools and the ability to install another catalog by clicking on the Add button.

Clicking on any catalog in the available list will then install it.

Collection

Bend Tool Inventory also supports the option to create custom collection(s) of tools which could be for a specific customer or cross catalog where multiple catalogs are in use

Refer to the TecZone Fold manual for more information
