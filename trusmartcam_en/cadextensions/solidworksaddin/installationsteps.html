Installation

Mandatory requirements

Signing in to TruSmartCAM desktop is mandatory when launching SolidWorks for the first time.

Ensure that the TruSmartCAM monitor is running in the windows notification area.

If TruSmartCAM desktop is not open, verify that the TruSmartCAM server service is running to allow the SolidWorks Add-in to function. If both are inactive, SolidWorks will display an error message upon launch.

Follow these steps to install and set up the TruSmartCAM Add-in for SolidWorks:

  1. Run the TruSmartCAM Setup, and on the Components selection page, enable SolidWorks Extension. (Skip the Server option if SolidWorks is installed on a TruSmartCAM client.)

  2. Complete the installation, then launch TruSmartCAM desktop and sign in with valid user credentials.

  3. Launch SolidWorks.

  4. In SolidWorks, go to Tools → TruSmartCAM → About TruSmartCAM, to verify the add-in version and details.

  5. To enable or disable the add-in at startup, go to Tools → Add-ins, and locate TruSmartCAM under Other Add-ins.
