Die Benutzeroberfläche

Die Benutzeroberfläche

Die Benutzeroberfläche von TruSmartCAM bietet Zugriff auf alle wichtigen Funktionen über die Befehlsleiste und Menüleiste und ermöglicht eine effiziente Navigation und Bedienung.

Im Bild:

1 - Befehlsleiste

2 - Menüleiste

3 - Unterbefehle

Befehlsleiste

Open - Importiert Teile, Aufträge, Bestellungen und Baugruppen in die Bibliothek. Details finden Sie unter Öffnen.

Language - Ändert die Sprache der Benutzeroberfläche. Details finden Sie unter Sprache.

Library - Öffnet die Bibliotheks-Startseite, auf der Teile und Baugruppen getrennt angezeigt werden. Details finden Sie unter Bibliothek.

Jobs - Zeigt importierte oder erstellte Aufträge an. Details finden Sie unter Aufträge.

Factory - Öffnet den zentralen Ort zur Konfiguration und Verwaltung von Routen, Materialien, Benutzern und Kundendaten für die Produktionseinrichtung. Details finden Sie unter Factory.

Info - Zeigt Informationen über die Software an.

Menüleiste

File - Bietet Optionen zum Öffnen von Teilen, Aufträgen, Bestellungen und Baugruppen sowie zum Beenden der Anwendung.

View - Ermöglicht das Wechseln zwischen Bildschirmen und den Zugriff auf verschiedene Ansichten.

Help - Bietet Zugriff auf das Hilfehandbuch und Software-Aktualisierungsoptionen.

Admin - Enthält administrative Einstellungen und Konfigurationsoptionen.

Unterbefehle

Parts - Zeigt die Elemente der Teilebibliothek an.

Assys - Wechselt zu Baugruppenelementen in der Bibliotheksansicht.

Export - Exportiert den aktuellen Teil- oder Bearbeitungsbericht als CSV.
