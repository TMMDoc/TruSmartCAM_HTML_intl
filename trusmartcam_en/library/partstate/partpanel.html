Part panel

This section explains part-related actions such as Re-import, Re-tool, Rename, Export outputs, etc., used to manage part instances.

Open in read-only mode for viewing
Opens the selected part in read-only mode.

Check-out the document for editing
Enables editing of the selected part.

Delete
Deletes the selected part.

New…​
Selected part can be converted to a new order, job or assembly.

Export Outputs
Exports the part output to the destination configured.

Re-import…​
Allows to import the selected part again.

Re-tool…​
Allows to reprocess the selected part.

Rename…​
To change the name of selected part.

Assign Material…​
Specifies the material associated with the selected part.

Assign Customer…​
Specifies the customer associated with the selected part.

Rolling direction
Defines the orientation (horizontal or vertical) along which the material is rolled.

Prev / Next / Toolings
Navigates to the previous or next part or tooling.

Sort and filter
Use these filters to view other parts that share the same attribute as the selected part: Same Material, Same Customer, Same Bend count, or Same State.

Re-import

  • Right-click any part and click Re-import…​.

  • All selected part tooling instances across all machines are cleared including auto-tooled and manually tooled instances.

  • Existing output files (CAD, Bend, and Cut) are removed.

  • Parts are re-imported, and fresh CAD validation and tooling are applied for all machines.

  • New output files are created based on the updated tooling result.

Re-tool

  • Right-click any part and click Re-tool…​.

  • All machines are checked on by default and grouped as Laser, Punch, Panel, and Press brake.

  • Click OK to proceed. This removes both auto-tooled and manually tooled instances for the selected machines, and deletes related output files.

  • When Retain existing outputs checkbox is selected, only parts whose outputs were removed earlier will be retooled, other outputs remain untouched. If unchecked, all output files for selected machines are removed and regenerated during retooling.

  • If CAM settings changed for any machine, select that machine and Re-tool the parts.

  • If a tool (LTT, punch, or bend) is changed or missing, update the machine and Re-tool only the affected parts.

Rename

This section explains how to rename one or multiple parts in TruSmartCAM.

  • Right-click the part and click Rename…​.

  • A rename confirmation dialog appears showing the current part name.

  • Enter the new name and click the tick ✓ icon to confirm, or click the cross ✗ icon to cancel the rename operation.

  • To rename multiple parts, select all required parts, then right-click any one of them and click Rename…​.

  • A confirmation dialog appears with the total number of selected parts.

  • For each part, enter the new name and click the tick ✓ icon to confirm one by one.

  • If any part was mistakenly selected, click the cross ✗ icon to cancel. Alternatively, you can right-click the part and click Cancel Rename to skip renaming.

Export outputs

This option exports the part output to the destination configured in Factory Settings. (See Bend Settings and Cut Settings)

The output location is defined separately for Bend, Fold, and Cut in their respective output settings. All toolings associated with bend, fold, and cut operations are exported for the entire part.

As shown below, the exported files are saved in the specified folder (for example, C:\PraxData\Out\Reports).

The exported files depend on the operation type. For Bend and Fold operations, NC files and reports are generated. For Cut and Punch operations, only the part files are exported.

Part assign material

Parts with the status Material not assigned need to be updated before further processing. The steps below explain how to assign material.

  • Right-click the part that displays the status Material not assigned and select Assign Material…​.

  • The Assign Material window opens.

  • The dialog has four sections:

    • Suggestions - Automatically suggests raw materials based on the imported part’s thickness.

    • All Materials - Lists all material types defined in TruSmartCAM.

    • All Raw-Materials - Displays all raw materials available in the system.

    • Used Raw-Materials - Displays the most recently used raw materials for quick access.

  • You can use the Search field at the top to quickly find materials. For example, type "steel" to display all matching materials across the three sections.

  • From the filtered list, select the required material (e.g., Steel-25) from the Raw-Material list.

  • The selected material will be assigned to the part.

Rolling direction

Rolling direction refers to the orientation of the grain in a part, which can affect how the part is cut and processed. This option allows you to specify the Rolling direction for a part in the Part Library. Based on the selected direction, the part will be placed in the correct orientation for cutting on the available machines.

  • Right-click any part tile in the Part Library or Job window.

  • Select the preferred rolling direction:

    • None

    • Horizontal

    • Vertical

  • The part gets retooled automatically for all available cut machines based on the selected rolling direction.
