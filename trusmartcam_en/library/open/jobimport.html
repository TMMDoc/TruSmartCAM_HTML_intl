Jobs importieren

Import aus CSV oder XLSX

Sie können Jobs auch in gängigen Textformaten importieren.

  • Klicken Sie auf File → Open → Import Jobs, der Ordner-Dialog erscheint. Wählen Sie C:\Program Files\Metamation\Praxis\Samples\Jobs\C0\C0.csv.

  • Wenn alle Teile im CSV-Verzeichnis oder in den Lookup-Ordner-Verzeichnissen vorhanden sind, erscheint das Fenster new job mit einer Auflistung aller Teile, die den CSV-Feldern entsprechen.

  • Fehlende Teile werden ignoriert und erscheinen nicht im Fenster new job.

  • Anfänglich ist der Teilestatus „Nascent".

  • Klicken Sie auf OK, die Teile werden in die Teilebibliothek importiert. CAD-Validierung, Biege- und Schneidwerkzeugbelegung werden durchgeführt, was im Pmonitor-Bildschirm überwacht werden kann.

Create a blank part for the missing CAD files

Beim Import von Jobs über eine Tabelle (.csv oder .xlsx) kann es vorkommen, dass Zeilen auf CAD-Dateien verweisen, die im System nicht verfügbar sind.

  • Um dies zu behandeln, gehen Sie zu Factory → Settings → Job, und aktivieren Sie unter Spreadsheet import settings die Option Create a blank part for the missing CAD files

Wenn diese Option ausgewählt ist:

  • Falls eine CAD-Datei für eine Zeile während des Job-Imports fehlt, erstellt das System automatisch ein leeres Teil anstelle der fehlenden CAD-Datei.

  • Im Fenster new job werden diese fehlenden Teile gelb hervorgehoben, um anzuzeigen, dass die zugehörige CAD-Datei fehlt.

  • Das fehlende Teil enthält weiterhin seine zugeordneten Felder (wie Teilename, Material, Dicke, Menge usw.) aus der CSV. Der Teilestatus wird jedoch als „Nascent" angezeigt, was bedeutet, dass es nur ein leerer Platzhalter ohne tatsächliche Geometrie ist.

Dies stellt sicher, dass der Job-Import ohne Fehler aufgrund fehlender CAD-Dateien fortgesetzt werden kann — Sie können das leere Teil später aktualisieren oder durch die korrekte CAD-Datei ersetzen, sobald diese verfügbar ist.

Do not show the new job dialog if the spreadsheet has no error

factory → Settings → Job → Spreadsheet import settings→ Do not show the New Job dialog if the spreadsheet has no error

  • Wenn dieses Kontrollkästchen aktiviert ist, erstellt das System den neuen Job automatisch ohne den Neuer-Job-Dialog anzuzeigen, sofern die Tabelle keine Fehler enthält (keine fehlenden Teile, fehlende Daten oder ungültige Daten).

  • Wenn dieses Kontrollkästchen deaktiviert ist, erscheint der Neuer-Job-Dialog immer zur Überprüfung und Bestätigung durch den Benutzer, auch wenn die Tabelle keine Fehler enthält.

Prefer raw-material column over material + thickness

factory → Settings → Job→Prefer Raw-Material column over Material + Thickness

  • Wenn dieses Kontrollkästchen aktiviert ist, erwartet das System, dass die Tabelle die Spalte „Raw-Material" verwendet (z. B. Steel-20). Das bedeutet, dass Materialtyp und Dicke in einem einzigen Wert in einer Spalte kombiniert sein sollten.

  • Wenn dieses Kontrollkästchen deaktiviert ist, erwartet das System, dass die Tabelle separate Spalten für Material und Dicke enthält (z. B. Steel und 2.00). Das System kombiniert diese intern während des Imports.

Überwachungsordner mit CSV/XLSX

  • Aktivieren Sie unter factory → Settings→ Watch settings die Optionen Enable 'Live Folder' und Watch sub-folders also.

  • Legen Sie den Pfad des Überwachungsordners fest (UNC oder zugeordnetes Laufwerk).

  • Aktivieren Sie unter Spreadsheet import settings die Option Import spreadsheet from 'Live Folder'.

  • Wählen Sie Import Jobs aus den Tabellenimport-Optionen.

  • Kopieren Sie C:\Program Files\Metamation\Praxis\Samples\Jobs\C0\C0.csv und fügen Sie die Datei in den Überwachungsordner ein.

  • Das System prüft die CSV-Datei und versucht, den Job automatisch zu erstellen.

  • Wenn die CAD-Datei fehlt, wird kein Job erstellt, da die erforderlichen CAD-Daten nicht gefunden wurden.

CAD-Lookup-Verzeichnis & rekursive CAD-Suche – nicht konfiguriert

  • Wenn unter factory → Settings → Job → Spreadsheet import settings kein CAD lookup directories festgelegt ist und Use recursive CAD search nicht aktiviert ist, sucht das System nicht automatisch nach CAD-Dateien.

  • Wenn das Kontrollkästchen Create a blank part for the missing CAD files aktiviert ist, wird der Job auch bei fehlenden CAD-Dateien trotzdem erstellt.

  • In diesem Fall werden alle Teile im Job als „Nascent"-Teile (leere Teile) erstellt — das bedeutet, sie haben keine Geometrie, da die CAD-Dateien nicht gefunden werden konnten.

CAD-Lookup-Verzeichnis & rekursive CAD-Suche – konfiguriert

Wenn CAD lookup directories auf (C:\Program Files\Metamation\Praxis\Samples) gesetzt ist und Use recursive CAD search unter factory → Settings → Job → Spreadsheet import settings aktiviert ist:

  • Das System durchsucht automatisch den angegebenen Ordner und alle Unterordner nach den in der Tabelle genannten erforderlichen CAD-Dateien.

  • Create a blank part for the missing CAD files ist ebenfalls aktiviert. Falls eine CAD-Datei trotzdem nicht gefunden wird, erstellt das System ein leeres Teil als Platzhalter.
