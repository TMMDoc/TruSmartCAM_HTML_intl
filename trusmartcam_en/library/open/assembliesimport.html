Import assemblies

Assemblies are groups of related parts combined into a single unit, allowing you to manage multiple parts together as one entity.

You can import assemblies using the following methods:

  1. Import via file

  2. Drag and drop

  3. Import from spreadsheet

Import via file

An assembly can be imported into the Part Library using the Import Assemblies…​ option.

  • Click the Open icon in the command bar and select Import Assemblies…​, or use File → Open → Import Assemblies…​.

  • The Open dialog appears. Select the required assembly file(s).

  • A preview of the selected assembly is displayed.

  • Click Open to import the assembly.

  • Imported assemblies are listed under the Assemblies view in the Part Library.

Drag and drop

  • Drag and drop a assembly directly into the TruSmartCAM Part Library.

Import from spreadsheet

  • Use File → Open → Import Assemblies…​ and select a CSV, XLSX, or XML file, or drag and drop the file.

  • Review and update assembly details and quantities before import. Use Add…​ or Remove…​ to modify.

  • After import, assemblies can be prepared and assigned Static Nests.
