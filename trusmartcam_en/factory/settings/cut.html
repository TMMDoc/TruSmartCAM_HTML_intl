Cut

Output settings

Save cut outputs - The tooling results are saved only if the tooling is OK and these export settings are enabled and configured. Enable this option for the Cut NC to be saved.

Output tooling simulation preview - Enable this option to generate the tooling simulation preview along with the other outputs. This is a short GIF animation for the laser/punch tooling.

Cut output destination - This is the destination for the NC and reports to be saved.

MetaCAM settings

Lookup MetaCAM materials by raw-material names (Enable this for DMM-like integrations) - When this style is enabled, TruSmartCAM generates pages by raw-material names and assigns them to parts at the time of auto-tooling.

MetaCAM sync folders - This Cut setting is used to add additional MetaCAM folders which should be synchronized along with the config.dat when the config-push Monitor command is used. The config-push saves the local files to the TruSmartCAM repository which is downloaded on the other computers when MetaCAM (re)starts. The Engine computer should synchronize it automatically when the auto tooling is done for a MetaCAM machine apart from the Tools, some of the other installation-specific candidates are the Etc, Reports, and Machines folders.

DMM config sync location - This is the location for the Dynamic Machine Management config sync to be set.

TecZone Settings

Enable multi-pass nesting - This will enable the laser to cut multiple times on the same contour (preferred for thicker material).
