Instalación

Instalación

Requisitos obligatorios

Iniciar sesión en el escritorio de TruSmartCAM es obligatorio al lanzar SolidWorks por primera vez.

Asegúrese de que el monitor de TruSmartCAM esté ejecutándose en el área de notificación de Windows.

Si el escritorio de TruSmartCAM no está abierto, verifique que el servicio del servidor de TruSmartCAM esté en ejecución para permitir que el Add-in de SolidWorks funcione. Si ambos están inactivos, SolidWorks mostrará un mensaje de error al iniciarse.

Siga estos pasos para instalar y configurar el Add-in de TruSmartCAM para SolidWorks:

  1. Ejecute la configuración de TruSmartCAM, y en la página de selección de componentes, habilite SolidWorks Extension. (Omita la opción Server si SolidWorks está instalado en un cliente de TruSmartCAM.)

  2. Complete la instalación, luego inicie el escritorio de TruSmartCAM e inicie sesión con credenciales de usuario válidas.

  3. Inicie SolidWorks.

  4. En SolidWorks, vaya a Tools → TruSmartCAM → About TruSmartCAM, para verificar la versión y los detalles del add-in.

  5. Para habilitar o deshabilitar el add-in al inicio, vaya a Tools → Add-ins, y localice TruSmartCAM bajo Other Add-ins.
