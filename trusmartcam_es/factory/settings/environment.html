Environment

Ajustes de entorno

Unidades predeterminadas de programa - El usuario puede elegir si la unidad predeterminada es mm o inch.

Decimales a mostrar - Este valor es para establecer cuántos lugares decimales se mostrarán.

Precisión decimal al exportar/importar - Este valor es para establecer cuántos lugares decimales deben ser exactos al importar o exportar.

Mostrar los utillajes de piezas en grupos - Use esta opción para seleccionar el grupo de herramental para mostrar el estado del herramental en la biblioteca de piezas.

Leyendas para iconos de barra de comandos - Habilitar esta opción proporcionará un título debajo de cada icono para referenciar a qué se referirá la página. Cuando esta opción está deshabilitada, todos los títulos para los interruptores se eliminan y solo se muestra el icono para cada página diferente.

Mostrar etiquetas de estado de tarea - Habilitar esta opción mostrará el número de piezas que están dentro de un job al final de la barra de estado de Job. Cuando esta opción está deshabilitada, el recuento de piezas se elimina y el color para mostrar el proceso permanecerá.

Mostrar piezas con nesting por color de pedido de tarea - Esta opción mostrará cada pieza diferente dentro de un Job como un color diferente. Al deshabilitar esta opción, todas las piezas se mostrarán en el mismo color gris.

Mostrar la duración en formato de intervalo de tiempo (HH:MM:SS) - Marque esta opción para mostrar la duración en formato HH:MM:SS. Cuando está habilitado, la duración en los detalles de layout y los reportes de job/layout se muestran en el formato seleccionado. Debido a las restricciones de espacio, la duración se muestra en un formato de segundos más compacto en los tiles de layout.
