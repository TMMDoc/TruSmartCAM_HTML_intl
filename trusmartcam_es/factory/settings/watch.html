Watch

Watch

La configuración de Watch permite que TruSmartCAM monitoree una carpeta designada e importe automáticamente archivos al sistema. Esto ayuda a automatizar la importación de Parts, Jobs, Assemblies y Orders sin intervención manual.

Visualizar ajustes

Habilitar 'Carpeta en directo' - Habilita el monitoreo automático de la carpeta especificada.

Visualizar esta carpeta - Especifica la carpeta que se monitoreará para archivos entrantes.

Visualizar también las subcarpetas - Incluye todas las subcarpetas en el proceso de monitoreo.

Tras la importación de archivos

Tras la importación - Después de que se importa el archivo, el usuario tiene dos opciones. El archivo se puede eliminar de la ubicación o mover a una ubicación diferente.

  • Delete imported file - Elimina el archivo de origen después de una importación exitosa.

  • Move imported file - Mueve el archivo de origen a la carpeta especificada en Move files to this folder después de una importación exitosa.

Mover archivos a esta carpeta - Especifica la carpeta de destino donde los archivos importados exitosamente deben moverse cuando After import está establecido en Move imported file.

Mover hojas de cálculo con errores a esta carpeta - Mueve las importaciones fallidas a la carpeta de error especificada.

Ajustes de importación de hojas de cálculo

Importar hojas de cálculo de 'Carpeta en directo' - Importa automáticamente archivos de hoja de cálculo detectados en la carpeta de watch.

No importar hojas de cálculo con errores - Evita que archivos que contienen errores de validación sean importados.

Opción de importación de hoja de cálculo - Selecciona el tipo de datos para importar desde el menú desplegable. Los formatos de hoja de cálculo compatibles pueden incluir CSV, XLSX o XML. Las opciones disponibles son:

  • Importar tareas - Importa datos de job desde la hoja de cálculo.

  • Importar pedidos - Importa datos de order desde la hoja de cálculo.

  • Importar grupos de módulos - Importa datos de assembly desde la hoja de cálculo.

  • Importar piezas - Importa datos de part desde la hoja de cálculo.

Información adicional

  • Use una ruta UNC (\\ServerName\FolderPath) o una unidad mapeada (M:\FolderPath) al configurar la carpeta de watch. No use rutas locales como C:\FolderPath.

  • La computadora que ejecuta el servicio Engine de TruSmartCAM debe tener acceso de lectura y escritura a la carpeta de watch configurada. Sin permisos suficientes, los archivos pueden no ser detectados, importados, movidos o eliminados exitosamente.

  • El mapeo de hoja de cálculo debe configurarse antes de habilitar la configuración de Watch. De lo contrario, TruSmartCAM no podrá procesar los archivos de hoja de cálculo detectados en la carpeta de watch.
