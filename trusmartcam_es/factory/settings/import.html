Import

Ajustes de salida

Guardar patrón plano - Habilite esta opción para que el patrón plano de una pieza importada se guarde.

Formato para patrón plano - Esta opción se usa para elegir formatos de archivo DXF o GEO.

Lugar de destino de patrón plano - Esta es la ubicación donde se guardará el patrón plano.

Importar ajustes

Unidades para archivos DXF - Esta es la unidad de medida con la que se importará un archivo DXF, el usuario puede elegir entre mm o inch.

Teclas de búsqueda de material - Se pueden asignar múltiples entradas de búsqueda usando un separador de coma entre ellas. Cuando se importa una pieza, TruSmartCAM usa estas claves para buscar valores de material. Los valores buscados luego se usan para resolver la materia prima. Los valores originales se almacenan en las Propiedades Misc de la pieza.

Teclas de búsqueda de revisión - Cuando se importa una pieza, el valor de revisión del archivo de pieza sin procesar (el archivo de pieza que se está importando) se compara con el valor de revisión existente en el repositorio. La importación se abandona si la revisión es la misma. El mapeo de campos personalizados ahora también es compatible con los archivos DXF. Para los archivos DXF, los datos de campo personalizado se extraen de la capa de texto en el dibujo o un adjunto PPI.

Utilizar comparación de geometría en misma revisión CAD - Compara la geometría al importar una pieza con la misma revisión CAD y solicita al usuario que reemplace o omita la importación si se detectan diferencias, Consulte Import configuration para más detalles.

Teclas de nombre de piezas - Esta opción funciona de manera similar a las claves de material y Teclas de búsqueda de revisión. Establezca una o más Teclas de nombre de piezas para usar el Nombre de la pieza de PPI en lugar del nombre de archivo como nombre de pieza.

Teclas de búsqueda de espesor - Se pueden asignar múltiples entradas de búsqueda usando un separador de coma entre ellas. Cuando se importa una pieza, TruSmartCAM usa estas claves para buscar valores de espesor. Los valores buscados luego se usan para resolver el espesor. Los valores originales se almacenan en las Propiedades Misc de la pieza.

Factor de corrección (fudge) de comparación de espesores - Esto se usa para buscar la materia prima cuando se importa una pieza. TruSmartCAM intenta resolver la materia prima después de que se analizan el nombre del material y el espesor. Para más información, consulte el tema Thickness fudge a continuación.

Identificar formas después de importar una plana - Cuando se importa una pieza, las formas internas dentro de la pieza pueden reconocerse e identificarse automáticamente.

Factor de corrección (fudge) de identificación de formas - El valor de fudge de identificación de forma se puede establecer en:

  1. Nivel global - Este valor se usa como predeterminado para todas las formas.

  2. Nivel de Forma Individual - El valor predeterminado se puede anular estableciendo el valor de fudge a nivel de forma individual. Seleccionar la forma en el diálogo de Shape Library muestra el valor de fudge. Por defecto, esto se establece en 0, y se usa el valor global predeterminado para tales formas.

Girar plano horizontalmente - Generalmente las mesas de máquinas de corte son rectangulares y más anchas. Para mejorar la viabilidad del herramental, rote la pieza a lo largo del ancho de la pieza después de importarla.

No girar pieza más corta que esta - Establezca esto como el número mínimo para permitir la rotación. Cualquier pieza más pequeña que el valor establecido no será rotada.

Varios ajustes

Habilitar preimportación de útiles de otros fabricantes - Habilite la opción de importación de herramientas externas de pre-importación para habilitar Creo Reader. TruSmartCAM carga las instancias de familia pre-extraídas de la pieza descargada. Tenga en cuenta que la extracción de instancias se realiza durante la carga de la pieza (como una operación de pre-importación), por lo que el software Creo debe estar instalado en la computadora desde donde se está cargando la pieza.

Ignorar color de dibujo en miniaturas planas para mayor claridad - Habilite esta opción para generar las miniaturas en escala de grises más nítidas. Los colores de capa y las entidades de texto se ignoran por el bien de la claridad de la imagen.

Usar solo un motor para todas las tareas de importación - Cuando está habilitado, solo se usa un núcleo de engine para tareas importantes.

Thickness fudge

El Factor de corrección (fudge) de comparación de espesores elige el valor de espesor de lámina más cercano cuando un espesor de lámina deseado no está disponible en el inventario comparando el valor de espesor de las láminas en el inventario.

Por ejemplo, supongamos lo siguiente:

  1. Tenemos una pieza con espesor de lámina de 1.6 mm y material de acero.

  2. El valor de Factor de corrección (fudge) de comparación de espesores es 0.1 mm.

  3. El inventario tiene láminas de acero con espesor 1.5 mm, 1.55 mm y 1.7 mm.

TruSmartCAM elige una lámina con el valor de espesor más cercano después de comparar el valor de Factor de corrección (fudge) de comparación de espesores (0.1 mm) con el elemento importado (que mide 1.6 mm). En este caso, se seleccionará 1.55 mm.
