Ajustes de salida

Guardar diseño CN e informes - Habilite esta opción para que el NC de layout y los reportes se guarden.

Guardar archivo de diseño - Habilite esta opción para que el archivo de layout se guarde.

Vista previa de simulación de salida en informes - Habilite esta opción para generar la vista previa de simulación de herramental junto con las otras salidas. Esta es una animación GIF corta para el herramental de laser/punch.

No eliminar las ediciones de programa al finalizar diseño/tarea - Habilitar esta opción omitirá la operación de eliminación de las salidas del programa cuando un job/layout ha sido marcado como completado.

Ajustes de nesting

Mezclar piezas de múltiples tareas - Soporta modo de anidamiento mixto. Las piezas de múltiples jobs deben anidarse juntas para una mejor eficiencia de empaquetado.

No liberar automáticamente diseños - Esto evitará que los layouts estén disponibles para producción hasta que todos los layouts estén en un estado liberado.

Autoliberar embalaje separación - Este valor se usa para decidir si un layout anidado se mantiene en la cola de aprobación y se vuelve a anidar cuando se planifican nuevas piezas o si se liberan para producción y las salidas relacionadas se exportan a la carpeta de salida.

Prefijo de nombre de chapa interactivo - Esta sección se usa para soportar anidamiento interactivo donde el usuario tiene más control sobre las piezas que se anidan y el resultado del anidamiento. Use un esquema de nombres, diferente del anidamiento automático, para los layouts anidados interactivamente.

Actualizar el inventario de hojas sobre la (des)liberación de nesting - Si esta opción está habilitada, el inventario de láminas se actualizará al enviar un anidamiento a una máquina. También retirar el anidamiento de la máquina devolverá la lámina al inventario.

Automatizar administración de existencias del restos - Cuando este interruptor está seleccionado, el software automatiza la gestión de inventario de remanentes. Si no está seleccionado, la gestión de remanentes se vuelve interactiva.

Autorefrescar diseños obsoletos - Cuando está marcado, el software actualiza todos los ui:EGTag.SelectOutOfDate[Out-of-date Layouts] vinculados a una pieza después de que se calculan sus soluciones de corte.

Nesting de TwinLine automático - Marque esta opción para habilitar la automatización de twin-line.

Ajustes de flujo de trabajo Pull

Use 'Flujo de trabajo Pull' - Esta opción es útil para usuarios que necesitan un sistema de planificación y anidamiento altamente autónomo. Cuando está habilitado, TruSmartCAM planifica el anidamiento bajo demanda.

Nestings por estación - Este valor es para establecer cuántos layouts por estación se deben crear.

Chapas máximas por nesting - Este valor se usa para controlar el recuento máximo de láminas contra el anidamiento único. Entonces, por ejemplo, si hay un pedido con una pieza grande (que llena toda la lámina), necesitada en grandes cantidades (digamos 500) - TruSmartCAM ya no crea un solo anidamiento con 500 copias en modo pull. En su lugar, crea 10 copias por anidamiento donde 10 es el máximo.

Programas de plegado por estación - Este valor es para establecer cuántos programas de bend por estación se deben crear.

Marcar la tarea completada cuando se retiran las salidas de programa - Habilite esta opción para que un layout se marque como Done cuando las salidas se eliminen de la ubicación de salida de destino.

Ajustes de importación de hojas de cálculo

Directorios de búsqueda CAD - TruSmartCAM intentará cargar la entrada de pieza mencionada en el CSV importado desde la biblioteca de piezas. Sin embargo, si la pieza no está disponible en la biblioteca de piezas, entonces intentará obtener las piezas de los directorios de búsqueda según el orden de directorios establecido en este diálogo.

Usar búsqueda CAD recursiva - Habilite esta opción para que la búsqueda se repita para resultados sucesivos.

Crear una pieza en blanco para los archivos CAD que faltan - Si las piezas no existen en la biblioteca de TruSmartCAM o en los directorios de búsqueda, TruSmartCAM creará una pieza en blanco para esas piezas no disponibles.

No mostrar el nuevo diálogo de tarea si la hoja de cálculo no contiene errores - Si las entradas CSV están limpias, es decir, sin discrepancia de datos de DB o todos los campos están listados correctamente, TruSmartCAM no mostrará la ventana de descripción general del job y procesará el job tan pronto como se importe o descargue el CSV.

Preferir columna de materia prima sobre material + espesor - Tipos de hoja de cálculo compatibles:

  1. Patrón 1: PartName, Quantity, DueDate, Priority, RawMaterial

  2. Patrón 2: PartName, Quantity, DueDate, Priority, Material, Thickness

  3. Patrón 3: PartName, Quantity, DueDate, Priority, Material, Thickness, RawMaterial

Si esta opción no está marcada, el Patrón 1 se considerará el tipo de hoja de cálculo predeterminado. También soportamos el Patrón 3 en caso de que el formato CSV sea del tipo Patrón 3.

Si esta opción está marcada y CSV es del tipo Patrón 2, entonces TruSmartCAM resolverá Raw Material de los campos Material y Thickness del CSV.

Ajustes de salida de hoja de cálculo

Exportar informe de finalización de tarea/diseño (.csv)* - Marque esta opción para habilitar la exportación CSV cuando un Layout o Job esté completo. Los reportes completados se guardan en el destino de Reports especificado en Factory → Settings → Reports settings.
