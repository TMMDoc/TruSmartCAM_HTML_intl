Supported file formats

TruSmartCAM soporta importación directa de CAD. La disponibilidad de formatos de importación depende de la configuración de la licencia. Los formatos compatibles se clasifican como sigue.

La lista de formatos CAD compatibles y la compatibilidad de versión de archivo depende del nivel de licencia activo. Las versiones compatibles se actualizan periódicamente durante el año. Antes de actualizar su software CAD a una versión más reciente, se recomienda verificar la lista de compatibilidad para garantizar el soporte continuo.
