Requisito de Puerto

Server de TruSmartCAM : 7371

El servidor espera las solicitudes del cliente en este puerto TCP. Se deben permitir conexiones entrantes a este puerto en la Computadora del Server.

Servidor de Licencia Flotante : 5776

El Servidor de Licencia Flotante espera solicitudes de licenciamiento en este puerto. Se deben permitir conexiones entrantes a este puerto en la computadora del servidor de licencia flotante (Server de TruSmartCAM).

Conexión de licencia a la Nube : 7567

Este puerto es usado por TecZone (Software de Doblado) y el Servidor de Licencia Flotante para enviar solicitudes de licenciamiento a Metamation Cloud. Por lo tanto, se debe permitir la conexión saliente en este puerto para activaciones y renovaciones de licencias.

Requisitos de Instalación y Soporte

  • Se requieren privilegios administrativos para instalar la aplicación TruSmartCAM.

  • Herramientas de diagnóstico remoto a través de VPN de Contractor o una conexión segura.
