Server installation

Siga las instrucciones para instalar la instalación del Server de TruSmartCAM junto con SQL.

Derechos de Administrador

Para ejecutar esta instalación, el usuario debe tener privilegios administrativos.

  • Ejecute Setup.TruSmartCAM.<version>.exe para comenzar la instalación.

Version es un número.

  • Lea el acuerdo de licencia y elija I accept the agreement y luego Haga clic en Siguiente .

Paso 1 : Ubicación de Instalación

  • Mencione el directorio de instalación o acepte el predeterminado de instalación C:\Program Files\Metamation\Praxis y haga clic en Siguiente.

    • C:\Program Files\Metamation\Praxis - La carpeta de instalación tendrá las siguientes carpetas : Bin, MetaCAM, Samples y Tracker.

    • C:\Program Data\Metamation - La Carpeta de Datos tendrá las siguientes carpetas : TruSmartCAM, TecZone y MetaCAM.

Paso 2 : Selección de Componentes

Tipo de Instalación Componentes

Full

Server + Desktop client + Engine

Client

Desktop Client

Server

Server + Desktop client

Station

Selecciona todas las Terminales de Estaciones como TecZone, MetaCAM, RightAngle(Controlador de Doblado), Vulcan(Controlador de Máquina Láser)

Custom

Selecciona por elección del usuario.

  • Elija Full Installation. Seleccione Code Senders y Bend Code Senders de TruSmartCAM Tools. Ahora el tipo de Instalación cambia a Personalizar. Haga clic en Siguiente.

Paso 3 : Elegir Instalar SQL Server

La instalación de SQL Express 2022 es necesaria solo para el Server de TruSmartCAM. Esta opción no está disponible para instalaciones de Client y Sync. También esto está Marcado ON e instalado por primera vez y se puede ignorar después. Si ya está SQL instalado en el servidor de TruSmartCAM, el usuario puede ignorar el Paso 3.

  • En Select Additional Tasks : Haga clic en Install SQL Express y Haga clic en Siguiente.

  • En Ready to Install : Resume todas las características de instalación y Haga clic en Install.

Paso 4 : Instalación de SQL Server

Una vez que los componentes de TruSmartCAM están instalados, el instalador procede con la instalación del servidor SQL. Acepte el Acuerdo de Licencia para proceder e instalar SQL express con las opciones predeterminadas.

Una vez completada la instalación Haga clic en Close y complete la instalación.

Dependiendo de la versión de Windows, cerrar la instalación de SQL puede solicitar un reinicio.

Paso 5 : Accesos directos de escritorio de Windows

  • Después de la instalación de SQL, si el Sistema no se reinicia, entonces seleccione la casilla de verificación Launch TruSmartCAM y Haga clic en Finish.

  • Si el Sistema se reinicia, entonces Lance el icono de acceso directo de TruSmartCAM en el Escritorio o escriba TruSmartCAM en Windows Start Program y Run as administrator.

Instalación del servidor : Los iconos de acceso directo de TruSmartCAM y TruSmartCAM License están disponibles en el Escritorio. + Instalación Client/SYNC : Solo el icono de acceso directo de TruSmartCAM está disponible en el Escritorio.

Paso 6 : Registro de Licencia

  • En el campo Serial Number, escriba la clave serial de TruSmartCAM proporcionada por el Equipo de Servicio de Trumpf Metamation. Escriba <Nombre de Empresa> en el campo "Registered to" y E-Mail. Haga clic en OK.

En caso de que el firewall de Red bloquee el registro de licencia o No haya internet en la PC del Server de TruSmartCAM, entonces la licencia de TruSmartCAM se realiza mediante activación Offline. Envíe por correo electrónico el archivo Prax.request desde la carpeta C:\ProgramData\Metamation\Praxis al Equipo de Servicio de Trumpf Metamation. El Equipo de Trumpf Metamation proporcionará el archivo Prax.lock que debe colocarse en esta carpeta.

Paso 7 : Configuración inicial única

  • Escriba el Nombre de la Base de Datos y Establezca la Ubicación de Almacenamiento de Datos. Seleccione la casilla de verificación Usar servidor SQL y seleccione el nombre del servidor SQL para conectarse a la Base de Datos. Haga clic en OK.

  • El Cliente de Escritorio de TruSmartCAM se abre mostrando Database Connection Succeeded en la parte inferior.

  • Los archivos de Configuración de TruSmartCAM, TecZone y MetaCAM fueron creados con máquinas de configuración predeterminadas o contra las máquinas de licencia de TruSmartCAM. La misma configuración se compartirá en todos los Clientes de TruSmartCAM.

La pantalla de bienvenida muestra el LoginName de Usuario de Windows como Login y Name de TruSmartCAM. El primer Usuario de TruSmartCAM se considera como Admin del lugar de emplazamiento. Proporcione Email y Contraseña. Haga clic en OK .

Seleccione la casilla de verificación Recuérdame para guardar las credenciales de inicio de sesión para la próxima vez. Desmarcar la casilla de verificación Recuérdame solicitará el diálogo de inicio de sesión y contraseña cada vez que se lance la aplicación TruSmartCAM.

Paso 8 : Inicio de sesión en TruSmartCAM

  • Cierre TruSmartCAM y vuelva a abrirlo. El sistema le solicita que ingrese las credencias de inicio de sesión de TruSmartCAM.

  • El nombre de usuario registrado se muestra junto al icono de usuario en el encabezado de la aplicación TruSmartCAM.

  • Cuando coloca el cursor sobre el icono de usuario, el sistema muestra:

    • El rol de usuario (por ejemplo: Programmer, Admin).

    • Un acceso directo para cerrar sesión: Ctrl+L.

  • Al hacer clic en el icono de usuario se abre un menú que le permite:

    • Editar información…​ - Actualizar los detalles de su perfil.

    • Modificar contraseña…​ - Cambiar la contraseña de su cuenta.

    • Cerrar sesión - Cerrar sesión de la aplicación.

    • Cancelar - Cerrar el menú sin hacer cambios.

Paso 9 : Configuración de Máquina de Factory

  • Licencia de TruSmartCAM [Todas las máquinas habilitan el bit de licencia] : La Máquina y el Material predeterminados de TruSmartCAM se consideran como Configuración de Factory.

  • Licencia de TruSmartCAM [Sin demo y algunos bits de máquina habilitados] : Esas Pocas máquinas y el Material predeterminado se consideran como Configuración de Factory.

  • (Open → Import) una nueva Part desde C:\Program Files\Metamation\Praxis\Samples\Parts y verifique si la Part se importa exitosamente.

Nota 1 : Monitor de TruSmartCAM

  • Monitor de TruSmartCAM con Instalación de Engine (derechos de administrador del sitio):

    • El Monitor de TruSmartCAM con Engine junto con Derechos de Administrador del Sitio de TruSmartCAM tendrá las siguientes opciones:

      1. Push de configuración de TruSmartCAM, configuración de TecZone, configuración de MetaCAM en todos los Clientes de TruSmartCAM. También tiene derechos para crear Nuevo Login de Usuario de TruSmartCAM.

      2. Show Agents mostrará el total de engine disponibles para realizar tareas de automatización que están contra la Licencia de TruSmartCAM.

  • Monitor de TruSmartCAM sin Instalación de Engine (derechos de administrador del sitio):

  • Monitor de TruSmartCAM sin Instalación de Engine (derechos de no administrador del sitio):

La PC que ejecuta la aplicación TruSmartCAM debe tener instalado el componente Engine de TruSmartCAM [PC del Server o en cualquier instalación del Client]. La PC con el componente Engine de TruSmartCAM instalado debe estar ejecutándose 24x7 para realizar todas las tareas de automatización.

Nota 2 : Lock Tracker

  • Haga clic en el acceso directo TruSmartCAM License disponible en el Escritorio del Server de TruSmartCAM. Alternativamente, abra la url http://localhost:7171/metamation/locktrack en cualquier navegador web.

Esto abrirá el módulo Lock Tracker que tiene información de Licencia, información de sesión actual de TruSmartCAM y Log.

Página HOME de LockTracker : Esto muestra el Número de Serial de TruSmartCAM, información de Expiración del Serial y información de bits de Serial disponibles.

Página SESSIONS de LockTracker : Información de conexión del Server de TruSmartCAM actual como PC cliente de Escritorio de TruSmartCAM, Estado de PMonitor, PEngine.

Página LOGs de LockTracker : Inicio y cierre de sesión del cliente TruSmartCAM | Inicio de Pmonitor y estado de conexión/desconexión de PEngine, etc., se pueden ver en esta página de Logs.

Nota 3 : OpenGL

  • La aplicación TruSmartCAM mientras se abre se cierra inmediatamente y también pmonitor no se muestra en la barra de tareas.

  • Esto representa un problema de OpenGl.

  • Otra forma de confirmar esto: Abra C:\Program Files\Metamation\Praxis\Bin\Flux.exe que muestra un diálogo de error sobre el problema de OpenGL.

  • Actualice C:\ProgramData\Metamation\Praxis\Prax.ini, sección de opciones con GLVersion=1 hace que la aplicación TruSmartCAM se abra exitosamente.

[Options]
GLVersion=1.0
