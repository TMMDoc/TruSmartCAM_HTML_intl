Importar piezas

Antes de importar piezas, asegúrese de que los formatos de archivo sean compatibles con TruSmartCAM. Consulte formatos de archivo soportados para más detalles.

Si el material y el espesor no se asignan a la pieza después de importar, asigne material usando el panel de piezas.

Métodos de importación

Hay tres métodos para importar piezas a la Biblioteca de Piezas:

  1. Importar mediante archivo

  2. Arrastrar y soltar

  3. Carpeta vigilada

Importar mediante archivo

Un archivo de pieza 2D o 3D puede importarse a la Biblioteca de Piezas usando la opción Importar piezas.

  • Haga clic en los iconos abrir desde la barra de comandos y seleccione la opción Importar piezas.

  • Aparece el diálogo Open — seleccione cualquier archivo CAD (2D o 3D).

  • Se muestra una vista previa del archivo CAD seleccionado.

  • Seleccione la pieza requerida y haga clic en Open.

  • El sistema provee herramientas automáticamente a la pieza para todas las máquinas disponibles con estado de herramientas actualizado.

Arrastrar y soltar

  • Arrastre y suelte un archivo de pieza directamente en la Biblioteca de Piezas de TruSmartCAM.

Carpeta vigilada

La configuración de vigilancia le permite monitorear una carpeta para archivos nuevos e importarlos automáticamente a la Biblioteca de Piezas.

Para configurar, consulte la página Configuración de vigilancia.

  • Use la ruta UNC (Convención de Nomenclatura Universal) \\192.168.1.11\<Ruta de carpeta> o una unidad mapeada M:\<Ruta de carpeta>.

  • La PC que ejecuta el Motor debe tener acceso de lectura/escritura a esta carpeta.

  • No use una ruta local como C:\<Ruta de carpeta> porque el servicio Watch busca archivos en la PC del Motor — no en la PC local.

Configuración de importación

Utilizar comparación de geometría en misma revisión CAD

Al importar una pieza con la misma revisión CAD, TruSmartCAM puede comparar la geometría del archivo CAD con la pieza existente.

  • Para habilitar esto, configure Teclas de búsqueda de revisión (por ejemplo, REV) en fábrica → ajustes →Importar/Exportar. Luego, habilite la opción Utilizar comparación de geometría en misma revisión CAD.

La comparación de geometría en la misma revisión CAD funciona solo cuando Teclas de búsqueda de revisión están configuradas.

  • Ahora, importe una pieza a la biblioteca de piezas con una revisión específica (por ejemplo, Revisión 1).

  • La misma pieza se importa nuevamente con el mismo valor de revisión.

  • TruSmartCAM compara la geometría del archivo CAD recién importado con la pieza existente.

  • Si se detecta una diferencia de geometría, TruSmartCAM muestra un mensaje preguntando si desea:

    • Sustituir la pieza existente con la nueva geometría, manteniendo el mismo valor de revisión.

    • Ignorar la importación, reteniendo la pieza existente y su geometría.

Esto permite a los usuarios actualizar la geometría de la pieza sin cambiar el valor de revisión, mientras se mantiene el control sobre si se reemplazan las piezas existentes.

Las piezas importadas se agregan automáticamente a la biblioteca para su uso posterior. Para más información, consulte Biblioteca.
