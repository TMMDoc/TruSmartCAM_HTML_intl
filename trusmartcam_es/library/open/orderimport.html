Importar pedidos

Los pedidos pueden crearse o importarse usando los siguientes métodos:

  1. Importar pedidos desde archivos de hoja de cálculo.

  2. Importar pedidos automáticamente mediante configuración de vigilancia.

  3. Crear pedidos desde piezas existentes en la biblioteca de piezas.

  4. Crear pedidos desde anidamientos estáticos de ensamblaje.

Importar pedidos desde hoja de cálculo

  • Navegue a abrir → Importar _pedidos…​.

  • Los formatos de hoja de cálculo compatibles incluyen CSV, XLSX y XML.

  • Seleccione el archivo de hoja de cálculo requerido y haga clic en abrir.

  • El proceso de importación es similar al flujo de trabajo de importación de trabajo/ensamblaje.

También puede arrastrar y soltar directamente la hoja de cálculo en la página Orders para importar pedidos.

Importar pedidos usando configuración de vigilancia

  • Habilite la opción Importar hojas de cálculo de 'Carpeta en directo' en Visualizar ajustes.

  • Seleccione Importar pedidos en la Opción de importación de hoja de cálculo.

  • Configure la ubicación de carpeta desde la cual se deben importar los pedidos.

  • El sistema monitorea continuamente la carpeta configurada.

  • Los archivos de hoja de cálculo recién agregados se importan automáticamente a Orders.

Para pasos de configuración detallados, consulte la documentación de Configuración de vigilancia .

Crear pedidos desde biblioteca de piezas

  • Seleccione las piezas requeridas de la Biblioteca de Piezas.

  • Haga clic derecho y elija Nuevo…​ → Pedido para abrir el diálogo Nuevos pedidos.

  • Ingrese los detalles requeridos como cantidad, fecha de vencimiento y prioridad.

  • Haga clic en OK para crear los pedidos.

  • El sistema cambia automáticamente a la página Orders.

  • Los pedidos creados se muestran en la página Orders Ready para creación de trabajo.

Crear pedidos desde ensamblaje

  • Haga clic derecho en el ensamblaje requerido.

  • Seleccione Nuevo…​ → Pedido para abrir el diálogo Nuevos pedidos.

  • Ingrese el Recuento de montaje requerido.

  • Haga clic en OK para crear los pedidos.
