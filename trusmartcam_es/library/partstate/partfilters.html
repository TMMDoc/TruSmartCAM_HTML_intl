Part filters

Part filters

The Part Library Filters let you quickly search for and locate parts in the library by applying specific conditions. By narrowing results to only the parts that meet your criteria, these filters make it easier to manage and access relevant parts efficiently.

Estado

  • Use the Estado filter to search parts based on tooling status.

  • The dropdown displays tooling states as colored boxes and all available machines and their applicable errors.

  • The colored boxes represent the following states:

    • White: Tooling OK

    • Orange: Tooling has error

    • Yellow: Tooling has warning

    • Grey: Tooling suppressed

  • If a part has narrow flange or tooling not assigned errors, selecting these will display the respective parts.

  • Parts with suppressed tooling on the selected machine will be displayed by clicking on the grey colored box.

Plegar montajes

  • Use the Plegar montajes filter to search parts based on the bend tool used.

  • The dropdown shows all available bend tools and can be filtered by tool name, type (punch, gooseneck, radius, etc.) or description.

  • Select multiple tools and click Combinar todo|&** to sort the tools which use both the selected tools.

Útiles usados

  • Filtering by Útiles usados works similar to filtering by Plegar montajes.

  • Tool icons are shown in proportional sizes (not exact, but relative).

  • When the tooling preview is open, hovering the mouse over a tool in the filter list highlights that tool in the preview pane.

  • Multiple tools can be selected and applied using Combinar todo|&**.

Estado

  • Use the Estado filter to search parts based on their current state.

  • The dropdown displays all applicable states depending on the library parts available.

  • The list of states changes dynamically according to the selected parts and their status.

  • For example, select Material Missing to view parts in an unresolved material state.
