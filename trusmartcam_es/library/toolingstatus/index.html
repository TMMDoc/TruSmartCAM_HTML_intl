Estado de herramientas

En TruSmartCAM, cada mosaico de pieza indica visualmente el estado de herramientas de esa pieza para todas las máquinas disponibles. Esto ayuda a los usuarios a ver rápidamente para qué máquinas una pieza tiene herramientas, y si las herramientas están listas, requieren revisión o faltan.

  • El estado de herramientas se muestra directamente en el mosaico de la pieza mediante iconos e indicadores de color.

  • Cada icono representa una tecnología de máquina (por ejemplo, Punzonado, Láser).

  • El color del icono indica el estado actual de las herramientas para esa máquina.

La siguiente tabla explica el significado de cada icono y color, ayudando a los usuarios a interpretar el estado de herramientas con precisión.

Errores de herramientas

Un error CAM (Fabricación Asistida por Computadora) ocurre durante la etapa de preparación de herramientas o fabricación. Estos errores surgen debido a problemas como colisiones de herramientas, herramientas faltantes o condiciones de corte inválidas.

Errores de herramientas de plegado

  • Colisiones detectadas y Error de pinza : La configuración de plegado causa colisiones de herramienta o sujetador durante la simulación.

  • Error de sobrecarga y Brida estrecha o perforación demasiado cerca : Las herramientas de plegado están sobrecargadas, u orificios están ubicados demasiado cerca de la línea de plegado, lo que puede dañar la pieza o las herramientas.

  • Necesita revisión y Duración de punzonado/estampado corta : La configuración de herramientas necesita inspección manual, o la longitud de matriz/punzón seleccionada es más pequeña de lo requerido para la pieza.

  • Utillaje no asignado, Tope trasero insuficiente, o Pieza demasiado grande para esta máquina : Las herramientas de plegado requeridas no están disponibles, el tope trasero no está configurado correctamente, o la pieza es demasiado grande para la máquina seleccionada.

Errores de herramientas de corte

  • Falta requisito previo de corte : La condición de corte no está mapeada para la materia prima, por lo que no se puede realizar el corte.

  • Contornos no equipados o solo parcialmente : Algunas formas o bordes se pierden o solo están parcialmente cubiertos durante el proceso de herramientas, haciendo la pieza incompleta para el corte.
