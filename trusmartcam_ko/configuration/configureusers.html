사용자 구성

  • 다양한 역할을 가진 TruSmartCAM 클라이언트는 공장 구성을 제어하는 데 사용됩니다.

    • 새 기계, 재료 및 시트 추가.

    • 기계 구성 편집.

    • 부품, 작업 및 어셈블리 편집.

    • 네스트 시트 편집/업데이트, 작업 계획.

    • 사용자 추가/편집.

  • Pserver 설치 PC는 첫 번째 TruSmartCAM 사용자로 간주됩니다.

  • TruSmartCAM는 Windows 자격 증명을 TruSmartCAM 사용자 이름으로 사용하며, 이는 사이트 관리자 [최상위 사용자 권한]입니다.

  • 다음 설치, 즉 TruSmartCAM 클라이언트 설치 컴퓨터 사용자 이름은 프로그래머로 간주됩니다.

사이트 관리자만이 다른 사용자 정보를 편집하고 암호를 변경할 수 있습니다.
