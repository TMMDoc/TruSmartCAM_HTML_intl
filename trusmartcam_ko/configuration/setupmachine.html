기계 추가/제거

편집 명령을 사용하여 기계 선택 대화 상자를 시작합니다. 현재 선택 항목과 함께 모든 절단, 벤딩 기계의 계층적 목록을 표시합니다.

  • 계층적 트리 뷰에서 간략한 기계 이름이 기계 레이블로 표시되고 전체 이름이 툴팁에 표시됩니다. 폴더 툴팁은 경로를 표시합니다(예: Trumpf • C Series • C60)

  • 비Trumpf 프레스 브레이크의 경우 브랜드별 아이콘이 표시됩니다. 기계 툴팁에는 기계 브랜드도 표시됩니다(예 - Brand: Bystronic).

  • 검색 상자를 사용하여 이름, 전체 이름, 브랜드 등으로 기계를 검색할 수 있습니다. 검색 중에도 기계 선택이 유지되므로 동시에 선택하고 검색할 수 있습니다.

  • 검색을 사용하여 기계 기술, 크기로도 검색할 수 있습니다. 따라서 검색 상자에 'fold’를 입력하면 모든 폴딩(패널 벤더) 기계가 표시됩니다. 마찬가지로 10ft를 입력하면 10ft 테이블 길이의 모든 프레스 브레이크가 표시됩니다(이는 단위에 민감합니다. 미터법 단위의 경우 m을 사용하십시오). 이름에서 사용 가능한 경우 축 수도 검색에 사용할 수 있습니다(예: 2A, 5A 등)

  • 모두 리셋 명령은 기계 선택을 데모 구성으로 재설정합니다.
