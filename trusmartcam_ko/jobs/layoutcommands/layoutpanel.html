레이아웃 패널

이 섹션에서는 기계로 전송…​, 기계에서 불러오기…​, 출력물 내보내기, 완료로 표시…​, 파트들 추가…​, 적용/최적화…​, 레이아웃들 분할…​ 및 레이아웃 인스턴스를 관리하는 데 사용되는 기타 옵션과 같은 레이아웃 관련 작업에 대해 설명합니다.

보기용 읽기 전용으로 열기
선택한 레이아웃을 읽기 전용 모드로 엽니다.

Check-out the document for editing
선택한 레이아웃의 편집을 활성화합니다.

삭제
선택한 레이아웃을 삭제합니다.

보고서 보기
선택한 레이아웃에 대한 표준 레이아웃 보고서 및 라벨별 보고서를 엽니다.

기계로 전송…​
선택한 레이아웃을 기계로 보내고 출력 파일(fxlyt, NC, report, csv)을 작업 출력 위치로 내보냅니다.

파트들 추가…​
레이아웃 재포장 마법사를 사용하여 제대로 포장되지 않은 레이아웃에 더 많은 부품을 추가합니다.

적용/최적화…​
선택한 레이아웃을 다시 네스팅하여 부품을 압축하고 사용 가능한 영역을 확보합니다.

LTT 변경…​
선택한 레이아웃에 할당된 시트를 변경합니다.

레이아웃들 분할…​
선택한 레이아웃을 두 개 이상의 개별 레이아웃으로 분할합니다.

Rename layout…​
선택한 레이아웃의 이름을 변경합니다.

통합 작업 / 이전 / 다음 / 파트들
상위 작업으로 이동하거나 선택한 작업 내에서 이전 또는 다음 레이아웃으로 이동하거나 레이아웃에 포함된 부품으로 이동합니다.

Sort and filter
이러한 필터를 사용하여 선택한 레이아웃과 동일한 속성을 공유하는 다른 레이아웃을 볼 수 있습니다: Same Machine, Same Material, Same LTT, Same Priority, Same State, Same Sheet, Same Part, Same Remnant State 또는 Same Rolling direction.

기계로 전송…​

  • 네스트가 준비되면(채워지거나 긴급하게 필요한 경우) 기계로 보낼 수 있습니다.

  • 네스트를 마우스 오른쪽 버튼으로 클릭하고 *기계로 전송…​*를 선택합니다.

  • 네스트 이름을 따서 자동으로 폴더가 생성됩니다.

  • 출력 파일(fxlyt, NC, report, csv)이 JOB 출력 위치로 내보내집니다.

  • 기계로 보낸 네스트는 파란색 체크 표시와 함께 표시됩니다.

릴리스된 네스트는 더 이상 재포장을 위해 재시도되지 않습니다.

기계에서 불러오기…​

  • 출력에 유효성 검사 또는 업데이트가 필요한 경우(예: 더 많은 부품 추가):

    • 내보낸 레이아웃을 마우스 오른쪽 버튼으로 클릭합니다.

    • *기계에서 불러오기…​*를 선택합니다.

  • 출력 파일과 폴더(fxlyt, NC, report, csv)가 삭제됩니다.

  • 네스트를 다시 편집할 수 있게 됩니다.

  • 변경한 후 다시 기계로 보낼 수 있습니다.

출력물 내보내기

  • 기계로 보낸 레이아웃(네스트)을 마우스 오른쪽 버튼으로 클릭하고 *출력물 내보내기*를 클릭합니다.

  • Export Outputs 옵션은 NC, Report 및 Layout 파일을 각 기계 출력 폴더로 내보냅니다.

  • 이 옵션은 레이아웃의 현재 단위 설정으로 NC 파일을 다시 생성합니다.

Export Outputs는 완료된 레이아웃 및 재포장 대기열에서 대기 중인 레이아웃에서는 사용할 수 없습니다.

Mark complete

  • 네스트가 기계에서 처리되면 네스트를 마우스 오른쪽 버튼으로 클릭하고 *완료로 표시…​*를 선택합니다.

  • 여러 네스트를 선택하고 동시에 완료로 표시할 수 있습니다.

  • 네스트가 완료로 표시되면:

    • 회색으로 표시되고 목록 끝으로 이동하며 녹색 체크 표시와 함께 표시됩니다.

    • 레이아웃이 활성 레이아웃 목록에서 제거됩니다.

    • 부품 및 작업 상태가 업데이트되고 처리된 수량이 완료 상태로 이동합니다.

    • 해당 레이아웃에 대한 출력 파일이 기계 출력 위치에서 제거되어 중복 사용을 방지합니다.

파트들 추가…​

  • 이 옵션을 사용하여 상대적으로 제대로 포장되지 않은 레이아웃에 더 많은 부품을 추가합니다. 대화형 네스팅과 마찬가지로 이것은 마법사 기반 옵션이며 단일 또는 여러 레이아웃에 동시에 사용할 수 있습니다.

  • 이 옵션을 사용하면 TruSmartCAM가 메인 작업 영역에 Layout repack wizard를 마운트합니다. Ctrl 키를 누른 상태에서 레이아웃에 포장할 하나 이상의 부품을 선택하고 *다음*를 누릅니다.

  • 선택한 부품이 편집 가능한 수량 열과 함께 다음 단계에 표시됩니다. 필요한 경우 Update Qty 필드에서 부품 수량을 업데이트합니다. *다음*를 클릭하여 네스팅으로 진행합니다. 수량은 증가만 가능하고 감소할 수 없습니다.

  • 모든 레이아웃이 개별적으로 네스팅되고 네스팅된 결과가 새로 포장된 부품과 함께 표시됩니다. 레이아웃을 선택하면 기존의 모든 레이아웃 부품이 표시됩니다. 재포장 중에 새 부품이 배치되지 않으면 결과가 자동으로 삭제됩니다. 여러 복사본이 있는 레이아웃은 재포장 후 다른 패턴을 생성할 수 있습니다.

적용/최적화…​

  • 이 마법사 기반 명령을 사용하여 원래 기계가 다운되었거나 기계 작업 부하의 균형을 맞춰야 할 때 레이아웃을 한 기계에서 다른 기계로 조정할 수 있습니다.

  • 계획한 시트가 재고가 없고 레이아웃을 사용 가능한 다른 시트에서 다시 계획해야 하는 경우 사용할 수 있습니다.

  • 재료를 더 효율적으로 사용하기 위해 더 작은 시트에 포장된 레이아웃을 더 큰 시트로 병합할 수 있습니다.

  • 큰 시트 재고를 사용할 수 없는 경우 더 큰 시트에 포장된 레이아웃을 더 작은 시트로 분할할 수 있습니다.

  • 여러 레이아웃(다른 기계의 것도 포함)을 함께 병합하여 포장 효율성을 높일 수 있습니다.

  • 사용하려면 하나 이상의 레이아웃을 선택하고 *적용/최적화…​*를 클릭하여 작업 영역에서 마법사를 엽니다.

  • 왼쪽 창에는 선택한 모든 레이아웃이 표시되고 Parts 창에는 레이아웃 부품이 표시됩니다.

  • 대상 기계와 시트 재료를 선택한 다음 *다음*를 클릭하여 재네스팅을 수행합니다.

  • 새 네스트 결과가 결과 창에 표시됩니다.

  • *다음*를 클릭하여 조정되거나 압축된 레이아웃을 저장합니다.

Splitting a layout

레이아웃들 분할…​ 옵션을 사용하면 툴링이나 네스팅을 다시 실행하지 않고 수량을 수정하여 기존 레이아웃을 여러 레이아웃으로 분할할 수 있습니다.

이 기능은 다음 시나리오에서 특히 유용합니다:

  1. 총 수량의 일부를 즉시 생산하고 나머지 수량을 나중에 생산하도록 예약해야 하는 경우.

  2. 워크플로우 및 리소스 활용을 최적화하기 위해 생산 요구 사항을 여러 교대 근무 또는 날짜에 분산해야 하는 경우.

  3. 특정 기계를 일시적으로 사용할 수 없거나 사용 중이며 현재 생산 제약 조건을 수용하도록 레이아웃을 조정해야 하는 경우.

레이아웃을 분할하려면 다음 단계를 수행합니다:

  • Layouts 보기에서 필요한 레이아웃을 선택합니다.

  • *레이아웃들 분할…​*를 클릭합니다.

  • *새 복사본*에 필요한 값을 입력합니다.

  • *OK*를 클릭합니다.

선택한 레이아웃이 이제 조정된 수량으로 두 개의 레이아웃으로 분할됩니다.
