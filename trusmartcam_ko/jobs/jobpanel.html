작업 패널

이 섹션에서는 Export outputs, 통합 작업 리셋…​, 완료로 표시…​, 통합 작업 내보내기 중…​ 및 작업 인스턴스를 관리하는 데 사용되는 기타 옵션과 같은 작업 관련 작업에 대해 설명합니다.

보기용 읽기 전용으로 열기
선택한 작업을 읽기 전용 모드로 엽니다.

Check-out the document for editing
선택한 작업의 편집을 활성화합니다.

삭제
선택한 작업을 삭제합니다.

보고서 보기
선택한 작업에 대한 생산 보고서를 엽니다.

출력물 내보내기
선택한 작업에 대해 생성된 기계 준비 출력 파일(NC 파일 및 보고서)을 내보냅니다.

Plan job
선택한 작업의 부품에 대한 네스팅된 레이아웃을 생성합니다.

통합 작업 리셋…​
작업에 대해 생성된 모든 레이아웃을 제거하고 부품을 계획 가능한 상태로 재설정합니다.

완료로 표시…​
생산 후 작업과 모든 네스트를 완료로 표시합니다.

통합 작업 내보내기 중…​
재사용하거나 다른 시스템으로 가져올 수 있는 패키지로 전체 작업을 내보냅니다.

이전 / 다음 / 레이아웃
선택한 작업 내에서 이전 또는 다음 레이아웃으로 이동합니다.

Sort and filter
이러한 필터를 사용하여 동일한 재료, 고객, 납기 또는 우선 순위를 공유하는 다른 작업을 볼 수 있습니다.

출력물 내보내기

  • 필요한 작업을 선택합니다.

  • *출력물 내보내기*를 클릭합니다.

  • 선택한 작업에 대해 생성된 기계 준비 출력 파일을 내보냅니다.

  • Export Outputs는 *공장 설정*에 구성된 기본 위치 또는 *찾아보기…​*를 사용하여 사용자 정의 위치에 저장할 수 있습니다.

  • 작업에 대한 기계 출력 파일(NC 및 보고서 등)만 내보냅니다.

  • 이러한 파일은 생산에 직접 사용되며 기계로 보낼 수 있습니다.

통합 작업 리셋…​

생성된 모든 레이아웃을 제거하고 부품을 다시 계획할 수 있도록 하여 선택한 작업을 재설정합니다.

작업을 재설정하려면:

  • 작업을 마우스 오른쪽 버튼으로 클릭하고 *통합 작업 리셋…​*를 선택합니다.

  • 선택한 작업에 대해 기존의 모든 네스팅된 레이아웃을 삭제하고 모든 부품을 계획 가능한 상태로 재설정합니다.

  • 레이아웃에 다른 작업의 부품이 포함되어 있으면 해당 부품도 재설정됩니다.

통합 작업 내보내기 중…​

  • 필요한 작업을 선택합니다.

  • *통합 작업 내보내기 중…​*를 클릭합니다.

  • 이 옵션은 재사용하거나 다른 시스템으로 가져올 수 있는 패키지로 전체 작업을 내보냅니다.

  • 내보낸 데이터에는 작업을 다시 만드는 데 필요한 모든 정보가 포함됩니다:

    • 부품 세부 정보 및 수량 (CSV 파일).

    • 레이아웃 및 네스팅 데이터 (CSV / FXYLT 파일).

  • 내보낸 파일은 구성된 내보내기 위치에 작업 폴더로 저장됩니다.

  • 이 옵션은 전체 작업을 전송, 공유 또는 백업하는 데 사용됩니다.

완료로 표시…​

  • 작업이 기계에서 처리되면 Job panel을 열고 *완료로 표시…​*를 선택합니다.

  • 작업이 완료로 표시되면:

    • 작업 아래의 모든 네스트가 완료로 표시됩니다.

    • 작업이 회색으로 표시되고 목록 끝으로 이동하며 녹색 체크 표시와 함께 표시됩니다.

    • 작업 아래의 모든 레이아웃이 활성 레이아웃 목록에서 제거됩니다.

    • 부품 및 작업 상태가 업데이트되고 처리된 수량이 완료 상태로 이동합니다.

    • 작업의 모든 레이아웃에 대한 출력 파일이 기계 출력 위치에서 제거되어 중복 사용을 방지합니다.
