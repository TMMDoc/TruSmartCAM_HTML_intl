작업 필터

작업 페이지 필터는 선택한 조건과 일치하는 항목만 표시하여 작업, 활성 부품 및 네스트를 찾는 데 도움이 됩니다. 작업이나 부품을 찾고, 정렬하고, 관리하는 것을 단순화하여 워크플로우의 전반적인 효율성을 향상시킵니다.

사용 가능한 필터의 예

우선순위 및 마감일

  • 우선순위 필터를 사용하면 할당된 우선 순위(긴급, 높음, 낮음, 보통)를 기반으로 부품 또는 작업을 검색하고 표시할 수 있습니다.

  • 특정 우선 순위 수준을 선택하면 해당 특정 우선 순위가 있는 부품만 강조 표시되거나 결과에 표시됩니다.

  • 예를 들어 높음 Priority를 선택하면 시스템은 네스트에서 High Priority로 표시된 부품만 필터링하고 강조 표시합니다.

  • 마감일 필터를 사용하면 할당된 날짜를 기준으로 부품 또는 작업을 검색하고 표시할 수 있습니다.

  • 특정 납기를 선택하면 해당 특정 납기가 있는 부품만 강조 표시되거나 결과에 표시됩니다.

  • 오늘, 어제, 내일 등과 같은 사전 정의된 납기 필터 외에도 yymmdd 형식으로 검색 텍스트 상자에 사용자 정의 날짜 값/범위를 입력할 수도 있습니다.

  • 날짜 값 뒤에 부등식 연산자 (<, <=, >, >=, =)가 있는 표현식을 사용할 수도 있습니다. 예: >=250901은 2025년 9월 1일 또는 그 이후에 만료되는 모든 항목을 검색합니다.

필터 목록이 검색 콘텐츠를 숨기면 투명도 토글|*마우스 오른쪽 버튼 클릭* 명령을 사용하여 투명하게 만들 수 있습니다.

파트 이름/ Layout name

  • 파트 이름 필터는 작업 및 네스트 페이지에서 사용할 수 있습니다.

  • 부품 이름을 입력하면 올바른 부품을 빠르게 선택하는 데 도움이 되는 제안 목록이 표시됩니다.

  • 제안 목록에서 하나 이상의 항목을 선택하면 검색 결과가 선택한 부품으로만 좁혀집니다.

  • 작업 페이지에서 이 동작은 레이아웃 이름을 포함하도록 확장되어 사용자가 부품 이름 또는 레이아웃 이름으로 검색할 수 있습니다.

잔재

  • 잔재 필터를 사용하여 잔재 가용성에 따라 레이아웃을 검색합니다.

  • 드롭다운에 다음 옵션이 표시됩니다:

    • 잔재 없음 - 잔재가 생성되지 않은 레이아웃을 나열합니다.

    • 존재함 - 잔재가 이미 존재하는 레이아웃을 나열합니다.

    • 추가됨 - 새 잔재가 추가된 레이아웃을 나열합니다.

  • 남은 시트 재료를 효율적으로 식별하고 관리하는 데 도움이 됩니다.
