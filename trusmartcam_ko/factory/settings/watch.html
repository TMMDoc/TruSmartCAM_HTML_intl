Watch

Watch Settings를 사용하면 TruSmartCAM이 지정된 폴더를 모니터링하고 파일을 시스템으로 자동으로 가져올 수 있습니다. 이를 통해 수동 개입 없이 Parts, Jobs, Assemblies 및 Orders의 가져오기를 자동화할 수 있습니다.

감시 설정

'라이브 폴더' 활성화 - 지정된 폴더의 자동 모니터링을 활성화합니다.

이 폴더 감시 - 들어오는 파일을 모니터링할 폴더를 지정합니다.

하위 폴더들도 감시 - 모니터링 프로세스에 모든 하위 폴더를 포함합니다.

파일 가져오기 후

가져오기 후 - 파일을 가져온 후 사용자에게는 두 가지 옵션이 있습니다. 파일을 위치에서 삭제하거나 다른 위치로 이동할 수 있습니다.

  • Delete imported file - 성공적으로 가져온 후 소스 파일을 제거합니다.

  • Move imported file - 성공적으로 가져온 후 소스 파일을 *Move files to this folder*에 지정된 폴더로 이동합니다.

파일들을 이 폴더로 이동 - *After import*가 *Move imported file*로 설정된 경우 성공적으로 가져온 파일을 이동할 대상 폴더를 지정합니다.

오류가 있는 스프레드시트를 이 폴더로 이동 - 실패한 가져오기를 지정된 오류 폴더로 이동합니다.

스프레드시트 가져오기 설정

'라이브 폴더’에서 스프레드시트 가져오기 - 감시 폴더에서 감지된 스프레드시트 파일을 자동으로 가져옵니다.

오류가 있는 스프레드시트 가져오기 안 함 - 유효성 검사 오류가 포함된 파일을 가져오지 않도록 방지합니다.

스프레드시트 가져오기 옵션 - 드롭다운에서 가져올 데이터 유형을 선택합니다. 지원되는 스프레드시트 형식에는 CSV, XLSX 또는 XML이 포함될 수 있습니다. 사용 가능한 옵션은 다음과 같습니다:

  • 통합 작업들 가져오기 - 스프레드시트에서 작업 데이터를 가져옵니다.

  • 오더들 가져오기 - 스프레드시트에서 주문 데이터를 가져옵니다.

  • 조립품들 가져오기 - 스프레드시트에서 어셈블리 데이터를 가져옵니다.

  • 파트들 가져오기 - 스프레드시트에서 부품 데이터를 가져옵니다.

Additional information

  • 감시 폴더를 구성할 때 UNC 경로 (\\ServerName\FolderPath) 또는 매핑된 드라이브 `(M:\FolderPath)`를 사용하십시오. `C:\FolderPath`와 같은 로컬 경로를 사용하지 마십시오.

  • TruSmartCAM Engine 서비스를 실행하는 컴퓨터에는 구성된 감시 폴더에 대한 읽기 및 쓰기 액세스 권한이 있어야 합니다. 충분한 권한이 없으면 파일이 감지, 가져오기, 이동 또는 성공적으로 삭제되지 않을 수 있습니다.

  • Watch Settings를 활성화하기 전에 스프레드시트 매핑을 구성해야 합니다. 그렇지 않으면 TruSmartCAM이 감시 폴더에서 감지된 스프레드시트 파일을 처리할 수 없습니다.
