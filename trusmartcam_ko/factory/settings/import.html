Import

출력 설정

전개도 저장 - 가져온 부품의 전개 패턴을 저장하려면 이 옵션을 활성화하십시오.

전개도 형식 - 이 옵션은 DXF 또는 GEO 파일 형식을 선택하는 데 사용됩니다.

전개도 출력 대상 - 전개 패턴이 저장될 위치입니다.

가져오기 설정

DXF 파일용 단위들 - DXF 파일을 가져올 측정 단위이며, 사용자는 mm 또는 inch 중에서 선택할 수 있습니다.

재료 조회 키 - 쉼표 구분 기호를 사용하여 여러 조회 항목을 할당할 수 있습니다. 부품을 가져올 때 TruSmartCAM은 이러한 키를 사용하여 재료 값을 검색합니다. 검색된 값은 원자재를 확인하는 데 사용됩니다. 원래 값은 부품 Misc Properties에 저장됩니다.

리비전 조회 키 - 부품을 가져올 때 원시 부품 파일(가져오는 부품 파일)의 개정 값이 리포지토리의 기존 개정 값과 비교됩니다. 개정이 동일하면 가져오기가 중단됩니다. 이제 DXF 파일에 대해서도 사용자 정의 필드 매핑이 지원됩니다. DXF 파일의 경우 사용자 정의 필드 데이터는 도면의 텍스트 레이어 또는 PPI 첨부 파일에서 추출됩니다.

동일 CAD 리비전에서 형상 비교 사용 - 동일한 CAD 개정으로 부품을 가져올 때 형상을 비교하고 차이가 감지되면 사용자에게 가져오기를 교체하거나 건너뛸지 묻습니다. 자세한 내용은 Import configuration을 참조하십시오.

파트 이름 키 - 이 옵션은 재료 및 리비전 조회 키와 유사하게 작동합니다. 파일 이름 대신 PPI 파트 이름을 부품 이름으로 사용하려면 하나 이상의 파트 이름 키를 설정하십시오.

두께 조회 키 - 쉼표 구분 기호를 사용하여 여러 조회 항목을 할당할 수 있습니다. 부품을 가져올 때 TruSmartCAM은 이러한 키를 사용하여 두께 값을 검색합니다. 검색된 값은 두께를 확인하는 데 사용됩니다. 원래 값은 부품 Misc Properties에 저장됩니다.

두께 비교 허용 오차 - 이것은 부품을 가져올 때 원자재를 조회하는 데 사용됩니다. TruSmartCAM은 재료 이름과 두께가 구문 분석된 후 원자재를 확인하려고 시도합니다. 자세한 내용은 아래 Thickness fudge 주제를 참조하십시오.

전개도 가져오기 후 형상들 식별 - 부품을 가져올 때 부품 내의 내부 형상을 자동으로 인식하고 식별할 수 있습니다.

형상 인식 허용 오차 - 형상 식별 허용 오차 값은 다음 위치에서 설정할 수 있습니다:

  1. Global level - 이 값은 모든 형상의 기본값으로 사용됩니다.

  2. Individual Shape Level - 개별 형상 수준에서 허용 오차 값을 설정하여 기본값을 재정의할 수 있습니다. Shape Library 대화 상자에서 형상을 선택하면 허용 오차 값이 표시됩니다. 기본적으로 이것은 0으로 설정되며 이러한 형상에는 기본 전역 값이 사용됩니다.

전개도 수평 회전 - 일반적으로 절단 기계 테이블은 직사각형이고 더 넓습니다. 툴링 가능성을 향상시키기 위해 부품을 가져온 후 부품 너비를 따라 부품을 회전시킵니다.

파트 회전 임계값 - 회전을 허용하는 최소 숫자로 설정합니다. 설정된 값보다 작은 부품은 회전되지 않습니다.

기타 설정

사전 가져오기 외부 툴들 활성화 - Creo Reader를 활성화하려면 사전 가져오기 외부 도구 가져오기 옵션을 활성화하십시오. TruSmartCAM은 삭제된 부품의 사전 추출된 패밀리 인스턴스를 업로드합니다. 인스턴스 추출은 부품 업로드 중에(사전 가져오기 작업으로) 수행되므로 부품을 업로드하는 컴퓨터에 Creo 소프트웨어가 설치되어 있어야 합니다.

가독성을 위해 전개도 썸네일의 도면 색상 무시 - 회색조의 더 선명한 썸네일을 생성하려면 이 옵션을 활성화하십시오. 이미지 선명도를 위해 레이어 색상 및 텍스트 엔터티는 무시됩니다.

모든 가져오기 과제들에 하나의 엔진만 사용 - 활성화하면 중요한 작업에 하나의 엔진 코어만 사용됩니다.

Thickness fudge

*두께 비교 허용 오차*는 재고에서 시트의 두께 값을 비교하여 원하는 시트 두께를 재고에서 사용할 수 없을 때 가장 가까운 시트 두께 값을 선택합니다.

예를 들어 다음을 가정해 봅시다:

  1. 시트 두께가 1.6 mm이고 재료가 강철인 부품이 있습니다.

  2. 두께 비교 허용 오차 값은 0.1 mm입니다.

  3. 재고에는 두께가 1.5 mm, 1.55 mm 및 1.7 mm인 강철 시트가 있습니다.

TruSmartCAM은 두께 비교 허용 오차 값(0.1 mm)을 가져온 항목(1.6 mm 측정)과 비교한 후 가장 가까운 두께 값을 가진 시트를 선택합니다. 이 경우 1.55 mm가 선택됩니다.
