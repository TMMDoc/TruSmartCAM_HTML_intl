출력 설정

NC 레이아웃 저장 - 레이아웃 NC 및 보고서를 저장하려면 이 옵션을 활성화하십시오.

레이아웃 파일 및 보고서들 저장 - 레이아웃 파일을 저장하려면 이 옵션을 활성화하십시오.

시뮬레이션 미리보기 보고서들 출력 - 다른 출력과 함께 툴링 시뮬레이션 미리보기를 생성하려면 이 옵션을 활성화하십시오. 이것은 레이저/펀치 툴링을 위한 짧은 GIF 애니메이션입니다.

레이아웃/통합 작업 완료 시 프로그램 출력물을 삭제하지 마십시오 - 이 옵션을 활성화하면 작업/레이아웃이 완료로 표시될 때 프로그램 출력에 대한 삭제 작업을 건너뜁니다.

네스팅 설정

여러 통합 작업에서 파트들 혼합 - 혼합 모드 네스팅을 지원합니다. 더 나은 패킹 효율을 위해 여러 작업의 부품을 함께 네스팅해야 합니다.

레이아웃들을 자동 해제하지 마십시오 - 이것은 모든 레이아웃이 릴리스 상태가 될 때까지 프로덕션에 레이아웃을 사용할 수 없도록 중지합니다.

재네스팅 패킹 마감 시한 - 이 값은 네스팅된 레이아웃을 승인 대기열에 보관하고 새 부품이 계획될 때 다시 네스팅할지 또는 프로덕션에 릴리스하고 관련 출력을 출력 폴더로 내보낼지 결정하는 데 사용됩니다.

인터랙티브 시트 이름 접두사 - 이 섹션은 사용자가 네스팅되는 부품과 네스트 결과를 더 잘 제어할 수 있는 대화형 네스팅을 지원하는 데 사용됩니다. 대화형으로 네스팅된 레이아웃에 대해 자동 네스팅과 다른 명명 체계를 사용합니다.

네스팅 확정/취소 시 시트 재고 업데이트 - 이 옵션이 활성화되면 네스트를 기계로 보낼 때 시트 재고가 업데이트됩니다. 또한 기계에서 네스트를 회수하면 시트가 재고로 반환됩니다.

자동 잔재 재고 관리 - 이 스위치를 선택하면 소프트웨어가 잔재 재고 관리를 자동화합니다. 선택하지 않으면 잔재 관리가 대화형이 됩니다.

최신이 아닌 레이아웃 자동 새로고침 - 선택하면 소프트웨어는 절단 솔루션이 계산된 후 부품과 연결된 모든 최신이 아닌 레이아웃들를 새로 고칩니다.

트윈라인 네스팅 자동화 - 트윈 라인 자동화를 활성화하려면 이 옵션을 선택하십시오.

풀 워크플로 설정

'풀 워크플로' 사용 - 이 옵션은 고도로 자율적인 계획 및 네스팅 시스템이 필요한 사용자에게 유용합니다. 활성화하면 TruSmartCAM이 주문형으로 네스팅을 계획합니다.

스테이션당 네스팅 수 - 이 값은 스테이션당 생성해야 하는 레이아웃 수를 설정하는 데 사용됩니다.

네스팅당 최대 시트 수 - 이 값은 고유 네스트에 대한 최대 시트 수를 제어하는 데 사용됩니다. 예를 들어 대량(예: 500개)으로 필요한 큰 부품(전체 시트를 채우는)이 있는 주문이 있는 경우 - TruSmartCAM은 더 이상 풀 모드에서 500개의 복사본이 있는 단일 네스트를 생성하지 않습니다. 대신 10이 최대값인 경우 네스트당 10개의 복사본을 생성합니다.

스테이션당 벤딩 프로그램들 - 이 값은 스테이션당 생성해야 하는 벤드 프로그램 수를 설정하는 데 사용됩니다.

프로그램 출력물 제거 시 작업을 완료로 표시 - 프로그램 출력이 대상 출력 대상에서 제거될 때 레이아웃을 완료로 표시하려면 이 옵션을 활성화하십시오.

스프레드시트 가져오기 설정

CAD 조회 디렉터리 - TruSmartCAM은 부품 라이브러리에서 가져온 CSV에 언급된 부품 항목을 로드하려고 시도합니다. 그러나 부품이 부품 라이브러리에서 사용할 수 없는 경우 이 대화 상자에 설정된 디렉터리 순서에 따라 조회 디렉터리에서 부품을 가져오려고 시도합니다.

재귀적 CAD 검색 사용 - 연속 결과에 대해 검색을 반복하려면 이 옵션을 활성화하십시오.

누락된 CAD 파일들에 대한 빈 파트 생성 - 부품이 TruSmartCAM 라이브러리 또는 조회 디렉터리에 없는 경우 TruSmartCAM은 해당 사용할 수 없는 부품에 대해 빈 부품을 생성합니다.

스프레드시트에 오류가 없는 경우 새 통합 작업 대화상자 표시 안 함 - CSV 항목이 깨끗한 경우 즉 DB의 데이터 불일치가 없거나 모든 필드가 제대로 나열된 경우 TruSmartCAM은 작업 개요 창을 표시하지 않고 CSV를 가져오거나 삭제하는 즉시 작업을 처리합니다.

미가공 재료를 재료 + 두께보다 우선 사용 - 지원되는 스프레드시트 유형:

  1. Pattern 1: PartName, Quantity, DueDate, Priority, RawMaterial

  2. Pattern 2: PartName, Quantity, DueDate, Priority, Material, Thickness

  3. Pattern 3: PartName, Quantity, DueDate, Priority, Material, Thickness, RawMaterial

이 옵션을 선택하지 않으면 Pattern 1이 기본 스프레드시트 유형으로 간주됩니다. CSV 형식이 Pattern 3 유형인 경우 Pattern 3도 지원합니다.

이 옵션을 선택하고 CSV가 Pattern 2 유형인 경우 TruSmartCAM은 CSV의 Material 및 Thickness 필드에서 Raw Material을 확인합니다.

스프레드시트 출력 설정

통합 작업/레이아웃 완료 보고서 내보내기 (.csv)* - 레이아웃 또는 작업이 완료될 때 CSV 내보내기를 활성화하려면 이 옵션을 선택하십시오. 완료된 보고서는 Factory → Settings → Reports 설정에 지정된 Reports 대상에 저장됩니다.
