Parts

TruSmartCAM의 Part Library는 모든 가져온 부품이 상호 작용 타일로 표시되는 중앙 집중식 작업 공간 역할을 합니다. 이를 통해 사용자는 부품 세부 정보를 빠르게 확인하고, 기계 전체의 툴링 상태를 확인하고, 검색, 탐색 또는 부품별 정보 열기와 같은 작업을 수행할 수 있습니다.

부품 가져오기에 대한 자세한 내용은 Import parts를 참조하십시오.

  • 단축키 P를 사용하여 라이브러리 로 이동합니다.

  • Library 보기는 모든 가져온 부품을 타일로 표시합니다.

  • 각 타일은 다음을 표시합니다:

    • 부품 이미지.

    • 기본 부품 속성 — 이름, 크기, 재료, 두께 등.

    • 사용 가능한 공장 기계의 툴링 상태.

  • 보기는 실시간 데이터로 자동 새로 고침되며 F5를 눌러 언제든지 강제로 새로 고칠 수 있습니다.

  • 검색 상자를 사용하여 Part-Name 또는 Material로 부품을 검색할 수 있습니다:

    • 검색 상자 내부를 클릭하거나 Ctrl+F를 누릅니다.

    • 검색할 텍스트를 입력하면 TruSmartCAM이 목록을 실시간으로 업데이트합니다.

    • Esc를 눌러 검색 텍스트를 재설정합니다.

Part detail page

  • 타일을 두 번 클릭하거나 타일을 선택한 후 Enter를 누르면 보기가 세부 정보로 변경됩니다.

  • 부품을 클릭하여 기계 경로를 확인합니다. 경로 내에서 기계를 선택하면 기계 패널이 열리고 툴링을 검토하고 수정할 수 있습니다.

  • 또한 오른쪽의 세부 정보 페이지에는 툴링 매개변수 및 사용된 기계에 대한 정보가 표시됩니다.
