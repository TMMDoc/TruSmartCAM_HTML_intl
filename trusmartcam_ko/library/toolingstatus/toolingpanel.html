Tooling panel

이 섹션에서는 툴링 인스턴스를 관리하는 데 사용되는 출력물 내보내기, Check Tooling, 툴링 억제, 기본 전개도로 설정…​, 툴링 복사…​ 및 기타 옵션과 같은 툴링 관련 작업을 설명합니다.

보기용 읽기 전용으로 열기
선택한 툴링을 읽기 전용 모드로 엽니다.

Check-out the document for editing
선택한 툴링을 편집할 수 있도록 합니다.

출력물 내보내기
선택한 툴링 출력을 구성된 대상으로 내보냅니다.

Check Tooling
변경하지 않고 기존 툴링을 검증하고 검증이 실패하면 상태를 업데이트합니다.

툴링 복사…​
선택한 참조 툴링에서 펀치, 시퀀스 및 백 게이지 설정을 복사하고 가능한 경우 다른 솔루션에 적용합니다.

기본 전개도로 설정…​
절단 솔루션 생성을 위한 참조 플랫 크기로 선택한 벤드 또는 폴드 툴링을 표시합니다.

Re-tool…​
다른 툴링에 영향을 주지 않고 부품에 대해 선택한 툴링을 다시 처리합니다.

툴링 억제
추가 처리 및 출력 생성에서 제외되도록 선택한 툴링을 억제합니다.

이전 / 다음 / 파트
이전 또는 다음 툴링으로 이동하거나 상위 부품으로 돌아갑니다.

Sort and filter
빠른 참조를 위해 선택한 툴링과 유사한 특성을 공유하는 다른 툴링을 표시합니다.

출력물 내보내기

이 옵션은 부품 출력을 공장 설정 에 구성된 대상으로 내보냅니다. ( Bend Settings 및 Cut Settings 참조)

전체 부품이 아닌 선택한 툴링 작업과 연결된 툴링만 내보냅니다.

내보낸 파일은 작업 유형에 따라 다릅니다. Bend 및 Fold 작업의 경우 NC 파일 및 보고서가 생성됩니다. Cut 및 Punch 작업의 경우 부품 파일만 내보냅니다.

Check Tooling

  • CAM Settings 변경이 있거나 벤드 도구가 손실된 경우 부품을 다시 툴링할 필요가 없습니다.

  • 대신 Check Tooling을 사용하여 변경하지 않고 기존 툴링을 검증합니다.

  • 검증이 실패하면 해당 인스턴스의 툴링 상태가 업데이트되고 관련 출력 파일이 제거됩니다.

  • Check Tooling은 오류 무시 를 사용하여 이전에 수동으로 재정의된 오류가 있는 부품을 감지하고 필요한 경우 상태를 오류로 업데이트합니다.

툴링 복사…​

복사 툴링을 사용하면 한 솔루션의 벤드 툴링을 다른 솔루션에 빠르게 적용하여 시간과 노력을 절약할 수 있습니다.

이것은 선택한 참조 툴링(솔루션)에서 펀치, 시퀀스 및 백 게이지 설정을 복사하고 다른 모든 솔루션에 동일한 설정을 적용하려고 시도합니다.

  • 복사된 설정이 대상 기계에 대해 실행 가능한 경우 저장되고 사용됩니다.

  • 설정이 실행 가능하지 않은 경우 해당 기계의 원래 툴링은 변경되지 않은 상태로 유지됩니다.

예를 들어 다른 기계에 대해 다른 툴링이 있을 수 있습니다:

  • 여기서 Machine 5230SX 6A는 Gooseneck 펀치를 사용하고 Machine 5085X 6A B23은 Pointed 펀치를 사용합니다.

Machine 5230SX 6A에서 툴링 복사…​ 를 사용하면 Gooseneck 설정을 다른 모든 기계에 복사하려고 시도합니다.

동일한 펀치, 시퀀스 및 백 게이지 설정이 가능한 경우 적용됩니다. 복사된 설정을 사용할 수 없는 경우 원래 툴링은 변경되지 않은 상태로 유지됩니다.

복사 툴링을 적용한 후 Gooseneck 설정이 실행 가능한 다른 모든 기계에 성공적으로 적용되었습니다.

기본 전개도로 설정…​

기본 전개도로 설정…​ 기능은 절단 솔루션 생성을 위한 참조 플랫 크기로 사용되는 벤드 또는 폴드 툴링을 정의합니다. 툴링이 Base-Flat로 표시되면 TruSmartCAM은 해당 기계를 기반으로 부품의 플랫 크기를 업데이트하고 그에 따라 절단 솔루션을 다시 생성합니다. Base-Flat 툴링은 UI에서 파란색 테두리로 표시됩니다.

다른 벤드 기계는 동일한 부품에 대해 플랫 크기의 약간의 변형을 생성할 수 있습니다. 툴링을 Base-Flat로 설정하면 TruSmartCAM은 해당 기계의 플랫 크기를 마스터 참조로 사용하고 절단 솔루션을 자동으로 다시 계산합니다.

여기서 다양한 기계에서 동일한 부품에 대해 다른 플랫 크기를 볼 수 있습니다:

이 예에서 5130X 6A B23 툴링이 Base-Flat로 설정되고 업데이트된 플랫 크기가 부품 속성에 적용됩니다:

자동 base flat 선택

공장 설정 벤딩과정에서 전개도 보정 적용 이 활성화된 경우:

  • TruSmartCAM은 부품 가져오기 중에 벤드 툴링(최대 벤드 길이 테이블 포함)을 자동으로 Base-Flat로 설정합니다.

  • 선택한 벤드 툴링에 오류가 발생하거나 실행 가능하지 않은 경우 TruSmartCAM은 지능적으로 전환하고 다른 유효한 벤드 툴링을 Base-Flat로 설정합니다. 이렇게 하면 수동 개입 없이 플랫 크기 참조 및 절단 솔루션이 항상 일관되고 유효하게 유지됩니다.

향상된 base flat 관리

TruSmartCAM은 벤드 또는 폴드 툴링 편집을 저장할 때 Praxis에 저장 대화 상자에서 Base-Flat 옵션을 제공합니다. 이러한 옵션은 OK 상태의 툴링에 대해서만 나타나며 플랫 크기와 절단 솔루션이 정확하게 유지되도록 합니다.

기본 전개도(Base-Flat)로 설정하고 절단 프로그램 계산

이 옵션은 툴링을 처음으로 Base-Flat로 표시하고 해당 절단 솔루션을 자동으로 생성하는 데 사용됩니다.

다음과 같은 경우 나타납니다:

  • 툴링이 현재 Base-Flat로 표시되지 않음.

  • 사용자가 편집을 위해 툴링을 체크 아웃하고 변경 사항을 저장 중.

이 옵션을 사용하려면 벤드 또는 폴드 툴링을 체크 아웃하고 벤드 매개변수 또는 플랫 크기 변경과 같은 필요한 수정을 수행합니다. Ctrl+S를 눌러 변경 사항을 저장합니다. Praxis에 저장 대화 상자에서 기본 전개도(Base-Flat)로 설정하고 절단 프로그램 계산 을 선택하고 OK 를 클릭합니다. TruSmartCAM은 툴링을 Base-Flat로 설정하고 업데이트된 플랫 크기를 사용하여 절단 솔루션을 생성합니다.

기본 전개도를 업데이트하고 절단 프로그램 재계산

이 옵션은 툴링이 이미 Base-Flat로 표시되고 편집되어 플랫 크기와 절단 솔루션을 다시 생성해야 하는 경우 사용됩니다.

다음과 같은 경우 나타납니다:

  • 툴링이 이미 Base-Flat로 표시됨.

  • 사용자가 이를 체크 아웃하고 편집한 후 변경 사항을 저장 중.

기존 Base-Flat 툴링을 편집할 때 필요에 따라 벤드 매개변수 또는 플랫 크기 형상을 업데이트합니다. Praxis에 저장 대화 상자에서 기본 전개도를 업데이트하고 절단 프로그램 재계산 을 선택합니다. TruSmartCAM은 Base-Flat 툴링을 업데이트하고 최신 변경 사항을 반영하도록 관련된 모든 절단 솔루션을 다시 생성합니다.

한 번에 하나의 벤드 툴링만 Base-Flat로 설정할 수 있습니다.

Base-Flat가 변경되면 기존 절단 솔루션이 자동으로 다시 처리됩니다.

이 기능은 TruSmartCAM 내의 벤드 및 폴드 툴링 모두에 사용할 수 있습니다.

다시 툴…​

이 옵션은 부품에 대해 선택한 툴링을 다시 처리합니다.

Tooling 패널에서 필요한 툴링을 마우스 오른쪽 버튼으로 클릭하고 다시 툴…​ 를 클릭합니다

계속하기 전에 확인 대화 상자가 표시됩니다.

예 를 클릭하여 다시 툴링을 계속하거나 아니오 를 클릭하여 작업을 취소합니다.

선택한 툴링만 다시 툴링됩니다. 부품과 연결된 다른 툴링은 영향을 받지 않습니다. 선택한 툴링과 관련된 기존 출력은 다시 툴링 프로세스 중에 제거되고 다시 생성됩니다.

툴링 데이터가 오래되었거나 잘못되었거나 기계 또는 툴링 구성이 변경된 경우 이 옵션을 사용합니다.

오류 무시

  • 툴링된 인스턴스에 경고가 표시되고 여전히 계속 진행하려는 경우 오류 무시 를 클릭하여 경고를 재정의할 수 있습니다.

  • 이렇게 하면 툴링 문제가 있어도 시스템이 해당 인스턴스에 대한 출력 파일을 내보낼 수 있습니다.

경고를 안전하게 무시할 수 있다고 확신하는 경우에만 *Ignore Errors*를 사용하십시오. 실제 오류를 재정의하면 기계 손상 또는 안전 위험이 발생할 수 있습니다.

Suppress/Unsuppress tooling

  • 기계가 비활성 상태이거나 특정 부품을 특정 기계에서 처리하지 않으려는 경우 툴링 억제 을 사용하여 해당 기계에서 해당 부품을 네스팅에서 제외합니다.

  • 억제 후 툴링된 인스턴스 위로 마우스를 가져가면 다음과 같은 메시지가 표시됩니다: "Tooling suppressed by <User name> on <Date Time>".

  • Suppress tooling은 해당 기계 및 부품에 대한 출력 파일을 제거합니다.

  • 툴링 억제 해제 을 사용하여 해당 기계에서 부품에 대한 툴링 상태를 복원합니다.

  • 억제 해제되면 해당 출력 파일이 다시 생성됩니다.
