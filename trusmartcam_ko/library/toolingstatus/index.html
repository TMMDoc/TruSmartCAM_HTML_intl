Tooling status

TruSmartCAM에서 각 부품 타일은 사용 가능한 모든 기계에 대한 해당 부품의 툴링 상태를 시각적으로 나타냅니다. 이를 통해 사용자는 부품이 어떤 기계에 대해 툴링되었는지, 툴링이 준비되었는지, 검토가 필요한지 또는 누락되었는지를 빠르게 확인할 수 있습니다.

  • 툴링 상태는 아이콘과 색상 표시기를 사용하여 부품 타일에 직접 표시됩니다.

  • 각 아이콘은 기계 기술(예: Punching, Laser)을 나타냅니다.

  • 아이콘의 색상은 해당 기계의 현재 툴링 상태를 나타냅니다.

아래 표는 각 아이콘과 색상의 의미를 설명하여 사용자가 툴링 상태를 정확하게 해석할 수 있도록 합니다.

툴링 오류

CAM (Computer-Aided Manufacturing) 오류는 툴링 또는 제조 준비 단계에서 발생합니다. 이러한 오류는 도구 충돌, 누락된 툴링 또는 잘못된 절단 조건과 같은 문제로 인해 발생합니다.

벤드 툴링 오류

  • 충돌 인식됨 및 그리퍼 오류 : 벤딩 설정이 시뮬레이션 중에 도구 또는 그리퍼 충돌을 발생시킵니다.

  • 과부하 오류 및 좁은 플랜지 또는 구멍 근접 : 벤드 도구가 과부하되거나 홀이 벤드 라인에 너무 가까이 있어 부품이나 도구가 손상될 수 있습니다.

  • 검토 필요 및 펀치/다이 길이 부족 : 툴링 설정에 수동 검사가 필요하거나 선택한 다이/펀치 스팬이 부품에 필요한 것보다 작습니다.

  • 툴링 할당되지 않음, 백게이지 불량 또는 Part too big for this machine : 필요한 벤딩 도구를 사용할 수 없거나 백게이지가 제대로 구성되지 않았거나 부품이 선택한 기계에 비해 너무 큽니다.

절단 툴링 오류

  • 절단 조건 누락 : 원자재에 대한 절단 조건이 매핑되지 않아 절단을 수행할 수 없습니다.

  • 툴링 미완료 또는 일부만 툴링된 컨투어들 : 일부 모양 또는 엣지가 누락되거나 툴링 프로세스 중에 부분적으로만 커버되어 부품이 절단에 불완전합니다.
