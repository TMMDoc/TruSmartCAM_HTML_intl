Tooling validation

이 섹션에서는 TruSmartCAM의 부품에 대한 툴링 정보에 액세스하고 수정하는 방법을 설명합니다.

  • 부품 타일을 마우스 오른쪽 버튼으로 클릭하여 부품 패널을 열고 툴링 옵션에 액세스합니다.

  • 툴링들 를 클릭하여 사용 가능한 첫 번째 툴링된 인스턴스로 전환합니다.

  • 또는 툴링된 인스턴스를 직접 클릭하여 기계 세부 정보 및 툴링 옵션을 확인합니다.

  • 를 사용하여 보기용 읽기 전용으로 열기 합니다.

  • 를 사용하여 편집을 위한 문서 체크아웃 합니다.

  • 선택한 툴링된 인스턴스가 TecZone 또는 MetaCAM에서 열립니다.

  • 수동 툴링 검사를 수행하고 필요한 변경을 수행합니다.

  • 저장하려고 하면 Unsaved Changes 대화 상자가 편집 내용을 확인하라는 메시지를 표시합니다.

  • Yes 를 클릭하여 저장을 계속합니다.

  • 시스템은 저장 후 Tooling Validation을 다시 수행합니다.

  • 오류가 없으면 TruSmartCAM에 변경 사항을 저장하는 옵션이 나타납니다.

  • 툴링 오류가 발견되면 오류 메시지가 표시됩니다:

  • 저장 후 부품 라이브러리에서 부품 툴링 상태가 업데이트됩니다.

  • 툴링된 인스턴스의 녹색은 체크인되었음을 나타냅니다.

  • Tooled Instance Author는 마지막으로 편집한 사람으로 업데이트됩니다.

  • 사용자 편집됨 필드는 툴링된 인스턴스가 수동으로 수정되었는지 여부를 나타냅니다. 사용자가 인스턴스를 체크 아웃하고 변경하면 저장 시 이 필드가 자동으로 Yes로 설정됩니다.

  • 부품이 현재 편집 중인 경우 타일에 사용자 아이콘이 나타납니다.

  • 타일 위로 마우스를 가져가면 누가 체크 아웃했는지와 체크 아웃한 시간이 표시됩니다.

  • 부품이 체크 아웃되어 있는 동안 동일한 툴링된 인스턴스에 대해 편집을 위한 문서 체크아웃 옵션이 비활성화되고 체크아웃 취소 옵션이 활성화됩니다.
