Import assemblies

어셈블리는 관련 부품을 하나의 단위로 결합한 그룹으로, 여러 부품을 하나의 엔티티로 함께 관리할 수 있습니다.

다음 방법을 사용하여 어셈블리를 가져올 수 있습니다:

  1. 파일을 통한 가져오기

  2. 드래그 앤 드롭

  3. 스프레드시트에서 가져오기

파일을 통한 가져오기

조립품들 가져오기 옵션을 사용하여 Part Library로 어셈블리를 가져올 수 있습니다.

  • 명령 표시줄에서 Open 아이콘을 클릭하고 조립품들 가져오기 를 선택하거나 _File → 열기 → 조립품들 가져오기 를 사용합니다.

  • Open 대화 상자가 나타납니다. 필요한 어셈블리 파일을 선택합니다.

  • 선택한 어셈블리의 미리 보기가 표시됩니다.

  • Open을 클릭하여 어셈블리를 가져옵니다.

  • 가져온 어셈블리는 Part Library의 Assemblies 보기에 나열됩니다.

드래그 앤 드롭

  • 어셈블리를 TruSmartCAM Part Library로 직접 드래그 앤 드롭합니다.

스프레드시트에서 가져오기

  • _File → 열기 → 조립품들 가져오기 를 사용하고 CSV, XLSX 또는 XML 파일을 선택하거나 파일을 드래그 앤 드롭합니다.

  • 가져오기 전에 어셈블리 세부 정보 및 수량을 검토하고 업데이트합니다. _Add…​ 또는 _Remove…​ 를 사용하여 수정합니다.

  • 가져오기 후 어셈블리를 준비하고 Static Nests를 할당할 수 있습니다.
