csv 또는 xlsx에서 가져오기

일반 텍스트 형식으로 작업을 가져올 수도 있습니다

  • File → Open → 통합 작업들 가져오기 를 클릭하면 Open Folder 대화 상자가 나타납니다. `C:\Program Files\Metamation\Praxis\Samples\Jobs\C0\C0.csv`를 선택합니다.

  • csv 위치 또는 조회 폴더 디렉터리에 모든 부품이 존재하면 csv 필드와 일치하는 모든 부품을 나열하는 새 통합 작업 창이 나타납니다.

  • 누락된 부품은 무시되며 새 통합 작업 창에 표시되지 않습니다.

  • 처음에 Part 상태는 Nascent입니다.

  • OK 를 클릭하면 부품이 Part Library로 가져옵니다. CAD 검증, bend 및 cut 툴링 인스턴스에 대한 툴링이 수행되며 Pmonitor 화면에서 모니터링할 수 있습니다.

누락된 CAD 파일들에 대한 빈 파트 생성

스프레드시트(.csv 또는 .xlsx)를 사용하여 작업을 가져올 때 시스템에서 사용할 수 없는 CAD 파일을 참조하는 행이 있을 수 있습니다.

  • 이를 처리하려면 공장 → 설정 → 통합 작업*로 이동하고 *스프레드시트 가져오기 에서 누락된 CAD 파일들에 대한 빈 파트 생성 를 활성화합니다

이 옵션을 선택하면:

  • 작업 가져오기 중에 CAD 파일이 누락된 경우 시스템은 누락된 CAD 파일 대신 빈 부품을 자동으로 생성합니다.

  • 새 통합 작업 창에서 이러한 누락된 부품은 관련 CAD 파일이 누락되었음을 나타내기 위해 노란색으로 시각적으로 강조 표시됩니다.

  • 누락된 부품에는 CSV의 매핑된 필드(예: 부품 이름, 재료, 두께, 수량 등)가 여전히 있습니다. 그러나 부품 상태는 Nascent로 표시되며 이는 실제 형상이 없는 빈 자리 표시자임을 의미합니다.

이렇게 하면 CAD 파일 누락으로 인한 오류 없이 작업 가져오기를 계속 진행할 수 있습니다 — 나중에 올바른 CAD 파일을 사용할 수 있게 되면 빈 부품을 업데이트하거나 교체할 수 있습니다.

스프레드시트에 오류가 없는 경우 새 통합 작업 대화상자 표시 안 함

공장 → 설정 → 통합 작업 → 스프레드시트 가져오기→ 스프레드시트에 오류가 없는 경우 새 통합 작업 대화상자 표시 안 함

  • 이 확인란을 선택하면 스프레드시트에 오류가 없는 경우(누락된 부품, 누락된 날짜 또는 잘못된 데이터 없음) 시스템이 New Job 대화 상자를 표시하지 않고 자동으로 새 작업을 생성합니다.

  • 이 확인란의 선택을 취소하면 스프레드시트에 오류가 없는 경우에도 사용자 검토 및 확인을 위해 New Job 대화 상자가 항상 나타납니다.

Prefer raw-material column over material + thickness

공장 → 설정 → 통합 작업→Prefer Raw-Material column over Material + Thickness

  • 이 확인란을 선택하면 시스템은 스프레드시트가 Raw-Material 열을 사용할 것으로 예상합니다(예: Steel-20). 이는 재료 유형과 두께를 하나의 열에 하나의 값으로 결합해야 함을 의미합니다.

  • 이 확인란의 선택을 취소하면 시스템은 스프레드시트에 Material 및 Thickness에 대한 별도의 열이 있을 것으로 예상합니다(예: Steel 및 2.00). 시스템은 가져오기 중에 내부적으로 결합합니다.

CSV/XLSX를 사용한 Watch folder

  • 공장 → 설정→ 감시 에서 '라이브 폴더' 활성화 및 하위 폴더들도 감시 를 확인합니다.

  • watch folder 경로(UNC 또는 매핑된 드라이브)를 설정합니다.

  • 스프레드시트 가져오기 에서 '라이브 폴더’에서 스프레드시트 가져오기 를 확인합니다.

  • Spreadsheet Import 옵션에서 통합 작업들 가져오기 를 선택합니다.

  • `C:\Program Files\Metamation\Praxis\Samples\Jobs\C0\C0.csv`를 복사하여 Watch Folder에 붙여넣습니다.

  • 시스템은 CSV를 확인하고 자동으로 작업을 생성하려고 시도합니다.

  • CAD 파일이 누락된 경우 필요한 CAD 데이터를 찾을 수 없으므로 작업이 생성되지 않습니다.

CAD lookup directory 및 recursive CAD search - 구성되지 않음

  • 공장 → 설정 → 통합 작업 → 스프레드시트 가져오기*에서 *CAD lookup directories 가 설정되지 않고 재귀적 CAD 검색 사용 가 사용되지 않으면 시스템은 CAD 파일을 자동으로 검색하지 않습니다.

  • 누락된 CAD 파일들에 대한 빈 파트 생성 확인란이 선택되어 있으면 CAD 파일이 누락되어도 작업이 생성됩니다.

  • 이 경우 작업의 모든 부품은 nascent(빈) 부품으로 생성됩니다 — CAD 파일을 찾을 수 없어서 형상이 없음을 의미합니다.

CAD lookup directory 및 recursive CAD search - 구성됨

CAD 조회 디렉터리 가 (C:\Program Files\Metamation\Praxis\Samples)로 설정되고 재귀적 CAD 검색 사용 가 공장 → 설정 → 통합 작업 → *스프레드시트 가져오기*에서 활성화된 경우:

  • 시스템은 스프레드시트에 언급된 필요한 CAD 파일에 대해 지정된 폴더 및 모든 하위 폴더를 자동으로 검색합니다.

  • 누락된 CAD 파일들에 대한 빈 파트 생성 도 활성화되어 있으므로 CAD 파일을 찾을 수 없는 경우 시스템은 자리 표시자로 빈 부품을 생성합니다.
