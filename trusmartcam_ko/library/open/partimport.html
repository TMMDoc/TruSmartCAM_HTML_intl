Import parts

부품을 가져오기 전에 파일 형식이 TruSmartCAM과 호환되는지 확인하십시오. 자세한 내용은 supported file formats를 참조하십시오.

가져오기 후 재료 및 두께가 부품에 매핑되지 않은 경우 부품 패널을 사용하여 재료를 할당하십시오.

가져오기 방법

Part Library로 부품을 가져오는 세 가지 방법이 있습니다:

  1. 파일을 통한 가져오기

  2. 드래그 앤 드롭

  3. Watch folder

파일을 통한 가져오기

파트들 가져오기 옵션을 사용하여 2D 또는 3D 부품 파일을 Part Library로 가져올 수 있습니다.

  • 명령 표시줄에서 열기 아이콘을 클릭하고 파트들 가져오기 옵션을 선택합니다.

  • Open 대화 상자가 나타납니다 — CAD 파일(2D 또는 3D)을 선택합니다.

  • 선택한 CAD 파일의 미리 보기가 표시됩니다.

  • 필요한 부품을 선택하고 *Open*을 클릭합니다.

  • 시스템은 업데이트된 툴링 상태와 함께 사용 가능한 모든 기계에 대해 부품을 자동으로 툴링합니다.

드래그 앤 드롭

  • 부품 파일을 TruSmartCAM Part Library로 직접 드래그 앤 드롭합니다.

Watch folder

Watch 설정을 사용하면 새 파일에 대한 폴더를 모니터링하고 자동으로 Part Library로 가져올 수 있습니다.

구성하려면 Watch settings 페이지를 참조하십시오.

  • UNC (Universal Naming Convention) 경로 \\192.168.1.11\<Folder path> 또는 매핑된 드라이브 `M:\<Folder Path>`를 사용하십시오.

  • Engine이 실행 중인 PC에는 이 폴더에 대한 읽기/쓰기 액세스 권한이 있어야 합니다.

  • `C:\<Folder Path>`와 같은 로컬 경로를 사용하지 마십시오. Watch 서비스는 로컬 PC가 아닌 Engine PC에서 파일을 찾기 때문입니다.

가져오기 구성

동일 CAD 리비전에서 형상 비교 사용

동일한 CAD 리비전으로 부품을 가져올 때 TruSmartCAM은 CAD 파일의 형상을 기존 부품과 비교할 수 있습니다.

  • 이를 활성화하려면 공장 → 설정 →가져오기/내보내기*에서 *리비전 조회 키 (예: REV)를 구성합니다. 그런 다음 동일 CAD 리비전에서 형상 비교 사용 옵션을 활성화합니다.

동일한 CAD 리비전에 대한 형상 비교는 리비전 조회 키 가 구성된 경우에만 작동합니다.

  • 이제 특정 리비전(예: Revision 1)으로 부품 라이브러리에 부품을 가져옵니다.

  • 동일한 부품을 동일한 리비전 값으로 다시 가져옵니다.

  • TruSmartCAM은 새로 가져온 CAD 파일의 형상을 기존 부품과 비교합니다.

  • 형상 차이가 감지되면 TruSmartCAM은 다음 중 하나를 선택하라는 프롬프트를 표시합니다:

    • 교체 동일한 리비전 값을 유지하면서 기존 부품을 새 형상으로 교체합니다.

    • 건너뛰기 가져오기를 건너뛰고 기존 부품 및 해당 형상을 유지합니다.

이를 통해 사용자는 리비전 값을 변경하지 않고도 부품 형상을 업데이트할 수 있으며 기존 부품을 교체할지 여부에 대한 제어를 유지할 수 있습니다.

가져온 부품은 추가 사용을 위해 자동으로 라이브러리에 추가됩니다. 자세한 내용은 Library를 참조하십시오.
