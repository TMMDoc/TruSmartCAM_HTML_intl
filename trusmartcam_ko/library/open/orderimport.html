Import orders

다음 방법을 사용하여 주문을 생성하거나 가져올 수 있습니다:

  1. 스프레드시트 파일에서 주문 가져오기.

  2. watch 설정을 통해 자동으로 주문 가져오기.

  3. 부품 라이브러리의 기존 부품에서 주문 생성.

  4. 어셈블리 정적 네스트에서 주문 생성.

스프레드시트에서 주문 가져오기

  • 열기 → Import _Orders…​ 로 이동합니다.

  • 지원되는 스프레드시트 형식에는 CSV, XLSX 및 XML이 포함됩니다.

  • 필요한 스프레드시트 파일을 선택하고 열기 을 클릭합니다.

  • 가져오기 프로세스는 Job/Assembly 가져오기 워크플로우와 유사합니다.

스프레드시트를 Orders 페이지로 직접 드래그 앤 드롭하여 주문을 가져올 수도 있습니다.

watch 설정을 사용하여 주문 가져오기

  • 감시 설정 에서 '라이브 폴더’에서 스프레드시트 가져오기 옵션을 활성화합니다.

  • 스프레드시트 가져오기 옵션 에서 오더들 가져오기 를 선택합니다.

  • 주문을 가져올 폴더 위치를 구성합니다.

  • 시스템은 구성된 폴더를 지속적으로 모니터링합니다.

  • 새로 추가된 스프레드시트 파일은 자동으로 Orders로 가져옵니다.

자세한 구성 단계는 Watch settings 문서를 참조하십시오.

부품 라이브러리에서 주문 생성

  • Part Library에서 필요한 부품을 선택합니다.

  • 마우스 오른쪽 버튼을 클릭하고 새로 만들기…​ → 오더 를 선택하여 새 오더들 대화 상자를 엽니다.

  • 수량, 마감일 및 우선 순위와 같은 필요한 세부 정보를 입력합니다.

  • OK 를 클릭하여 주문을 생성합니다.

  • 시스템이 자동으로 Orders 페이지로 전환됩니다.

  • 생성된 주문은 작업 생성을 위해 Orders Ready 페이지에 표시됩니다.

어셈블리에서 주문 생성

  • 필요한 Assembly를 마우스 오른쪽 버튼으로 클릭합니다.

  • 새로 만들기…​ → 오더 를 선택하여 새 오더들 대화 상자를 엽니다.

  • 필요한 조립품 수량 를 입력합니다.

  • OK 를 클릭하여 주문을 생성합니다.
