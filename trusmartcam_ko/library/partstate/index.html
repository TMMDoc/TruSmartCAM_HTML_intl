부품 상태는 부품 타일에 부품 이름 아래에 직접 표시되어 부품의 현재 검증 조건을 나타냅니다.

부품 상태 유형은 부품 수준 검증 검사의 결과로 부품이 가질 수 있는 다양한 조건을 설명합니다. 이러한 상태는 부품의 설계, 재료 또는 형상과 관련된 문제를 나타내며 툴링을 진행하기 전에 해결해야 합니다.

Part errors

A CAD (Computer-Aided Design) error occurs when there is an issue in the parts design or geometry, such as open contours, missing geometry, or unassigned material. These errors originate from the CAD model and prevent the part from being processed further.

  • 벤딩 실행 불가 : TruSmartCAM automatically tries to find a valid bending method for the part. If it cannot find any suitable bend solution, the part is marked as bend infeasible.

  • Cut infeasible : If bend warnings are ignored, TruSmartCAM then attempts to find a cutting method. If no valid cutting solution is found, the part is marked as cut infeasible.

Material errors

  • 재료가 존재하지 않습니다 : The part’s thickness is read from the CAD model and matched with materials in the TruSmartCAM Raw Material database. If no matching material is found, the part is marked with a 재료가 존재하지 않습니다 error.

  • 재료가 할당되지 않음 : If the CAD model does not have a material mapping field or if the field is empty, TruSmartCAM cannot assign a material. In this case, the part is marked with a 재료가 할당되지 않음 error.

Geometry errors

  • 열린 컨투어 인식됨 : The CAD file has one or more open (incomplete) shapes, which prevents proper tooling.

  • 다중 외측 컨투어 인식됨 : The CAD file has more than one outer closed loop, which creates confusion during processing.

  • Nascent error : When a part is loaded from a spreadsheet (CSV or XLSX), but the actual CAD file is missing, it is marked as a nascent part.

  • 형상 누락 : The CAD model has no valid geometry or the geometry is unreadable, making it unusable for tooling.
