서버 설치

SQL과 함께 TruSmartCAM 서버를 설치하려면 다음 지침을 따르십시오.

관리자 권한

이 설치를 실행하려면 사용자에게 관리자 권한이 있어야 합니다.

  • 설치를 시작하려면 *Setup.TruSmartCAM.<version>.exe*를 실행합니다.

Version은 숫자입니다.

  • 라이선스 계약을 읽고 *I accept the agreement*를 선택한 다음 다음 를 클릭합니다.

1단계 : 설치 위치

  • 설치 디렉터리를 지정하거나 기본 설치 경로인 `C:\Program Files\Metamation\Praxis`를 수락하고 다음를 클릭합니다.

    • C:\Program Files\Metamation\Praxis - 설치 폴더에는 다음 폴더가 포함됩니다: Bin, MetaCAM, Samples 및 Tracker.

    • C:\Program Data\Metamation - 데이터 폴더에는 다음 폴더가 포함됩니다: TruSmartCAM, TecZone 및 MetaCAM.

2단계 : 구성 요소 선택

설치 유형 구성 요소

Full

Server + Desktop client + Engine

Client

Desktop Client

Server

Server + Desktop client

Station

Picks all Stations Terminal like TecZone, MetaCAM, RightAngle(Bend Controller), Vulcan(Laser Machine Controller)

Custom

Picks by user choice.

  • Full Installation을 선택합니다. TruSmartCAM Tools에서 Code Senders 및 *Bend Code Senders*를 선택합니다. 이제 설치 유형이 *사용자 정의*로 변경됩니다. 다음를 클릭합니다.

3단계 : SQL Server 설치 선택

SQL Express 2022 설치는 TruSmartCAM 서버에만 필요합니다. 이 옵션은 Client 및 Sync 설치에서는 사용할 수 없습니다. 또한 처음 설치 시 선택되어 설치되며 이후에는 무시할 수 있습니다. TruSmartCAM 서버에 SQL이 이미 설치되어 있는 경우 사용자는 3단계를 무시할 수 있습니다.

  • Select Additional Tasks에서: *Install SQL Express*를 클릭하고 다음를 클릭합니다.

  • Ready to Install에서: 모든 설치 기능을 요약하고 Install을 클릭합니다.

4단계 : SQL Server 설치

TruSmartCAM 구성 요소가 설치되면 설치 프로그램은 SQL 서버 설치를 진행합니다. 라이선스 계약에 동의하여 진행하고 기본 옵션으로 SQL express를 설치합니다.

설치가 완료되면 Close를 클릭하고 설치를 완료합니다.

Windows 버전에 따라 SQL 설치를 닫으면 다시 시작하라는 메시지가 표시될 수 있습니다.

5단계 : Windows 바탕 화면 바로 가기

  • SQL 설치 후 시스템이 다시 시작되지 않으면 Launch TruSmartCAM 확인란을 선택하고 *Finish*를 클릭합니다.

  • 시스템이 다시 시작되면 바탕 화면에서 TruSmartCAM 바로 가기 아이콘을 시작하거나 Windows 시작 프로그램에서 TruSmartCAM을 입력하고 *관리자 권한으로 실행*합니다.

서버 설치 : TruSmartCAM 및 TruSmartCAM License 바로 가기 아이콘이 바탕 화면에서 사용 가능합니다. + Client/SYNC 설치 : TruSmartCAM 바로 가기 아이콘만 바탕 화면에서 사용 가능합니다.

6단계 : 라이선스 등록

  • Serial Number 필드에 Trumpf Metamation Service Team에서 제공한 _TruSmartCAM Serial key_를 입력합니다. "Registered to" 필드와 E-Mail에 *<Company Name>*을 입력합니다. *OK*를 클릭합니다.

네트워크 방화벽이 라이선스 등록을 차단하거나 TruSmartCAM 서버 PC에 인터넷이 없는 경우 오프라인 활성화를 통해 TruSmartCAM 라이선스가 수행됩니다. C:\ProgramData\Metamation\Praxis 폴더에서 *Prax.request*를 Trumpf Metamation Service Team에 이메일로 보냅니다. Trumpf Metamation Team은 이 폴더에 저장해야 하는 *Prax.lock*을 제공합니다.

7단계 : 초기 1회 구성

  • Database Name을 입력하고 Data Storage Location을 설정합니다. SQL 서버 사용 확인란을 선택하고 데이터베이스에 연결할 SQL 서버 이름을 선택합니다. *OK*를 클릭합니다.

  • TruSmartCAM Desktop Client가 열리고 왼쪽 하단에 *Database Connection Succeeded*가 표시됩니다.

  • TruSmartCAM, TecZone 및 MetaCAM 구성 파일이 기본 구성 머신 또는 TruSmartCAM 라이선스 머신에 대해 생성되었습니다. 동일한 구성이 모든 TruSmartCAM 클라이언트에서 공유됩니다.

환영 화면이 Windows User LoginName을 TruSmartCAM Login 및 Name으로 표시합니다. 첫 번째 TruSmartCAM 사용자는 사이트 관리자*로 간주됩니다. Email 및 Password를 입력합니다. *OK 를 클릭합니다.

다음 번에 로그인 자격 증명을 저장하려면 로그인 기억하기 확인란을 선택합니다. 로그인 기억하기 확인란을 선택 해제하면 TruSmartCAM 애플리케이션이 시작될 때마다 로그인 및 비밀번호 대화 상자가 표시됩니다.

8단계 : TruSmartCAM 로그인

  • TruSmartCAM을 닫고 다시 엽니다. 시스템에서 TruSmartCAM 로그인 자격 증명을 입력하라는 메시지가 표시됩니다.

  • 로그인한 사용자 이름이 TruSmartCAM 애플리케이션 헤더의 사용자 아이콘 옆에 표시됩니다.

  • 사용자 아이콘 위로 마우스를 가져가면 시스템에 다음이 표시됩니다:

    • 사용자 역할(예: Programmer, Admin).

    • 로그아웃 바로 가기: Ctrl+L.

  • 사용자 아이콘을 클릭하면 다음을 수행할 수 있는 메뉴가 열립니다:

    • 정보 편집…​ - 프로필 세부 정보를 업데이트합니다.

    • 비밀번호 변경…​ - 계정 비밀번호를 변경합니다.

    • 로그아웃 - 애플리케이션에서 로그아웃합니다.

    • 취소 - 변경 사항 없이 메뉴를 닫습니다.

9단계 : Factory Machine 구성

  • TruSmartCAM License [모든 머신 라이선스 비트 활성화] : 기본 TruSmartCAM Machine 및 Material이 Factory Configuration으로 간주됩니다.

  • TruSmartCAM License [데모 없음 및 일부 머신 비트 활성화] : 이러한 일부 머신 및 기본 Material이 Factory Configuration으로 간주됩니다.

  • `C:\Program Files\Metamation\Praxis\Samples\Parts`에서 새 Part를 (Open → Import)하고 Part가 성공적으로 가져와졌는지 확인합니다.

참고 1 : TruSmartCAM Monitor

  • Engine 설치가 포함된 TruSmartCAM Monitor (Site admin 권한):

    • Site Admin TruSmartCAM 권한이 있는 Engine이 포함된 TruSmartCAM Monitor에는 다음 옵션이 있습니다:

      1. 모든 TruSmartCAM 클라이언트에 TruSmartCAM 구성, TecZone, MetaCAM 구성을 푸시합니다. 또한 새 TruSmartCAM 사용자 로그인을 생성할 권한이 있습니다.

      2. Show Agents는 TruSmartCAM License에 대한 자동화 작업을 수행할 수 있는 총 엔진을 표시합니다.

  • Engine 설치가 없는 TruSmartCAM Monitor (Site admin 권한):

  • Engine 설치가 없는 TruSmartCAM Monitor (Non Site Admin 권한):

TruSmartCAM 애플리케이션을 실행하는 PC에는 Engine TruSmartCAM 구성 요소가 설치되어 있어야 합니다[서버 PC 또는 모든 클라이언트 설치]. Engine TruSmartCAM 구성 요소가 설치된 PC는 모든 자동화 작업을 수행하기 위해 24x7로 실행되어야 합니다.

참고 2 : Lock Tracker

  • TruSmartCAM 서버 바탕 화면에서 사용 가능한 TruSmartCAM License 바로 가기를 클릭합니다. 또는 웹 브라우저에서 http://localhost:7171/metamation/locktrack url을 엽니다.

이렇게 하면 라이선스 정보, TruSmartCAM 현재 세션 정보 및 로그가 있는 Lock Tracker 모듈이 열립니다.

LockTracker HOME Page : TruSmartCAM Serial Number, Serial 만료 정보 및 Serial 비트 사용 가능 정보를 표시합니다.

LockTracker SESSIONS Page : TruSmartCAM Desktop client PC, PMonitor, PEngine 상태와 같은 현재 TruSmartCAM 서버 연결 정보입니다.

LockTracker LOGs Page : TruSmartCAM 클라이언트 시작 및 로그오프 | Pmonitor 시작 및 PEngine 연결/연결 해제 상태 등을 이 로그 페이지에서 볼 수 있습니다.

참고 3 : OpenGL

  • TruSmartCAM 애플리케이션을 열 때 즉시 닫히고 pmonitor가 작업 표시줄에 표시되지 않습니다.

  • 이것은 OpenGl 문제를 나타냅니다.

  • 이를 확인하는 또 다른 방법: C:\Program Files\Metamation\Praxis\Bin\Flux.exe를 열면 OpenGL 문제에 대한 오류 대화 상자가 표시됩니다.

  • C:\ProgramData\Metamation\Praxis\Prax.ini, options 섹션을 *GLVersion=1*로 업데이트하면 TruSmartCAM 애플리케이션이 성공적으로 열립니다.

[Options]
GLVersion=1.0
