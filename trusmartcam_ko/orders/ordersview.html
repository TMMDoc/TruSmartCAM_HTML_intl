주문 보기

주문 페이지 오른쪽의 옵션은 처리 단계에 따라 주문의 다양한 보기를 나타냅니다. 보기를 선택하면 표시되는 주문이 업데이트되어 워크플로의 특정 단계에 집중할 수 있습니다.

Ready

Ready 보기를 선택하면 준비되었지만 아직 작업으로 변환되지 않은 주문이 표시됩니다.

  • 작업 생성 준비가 된 주문을 표시합니다.

  • 처리 대기 중인 주문을 식별하는 데 유용합니다.

Active

Active 보기를 선택하면 작업으로 변환되어 현재 처리 중인 주문이 표시됩니다.

  • 이미 작업 단계로 이동된 주문을 나타냅니다.

  • 실행 중인 주문을 추적하는 데 도움이 됩니다.

  • 진행 중인 생산을 모니터링하는 데 유용합니다.

Done

Done 보기를 선택하면 처리가 완료된 주문이 표시됩니다.

  • 관련 작업이 완료된 주문을 표시합니다.

  • 워크플로가 완료되었음을 나타냅니다.

  • 검토 및 기록 보관에 유용합니다.

All

All 보기를 선택하면 단계에 관계없이 모든 주문이 표시됩니다.

  • 모든 주문에 대한 완전한 개요를 제공합니다.

  • 모든 단계에서 검색 및 관리에 유용합니다.

주문 상태 표시줄

각 주문 타일 아래에 표시되는 상태 표시줄은 주문이 작업에 연결되어 있는지 여부를 나타냅니다. 두 가지 유형이 있습니다:

  • 표시줄 없음 - 주문이 작업에 할당되지 않았습니다.

  • 표시줄 있음 - 주문이 작업의 일부이며 표시줄은 아래 나열된 해당 작업 상태를 표시합니다.
