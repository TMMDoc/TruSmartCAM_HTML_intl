필수 요구 사항

SolidWorks를 처음 실행할 때 TruSmartCAM 데스크톱에 로그인하는 것이 필수입니다.

Windows 알림 영역에서 TruSmartCAM 모니터가 실행 중인지 확인하십시오.

TruSmartCAM 데스크톱이 열려 있지 않은 경우 TruSmartCAM 서버 서비스가 실행 중인지 확인하여 SolidWorks 추가 기능이 작동할 수 있도록 합니다. 두 가지 모두 비활성화된 경우 SolidWorks는 시작 시 오류 메시지를 표시합니다.

다음 단계에 따라 SolidWorks용 TruSmartCAM 추가 기능을 설치하고 설정하십시오:

  1. TruSmartCAM 설정을 실행하고 구성 요소 선택 페이지에서 *SolidWorks Extension*을 활성화합니다. (SolidWorks가 TruSmartCAM 클라이언트에 설치된 경우 서버 옵션을 건너뜁니다.)

  2. 설치를 완료한 후 TruSmartCAM 데스크톱을 실행하고 유효한 사용자 자격 증명으로 로그인합니다.

  3. SolidWorks를 실행합니다.

  4. SolidWorks에서 Tools → TruSmartCAM → *About TruSmartCAM*로 이동하여 추가 기능 버전 및 세부 정보를 확인합니다.

  5. 시작 시 추가 기능을 활성화하거나 비활성화하려면 Tools → Add-ins*로 이동하여 *Other Add-ins 아래에서 *TruSmartCAM*을 찾습니다.
