부품 저장

SolidWorks를 실행하고 판금 부품(.sldprt)을 엽니다.

그런 다음 다음 단계를 따르십시오:

  1. SolidWorks에서 TruSmartCAM 탭으로 전환하여 TruSmartCAM 도구 모음에 액세스합니다.

  2. TruSmartCAM 도구 모음에서 *Praxis에 저장*를 클릭합니다.

Praxis에 저장 대화 상자에서 적절한 TruSmartCAM 원자재를 선택한 다음 *OK*를 클릭하여 부품을 TruSmartCAM 부품 라이브러리에 업로드하고 저장합니다.

이제 부품이 TruSmartCAM에 저장됩니다.

TruSmartCAM에는 `C:\Program Files\Metamation\ TruSmartCAM \Samples\Parts`에 위치한 몇 가지 샘플 부품이 포함되어 있습니다. 필요한 경우 테스트 목적으로 사용할 수 있습니다.

자동 재료 해석

이 기능은 업로드 중에 SolidWorks 재료 및 부품 두께 값을 해당 TruSmartCAM 원자재로 자동으로 변환합니다.

재료 매핑을 구성하려면:

  1. TruSmartCAM 도구 모음에서 옵션*를 클릭하여 *옵션 대화 상자를 엽니다.

  2. 재료 매핑 페이지로 이동합니다.

  3. 왼쪽 창에서 SolidWorks 재료를 선택합니다.

  4. 오른쪽 창에서 해당 TruSmartCAM 재료를 선택합니다.

  5. *매핑*을 클릭하여 매핑을 생성합니다.

    • 매핑된 쌍이 하단의 매핑 목록에 나타납니다.

  6. 매핑을 제거하려면 선택하고 *매핑 해제*을 클릭합니다.

두께 오차 설정

두께 오차 값은 TruSmartCAM이 SolidWorks의 판금 모델 두께를 TruSmartCAM 데이터베이스의 기존 원자재 두께와 일치시키는 허용 오차 범위를 정의합니다. 이는 설계 반올림, 단위 변환 등으로 인한 모델 두께의 작은 변동을 보상합니다.

SolidWorks 부품 두께가 TruSmartCAM의 재료와 정확히 일치하지 않는 경우 오차 값을 사용하면 가장 가까운 사용 가능한 재료가 자동으로 선택됩니다.

예시

공칭 2.0 mm 재료 및 0.05 mm의 오차 값의 경우:

  • TruSmartCAM은 1.95 mm와 2.05 mm 사이의 모든 SolidWorks 모델 두께를 유효한 일치로 수락합니다.

  • 이 범위를 벗어난 편차는 부품 업로드 중에 수동 재료 선택 프롬프트를 트리거합니다.

허용 오차 값을 보거나 수정하려면:

  • TruSmartCAM 도구 모음에서 *옵션*를 클릭한 다음 *기타 설정*로 이동하여 허용 오차 값을 확인한 후 *Praxis에 저장*를 클릭합니다.

  • Praxis에 저장 대화 상자에서 *OK*를 클릭하면 부품이 TruSmartCAM에 저장됩니다.

사용자 정의 부품 속성 읽기

이 설정을 사용하면 TruSmartCAM이 SolidWorks 부품 파일에서 직접 Material, Revision 및 기타 사용자 정의 속성과 같은 사용자 정의 부품 정보를 자동으로 읽을 수 있습니다.

설정 방법:

  • TruSmartCAM 도구 모음에서 옵션*를 클릭하여 *옵션 대화 상자를 연 다음 *사용자 정의 필드들*로 이동합니다.

  • 매핑 및 매핑 해제 버튼을 사용하여 필요한 TruSmartCAM 필드를 해당 SolidWorks Custom 또는 Configuration 필드와 연결합니다.

표시되는 값은 현재 활성화된 SolidWorks 부품에서 로드됩니다.

  • 매핑이 완료되면 선택한 사용자 정의 속성이 부품을 TruSmartCAM으로 가져오거나 업데이트할 때마다 SolidWorks에서 자동으로 읽혀집니다. 이제 *Praxis에 저장*를 클릭합니다.

여기에 정의된 매핑은 TruSmartCAM 리포지토리에 저장되며 네트워크 내의 모든 TruSmartCAM SolidWorks 설치에서 자동으로 동기화됩니다.

  • Praxis에 저장 대화 상자에서 *OK*를 클릭하면 부품이 TruSmartCAM에 저장됩니다.
