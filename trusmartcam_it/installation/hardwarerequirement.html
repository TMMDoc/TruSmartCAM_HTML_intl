Requisiti porte

TruSmartCAM Server : 7371

Il server attende le richieste del client su questa porta TCP. Le connessioni in ingresso devono essere consentite su questa porta sul computer del server.

Floating License Server : 5776

Il Floating License Server attende le richieste di licenza su questa porta. Le connessioni in ingresso devono essere consentite su questa porta sul computer del server di licenza floating (Server TruSmartCAM).

Connessione licenza al Cloud : 7567

Questa porta viene utilizzata da TecZone (software di piegatura) e dal Floating License Server per inviare richieste di licenza al Cloud Metamation. Quindi la connessione in uscita deve essere consentita su questa porta per le attivazioni e i rinnovi delle licenze.

Requisiti di installazione e supporto

  • Sono necessari privilegi amministrativi per installare l’applicazione TruSmartCAM.

  • Strumenti diagnostici remoti tramite VPN Contractor o una connessione sicura.
