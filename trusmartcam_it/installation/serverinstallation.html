Server installation

Seguire le istruzioni per installare l’installazione del server TruSmartCAM insieme a SQL.

Diritti di amministratore

Per eseguire questa installazione, l’utente deve avere privilegi amministrativi.

  • Eseguire Setup.TruSmartCAM.<version>.exe per iniziare l’installazione.

La versione è un numero.

  • Leggere il contratto di licenza e scegliere I accept the agreement e quindi fare clic su Succ .

Passaggio 1 : Posizione di installazione

  • Menzionare la directory di installazione o accettare l’impostazione predefinita di installazione C:\Program Files\Metamation\Praxis e fare clic su Succ.

    • C:\Program Files\Metamation\Praxis - La cartella di installazione avrà le seguenti cartelle : Bin, MetaCAM, Samples e Tracker.

    • C:\Program Data\Metamation - La cartella dati avrà le seguenti cartelle : TruSmartCAM, TecZone e MetaCAM.

Passaggio 2 : Selezione componenti

Tipo di installazione Componenti

Full

Server + Desktop client + Engine

Client

Desktop Client

Server

Server + Desktop client

Station

Seleziona tutti i terminali delle stazioni come TecZone, MetaCAM, RightAngle(Bend Controller), Vulcan(Laser Machine Controller)

Custom

Seleziona per scelta dell’utente.

  • Scegliere Full Installation. Selezionare Code Senders e Bend Code Senders da TruSmartCAM Tools. Ora il tipo di installazione cambia in Customizza. Fare clic su Succ.

Passaggio 3 : Scegli Installa SQL Server

L’installazione di SQL Express 2022 è necessaria solo per il server TruSmartCAM. Questa opzione non è disponibile per le installazioni Client e Sync. Inoltre questa è selezionata ON e installata per la prima volta e può essere ignorata successivamente. Se SQL è già installato nel server TruSmartCAM, l’utente può ignorare il passaggio 3.

  • In Select Additional Tasks : Fare clic su Install SQL Express e fare clic su Succ.

  • In Ready to Install : Riepilogare tutte le funzionalità di installazione e fare clic su Install.

Passaggio 4 : Installazione di SQL Server

Una volta installati i componenti TruSmartCAM, il programma di installazione procede con l’installazione del server SQL. Accettare il contratto di licenza per procedere e installare SQL express con le opzioni predefinite.

Una volta completata l’installazione, fare clic su Close e completare l’installazione.

A seconda della versione di Windows, la chiusura dell’installazione di SQL potrebbe richiedere un riavvio.

Passaggio 5 : Collegamenti desktop di Windows

  • Dopo l’installazione di SQL, se il sistema non si riavvia, selezionare la casella di controllo Launch TruSmartCAM e fare clic su Finish.

  • Se il sistema si riavvia, avviare l’icona del collegamento TruSmartCAM sul desktop o digitare TruSmartCAM in Windows Start Program e Run as administrator.

Installazione server : TruSmartCAM e TruSmartCAM License Icone di collegamento sono disponibili sul desktop. + Installazione Client/SYNC : Solo l’icona di collegamento TruSmartCAM disponibile sul desktop.

Passaggio 6 : Registrazione licenza

  • Nel campo Serial Number, digitare la chiave seriale TruSmartCAM fornita da Trumpf Metamation Service Team. Digitare <Company Name> nel campo "Registered to" e E-Mail. Fare clic su OK.

In caso di firewall di rete che blocchi la registrazione della licenza o assenza di Internet nel PC del server TruSmartCAM, la licenza TruSmartCAM viene eseguita tramite attivazione offline. Inviare via email il file Prax.request dalla cartella C:\ProgramData\Metamation\Praxis al Trumpf Metamation Service Team. Il team Trumpf Metamation fornirà il file Prax.lock che deve essere rilasciato in questa cartella.

Passaggio 7 : Configurazione iniziale una tantum

  • Digitare il nome del database e impostare la posizione di archiviazione dei dati. Selezionare la casella di controllo Usare il server SQL e selezionare il nome del server SQL per connettersi al database. Fare clic su OK.

  • Il client desktop TruSmartCAM si apre mostrando Database Connection Succeeded in basso.

  • I file di configurazione TruSmartCAM, TecZone e MetaCAM sono stati creati con macchine di configurazione predefinite o contro le macchine di licenza TruSmartCAM. La stessa configurazione verrà condivisa tra tutti i client TruSmartCAM.

La schermata di benvenuto mostra il nome di accesso utente Windows come accesso e nome TruSmartCAM. Il primo utente TruSmartCAM è considerato come Amministratore del sito. Fornire email e password. Fare clic su OK .

Selezionare la casella di controllo Ricordami per salvare le credenziali di accesso per la prossima volta. Deselezionare la casella di controllo Ricordami richiederà la finestra di dialogo di accesso e password ogni volta che viene avviata l’applicazione TruSmartCAM.

Passaggio 8 : Accesso a TruSmartCAM

  • Chiudere TruSmartCAM e riaprirlo. Il sistema richiede di inserire le credenziali di accesso TruSmartCAM.

  • Il nome utente connesso viene visualizzato accanto all’icona utente nell’intestazione dell’applicazione TruSmartCAM.

  • Quando si passa il mouse sopra l’icona utente, il sistema visualizza:

    • Il ruolo utente (ad esempio: Programmer, Admin).

    • Un collegamento per disconnettersi: Ctrl+L.

  • Facendo clic sull’icona utente si apre un menu che consente di:

    • Modifica informazione…​ - Aggiornare i dettagli del profilo.

    • Modifica password…​ - Modificare la password dell’account.

    • Esci - Disconnettersi dall’applicazione.

    • Annulla - Chiudere il menu senza apportare modifiche.

Passaggio 9 : Configurazione delle macchine in fabbrica

  • Licenza TruSmartCAM [Tutte le macchine abilitano il bit di licenza] : Macchina e materiale predefiniti TruSmartCAM sono considerati come configurazione della fabbrica.

  • Licenza TruSmartCAM [Nessuna demo e pochi bit macchina abilitati] : Quelle poche macchine e il materiale predefinito sono considerati come configurazione della fabbrica.

  • (Open → Import) una nuova parte da C:\Program Files\Metamation\Praxis\Samples\Parts e verificare se la parte viene importata con successo.

Nota 1 : TruSmartCAM Monitor

  • TruSmartCAM Monitor con installazione Engine (diritti di amministratore del sito):

    • TruSmartCAM Monitor con Engine insieme ai diritti di amministratore del sito TruSmartCAM avrà le seguenti opzioni:

      1. Push configurazione TruSmartCAM, configurazione TecZone, MetaCAM tra tutti i client TruSmartCAM. Inoltre, ha i diritti per creare un nuovo accesso utente TruSmartCAM.

      2. Show Agents mostrerà il motore totale disponibile per eseguire attività di automazione che è contro la licenza TruSmartCAM.

  • TruSmartCAM Monitor senza installazione Engine (diritti di amministratore del sito):

  • TruSmartCAM Monitor senza installazione Engine (diritti non di amministratore del sito):

Il PC che esegue l’applicazione TruSmartCAM deve avere il componente Engine TruSmartCAM installato [PC server o in qualsiasi installazione client]. Il PC con componente Engine TruSmartCAM installato dovrebbe essere in esecuzione 24x7 per eseguire tutte le attività di automazione.

Nota 2 : Lock Tracker

  • Fare clic sul collegamento TruSmartCAM License disponibile sul desktop del server TruSmartCAM. In alternativa, aprire l’URL http://localhost:7171/metamation/locktrack in qualsiasi browser web.

Questo aprirà il modulo Lock Tracker che contiene informazioni sulla licenza, informazioni sulla sessione corrente di TruSmartCAM e log.

Pagina HOME di LockTracker : Questo mostra il numero di serie TruSmartCAM, la scadenza del seriale informazioni e informazioni sui bit seriali disponibili.

Pagina SESSIONS di LockTracker : Informazioni sulla connessione al server TruSmartCAM corrente come PC client desktop TruSmartCAM, PMonitor, stato PEngine.

Pagina LOGs di LockTracker : Avvio e disconnessione del client TruSmartCAM | Avvio di Pmonitor e stato di connessione/disconnessione PEngine ecc., possono essere visualizzati in questa pagina Logs.

Nota 3 : OpenGL

  • L’applicazione TruSmartCAM durante l’apertura viene chiusa immediatamente e anche pmonitor non viene mostrato nella barra delle applicazioni.

  • Questo rappresenta un problema OpenGl.

  • Un altro modo per confermarlo: Aprire C:\Program Files\Metamation\Praxis\Bin\Flux.exe che visualizza la finestra di dialogo di errore sul problema OpenGL.

  • Aggiornare C:\ProgramData\Metamation\Praxis\Prax.ini, sezione opzioni con GLVersion=1 per far aprire correttamente l’applicazione TruSmartCAM.

[Options]
GLVersion=1.0
