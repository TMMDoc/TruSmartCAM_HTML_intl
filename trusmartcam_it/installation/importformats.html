Supported file formats

TruSmartCAM supporta l’importazione diretta CAD. La disponibilità dei formati di importazione dipende dalla configurazione della licenza. I formati supportati sono classificati come segue.

L’elenco dei formati CAD supportati e la compatibilità delle versioni dei file dipendono dal livello di licenza attivo. Le versioni supportate vengono aggiornate periodicamente durante l’anno. Prima di aggiornare il software CAD a una release più recente, si consiglia di verificare l’elenco di compatibilità per garantire il supporto continuato.
