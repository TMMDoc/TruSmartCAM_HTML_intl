Importa pezzi

Prima di importare i pezzi, assicurarsi che i formati dei file siano compatibili con TruSmartCAM. Vedere supported file formats per maggiori dettagli.

Se il materiale e lo spessore non sono mappati al pezzo dopo l’importazione, assegnare il materiale utilizzando il pannello dei pezzi.

Metodi di importazione

Esistono tre metodi per importare i pezzi nella libreria dei pezzi:

  1. Importa tramite file

  2. Trascina e rilascia

  3. Cartella di monitoraggio

Importa tramite file

Un file di pezzo 2D o 3D può essere importato nella libreria dei pezzi utilizzando l’opzione Importa parti.

  • Fare clic sulle icone aperto dalla barra dei comandi e selezionare l’opzione Importa parti.

  • Viene visualizzata la finestra di dialogo Open — selezionare un file CAD qualsiasi (2D o 3D).

  • Viene visualizzata un’anteprima del file CAD selezionato.

  • Selezionare il pezzo richiesto e fare clic su Open.

  • Il sistema attrezzerà automaticamente il pezzo per tutte le macchine disponibili con stato di attrezzaggio aggiornato.

Trascina e rilascia

  • Trascinare e rilasciare un file di pezzo direttamente nella libreria dei pezzi TruSmartCAM.

Cartella di monitoraggio

Le impostazioni di monitoraggio consentono di monitorare una cartella per nuovi file e importarli automaticamente nella libreria dei pezzi.

Per configurare, vedere la pagina Watch settings.

  • Utilizzare il percorso UNC (Universal Naming Convention) \\192.168.1.11\<Folder path> o un' unità mappata M:\<Folder Path>.

  • Il PC su cui è in esecuzione il motore deve avere accesso in lettura/scrittura a questa cartella.

  • Non utilizzare un percorso locale come C:\<Folder Path> perché il servizio di monitoraggio cerca i file sul PC del motore — non sul PC locale.

Configurazione importazione

Usa il confronto geometrie sulla stessa revisione CAD

Quando si importa un pezzo con la stessa revisione CAD, TruSmartCAM può confrontare la geometria del file CAD con il pezzo esistente.

  • Per abilitare questo, configurare Chiavi di ricerca revisione (ad esempio, REV) in fabbrica → impostazioni →Importazione/esportazione. Quindi, abilitare l’opzione Usa il confronto geometrie sulla stessa revisione CAD.

Il confronto della geometria sulla stessa revisione CAD funziona solo quando sono configurate Chiavi di ricerca revisione.

  • Ora, importare un pezzo nella libreria dei pezzi con una revisione specifica (ad esempio, Revisione 1).

  • Lo stesso pezzo viene importato di nuovo con lo stesso valore di revisione.

  • TruSmartCAM confronta la geometria del file CAD appena importato con il pezzo esistente.

  • Se viene rilevata una differenza di geometria, TruSmartCAM visualizza un prompt chiedendo se:

    • Sostituisci il pezzo esistente con la nuova geometria, mantenendo lo stesso valore di revisione.

    • Salta l’importazione, mantenendo il pezzo esistente e la sua geometria.

Ciò consente agli utenti di aggiornare la geometria del pezzo senza modificare il valore di revisione, pur mantenendo il controllo su se i pezzi esistenti vengono sostituiti.

I pezzi importati vengono automaticamente aggiunti alla libreria per un ulteriore utilizzo. Per ulteriori informazioni, vedere Library.
