Importa lavori

Importa da csv o xlsx

È anche possibile importare lavori in formati di testo comuni

  • Fare clic su File → Open → Importa job, viene visualizzata la finestra di dialogo Apri cartella. Selezionare C:\Program Files\Metamation\Praxis\Samples\Jobs\C0\C0.csv.

  • Se tutti i pezzi esistono nella posizione csv o nelle directory della cartella di ricerca, viene visualizzata la finestra nuovo job elencando tutti i pezzi corrispondenti ai campi csv.

  • I pezzi mancanti vengono ignorati e non appaiono nella finestra nuovo job.

  • Inizialmente lo stato del pezzo è Nascent.

  • Fare clic su OK, ora i pezzi vengono importati nella libreria dei pezzi. Viene eseguita la convalida CAD, l’attrezzaggio per l’istanza di attrezzaggio piegatura e taglio, che può essere monitorato nella schermata Pmonitor.

Creare un pezzo vuoto per i file CAD mancanti

Quando si importano lavori utilizzando un foglio di calcolo (.csv o .xlsx), a volte si possono avere righe che fanno riferimento a file CAD che non sono disponibili nel sistema.

  • Per gestire questo, andare su fabbrica → Impostazioni → Job, in Importare fogli di calcolo abilitare Creare un pezzo vuoto per i file CAD mancanti

Quando questa opzione è selezionata:

  • Se un file CAD manca per qualsiasi riga durante l’importazione del lavoro, il sistema creerà automaticamente un pezzo vuoto al posto del file CAD mancante.

  • Nella finestra nuovo job, questi pezzi mancanti sono visivamente evidenziati in giallo per indicare che il file CAD associato è mancante.

  • Il pezzo mancante avrà ancora i suoi campi mappati (come nome del pezzo, materiale, spessore, quantità, ecc.) dal CSV. Tuttavia, il suo stato del pezzo sarà mostrato come Nascent, il che significa che è solo un segnaposto vuoto senza geometria effettiva ancora.

Ciò garantisce che l’importazione del lavoro possa comunque procedere senza errori dovuti a file CAD mancanti — è possibile aggiornare o sostituire successivamente il pezzo vuoto con il file CAD corretto quando diventa disponibile.

Non visualizzare la finestra di dialogo Nuovo job se il foglio di calcolo non presenta errori

fabbrica → Impostazioni → Job → Importare fogli di calcolo→ Non visualizzare la finestra di dialogo Nuovo job se il foglio di calcolo non presenta errori

  • Quando questa casella di controllo è selezionata, il sistema crea automaticamente il nuovo lavoro senza mostrare la finestra di dialogo Nuovo lavoro se il foglio di calcolo non ha errori (nessun pezzo mancante, date mancanti o dati non validi).

  • Quando questa casella di controllo è deselezionata, la finestra di dialogo Nuovo lavoro appare sempre per la revisione dell’utente e la conferma, anche se il foglio di calcolo non ha errori.

Prefer raw-material column over material + thickness

fabbrica → Impostazioni → Job→Prefer Raw-Material column over Material + Thickness

  • Quando questa casella di controllo è selezionata, il sistema si aspetta che il foglio di calcolo utilizzi la colonna Raw-Material (ad esempio, Steel-20). Ciò significa che il tipo di materiale e lo spessore devono essere combinati in un singolo valore in una colonna.

  • Quando questa casella di controllo è deselezionata, il sistema si aspetta che il foglio di calcolo abbia colonne separate per materiale e spessore (ad esempio, Steel e 2.00). Il sistema le combina internamente durante l’importazione.

Cartella di monitoraggio utilizzando CSV/XLSX

  • In fabbrica → Impostazioni→ Visualizza selezionare Abilita 'Cartella Live' e Visualizza anch sotto-cartelle.

  • Impostare il percorso della cartella di monitoraggio (UNC o unità mappata).

  • In Importare fogli di calcolo, selezionare Importa foglio di calcolo da 'Cartella Live'.

  • Selezionare Importa job dalle opzioni Importazione foglio di calcolo.

  • Copiare C:\Program Files\Metamation\Praxis\Samples\Jobs\C0\C0.csv e incollarlo nella cartella di monitoraggio.

  • Il sistema controlla il CSV e tenta di creare automaticamente il lavoro.

  • Se il file CAD manca, non viene creato alcun lavoro perché i dati CAD richiesti non vengono trovati.

Directory di ricerca CAD e ricerca CAD ricorsiva - non configurata

  • In fabbrica → Impostazioni → Job → Importare fogli di calcolo, se non è impostata alcuna CAD lookup directories e Utilizzare la ricerca CAD ricorrente non viene utilizzata, il sistema non cerca automaticamente i file CAD.

  • Se è selezionata la casella di controllo Creare un pezzo vuoto per i file CAD mancanti, il lavoro verrà comunque creato anche quando i file CAD sono mancanti.

  • In questo caso, tutti i pezzi nel lavoro verranno creati come pezzi nascenti (vuoti) — il che significa che non hanno geometria perché i file CAD non sono stati trovati.

Directory di ricerca CAD e ricerca CAD ricorsiva - configurata

Quando Directory di ricerca CAD è impostato su (C:\Program Files\Metamation\Praxis\Samples) e Utilizzare la ricerca CAD ricorrente è abilitato in fabbrica → Impostazioni → Job → Importare fogli di calcolo:

  • Il sistema cerca automaticamente la cartella specificata e tutte le sue sottocartelle per i file CAD richiesti menzionati nel foglio di calcolo.

  • Anche Creare un pezzo vuoto per i file CAD mancanti è abilitato, quindi se un file CAD non viene ancora trovato, il sistema creerà un pezzo vuoto come segnaposto.
