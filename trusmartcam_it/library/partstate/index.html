Stato parti

Quando una parte viene importata in TruSmartCAM, la tecnologia lamiera rilevante viene elaborata automaticamente per la parte. Come parte di questo processo, TruSmartCAM esegue

Lo stato della parte viene visualizzato direttamente sul riquadro della parte, sotto il nome della parte, indicando la condizione di convalida corrente della parte.

I tipi di stato delle parti descrivono le diverse condizioni in cui può trovarsi una parte come risultato dei controlli di convalida a livello di parte. Questi stati indicano problemi relativi alla progettazione, al materiale o alla geometria della parte e devono essere risolti prima di procedere con l’attrezzaggio.

Part errors

A CAD (Computer-Aided Design) error occurs when there is an issue in the parts design or geometry, such as open contours, missing geometry, or unassigned material. These errors originate from the CAD model and prevent the part from being processed further.

  • Piegatura irrealizzabile : TruSmartCAM automatically tries to find a valid bending method for the part. If it cannot find any suitable bend solution, the part is marked as bend infeasible.

  • Cut infeasible : If bend warnings are ignored, TruSmartCAM then attempts to find a cutting method. If no valid cutting solution is found, the part is marked as cut infeasible.

Material errors

  • Il materiale non esiste : The part’s thickness is read from the CAD model and matched with materials in the TruSmartCAM Raw Material database. If no matching material is found, the part is marked with a Il materiale non esiste error.

  • Materiale non assegnato : If the CAD model does not have a material mapping field or if the field is empty, TruSmartCAM cannot assign a material. In this case, the part is marked with a Materiale non assegnato error.

Geometry errors

  • Rilevati contorni aperti : The CAD file has one or more open (incomplete) shapes, which prevents proper tooling.

  • Rilevati contorni esterni multipli : The CAD file has more than one outer closed loop, which creates confusion during processing.

  • Nascent error : When a part is loaded from a spreadsheet (CSV or XLSX), but the actual CAD file is missing, it is marked as a nascent part.

  • Geometria mancante : The CAD model has no valid geometry or the geometry is unreadable, making it unusable for tooling.
