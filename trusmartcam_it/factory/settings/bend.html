Bend

È possibile accedere alle impostazioni Bend facendo clic sull’icona Factory e quindi selezionando Settings, dove si vedrà un elenco di tutte le impostazioni ambientali disponibili del software.

Impostazioni output

Salva piegatura NC e report - I risultati dell’attrezzatura sono salvati solo se l’attrezzatura è OK e queste impostazioni di esportazione sono abilitate e configurate. Abilitare questa opzione per riservare NC e report di piegatura.

Mostra il nome del materiale grezzo nei report di piegatura - TruSmartCAM stamperà i nomi dei materiali grezzi nei report di piegatura.

Salva il report CSV sul setup dell’utensile di piegatura - Esporta i dettagli di configurazione degli utensili di piegatura in un file CSV

Salva piegatura FX insieme a NC - Se questa opzione è abilitata, quando la piegatura viene eseguita all’interno di TruSmartCAM, l’output della parte verrà salvato nella directory insieme ai file NC e al report per ogni configurazione.

Destinazione output di piegatura - Questa è la destinazione per i file NC, report e FX da salvare.

Impostazioni di processo

Applicare la correzione piatta durante la piegatura - Quando questa opzione è selezionata, TruSmartCAM verifica se le dimensioni piane sono state aggiornate durante l’attrezzatura di piegatura e riaggiorna la copia master utilizzata per il processo di taglio.

Mantenere gli utensili di piegatura obsoleti quando viene importato un file FX - Abilitare questa opzione per mantenere l’attrezzatura esistente quando un file FX legacy viene importato in TruSmartCAM.
