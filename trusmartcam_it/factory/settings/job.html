Impostazioni output

Salva layout NC e report - Abilitare questa opzione per salvare NC e report di layout.

Salva file di layout - Abilitare questa opzione per salvare il file di layout.

Anteprima della simulazione di uscita nei report - Abilitare questa opzione per generare l’anteprima della simulazione dell’attrezzatura insieme agli altri output. Si tratta di una breve animazione GIF per l’attrezzatura laser/punzonatrice.

Non cancellare le emissioni del programma al completamento del layout/job - Abilitare questa opzione ignorerà l’operazione di eliminazione per gli output del programma quando un lavoro/layout viene contrassegnato come completato.

Impostazioni di nesting

Mescola pezzi da job multipli - Supporta il nesting in modalità mista. Le parti di più lavori dovrebbero essere nidificate insieme per una migliore efficienza di imballaggio.

Non rilasciare automaticamente i layout - Questo impedirà che i layout siano disponibili per la produzione finché tutti i layout non saranno in uno stato rilasciato.

Taglio automatico della guarnizione - Questo valore viene utilizzato per decidere se un layout nidificato viene tenuto nella coda di approvazione e viene ri-nidificato quando vengono pianificate nuove parti o vengono rilasciate per la produzione e gli output correlati vengono esportati nella cartella di output.

Prefisso interattivo del nome lamiera - Questa sezione viene utilizzata per supportare il nesting interattivo in cui l’utente ha più controllo sulle parti da nidificare e sul risultato del nesting. Utilizzare uno schema di denominazione, diverso dal nesting automatico, per i layout nidificati in modo interattivo.

Aggiornare le giacenze magazzino lamiere al rilascio (o meno) del annidare - Se questa opzione è abilitata, l’inventario delle lastre si aggiornerà quando si invia un nest a una macchina. Inoltre, richiamare il nest dalla macchina restituirà la lastra all’inventario.

Automatizzare la gestione giacenze dei residui - Quando questo interruttore è selezionato, il software automatizza la gestione dell’inventario dei ritagli. Se non selezionato, la gestione dei ritagli diventa interattiva.

Aggiornamento automatico layout non aggiornati - Quando selezionato, il software aggiorna tutti i ui:EGTag.SelectOutOfDate[Out-of-date Layouts] collegati a una parte dopo che le sue soluzioni di taglio sono state calcolate.

Nesting automatico con TwinLine - Selezionare questa opzione per abilitare l’automazione twin-line.

Impostazioni del workflow di spinta

Usare 'Workflow di spinta' - Questa opzione è utile per gli utenti che necessitano di un sistema di pianificazione e nesting altamente autonomo. Quando abilitato, TruSmartCAM pianifica il nesting su richiesta.

Nesting per stazione - Questo valore serve per impostare quanti layout per stazione devono essere creati.

Lamiere massime per nesting - Questo valore viene utilizzato per controllare il conteggio massimo delle lastre rispetto al nest univoco. Quindi, ad esempio, se c’è un ordine con una parte grande (che riempie l’intera lastra), necessaria in grandi quantità (diciamo 500) - TruSmartCAM non crea più un singolo nest con 500 copie in modalità pull. Invece, crea 10 copie per nest dove 10 è il massimo.

Programmi di piegatura per stazione - Questo valore serve per impostare quanti programmi di piegatura per stazione devono essere creati.

Contrassegna l’attività come completata quando le emissioni del programma vengono rimosse - Abilitare questa opzione per contrassegnare un layout come completato quando gli output vengono rimossi dalla destinazione di output di destinazione.

Impostazioni dell’importazione fogli di calcolo

Directory di ricerca CAD - TruSmartCAM tenterà di caricare la voce della parte menzionata nel CSV importato dalla libreria delle parti. Tuttavia, se la parte non è disponibile nella libreria delle parti, tenterà di recuperare le parti dalle directory di ricerca in base all’ordine delle directory impostate in questa finestra di dialogo.

Utilizzare la ricerca CAD ricorrente - Abilitare questa opzione per ripetere la ricerca per risultati successivi.

Creare un pezzo vuoto per i file CAD mancanti - Se le parti non esistono nella libreria TruSmartCAM o nelle directory di ricerca, TruSmartCAM creerà una parte vuota per quelle parti non disponibili.

Non visualizzare la finestra di dialogo Nuovo job se il foglio di calcolo non presenta errori - Se le voci CSV sono pulite, cioè nessuna discrepanza di dati dal DB o tutti i campi sono elencati correttamente, TruSmartCAM non mostrerà la finestra di panoramica del lavoro e elaborerà il lavoro non appena il CSV viene importato o rilasciato.

Preferire la colonna Materiale grezzo rispetto a Materiale + Spessore - Tipi di fogli di calcolo supportati:

  1. Pattern 1: PartName, Quantity, DueDate, Priority, RawMaterial

  2. Pattern 2: PartName, Quantity, DueDate, Priority, Material, Thickness

  3. Pattern 3: PartName, Quantity, DueDate, Priority, Material, Thickness, RawMaterial

Se questa opzione non è selezionata, Pattern 1 sarà considerato il tipo di foglio di calcolo predefinito. Supportiamo anche Pattern 3 nel caso in cui il formato CSV sia di tipo Pattern 3.

Se questa opzione è selezionata e CSV è di tipo Pattern 2, allora TruSmartCAM risolverà il materiale grezzo dai campi Material e Thickness del CSV.

Impostazioni output dei fogli di calcolo

Esportazione del report di completamento del job/layout (.csv)* - Selezionare questa opzione per abilitare l’esportazione CSV quando un Layout o Job è completo. I report completati vengono salvati nella destinazione Reports specificata in Factory → Settings → Reports settings.
