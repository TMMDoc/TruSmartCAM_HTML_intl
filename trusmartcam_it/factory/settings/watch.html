Watch

Watch

Le impostazioni Watch consentono a TruSmartCAM di monitorare una cartella designata e importare automaticamente i file nel sistema. Questo aiuta ad automatizzare l’importazione di parti, lavori, assiemi e ordini senza intervento manuale.

Visualizza impostazioni

Abilita 'Cartella Live' - Abilita il monitoraggio automatico della cartella specificata.

Visualizza questa cartella - Specifica la cartella da monitorare per i file in arrivo.

Visualizza anch sotto-cartelle - Include tutte le sottocartelle nel processo di monitoraggio.

Dopo l’importazione del file

Dopo l’importazione - Dopo l’importazione del file, l’utente ha due opzioni. Il file può essere eliminato dalla posizione o spostato in una posizione diversa.

  • Delete imported file - Rimuove il file di origine dopo un’importazione riuscita.

  • Move imported file - Sposta il file di origine nella cartella specificata in Move files to this folder dopo un’importazione riuscita.

Sposta file in questa cartella - Specifica la cartella di destinazione in cui devono essere spostati i file importati con successo quando After import è impostato su Move imported file.

Sposta fogli di calcolo con errori in questa cartella - Sposta le importazioni non riuscite nella cartella di errore specificata.

Impostazioni dell’importazione fogli di calcolo

Importa foglio di calcolo da 'Cartella Live' - Importa automaticamente i file di fogli di calcolo rilevati nella cartella watch.

Non importare fogli di calcolo con errori - Impedisce l’importazione di file contenenti errori di convalida.

Opzione importazione foglio di calcolo - Seleziona il tipo di dati da importare dall’elenco a discesa. I formati di fogli di calcolo supportati possono includere CSV, XLSX o XML. Le opzioni disponibili sono:

  • Importa job - Importa i dati dei lavori dal foglio di calcolo.

  • Importa ordini - Importa i dati degli ordini dal foglio di calcolo.

  • Importa gruppi costruttivi - Importa i dati degli assiemi dal foglio di calcolo.

  • Importa pezzi - Importa i dati delle parti dal foglio di calcolo.

Informazioni aggiuntive

  • Utilizzare un percorso UNC (\\ServerName\FolderPath) o un’unità mappata (M:\FolderPath) durante la configurazione della cartella watch. Non utilizzare percorsi locali come C:\FolderPath.

  • Il computer che esegue il servizio TruSmartCAM Engine deve avere accesso in lettura e scrittura alla cartella watch configurata. Senza autorizzazioni sufficienti, i file potrebbero non essere rilevati, importati, spostati o eliminati con successo.

  • La mappatura del foglio di calcolo deve essere configurata prima di abilitare le impostazioni Watch. In caso contrario, TruSmartCAM non sarà in grado di elaborare i file di fogli di calcolo rilevati nella cartella watch.
