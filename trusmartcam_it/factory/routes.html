Routes

In base alla licenza, TruSmartCAM avrà un set predefinito di macchine CAM e configurazioni di routing. È possibile personalizzare queste configurazioni per adattarle ai flussi di lavoro di produzione specifici.

Nella pagina routes, le macchine sono rappresentate come stazioni di lavorazione, ciascuna identificata da un’immagine e un nome.

I percorsi collegano le stazioni, mostrando i possibili percorsi del flusso di lavoro che una parte può seguire dall’inizio alla fine. È possibile visualizzare le proprietà della stazione passando il mouse sopra una stazione.

Facendo clic su una stazione si apre un pannello con informazioni dettagliate su tale stazione, inclusi nome, tipo e stato di disponibilità.

  • Per modificare la disponibilità della stazione, fare clic sulla stazione e attivare la casella di controllo _Abilitato nel pannello.

Esportare/Importare configurazioni di routing

La funzione Export/Import consente agli utenti di salvare le configurazioni di routing correnti e applicarle ad altri progetti o istanze.

Export CAD routes

Fare clic su Export CAD routes…​ per salvare tutte le regole di condizione di routing correnti in un file CSV.

Il CSV elenca i materiali e le loro condizioni di routing. Le colonne rappresentano i nomi delle macchine e le condizioni (ad esempio, Material, SpecialTool, HasBend, MaxBendLength). Un valore di 1 indica il routing verso una macchina; una cella vuota significa nessun routing.

È possibile modificare questo CSV e reimportarlo nel sistema.

Import CAD routes

Utilizzare Import CAD routes…​ per applicare un file CSV modificato.

Prima dell’importazione, eseguire Reset CAD routes per cancellare le configurazioni esistenti.

Il sistema legge il file e aggiorna le condizioni di routing. Le regole possono includere condizioni come HasBend e MaxBendLength.

Se HasBend = 1, la regola si applica alle parti con piegature.
Se MaxBendLength = 1, la regola limita ulteriormente in base alla lunghezza della piega più lunga della parte.

Esempio 1

La parte 00.13 ha una piega più lunga di 500 mm. Corrisponde alla prima regola, quindi TBC-5030-V2 è esclusa.

Esempio 2

La parte 00.20 ha una piega più lunga di 600 mm. Non corrisponde alla prima regola e segue la seconda regola, instradando verso tutte le macchine.

Reset CAD routes

Utilizzare Reset CAD routes per cancellare tutte le configurazioni esistenti prima di importare un nuovo CSV.

Se si importa senza ripristinare, le vecchie regole potrebbero entrare in conflitto con le nuove.
