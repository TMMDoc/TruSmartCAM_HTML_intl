Edit

Le opzioni modifica consentono di modificare varie impostazioni relative a macchine, materiali, utensili e preferenze di sistema all’interno di TecZone. Una volta selezionato, si verrà reindirizzati alla sezione di configurazione appropriata in TecZone per apportare le modifiche necessarie.

Le seguenti impostazioni possono essere modificate:

  • Macchine

  • Materiali CAM

  • Giacenze magazzino utensili di piegatura

  • Deduzioni di piegatura customizzate

  • Impostazioni fondenti
