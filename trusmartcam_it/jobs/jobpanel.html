Pannello commessa

Questa sezione spiega le azioni relative alle commesse come Export outputs, Ripristina job…​, Contrassegna come completo…​, Esporta job…​ e altre opzioni utilizzate per gestire le istanze delle commesse.

Aprire in modalità di sola lettura per la visualizzazione
Apre la commessa selezionata in modalità di sola lettura.

Check-out the document for editing
Abilita la modifica della commessa selezionata.

Cancella
Elimina la commessa selezionata.

Visualizza report
Apre il rapporto di produzione per la commessa selezionata.

Esporta output
Esporta i file di output pronti per la macchina (file NC e rapporti) generati per la commessa selezionata.

Plan job
Genera layout nidificati per le parti nella commessa selezionata.

Ripristina job…​
Rimuove tutti i layout generati per la commessa e reimposta le sue parti allo stato pianificabile.

Contrassegna come completo…​
Contrassegna la commessa e tutti i suoi nidi come completati dopo la produzione.

Esporta job…​
Esporta la commessa completa come pacchetto che può essere riutilizzato o importato in un altro sistema.

Prec / Succ / Layout
Naviga verso il layout precedente o successivo all’interno della commessa selezionata.

Sort and filter
Utilizzare questi filtri per visualizzare altre commesse che condividono lo stesso materiale, cliente, data di scadenza o priorità.

Esporta output

  • Selezionare la commessa richiesta.

  • Fare clic su Esporta output.

  • Questo esporta i file di output pronti per la macchina generati per la commessa selezionata.

  • Export Outputs può essere salvato nella posizione predefinita configurata in Impostazioni di default o in una posizione personalizzata utilizzando Sfoglia…​.

  • Esporta solo i file di output della macchina (come NC e rapporti) per la commessa.

  • Questi file vengono utilizzati direttamente per la produzione e possono essere inviati alla macchina.

Ripristina job…​

Reimposta la commessa selezionata rimuovendo tutti i layout generati e rendendo le parti disponibili per una nuova pianificazione.

Per reimpostare una commessa:

  • Fare clic con il pulsante destro del mouse sulla commessa e selezionare Ripristina job…​.

  • Questo elimina tutti i layout nidificati esistenti per la commessa selezionata e reimposta tutte le sue parti allo stato pianificabile.

  • Se i layout contengono parti di altre commesse, anche queste parti vengono reimpostate.

Esporta job…​

  • Selezionare la commessa richiesta.

  • Fare clic su Esporta job…​.

  • Questa opzione esporta la commessa completa come pacchetto che può essere riutilizzato o importato in un altro sistema.

  • I dati esportati includono tutte le informazioni necessarie per ricreare la commessa:

    • Dettagli e quantità delle parti (file CSV).

    • Dati di layout e nidificazione (file CSV / FXYLT).

  • I file esportati vengono salvati come cartella della commessa nella posizione di esportazione configurata.

  • Questa opzione viene utilizzata per trasferire, condividere o eseguire il backup dell’intera commessa.

Contrassegna come completo…​

  • Una volta che la commessa è stata elaborata sulla macchina, aprire il pannello Commessa e selezionare Contrassegna come completo…​.

  • Quando una commessa viene contrassegnata come completata:

    • Tutti i nidi della commessa vengono contrassegnati come completati.

    • La commessa viene visualizzata in grigio, spostata alla fine dell’elenco e mostrata con un segno di spunta verde.

    • Tutti i layout della commessa vengono rimossi dall’elenco dei layout attivi.

    • Gli stati delle parti e della commessa vengono aggiornati e le quantità elaborate passano allo stato completato.

    • I file di output per tutti i layout nella commessa vengono rimossi dalla posizione di output della macchina per evitare utilizzi duplicati.
