Manuelle Reststück-Verwaltung

Manuelles Reststück konfigurieren

  • Klicken Sie auf Werk → Einstellungen → Job.

  • Wählen Sie Tafelbestand bei Schachtelungsfreigabe (Nicht-Freigabe) aktualisieren aus.

Dadurch wird sichergestellt, dass Reststücke nachverfolgt und im Blechbestand aktualisiert werden, wann immer eine Verschachtelung abgeschlossen oder freigegeben wird.

Manuelles Reststück-Blech aus Auftrag

  • Klicken Sie mit der rechten Maustaste auf ein als abgeschlossen markiertes Layout (Layout mit grünem Häkchen).

  • Klicken Sie auf die Option Zu Resten hinzufügen…​.

  • Jetzt wird das Reststück-Blech manuell zur Blechdatenbank hinzugefügt.

Manuelles Reststück-Blech aus Auftrags-Verschachtelung

  • Wechseln Sie zu Job → Seite "Verschachtelungen". Verwenden Sie Filter und wählen Sie Status aus.

  • Wählen Sie "Abgeschlossen", um alle abgeschlossenen Layouts anzuzeigen.

  • Klicken Sie mit der rechten Maustaste auf das Layout, für das ein manuelles Reststück erstellt werden soll, und klicken Sie auf Zu Resten hinzufügen…​.

Reststücke entfernen

Wenn ein manuell erstelltes Reststück-Blech beschädigt oder verloren geht, können Sie es aus dem Blechbestand entfernen, ohne es direkt zu löschen.

  • Klicken Sie mit der rechten Maustaste auf das Reststück-Blech, das Sie entfernen möchten.

  • Klicken Sie auf die Option Aus Resten entfernen…​.

  • Das Reststück-Blech wird aus der Blechdatenbank entfernt.

  • Das teilweise schattierte quadratische Symbol, das anzeigt, dass das Reststück-Blech erstellt wurde, wird ebenfalls gelöscht, wenn das Reststück entfernt wird.

Reststück-Bleche (erstellt durch Auto/Manuell) werden mit hoher Priorität während der automatischen Verschachtelungsplanung oder interaktiven Verschachtelungsplanung verwendet.
