Serverinstallation

Befolgen Sie die Anweisungen zur Installation des TruSmartCAM-Servers zusammen mit SQL.

Administratorrechte

Um diese Installation auszuführen, muss der Benutzer über Administratorrechte verfügen.

  • Führen Sie Setup.TruSmartCAM.<version>.exe aus, um die Installation zu beginnen.

Version ist eine Nummer.

  • Lesen Sie die Lizenzvereinbarung und wählen Sie I accept the agreement und klicken Sie dann auf Weiter .

Schritt 1 : Installationsort

  • Geben Sie das Installationsverzeichnis an oder akzeptieren Sie den Standardinstallationsort C:\Program Files\Metamation\Praxis und klicken Sie auf Weiter.

    • C:\Program Files\Metamation\Praxis - Der Installationsordner enthält folgende Ordner: Bin, MetaCAM, Samples und Tracker.

    • C:\Program Data\Metamation - Der Datenordner enthält folgende Ordner: TruSmartCAM, TecZone und MetaCAM.

Schritt 2 : Komponentenauswahl

Installationstyp Komponenten

Full

Server + Desktop-Client + Engine

Client

Desktop-Client

Server

Server + Desktop-Client

Station

Wählt alle Stationsterminals wie TecZone, MetaCAM, RightAngle(Biegesteuerung), Vulcan(Lasermaschinensteuerung)

Custom

Wird vom Benutzer ausgewählt.

  • Wählen Sie Vollinstallation. Wählen Sie Code Senders und Bend Code Senders aus TruSmartCAM Tools. Jetzt ändert sich der Installationstyp zu Benutzerdefiniert. Klicken Sie auf Weiter.

Schritt 3 : SQL Server installieren auswählen

Die SQL Express 2022-Installation ist nur für den TruSmartCAM-Server erforderlich. Diese Option ist für Client- und Sync-Installationen nicht verfügbar. Dies ist auch beim ersten Mal aktiviert und wird installiert und kann danach ignoriert werden. Wenn bereits SQL auf dem TruSmartCAM-Server installiert ist, kann der Benutzer Schritt 3 ignorieren.

  • In Select Additional Tasks: Klicken Sie auf Install SQL Express und klicken Sie auf Weiter.

  • In Ready to Install: Fassen Sie alle Installationsmerkmale zusammen und klicken Sie auf Install.

Schritt 4 : SQL Server-Installation

Sobald die TruSmartCAM-Komponenten installiert sind, fährt das Installationsprogramm mit der SQL-Server-Installation fort. Akzeptieren Sie die Lizenzvereinbarung, um fortzufahren, und installieren Sie SQL Express mit den Standardoptionen.

Sobald die Installation abgeschlossen ist, klicken Sie auf Close und schließen Sie die Installation ab.

Abhängig von der Windows-Version kann das Schließen der SQL-Installation einen Neustart erfordern.

Schritt 5 : Windows-Desktop-Verknüpfungen

  • Wenn das System nach der SQL-Installation nicht neu startet, aktivieren Sie das Kontrollkästchen Launch TruSmartCAM und klicken Sie auf Finish.

  • Wenn das System neu startet, starten Sie die TruSmartCAM-Verknüpfung auf dem Desktop oder geben Sie TruSmartCAM in Windows Start Program ein und Führen Sie als Administrator aus.

Serverinstallation: TruSmartCAM- und TruSmartCAM License-Verknüpfungssymbole sind auf dem Desktop verfügbar. + Client/SYNC-Installation: Nur TruSmartCAM-Verknüpfungssymbol ist auf dem Desktop verfügbar.

Schritt 6 : Lizenzregistrierung

  • Geben Sie im Feld Serial Number den von Trumpf Metamation Service Team bereitgestellten TruSmartCAM Serial key ein. Geben Sie <Company Name> in das Feld "Registered to" und E-Mail ein. Klicken Sie auf OK.

Falls die Netzwerk-Firewall die Lizenzregistrierung blockiert oder kein Internet auf dem TruSmartCAM-Server-PC vorhanden ist, erfolgt die TruSmartCAM-Lizenzierung durch Offline-Aktivierung. Senden Sie die Prax.request aus dem Ordner C:\ProgramData\Metamation\Praxis per E-Mail an das Trumpf Metamation Service Team. Das Trumpf Metamation Team stellt die Prax.lock bereit, die in diesen Ordner abgelegt werden muss.

Schritt 7 : Erstmalige Konfiguration

  • Geben Sie den Datenbanknamen ein und legen Sie den Datenspeicherort fest. Aktivieren Sie das Kontrollkästchen SQL-Server verwenden und wählen Sie den SQL-Servernamen aus, um eine Verbindung zur Datenbank herzustellen. Klicken Sie auf OK.

  • Der TruSmartCAM-Desktop-Client öffnet sich und zeigt Database Connection Succeeded unten an.

  • TruSmartCAM-, TecZone- und MetaCAM-Konfigurationsdateien wurden mit Standard-Konfigurationsmaschinen oder gemäß den TruSmartCAM-Lizenzmaschinen erstellt. Dieselbe Konfiguration wird über alle TruSmartCAM-Clients hinweg geteilt.

Der Willkommensbildschirm wird mit dem Windows-Benutzer-LoginName als TruSmartCAM-Login und -Name angezeigt. Der erste TruSmartCAM-Benutzer wird als Standort-Admin betrachtet. Geben Sie E-Mail und Passwort ein. Klicken Sie auf OK .

Aktivieren Sie das Kontrollkästchen Mich erinnern , um die Anmeldedaten für das nächste Mal zu speichern. Deaktivieren Sie das Kontrollkästchen Mich erinnern, um bei jedem Start der TruSmartCAM-Anwendung zur Eingabe von Login und Passwort aufgefordert zu werden.

Schritt 8 : TruSmartCAM-Anmeldung

  • Schließen Sie TruSmartCAM und öffnen Sie es erneut. Das System fordert Sie auf, die TruSmartCAM- Anmeldedaten einzugeben.

  • Der angemeldete Benutzername wird neben dem Benutzersymbol im TruSmartCAM- Anwendungsheader angezeigt.

  • Wenn Sie den Mauszeiger über das Benutzersymbol bewegen, zeigt das System Folgendes an:

    • Die Benutzerrolle (z. B.: Programmer, Admin).

    • Eine Verknüpfung zum Abmelden: Ctrl+L.

  • Durch Klicken auf das Benutzersymbol wird ein Menü geöffnet, über das Sie Folgendes tun können:

    • Information bearbeiten… - Aktualisieren Sie Ihre Profildetails.

    • Passwort ändern…​ - Ändern Sie Ihr Kontopasswort.

    • Abmelden - Melden Sie sich von der Anwendung ab.

    • Abbrechen - Schließen Sie das Menü, ohne Änderungen vorzunehmen.

Schritt 9 : Werksmaschinenkonfiguration

  • TruSmartCAM-Lizenz [Alle Maschinen aktivieren Lizenzbit]: Standard-TruSmartCAM-Maschine und -Material werden als Werkskonfiguration betrachtet.

  • TruSmartCAM-Lizenz [Keine Demo und wenige Maschinenbits aktiviert]: Diese wenigen Maschinen und das Standardmaterial werden als Werkskonfiguration betrachtet.

  • (Öffnen → Importieren) Sie ein neues Teil aus C:\Program Files\Metamation\Praxis\Samples\Parts und überprüfen Sie, ob das Teil erfolgreich importiert wurde.

Hinweis 1 : TruSmartCAM Monitor

  • TruSmartCAM Monitor mit Engine-Installation (Site-Admin-Rechte):

    • TruSmartCAM Monitor mit Engine zusammen mit Site-Admin-TruSmartCAM-Rechten hat folgende Optionen:

      1. Push TruSmartCAM-Konfiguration, TecZone-, MetaCAM-Konfiguration über alle TruSmartCAM- Clients hinweg. Hat auch Rechte, neue TruSmartCAM-Benutzeranmeldungen zu erstellen.

      2. Show Agents zeigt die insgesamt verfügbare Engine für Automatisierungsaufgaben an, die der TruSmartCAM-Lizenz entspricht.

  • TruSmartCAM Monitor ohne Engine-Installation (Site-Admin-Rechte):

  • TruSmartCAM Monitor ohne Engine-Installation (Nicht-Site-Admin-Rechte):

Der PC, auf dem die TruSmartCAM-Anwendung läuft, muss die Engine-TruSmartCAM-Komponente installiert haben [Server-PC oder in einer beliebigen Client-Installation]. Der PC mit installierter Engine-TruSmartCAM-Komponente sollte rund um die Uhr laufen, um alle Automatisierungsaufgaben auszuführen.

Hinweis 2 : Lock Tracker

  • Klicken Sie auf die auf dem TruSmartCAM-Server-Desktop verfügbare Verknüpfung TruSmartCAM License. Alternativ öffnen Sie die URL http://localhost:7171/metamation/locktrack in einem beliebigen Webbrowser.

Dadurch wird das Lock Tracker-Modul geöffnet, das Lizenzinformationen, aktuelle TruSmartCAM- Sitzungsinformationen und das Protokoll enthält.

LockTracker HOME Page: Zeigt die TruSmartCAM-Seriennummer, Ablaufinformationen zur Seriennummer und verfügbare Seriennummernbits an.

LockTracker SESSIONS Page: Aktuelle Verbindungsinformationen des TruSmartCAM-Servers wie TruSmartCAM-Desktop-Client-PC, PMonitor, PEngine-Status.

LockTracker LOGs Page: TruSmartCAM-Client-Start und Abmeldung | Pmonitor-Start und PEngine-Verbindungs-/Trennungsstatus usw. können auf dieser Protokollseite angezeigt werden.

Hinweis 3 : OpenGL

  • Die TruSmartCAM-Anwendung wird beim Öffnen sofort geschlossen, und auch Pmonitor wird nicht in der Taskleiste angezeigt.

  • Dies weist auf ein OpenGL-Problem hin.

  • Eine andere Möglichkeit, dies zu bestätigen: Öffnen Sie C:\Program Files\Metamation\Praxis\Bin\Flux.exe, das einen Fehlerdialog bezüglich des OpenGL-Problems anzeigt.

  • Aktualisieren Sie C:\ProgramData\Metamation\Praxis\Prax.ini, Optionsabschnitt mit GLVersion=1, damit die TruSmartCAM-Anwendung erfolgreich geöffnet wird.

[Options]
GLVersion=1.0
