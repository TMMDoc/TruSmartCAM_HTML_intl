Portanforderungen

TruSmartCAM Server : 7371

Der Server wartet auf diesem TCP-Port auf Clientanfragen. Eingehende Verbindungen sollten zu diesem Port auf dem Servercomputer zugelassen werden.

Floating License Server : 5776

Der Floating License Server wartet auf diesem Port auf Lizenzierungsanfragen. Eingehende Verbindungen sollten zu diesem Port auf dem Floating-Lizenzservercomputer zugelassen werden (TruSmartCAM Server).

Lizenzverbindung zur Cloud : 7567

Dieser Port wird von TecZone (Biegesoftware) und dem Floating License Server verwendet, um Lizenzierungsanfragen an die Metamation Cloud zu senden. Daher sollte eine ausgehende Verbindung auf diesem Port für Lizenzaktivierungen und -verlängerungen zugelassen werden.

Installations- und Supportanforderungen

  • Administratorrechte erforderlich, um die TruSmartCAM-Anwendung zu installieren.

  • Remote-Diagnosetools über Contractor-VPN oder eine sichere Verbindung.
