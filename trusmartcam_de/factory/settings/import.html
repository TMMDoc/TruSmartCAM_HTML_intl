Import

Import

Ausgabeeinstellungen

Flachmuster speichern - Aktivieren Sie diese Option, damit das Abwicklungsmuster eines importierten Teils gespeichert wird.

Format für Flachmuster - Diese Option wird verwendet, um entweder DXF- oder GEO-Dateiformate auszuwählen.

Zielort für Flachmuster - Dies ist der Speicherort, an dem das Abwicklungsmuster gespeichert wird.

Importeinstellungen

Einheiten für DXF-Dateien - Dies ist die Maßeinheit, mit der eine DXF- Datei importiert wird. Der Benutzer kann zwischen mm oder Zoll wählen.

Schlüsselwörter für Materialsuche - Mehrere Lookup-Einträge können unter Verwendung eines Komma-Trennzeichens zwischen ihnen zugewiesen werden. Wenn ein Teil importiert wird, verwendet TruSmartCAM diese Schlüssel, um nach Materialwerten zu suchen. Die gesuchten Werte werden dann verwendet, um das Rohmaterial aufzulösen. Die Originalwerte werden in den Teil- Misc Properties gespeichert.

Schlüsselwörter für Revisionssuche - Wenn ein Teil importiert wird, wird der Revisionswert aus der Roh-Teildatei (die importierte Teildatei) mit dem vorhandenen Revisionswert im Repository verglichen. Der Import wird abgebrochen, wenn die Revision identisch ist. Das Custom-Field-Mapping wird jetzt auch für die DXF-Dateien unterstützt. Für die DXF-Dateien werden die Custom-Field-Daten aus der Textebene in der Zeichnung oder einem PPI-Anhang extrahiert.

Geometrievergleich auf gleicher CAD-Version nutzen - Vergleicht die Geometrie beim Importieren eines Teils mit derselben CAD-Revision und fordert den Benutzer auf, das Teil zu ersetzen oder den Import zu überspringen, wenn Unterschiede erkannt werden. Siehe Import configuration für weitere Details.

Schlüsselwörter für Teilenamen - Diese Option funktioniert ähnlich wie die Material- und Schlüsselwörter für Revisionssuche. Legen Sie einen oder mehrere Schlüsselwörter für Teilenamen fest, um den PPI Teilename anstelle des Dateinamens als Teilnamen zu verwenden.

Schlüsselwörter für Suche nach Dicke - Mehrere Lookup-Einträge können unter Verwendung eines Komma-Trennzeichens zwischen ihnen zugewiesen werden. Wenn ein Teil importiert wird, verwendet TruSmartCAM diese Schlüssel, um nach Dickenwerten zu suchen. Die gesuchten Werte werden dann verwendet, um die Dicke aufzulösen. Die Originalwerte werden in den Teil-Misc Properties gespeichert.

Fudge-Wert für Vergleich der Dicke - Dies wird beim Nachschlagen des Rohmaterials verwendet, wenn ein Teil importiert wird. TruSmartCAM versucht, das Rohmaterial aufzulösen, nachdem der Materialname und die Dicke geparst wurden. Weitere Informationen finden Sie im Thema Thickness fudge weiter unten.

Formen nach Import eines Flachmusters identifizieren - Wenn ein Teil importiert wird, können innere Formen innerhalb des Teils automatisch erkannt und identifiziert werden.

Fudge-Wert für Formenidentifizierung - Der Formen-Identifikations-Fudge-Wert kann festgelegt werden auf:

  1. Globale Ebene - Dieser Wert wird als Standard für alle Formen verwendet.

  2. Individuelle Formen-Ebene - Der Standardwert kann überschrieben werden, indem der Fudge-Wert auf der individuellen Formen-Ebene festgelegt wird. Durch Auswahl der Form im Shape Library-Dialog wird der Fudge-Wert angezeigt. Standardmäßig ist dieser auf 0 gesetzt, und der globale Standardwert wird für solche Formen verwendet.

Flachmuster horizontal drehen - Im Allgemeinen sind Schneidmaschinentische rechteckig und breiter. Um die Werkzeugzuordnungsmöglichkeit zu verbessern, drehen Sie das Teil entlang der Teilbreite, nachdem das Teil importiert wurde.

Keine Teile drehen, die kürzer sind - Legen Sie dies als Mindestzahl fest, um die Rotation zu ermöglichen. Jedes Teil, das kleiner als der eingestellte Wert ist, wird nicht gedreht.

Diverse Einstellungen

Vorabimport von Fremdwerkzeugen erlauben - Aktivieren Sie die Pre-Import-External-Tools-Importoption, um Creo Reader zu aktivieren. TruSmartCAM lädt die vorab extrahierten Familieninstanzen des abgelegten Teils hoch. Bitte beachten Sie, dass die Instanzextraktion während des Teil-Uploads durchgeführt wird (als Pre-Import- Operation), daher muss die Creo-Software auf dem Computer installiert sein, von dem aus das Teil hochgeladen wird.

Zeichenfarbe in Flachmuster-Thumbnails für Deutlichkeit ignorieren - Aktivieren Sie diese Option, um die schärferen Graustufen-Thumbnails zu generieren. Die Ebenenfarben und Textentitäten werden zugunsten der Bildklarheit ignoriert.

Nur eine Engine für alle Importaufgaben verwenden - Wenn aktiviert, wird nur ein Engine-Kern für wichtige Aufgaben verwendet.

Thickness fudge

Der Fudge-Wert für Vergleich der Dicke wählt den nächstgelegenen Blechdickenwert aus, wenn eine gewünschte Blechdicke nicht im Inventar verfügbar ist, indem der Dickenwert von Blechen im Inventar verglichen wird.

Nehmen wir zum Beispiel Folgendes an:

  1. Wir haben ein Teil mit einer Blechdicke von 1,6 mm und dem Material Stahl.

  2. Der Fudge-Wert für Vergleich der Dicke-Wert beträgt 0,1 mm.

  3. Das Inventar enthält Stahlbleche mit den Dicken 1,5 mm, 1,55 mm und 1,7 mm.

TruSmartCAM wählt ein Blech mit dem nächstgelegenen Dickenwert aus, nachdem der Fudge-Wert für Vergleich der Dicke-Wert (0,1 mm) mit dem importierten Element (mit einer Messung von 1,6 mm) verglichen wurde. In diesem Fall wird 1,55 mm die Auswahl sein.
