Installation

Obligatorische Anforderungen

Die Anmeldung bei TruSmartCAM Desktop ist beim ersten Start von SolidWorks zwingend erforderlich.

Stellen Sie sicher, dass der TruSmartCAM-Monitor im Windows-Benachrichtigungsbereich ausgeführt wird.

Wenn TruSmartCAM Desktop nicht geöffnet ist, überprüfen Sie, ob der TruSmartCAM-Serverdienst ausgeführt wird, damit das SolidWorks Add-in funktionieren kann. Wenn beide inaktiv sind, zeigt SolidWorks beim Start eine Fehlermeldung an.

Befolgen Sie diese Schritte, um das TruSmartCAM Add-in für SolidWorks zu installieren und einzurichten:

  1. Führen Sie das TruSmartCAM-Setup aus und aktivieren Sie auf der Komponentenauswahlseite SolidWorks Extension. (Überspringen Sie die Server-Option, wenn SolidWorks auf einem TruSmartCAM-Client installiert ist.)

  2. Schließen Sie die Installation ab, starten Sie dann TruSmartCAM Desktop und melden Sie sich mit gültigen Benutzeranmeldeinformationen an.

  3. Starten Sie SolidWorks.

  4. Gehen Sie in SolidWorks zu Tools → TruSmartCAM → About TruSmartCAM, um die Add-in- Version und Details zu überprüfen.

  5. Um das Add-in beim Start zu aktivieren oder zu deaktivieren, gehen Sie zu Tools → Add-ins und suchen Sie TruSmartCAM unter Other Add-ins.
