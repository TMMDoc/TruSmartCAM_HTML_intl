Maschine hinzufügen/entfernen

Verwenden Sie den Befehl bearbeiten, um den Maschinenauswahldialog zu öffnen. Er zeigt eine hierarchische Liste aller Schneid- und Biegemaschinen zusammen mit den aktuellen Auswahlen an.

  • In der hierarchischen Baumansicht wird der vereinfachte Maschinenname als Maschinenlabel angezeigt und der vollständige Name wird im Tooltip angezeigt. Der Ordner-Tooltip zeigt den Pfad an (z.B.: Trumpf • C Series • C60)

  • Für Nicht-Trumpf-Abkantpressen wird ein markenspezifisches Symbol angezeigt. Der Maschinen- Tooltip zeigt auch die Maschinenmarke an (z.B. - Brand: Bystronic).

  • Das Suchfeld kann verwendet werden, um Maschinen nach Name, Vollständigem Namen, Marke usw. zu suchen. Die Maschinenauswahl wird während der Suche beibehalten, sodass Sie gleichzeitig auswählen und suchen können.

  • Die Suche kann auch zur Suche nach Maschinentechnologie und Größe verwendet werden. Wenn Sie also 'fold' in das Suchfeld eingeben, werden alle Faltmaschinen (Schwenkbieger) angezeigt. Ebenso zeigt die Eingabe von 10ft alle Abkantpressen mit 10ft Tischlänge an (dies ist einheitensensitiv. Verwenden Sie m für Meter bei metrischen Einheiten). Wenn im Namen verfügbar, kann auch die Achsenanzahl für die Suche verwendet werden (z.B.: 2A, 5A usw.)

  • Der Befehl Alles zurücksetzen setzt die Maschinenauswahl auf die Demo- Konfiguration zurück.
