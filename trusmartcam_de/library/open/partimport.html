Teile importieren

Stellen Sie vor dem Importieren von Teilen sicher, dass die Dateiformate mit TruSmartCAM kompatibel sind. Weitere Details finden Sie unter Unterstützte Dateiformate.

Wenn Material und Dicke nach dem Import nicht dem Teil zugeordnet sind, weisen Sie das Material über das Teilepanel zu.

Importmethoden

Es gibt drei Methoden zum Importieren von Teilen in die Teilebibliothek:

  1. Import über Datei

  2. Drag and Drop

  3. Überwachungsordner

Import über Datei

Eine 2D- oder 3D-Teiledatei kann über die Option Importiere Teile in die Teilebibliothek importiert werden.

  • Klicken Sie auf das Symbol Öffnen in der Befehlsleiste und wählen Sie die Option Importiere Teile.

  • Der Dialog Open erscheint — wählen Sie eine beliebige CAD-Datei (2D oder 3D).

  • Eine Vorschau der ausgewählten CAD-Datei wird angezeigt.

  • Wählen Sie das gewünschte Teil aus und klicken Sie auf Open.

  • Das System bestückt das Teil automatisch für alle verfügbaren Maschinen mit aktualisiertem Werkzeugstatus.

Drag and Drop

  • Ziehen Sie eine Teiledatei direkt per Drag and Drop in die TruSmartCAM-Teilebibliothek.

Überwachungsordner

Überwachungseinstellungen ermöglichen es Ihnen, einen Ordner auf neue Dateien zu überwachen und diese automatisch in die Teilebibliothek zu importieren.

Informationen zur Konfiguration finden Sie auf der Seite Überwachungseinstellungen.

  • Verwenden Sie einen UNC-Pfad (Universal Naming Convention) \\192.168.1.11\<Ordnerpfad> oder ein zugeordnetes Laufwerk M:\<Ordnerpfad>.

  • Der PC, auf dem die Engine läuft, muss Lese-/Schreibzugriff auf diesen Ordner haben.

  • Verwenden Sie keinen lokalen Pfad wie C:\<Ordnerpfad>, da der Überwachungsdienst auf dem Engine-PC nach Dateien sucht — nicht auf dem lokalen PC.

Importkonfiguration

Geometrievergleich auf gleicher CAD-Version nutzen

Beim Importieren eines Teils mit derselben CAD-Revision kann TruSmartCAM die Geometrie der CAD-Datei mit dem vorhandenen Teil vergleichen.

  • Um dies zu aktivieren, konfigurieren Sie Schlüsselwörter für Revisionssuche (z. B. REV) unter Werk → Einstellungen →Import/Export. Aktivieren Sie dann die Option Geometrievergleich auf gleicher CAD-Version nutzen.

Der Geometrievergleich bei derselben CAD-Revision funktioniert nur, wenn Schlüsselwörter für Revisionssuche konfiguriert sind.

  • Importieren Sie nun ein Teil in die Teilebibliothek mit einer bestimmten Revision (z. B. Revision 1).

  • Dasselbe Teil wird erneut mit demselben Revisionswert importiert.

  • TruSmartCAM vergleicht die Geometrie der neu importierten CAD-Datei mit dem vorhandenen Teil.

  • Wenn ein Geometrieunterschied festgestellt wird, zeigt TruSmartCAM eine Eingabeaufforderung an, die fragt, ob:

    • Ersetzen das vorhandene Teil durch die neue Geometrie ersetzt werden soll, wobei derselbe Revisionswert beibehalten wird.

    • Überspringen der Import übersprungen werden soll, wobei das vorhandene Teil und seine Geometrie beibehalten werden.

Dies ermöglicht es Benutzern, die Teilegeometrie zu aktualisieren, ohne den Revisionswert zu ändern, während sie dennoch die Kontrolle darüber behalten, ob vorhandene Teile ersetzt werden.

Importierte Teile werden automatisch zur Bibliothek hinzugefügt und können weiterverwendet werden. Weitere Informationen finden Sie unter Bibliothek.
