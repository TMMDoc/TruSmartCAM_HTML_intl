Bearbeitungsvalidierung

Dieser Abschnitt beschreibt, wie Sie auf die Bearbeitungsinformationen für Teile in TruSmartCAM zugreifen und diese ändern können.

  • Klicken Sie mit der rechten Maustaste auf eine beliebige Teilekachel, um das Teile-Panel zu öffnen und auf Bearbeitungsoptionen zuzugreifen.

  • Klicken Sie auf Werkzeuge, um zur ersten verfügbaren bearbeiteten Instanz zu wechseln.

  • Alternativ klicken Sie direkt auf eine bearbeitete Instanz, um Maschinendetails und Bearbeitungsoptionen anzuzeigen.

  • Verwenden Sie für Zur Anzeige im Lesemodus öffnen.

  • Verwenden Sie für Dokument zur Bearbeitung auschecken.

  • Die ausgewählte bearbeitete Instanz wird in TecZone oder MetaCAM geöffnet.

  • Führen Sie manuelle Bearbeitungsprüfungen durch und nehmen Sie die erforderlichen Änderungen vor.

  • Wenn Sie versuchen zu speichern, werden Sie durch einen Dialog Nicht gespeicherte Änderungen aufgefordert, Ihre Bearbeitungen zu bestätigen.

  • Klicken Sie auf Yes, um mit dem Speichern fortzufahren.

  • Das System führt nach dem Speichern erneut eine Bearbeitungsvalidierung durch.

  • Wenn keine Fehler vorliegen, erscheint die Option „Änderungen in TruSmartCAM speichern“.

  • Wenn Bearbeitungsfehler gefunden werden, wird eine Fehlermeldung angezeigt:

  • Nach dem Speichern wird der Bearbeitungsstatus des Teils in der Teilebibliothek aktualisiert.

  • Eine grüne Farbe auf der bearbeiteten Instanz zeigt an, dass sie eingecheckt ist.

  • Der Autor der bearbeiteten Instanz wird auf die letzte Person aktualisiert, die sie bearbeitet hat.

  • Das Feld Vom Benutzer bearbeitet gibt an, ob die bearbeitete Instanz manuell geändert wurde. Wenn ein Benutzer die Instanz auscheckt und Änderungen vornimmt, wird dieses Feld beim Speichern automatisch auf Yes gesetzt.

  • Wenn ein Teil gerade bearbeitet wird, erscheint ein Benutzersymbol auf seiner Kachel.

  • Wenn Sie mit der Maus über die Kachel fahren, wird angezeigt, wer es ausgecheckt hat und wann es ausgecheckt wurde.

  • Während das Teil ausgecheckt ist, ist die Option Dokument zur Bearbeitung auschecken für dieselbe bearbeitete Instanz deaktiviert, und die Option Checkout abbrechen wird aktiviert.
