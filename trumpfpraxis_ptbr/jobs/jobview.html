Visualização do trabalho

Esta seção explica as diferentes visualizações disponíveis na visualização de Jobs, incluindo Jobs, Parts, Nests, Edit, Simulate e Export.

Jobs

Selecionar a opção jobs exibe uma lista de todos os trabalhos atualmente configurados no software. Você pode clicar duas vezes em um trabalho para visualizar seus detalhes ou clicar com o botão direito para acessar opções adicionais.

  • Os trabalhos são exibidos em visualização de blocos, onde cada trabalho é exibido com propriedades básicas como ID do Job, Ref do Job, Nome do cliente, Total de peças, Data de Vencimento, barra de status do Job etc.

  • Cada bloco de trabalho também exibe imagens de peças em formato de cartão empilhado. Você pode rolar por todas as peças em um trabalho usando a roda do mouse.

Detalhes do trabalho

  • Clicar duas vezes em um bloco ou pressionar a tecla Enter após selecionar um bloco alterna a visualização para o Modo Detalhado.

  • A página de detalhes exibe a lista de peças juntamente com as chapas aninhadas (layouts), se houver.

  • Use as teclas de seta Cima/Baixo para alternar entre trabalhos na página de detalhes.

  • Rolar a roda do mouse sobre a pilha de peças destaca a peça tanto na lista de peças quanto nos layouts.

  • Selecionar um layout destaca o layout e todas as peças aninhadas nele na lista de peças. Os destaques se desligam automaticamente após um minuto.

Parts

A visualização Active Parts exibe todas as peças que estão atualmente em andamento dentro de trabalhos ativos.

Cada peça é representada como um bloco mostrando detalhes principais como nome da peça, dimensões, material, espessura, quantidade e data de vencimento.

Nests

A visualização Active Nests exibe todos os nests que estão atualmente em andamento dentro de trabalhos ativos.

Edit

A opção Edit permite modificar o item selecionado.

Com base na visualização atual, a instância selecionada é aberta para edição:

  • Na visualização Jobs, o trabalho selecionado é aberto para edição.

  • Na visualização Active Parts, a peça selecionada é aberta para edição.

  • Na visualização Active Nests, o nest selecionado é aberto para edição.

Export csv

A opção Export permite selecionar informações do trabalho e exportá-las para um arquivo CSV.

  • Clique no ícone jobs.

  • Clique na opção export no lado direito da tela.

  • Uma caixa de diálogo aparece mostrando campos disponíveis:

  • Selecione as caixas de seleção para os campos que deseja exportar.

  • Use a opção Group By se desejar agrupar os dados por um campo selecionado.

  • Clique em Export para gerar o arquivo CSV.
