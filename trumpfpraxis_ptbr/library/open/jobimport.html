Importar trabalhos

Importar de csv ou xlsx

Você também pode importar trabalhos em formatos de texto comuns

  • Clique em File → Open → Import Jobs, o diálogo Open Folder aparece. Selecione C:\Program Files\Metamation\Praxis\Samples\Jobs\C0\C0.csv.

  • Se todas as Peças existirem no local do csv ou nos diretórios de pasta de consulta, a janela new job aparece listando todas as peças que correspondem aos campos do csv.

  • Peças ausentes são ignoradas e não aparecem na janela new job.

  • Inicialmente o status da Peça é Nascent.

  • Clique em OK, agora as peças são importadas na Biblioteca de Peças. Validação CAD, ferramentas para instância de dobra e corte são realizadas, o que pode ser monitorado na tela do Pmonitor.

Create a blank part for the missing CAD files

Ao importar trabalhos usando uma planilha (.csv ou .xlsx), você pode às vezes ter linhas referenciando arquivos CAD que não estão disponíveis no sistema.

  • Para lidar com isso, vá para Factory → Settings → Job, em Spreadsheet import settings habilite Create a blank part for the missing CAD files

Quando esta opção é selecionada:

  • Se um arquivo CAD estiver ausente para qualquer linha durante a importação do trabalho, o sistema criará automaticamente uma peça em branco no lugar do arquivo CAD ausente.

  • Na janela new job, essas peças ausentes são destacadas visualmente em amarelo para indicar que o arquivo CAD associado está ausente.

  • A peça ausente ainda terá seus campos mapeados (como nome da peça, material, espessura, quantidade, etc.) do CSV. No entanto, seu estado de peça será mostrado como Nascent, significando que é apenas um espaço reservado vazio sem geometria real ainda.

Isso garante que a importação do trabalho ainda possa prosseguir sem erros devido a arquivos CAD ausentes — você pode atualizar ou substituir a peça em branco com o arquivo CAD correto posteriormente quando estiver disponível.

Do not show the new job dialog if the spreadsheet has no error

factory → Settings → Job → Spreadsheet import settings→ Do not show the New Job dialog if the spreadsheet has no error

  • Quando esta caixa de seleção é marcada, o sistema cria automaticamente o novo trabalho sem mostrar o diálogo New Job se a planilha não tiver erros (sem peças ausentes, datas ausentes ou dados inválidos).

  • Quando esta caixa de seleção é desmarcada, o diálogo New Job sempre aparece para revisão e confirmação do usuário, mesmo se a planilha não tiver erros.

Prefer raw-material column over material + thickness

factory → Settings → Job→Prefer Raw-Material column over Material + Thickness

  • Quando esta caixa de seleção é marcada, o sistema espera que a planilha use a coluna Raw-Material (por exemplo, Steel-20). Isso significa que o tipo de material e espessura devem ser combinados em um único valor em uma coluna.

  • Quando esta caixa de seleção é desmarcada, o sistema espera que a planilha tenha colunas separadas para Material e Thickness (por exemplo, Steel e 2.00). O sistema os combina internamente durante a importação.

Pasta monitorada usando CSV/XLSX

  • Em factory → Settings→ Watch settings marque Enable 'Live Folder' e Watch sub-folders also.

  • Defina o caminho da pasta monitorada (UNC ou unidade mapeada).

  • Em Spreadsheet import settings, marque Import spreadsheet from 'Live Folder'.

  • Selecione Import Jobs nas opções de Spreadsheet Import.

  • Copie C:\Program Files\Metamation\Praxis\Samples\Jobs\C0\C0.csv e cole na Pasta Monitorada.

  • O sistema verifica o CSV e tenta criar o trabalho automaticamente.

  • Se o arquivo CAD estiver ausente, nenhum trabalho é criado porque os dados CAD necessários não foram encontrados.

Diretório de consulta CAD e pesquisa CAD recursiva - não configurado

  • Em factory → Settings → Job → Spreadsheet import settings, se nenhum CAD lookup directories está definido e Use recursive CAD search não é usado, o sistema não pesquisa arquivos CAD automaticamente.

  • Se a caixa de seleção Create a blank part for the missing CAD files estiver marcada, o trabalho ainda será criado mesmo quando arquivos CAD estiverem ausentes.

  • Neste caso, todas as peças no trabalho serão criadas como peças nascent (em branco) — significando que não têm geometria porque os arquivos CAD não puderam ser encontrados.

Diretório de consulta CAD e pesquisa CAD recursiva - configurado

Quando CAD lookup directories está definido para (C:\Program Files\Metamation\Praxis\Samples) e Use recursive CAD search está habilitado em factory → Settings → Job → Spreadsheet import settings:

  • O sistema pesquisa automaticamente a pasta especificada e todas as suas subpastas para os arquivos CAD necessários mencionados na planilha.

  • Create a blank part for the missing CAD files também está habilitado, então se algum arquivo CAD ainda não for encontrado, o sistema criará uma peça em branco como espaço reservado.
