Různá vylepšení

Do doplňku SolidWorks byla provedena řada vylepšení pro zlepšení celkové uživatelské zkušenosti, výkonu a stability.

  • Sjednocené přihlašování: Doplněk nyní využívá přihlašovací rozhraní TruSmartCAM. Pro obě aplikace je použita stejná validace přihlášení a hesla. Funkce „Zapamatovat si mě" funguje bezproblémově v TruSmartCAM i doplňku SolidWorks, čímž eliminuje potřebu opakovaného přihlašování.

  • Požadavek na API licenci: Doplněk nyní vyžaduje dostupnou API licenci pro provoz.

    • Pokud není k dispozici API licence, zobrazí se chyba licence.

  • Validovaný oddělovač název-konfigurace: Text oddělovače název-konfigurace je nyní validován, aby bylo zajištěno, že jsou použity pouze platné znaky cesty. To zabraňuje neplatným znakům v názvech souborů při ukládání dílů s více konfiguracemi.

  • Vylepšený systém zpráv: Systémové zprávy, varování a chyby se nyní zobrazují v oblasti stavu vlevo dole v pracovním prostoru SolidWorks, podobně jako v rozhraní TruSmartCAM.
