Import zakázek

Import zakázek

Import z csv nebo xlsx

Zakázky můžete také importovat v běžných textových formátech

  • Klikněte na File → Open → Import úloh, zobrazí se dialogové okno Open Folder. Vyberte C:\Program Files\Metamation\Praxis\Samples\Jobs\C0\C0.csv.

  • Pokud všechny díly existují v umístění csv nebo v adresářích vyhledávací složky, objeví se okno nová úloha se seznamem všech dílů odpovídajících polím csv.

  • Chybějící díly jsou ignorovány a nezobrazí se v okně nová úloha.

  • Zpočátku je stav dílu Nascent.

  • Klikněte na OK, nyní jsou díly importovány do knihovny dílů. CAD validace, nástroje pro ohýbací a řezací instance nástrojů jsou provedeny, což lze sledovat na obrazovce Pmonitor.

Vytvořit prázdný díl pro chybějící CAD soubory

Při importu zakázek pomocí tabulky (.csv nebo .xlsx) můžete někdy mít řádky odkazující na CAD soubory, které nejsou v systému k dispozici.

  • Pro řešení tohoto problému přejděte do závod → Nastavení → Úloha, v Import tabulkového procesoru povolte Vytvořit prázdný díl pro chybějící CAD soubory

Když je tato možnost vybrána:

  • Pokud chybí CAD soubor pro jakýkoli řádek během importu zakázky, systém automaticky vytvoří prázdný díl namísto chybějícího CAD souboru.

  • V okně nová úloha jsou tyto chybějící díly vizuálně zvýrazněny žlutou barvou, aby bylo zřejmé, že přidružený CAD soubor chybí.

  • Chybějící díl bude stále mít svá mapovaná pole (jako název dílu, materiál, tloušťka, množství atd.) z CSV. Jeho stav dílu však bude zobrazen jako Nascent, což znamená, že se jedná pouze o prázdný zástupný symbol bez skutečné geometrie.

Tím je zajištěno, že import zakázky může stále pokračovat bez chyb kvůli chybějícím CAD souborům — prázdný díl můžete později aktualizovat nebo nahradit správným CAD souborem, jakmile bude k dispozici.

Nezobrazovat dialogové okno Nové úlohy když má tabulkový procesor chybu

závod → Nastavení → Úloha → Import tabulkového procesoru→ Nezobrazovat dialogové okno Nové úlohy když má tabulkový procesor chybu

  • Když je toto zaškrtávací políčko vybráno, systém automaticky vytvoří novou zakázku bez zobrazení dialogu New Job, pokud tabulka neobsahuje žádné chyby (žádné chybějící díly, chybějící data nebo neplatná data).

  • Když je toto zaškrtávací políčko zrušeno, dialog New Job se vždy zobrazí pro kontrolu a potvrzení uživatelem, i když tabulka neobsahuje žádné chyby.

Prefer raw-material column over material + thickness

závod → Nastavení → Úloha→Prefer Raw-Material column over Material + Thickness

  • Když je toto zaškrtávací políčko vybráno, systém očekává, že tabulka použije sloupec Raw-Material (například Steel-20). To znamená, že typ materiálu a tloušťka by měly být sloučeny do jedné hodnoty v jednom sloupci.

  • Když je toto zaškrtávací políčko zrušeno, systém očekává, že tabulka bude mít samostatné sloupce pro Material a Thickness (například Steel a 2.00). Systém je interně kombinuje během importu.

Sledovaná složka pomocí CSV/XLSX

  • V závod → Nastavení→ Sledovat zaškrtněte Umožnit 'Live složku' a Sledovat také pod-složky.

  • Nastavte cestu ke sledované složce (UNC nebo mapovaná jednotka).

  • V Import tabulkového procesoru zaškrtněte Import tabulkového procesoru z 'Live složky'.

  • Vyberte Import úloh z možností Spreadsheet Import.

  • Zkopírujte C:\Program Files\Metamation\Praxis\Samples\Jobs\C0\C0.csv a vložte jej do sledované složky.

  • Systém zkontroluje CSV a pokusí se automaticky vytvořit zakázku.

  • Pokud CAD soubor chybí, není vytvořena žádná zakázka, protože požadovaná CAD data nejsou nalezena.

Adresář vyhledávání CAD a rekurzivní vyhledávání CAD - není nakonfigurováno

  • V závod → Nastavení → Úloha → Import tabulkového procesoru, pokud není nastaven CAD lookup directories a Použít rekurzivní CAD hledání není používáno, systém automaticky nevyhledává CAD soubory.

  • Pokud je zaškrtnuto políčko Vytvořit prázdný díl pro chybějící CAD soubory, zakázka bude stále vytvořena, i když CAD soubory chybí.

  • V tomto případě budou všechny díly v zakázce vytvořeny jako nascent (prázdné) díly — což znamená, že nemají žádnou geometrii, protože CAD soubory nebyly nalezeny.

Adresář vyhledávání CAD a rekurzivní vyhledávání CAD - nakonfigurováno

Když je Vyhledat adresáře CAD nastaveno na (C:\Program Files\Metamation\Praxis\Samples) a Použít rekurzivní CAD hledání je povoleno v závod → Nastavení → Úloha → Import tabulkového procesoru:

  • Systém automaticky vyhledává požadované CAD soubory uvedené v tabulce v zadané složce a všech jejích podsložkách.

  • Vytvořit prázdný díl pro chybějící CAD soubory je také povoleno, takže pokud stále není nalezen žádný CAD soubor, systém vytvoří prázdný díl jako zástupný symbol.
