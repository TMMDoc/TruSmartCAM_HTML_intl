Import dílů

Import dílů

Před importem dílů se ujistěte, že formáty souborů jsou kompatibilní s TruSmartCAM. Další informace naleznete v podporované formáty souborů.

Pokud po importu nejsou k dílu přiřazeny materiál a tloušťka, přiřaďte materiál pomocí panelu dílů.

Metody importu

Existují tři metody importu dílů do knihovny dílů:

  1. Import pomocí souboru

  2. Přetažení myší

  3. Sledovaná složka

Import pomocí souboru

2D nebo 3D soubor dílu lze importovat do knihovny dílů pomocí možnosti Import dílů.

  • Klikněte na ikonu otevřít v příkazovém řádku a vyberte možnost Import dílů.

  • Zobrazí se dialogové okno Open — vyberte jakýkoli CAD soubor (2D nebo 3D).

  • Zobrazí se náhled vybraného CAD souboru.

  • Vyberte požadovaný díl a klikněte na Open.

  • Systém automaticky nástroje díl pro všechny dostupné stroje s aktualizovaným stavem nástrojového vybavení.

Přetažení myší

  • Přetáhněte soubor dílu přímo do knihovny dílů TruSmartCAM.

Sledovaná složka

Nastavení sledování umožňuje monitorovat složku pro nové soubory a automaticky je importovat do knihovny dílů.

Pro konfiguraci viz stránka nastavení sledování.

  • Používejte UNC (Universal Naming Convention) cestu \\192.168.1.11\<Folder path> nebo mapovanou jednotku M:\<Folder Path>.

  • Počítač s běžícím Engine musí mít práva čtení/zápisu do této složky.

  • Nepoužívejte místní cestu jako C:\<Folder Path>, protože služba sledování hledá soubory na počítači Engine — ne na místním počítači.

Konfigurace importu

Použít porovnání geometrie u stejné revize CAD

Při importu dílu se stejnou CAD revizí může TruSmartCAM porovnat geometrii CAD souboru s existujícím dílem.

  • Pro povolení této funkce nakonfigurujte Vyhledat revize - testovací pole (například REV) v závod → nastavení →Import / export. Poté povolte možnost Použít porovnání geometrie u stejné revize CAD.

Porovnání geometrie u stejné CAD revize funguje pouze tehdy, když jsou nakonfigurovány Vyhledat revize - testovací pole.

  • Nyní importujte díl do knihovny dílů s konkrétní revizí (například Revision 1).

  • Stejný díl je importován znovu se stejnou hodnotou revize.

  • TruSmartCAM porovná geometrii nově importovaného CAD souboru s existujícím dílem.

  • Pokud je zjištěn rozdíl v geometrii, TruSmartCAM zobrazí výzvu s dotazem, zda:

    • Nahradit existující díl novou geometrií při zachování stejné hodnoty revize.

    • Přeskočit import a zachovat existující díl a jeho geometrii.

To umožňuje uživatelům aktualizovat geometrii dílu bez změny hodnoty revize, přičemž si stále udržují kontrolu nad tím, zda jsou existující díly nahrazeny.

Importované díly jsou automaticky přidány do knihovny pro další použití. Další informace naleznete v Library.
