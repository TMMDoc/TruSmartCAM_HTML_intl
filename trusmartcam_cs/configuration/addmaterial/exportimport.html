Export/Import materiálu

Použití textového formátu

Pro export inventáře materiálů přejděte na závod → plechy, klikněte na Export / import nastavení surového materiálu, vyberte Export, zvolte .txt a uložte do cílové složky.

Otevřete exportovaný soubor .csv, přidejte nový surový materiál a plech s potřebnými údaji a soubor uložte.

Pro import aktualizovaného souboru přejděte na závod → plechy → Export / import nastavení surového materiálu, vyberte Import, zvolte upravený soubor TXT a nově přidané materiály a plechy se zobrazí v TruSmartCAM.

Použití formátu CSV

  • Klikněte na závod → plechy → Export / import nastavení surového materiálu → Exportovat plechy. Zvolte soubor CSV a uložte do cílové složky.

  • Přidejte plechy a parametry plechů, uložte soubor .csv.

  • Pomocí Importovat plechy zvolte upravený soubor .csv.

  • Nově přidané materiály a plechy se zobrazí v TruSmartCAM.

Import materiálu musí být speciálně vytvořený textový soubor, který odpovídá následujícímu formátu.

Material-Name=[BendMaterial],[Density],[FName];
RawMaterial-Thickness,[Name]=[Film],[Treatment],[Surface];
Sheet-Width,Height,Grain,Desc,HeatNumber=Qty,[Factory],[Shelf],[Cost],[Comments]

Příklad: Aluminum0.5,Aluminum-051500,1500,None,=112,,,22,2500,1250,Horizontal,=105,,,7,2500,1500,None,=100

Pro snadné vytvoření importního souboru nejprve exportujte materiály a poté v něm proveďte úpravy nebo zobrazte strukturu.
