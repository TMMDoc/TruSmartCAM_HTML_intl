Sledování

Nastavení sledování umožňuje TruSmartCAM monitorovat určenou složku a automaticky importovat soubory do systému. Toto pomáhá automatizovat import součástí, zakázek, sestav a objednávek bez manuálního zásahu.

Sledovat nastavení

Umožnit 'Live složku' - Povoluje automatické monitorování specifikované složky.

Sledovat tuto složku - Specifikuje složku, která má být monitorována pro příchozí soubory.

Sledovat také pod-složky - Zahrnuje všechny podsložky do procesu monitorování.

Po importu souboru

Po importu - Po importu souboru má uživatel dvě možnosti. Soubor může být smazán z umístění nebo přesunut do jiného umístění.

  • Delete imported file - Odstraní zdrojový soubor po úspěšném importu.

  • Move imported file - Přesune zdrojový soubor do složky specifikované v Move files to this folder po úspěšném importu.

Přesunout soubory do této složky - Specifikuje cílovou složku, kam by měly být úspěšně importované soubory přesunuty, když After import je nastaven na Move imported file.

Přesunout tabulkové procesory s chybami do této složky - Přesouvá neúspěšné importy do specifikované chybové složky.

Nastavení tabulkového procesoru

Import tabulkového procesoru z 'Live složky' - Automaticky importuje soubory tabulek detekované ve sledované složce.

Neimportovat tabulkový procesor s chybami - Zabraňuje importu souborů obsahujících validační chyby.

Opce import tabulkového procesoru - Vybírá typ dat k importu z rozbalovacího seznamu. Podporované formáty tabulek mohou zahrnovat CSV, XLSX nebo XML. Dostupné možnosti jsou:

  • Import úloh - Importuje data zakázek z tabulky.

  • Import složek - Importuje data objednávek z tabulky.

  • Import konstrukčních skupin - Importuje data sestav z tabulky.

  • Import dílů - Importuje data součástí z tabulky.

Additional information

  • Použijte UNC cestu (\\ServerName\FolderPath) nebo mapovaný disk (M:\FolderPath) při konfiguraci sledované složky. Nepoužívejte lokální cesty jako C:\FolderPath.

  • Počítač provozující službu TruSmartCAM Engine musí mít oprávnění ke čtení a zápisu do konfigurované sledované složky. Bez dostatečných oprávnění nemusí být soubory detekovány, importovány, přesunuty nebo úspěšně smazány.

  • Mapování tabulek musí být nakonfigurováno před povolením nastavení sledování. Jinak TruSmartCAM nebude schopen zpracovat soubory tabulek detekované ve sledované složce.
