Πίνακας εργασίας

Αυτή η ενότητα εξηγεί τις ενέργειες που σχετίζονται με τις εργασίες όπως Export outputs, Επαναφορά εργασίας…​, Σήμανση ως ολοκληρωμένη…​, Εξαγωγή εργασίας…​ και άλλες επιλογές που χρησιμοποιούνται για τη διαχείριση των παρουσιών εργασίας.

Άνοιγμα σε κατάσταση μόνο για ανάγνωση για προβολή
Ανοίγει την επιλεγμένη εργασία σε λειτουργία μόνο για ανάγνωση.

Check-out the document for editing
Ενεργοποιεί την επεξεργασία της επιλεγμένης εργασίας.

Διαγραφή
Διαγράφει την επιλεγμένη εργασία.

Προβολή αναφοράς
Ανοίγει την αναφορά παραγωγής για την επιλεγμένη εργασία.

Εξαγωγή αποτελεσμάτων
Εξάγει τα έτοιμα για μηχάνημα αρχεία εξόδου (αρχεία NC και αναφορές) που δημιουργήθηκαν για την επιλεγμένη εργασία.

Plan job
Δημιουργεί διατάξεις τοποθέτησης για τα εξαρτήματα στην επιλεγμένη εργασία.

Επαναφορά εργασίας…​
Αφαιρεί όλες τις δημιουργημένες διατάξεις για την εργασία και επαναφέρει τα εξαρτήματά της στην κατάσταση που μπορούν να σχεδιαστούν.

Σήμανση ως ολοκληρωμένη…​
Σημειώνει την εργασία και όλες τις τοποθετήσεις της ως ολοκληρωμένες μετά την παραγωγή.

Εξαγωγή εργασίας…​
Εξάγει την πλήρη εργασία ως πακέτο που μπορεί να επαναχρησιμοποιηθεί ή να εισαχθεί σε άλλο σύστημα.

Προηγ. / Επόμενο / Διάταξη
Πλοηγείται στην προηγούμενη ή την επόμενη διάταξη εντός της επιλεγμένης εργασίας.

Sort and filter
Χρησιμοποιήστε αυτά τα φίλτρα για να δείτε άλλες εργασίες που μοιράζονται το ίδιο υλικό, πελάτη, ημερομηνία παράδοσης ή προτεραιότητα.

Εξαγωγή αποτελεσμάτων

  • Επιλέξτε την απαιτούμενη εργασία.

  • Κάντε κλικ στο Εξαγωγή αποτελεσμάτων.

  • Αυτό εξάγει τα έτοιμα για μηχάνημα αρχεία εξόδου που δημιουργήθηκαν για την επιλεγμένη εργασία.

  • Τα Export Outputs μπορούν να αποθηκευτούν είτε στην προεπιλεγμένη τοποθεσία που έχει ρυθμιστεί στις Εργοστασιακές ρυθμίσεις είτε σε προσαρμοσμένη τοποθεσία χρησιμοποιώντας Περιήγηση…​.

  • Εξάγει μόνο αρχεία εξόδου μηχανήματος (όπως NC και αναφορές) για την εργασία.

  • Αυτά τα αρχεία χρησιμοποιούνται απευθείας για παραγωγή και μπορούν να σταλούν στο μηχάνημα.

Επαναφορά εργασίας…​

Επαναφέρει την επιλεγμένη εργασία αφαιρώντας όλες τις δημιουργημένες διατάξεις και καθιστώντας τα εξαρτήματα διαθέσιμα για σχεδιασμό ξανά.

Για να επαναφέρετε μια εργασία:

  • Κάντε δεξί κλικ στην εργασία και επιλέξτε Επαναφορά εργασίας…​.

  • Αυτό διαγράφει όλες τις υπάρχουσες διατάξεις τοποθέτησης για την επιλεγμένη εργασία και επαναφέρει όλα τα εξαρτήματά της στην κατάσταση που μπορούν να σχεδιαστούν.

  • Εάν οι διατάξεις περιέχουν εξαρτήματα από άλλες εργασίες, αυτά τα εξαρτήματα επίσης επαναφέρονται.

Εξαγωγή εργασίας…​

  • Επιλέξτε την απαιτούμενη εργασία.

  • Κάντε κλικ στο Εξαγωγή εργασίας…​.

  • Αυτή η επιλογή εξάγει την πλήρη εργασία ως πακέτο που μπορεί να επαναχρησιμοποιηθεί ή να εισαχθεί σε άλλο σύστημα.

  • Τα εξαγόμενα δεδομένα περιλαμβάνουν όλες τις πληροφορίες που απαιτούνται για την επαναδημιουργία της εργασίας:

    • Λεπτομέρειες και ποσότητες εξαρτημάτων (αρχεία CSV).

    • Δεδομένα διάταξης και τοποθέτησης (αρχεία CSV / FXYLT).

  • Τα εξαγόμενα αρχεία αποθηκεύονται ως φάκελος εργασίας στη ρυθμισμένη τοποθεσία εξαγωγής.

  • Αυτή η επιλογή χρησιμοποιείται για τη μεταφορά, την κοινοποίηση ή τη δημιουργία αντιγράφων ασφαλείας ολόκληρης της εργασίας.

Σήμανση ως ολοκληρωμένη…​

  • Μόλις η εργασία υποβληθεί σε επεξεργασία στο μηχάνημα, ανοίξτε τον πίνακα Job και επιλέξτε Σήμανση ως ολοκληρωμένη…​.

  • Όταν μια εργασία σημειωθεί ως ολοκληρωμένη:

    • Όλες οι τοποθετήσεις κάτω από την εργασία σημειώνονται ως ολοκληρωμένες.

    • Η εργασία γκριζάρει, μετακινείται στο τέλος της λίστας και εμφανίζεται με ένα πράσινο σημάδι ελέγχου.

    • Όλες οι διατάξεις κάτω από την εργασία αφαιρούνται από τη λίστα ενεργών διατάξεων.

    • Οι καταστάσεις εξαρτημάτων και εργασίας ενημερώνονται και οι επεξεργασμένες ποσότητες μετακινούνται στην ολοκληρωμένη κατάσταση.

    • Τα αρχεία εξόδου για όλες τις διατάξεις στην εργασία αφαιρούνται από την τοποθεσία εξόδου του μηχανήματος για να αποφευχθεί η διπλή χρήση.
