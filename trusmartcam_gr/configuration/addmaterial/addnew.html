Προσθήκη νέου υλικού και φύλλου

Η διαχείριση φύλλων είναι προσβάσιμη κάνοντας κλικ στο εικονίδιο εργοστασιακές και επιλέγοντας μεταλλικές πλάκες. Αυτό παρέχει μια επισκόπηση των υλικών που έχουν ρυθμιστεί. Η επιλογή οποιουδήποτε υλικού που απαριθμείται στα αριστερά εμφανίζει τα διαθέσιμα φύλλα, πάχη, μεγέθη και ποσότητες.

Για να προσθέσετε νέο υλικό, κάντε κλικ στο Προσθήκη νέων στοιχείων → Νέο υλικό…​ από τη σελίδα μεταλλικές πλάκες στο εργοστασιακές.

Εισαγάγετε Υλικό, Πρώτη ύλη, Πάχος και παραμέτρους υλικού. Κάντε κλικ στο OK.

Προβολή υλικού

Η επιλογή _Προβολή…​ στο εργοστασιακές → *μεταλλικές πλάκες έχει τις ακόλουθες επιλογές:

  • *Επέκταση όλων: Επεκτείνει όλες τις κατηγορίες υλικών.

  • *Σύμπτυξη όλων: Συμπτύσσει όλες τις κατηγορίες υλικών.

  • *Προβολή όλων των υλικών: Απαριθμεί όλα τα ακατέργαστα υλικά ανεξάρτητα από την απογραφή φύλλων.

  • *Προβολή υλικών αποθέματος: Απαριθμεί μόνο τα ακατέργαστα υλικά που έχουν απογραφή φύλλων.

Το πάχος ακατέργαστου υλικού που εμφανίζεται με έντονη γραφή αντιπροσωπεύει τη διαθεσιμότητα φύλλου.

Προσθήκη φύλλου

Για να προσθέσετε νέο φύλλο, κάντε κλικ στο Προσθήκη νέων στοιχείων → Νέα μεταλλική πλάκα από τη σελίδα Sheets στο *εργοστασιακές. Τροποποιήστε τις παραμέτρους του φύλλου.

  • Αναζήτηση…​ (Ctrl+F) (Ctrl + F)…​: Φιλτράρει φύλλα με βάση το μέγεθος, την κατεύθυνση ινών, τα σχόλια, τον αριθμό θερμότητας, το υπόλοιπο και την εξάτμιση.

  • Slider (+ / –): Προσαρμόζει το επίπεδο ζουμ της προεπισκόπησης φύλλου.

  • Filter by: Ελέγχει την προεπισκόπηση φύλλου με βάση την επιλεγμένη επιλογή (Πλάτος, Ύψος, Περιοχή, Ποσότητα, Grain, ή Τύπος).

Η επεξεργασία ή η διαγραφή ενός υλικού ή φύλλου μπορεί να γίνει μόνο πριν εκχωρηθεί ένα εξάρτημα σε αυτό το υλικό ή φωλιαστεί μια διάταξη σε αυτό το φύλλο.

Η αλλαγή του ονόματος του υλικού δεν είναι δυνατή σε αυτό το στάδιο. Επομένως, πρέπει να δημιουργηθεί νέο υλικό.
