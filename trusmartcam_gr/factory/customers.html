Customers

Η σελίδα Customers σας επιτρέπει να διαχειρίζεστε όλες τις εγγραφές πελατών στο σύστημα. Μπορείτε να προσθέσετε νέους πελάτες, να ενημερώσετε υπάρχοντα στοιχεία και να αφαιρέσετε πελάτες που δεν απαιτούνται πλέον.

  • Κάντε κλικ στο εργοστασιακές → Πελάτες.

  • Για να δημιουργήσετε έναν νέο πελάτη, κάντε κλικ στο Προσθήκη _νέου…​ και παρέχετε τις απαιτούμενες πληροφορίες. Το πεδίο Name είναι υποχρεωτικό και πρέπει να είναι μοναδικό, πράγμα που σημαίνει ότι δεν μπορούν δύο πελάτες να έχουν το ίδιο όνομα. Τα άλλα πεδία είναι προαιρετικά.

  • Για να ενημερώσετε τα στοιχεία ενός πελάτη, κάντε δεξί κλικ στον πελάτη και επιλέξτε Επεξεργασία πληροφοριών…​. Όλα τα πεδία εκτός από το Name μπορούν να τροποποιηθούν.

  • Για να αφαιρέσετε έναν πελάτη, κάντε δεξί κλικ στον πελάτη και επιλέξτε Διαγραφή.

Η διαγραφή είναι δυνατή μόνο εάν ο πελάτης δεν είναι συνδεδεμένος με κάποιο Part ή Job. Διαφορετικά, η επιλογή διαγραφής θα είναι απενεργοποιημένη.
