Απαιτήσεις Θυρών

TruSmartCAM Server : 7371

Ο Server αναμένει τα αιτήματα των client σε αυτήν τη θύρα TCP. Οι εισερχόμενες συνδέσεις πρέπει να επιτρέπονται σε αυτήν τη θύρα στον Υπολογιστή Server.

Floating License Server : 5776

Ο Floating License Server αναμένει αιτήματα αδειοδότησης σε αυτήν τη θύρα. Οι εισερχόμενες συνδέσεις πρέπει να επιτρέπονται σε αυτήν τη θύρα στον υπολογιστή floating license server (TruSmartCAM Server).

Σύνδεση άδειας χρήσης με το Cloud : 7567

Αυτή η θύρα χρησιμοποιείται από το TecZone (Λογισμικό Κάμψης) και τον Floating License Server για αποστολή αιτημάτων αδειοδότησης στο Metamation Cloud. Επομένως, η εξερχόμενη σύνδεση πρέπει να επιτρέπεται σε αυτήν τη θύρα για ενεργοποιήσεις και ανανεώσεις αδειών.

Απαιτήσεις Εγκατάστασης και Υποστήριξης

  • Απαιτούνται δικαιώματα διαχειριστή για την εγκατάσταση της εφαρμογής TruSmartCAM.

  • Εργαλεία απομακρυσμένης διάγνωσης μέσω Contractor VPN ή ασφαλούς σύνδεσης.
