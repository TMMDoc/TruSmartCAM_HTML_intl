订单视图

"订单"页面右侧的选项表示基于处理阶段的订单的不同视图。选择一个视图会更新显示的订单，使您能够专注于工作流的特定阶段。

Ready

选择 Ready 视图会显示已准备好但尚未转换为作业的订单。

  • 显示已准备好创建作业的订单。

  • 用于识别待处理的订单。

Active

选择 Active 视图会显示已转换为作业且当前正在处理的订单。

  • 表示已移至作业阶段的订单。

  • 帮助跟踪正在执行的订单。

  • 用于监控正在进行的生产。

Done

选择 Done 视图会显示已完成处理的订单。

  • 显示相关作业已完成的订单。

  • 指示工作流已完成。

  • 用于审查和记录保存。

All

选择 All 视图会显示所有订单，无论其处于哪个阶段。

  • 提供所有订单的完整概览。

  • 用于跨所有阶段进行搜索和管理。

订单状态栏

每个订单磁贴下方显示的状态栏指示订单是否链接到作业。有两种类型：

  • 无状态栏 - 订单未分配给任何作业。

  • 有状态栏 - 订单是作业的一部分，状态栏显示如下所列的相应作业状态。
