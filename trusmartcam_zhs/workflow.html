工作流程

工作流程

TruSmartCAM 是一个制造执行系统，用于管理和优化钣金制造流程。TruSmartCAM 中的工作流程包括一系列步骤，您可以按照这些步骤完成特定任务或流程。您可以自定义工作流程以适应不同的制造流程，并根据您的特定需求进行调整。TruSmartCAM 中的工作流程通常包括以下步骤：

1. 配置系统

配置 TruSmartCAM 以匹配您的制造环境。此基础步骤包括定义具有其功能和参数的机器、添加可用的材料和厚度范围、配置包含冲头和模具的工装库、为不同的机器配置创建设置表，以及管理具有适当权限的用户帐户。正确配置的系统可确保准确处理并防止后续阶段出现错误。此设置通常在初始部署期间执行一次，并随着工厂的发展进行定期更新。

有关更多信息，请参阅 Add/Remove machine、Material、 Sheets、Setup tooling 和 Configure users。

2. 将零件添加到库中

使用支持的格式（如 DXF、DWG 或 STEP 文件）将 2D 或 3D CAD 零件导入到零件库中。在导入期间，TruSmartCAM 会自动分析零件几何形状、验证可制造性、分配所需的钣金技术（激光、冲压或组合），并生成预览缩略图。导入的零件在库视图中显示为磁贴，其中彩色编码的图标指示零件状态、技术类型、材料要求和任何检测到的问题。此可视化系统允许快速一目了然地评估零件就绪状态。

有关更多信息，请参阅 Library。

3. 验证零件和工装

检查每个导入的零件是否存在潜在的生产问题。验证零件几何形状是否有效且可制造、工装分配是否正确且可用、材料和厚度选择是否与库存匹配，以及成型零件的弯曲顺序是否正确定义。如果检测到问题，请在 TecZone 中打开零件以修改几何形状、重新分配或优化工装、调整加工参数或更正材料规格。在此阶段解决问题可防止生产延迟并减少制造过程中的废品。

有关更多信息，请参阅 Part state 和 Tooling status。

4. 将零件组织到作业中

根据生产要求将经过验证的零件分组到作业中。作业代表一个逻辑生产单元，可能包括来自单个客户订单的零件、共享相同材料和厚度的零件、具有相似交货期限的零件，或根据您的生产策略的组合。作业为规划提供容器，允许对相关零件进行批量处理，通过套料实现高效的材料利用，并简化跟踪和报告。作业可以通过选择零件手动创建，也可以根据预定义的规则和过滤器自动创建。

有关更多信息，请参阅 Jobs。

5. 将作业规划到生产机器

创建作业后，将其分配给适当的生产机器。考虑机器功能，包括支持的技术（激光、冲压、转塔）、材料和厚度容量、可用工装和当前工作负载。如果存在多台合适的机器，请根据生产优先级、机器可用性、设置要求或成本优化进行选择。分配可以是手动的以提高灵活性，也可以是基于预配置的规则和机器调度算法的自动分配。

有关更多信息，请参阅 Plan job。

6. 为作业发布 NC

在此阶段，生成切割布局。您仍然可以编辑布局并使用 TecZone 向布局添加切割技术。之后，导出 NC 输出并将其传输到机器控制器或指定的网络位置。然后锁定作业以防止进一步修改，并将其状态更新为"已发布"以进行生产跟踪。

有关更多信息，请参阅 Send/Recall NC。

7. 完成生产并最终确定作业

在分配的机器上执行已发布的作业。在生产过程中，机器操作员遵循 NC 程序、在切割期间验证零件质量、管理材料装载和卸载，并处理任何机器警报或问题。生产完成后，在 TruSmartCAM 中将单个零件或整个作业标记为完成。此最后步骤更新库存记录、记录实际生产时间和材料使用情况、存档作业数据以进行报告和分析，并完成工作流程循环。

有关更多信息，请参阅 Mark complete。
