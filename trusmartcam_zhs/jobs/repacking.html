重新打包

TruSmartCAM 没有单独的重新打包模块。相反，重新打包通过作业级别选项进行处理，这为如何组合零件提供了灵活性。

有三种方式可以进行重新打包。

  • 使用 混合多个套料的零件 进行重新打包。

  • 使用 添加零件… 交互式向导进行手动重新打包。

  • 使用 TecZone 进行手动重新打包。

使用 混合多个套料的零件 进行重新打包

  • 要启用此选项，请转到 出厂 → 设置 → 套料 → 排样设置 → 混合多个套料的零件。

  • 启用后，此选项允许在自动排样或交互式排样期间将来自不同作业的零件分组在一起，从而产生组合打包或排样。

  • 例如：

    • 作业-1 有 1 个零件，作业-2 有 1 个零件。

    • 启用 混合多个套料的零件 后，TruSmartCAM 可以将这 2 个零件重新打包到单个优化的排样中。

  • 如果没有此选项，每个作业的零件在排样/打包期间保持分离，具有不同的布局名称。

  • 也可以在不启用此选项的情况下手动执行重新打包。

使用添加零件交互式向导进行手动重新打包

  • 从布局视图中，选择要重新打包的零件。

  • 右键单击零件，然后从布局面板中选择 添加零件…。

  • 按照向导中的提示完成重新打包过程。

  • 选择 显示其他套料零件 并将它们添加到当前作业。

  • 查看更改并确认重新打包。

有关更多详细信息，请参阅 添加零件。

使用 TecZone 进行手动重新打包

  • 通过单击 打开 TecZone。

  • 在 TecZone 界面中，单击添加。

  • 将显示一个新窗口，您可以在其中选择要重新打包的零件。单击 显示… 然后选择 其他套料零件。做出选择后，单击 选中。

  • 将显示一个新窗口，您可以在其中定义零件需要重新打包的位置。

  • 单击 将更改保存到 Praxis，重新打包的布局将在 TruSmartCAM 中可用。
