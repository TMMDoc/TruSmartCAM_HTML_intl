自动余料管理

启用余料

TruSmartCAM 支持在排样期间从布局自动创建余料板材。此功能仅适用于 TecZone Laser 机床。

  • 单击 出厂 → 机床 → 机床默认设置… → 骨架切割。

  • 选择 切割板材骨架 和 创建余料板材 选项以启用此功能。

启用后，排样板材的剩余区域将根据在 X 和 Y 方向分配的最小余料板材值切割成余料板材。

配置自动余料

  • 单击 出厂 → 设置 → 套料。

  • 选择 排样（未）释放时更新板材库存 和 自动残料库存管理。

自动余料

当您排样零件时，如果剩余板材区域满足最小板材值 X 和 Y，TruSmartCAM 会自动从剩余空间创建余料板材。

  • 在作业排样视图中，具有余料板材的排样显示为绿色边框轮廓和灰色背景。

  • 双击排样以打开排样详细信息页面。显示余料板材信息，包括余料板材尺寸、数量和面积。

  • 右键单击排样并选择编辑以在 TecZone 中打开它。排样显示零件之间的切割线和骨架切割线以及刻有余料尺寸。（材料、厚度和余料尺寸）。

余料尺寸刻字标记帮助操作员在不测量的情况下选择板材，使其易于在未来的排样中使用。

当您单击 标记完成… 时，余料板材显示绿色勾号，表示排样已完成。部分阴影的方形图标表示已创建余料板材并在板材库存中更新。

板材数据库已更新为余料板材的尺寸和数量，该尺寸和数量在板材尺寸末尾用星号 (*) 符号标识。余料板材具有绿色边框。

作业的自动规划或交互式规划将余料板材视为排样计划中的高优先级，确保首先使用它们以保持车间没有未使用的余料。
