导出/导入材料

使用文本格式

要导出库存材料，请转到 出厂 → 板材，点击 导出/导入原材料设置， 选择 Export，选择 .txt，并将其保存到目标文件夹。

打开导出的 .csv 文件，添加新的原材料和板材以及必要的详细信息，然后保存文件。

要导入更新的文件，请转到 出厂 → 板材 → 导出/导入原材料设置，选择 Import，选择修改后的 TXT 文件，新添加的材料和板材将出现在 TruSmartCAM 中。

使用 CSV 格式

  • 点击 出厂 → 板材 → 导出/导入原材料设置 → 导出板材。选择 CSV 文件并保存到目标文件夹。

  • 添加板材和板材参数，保存 .csv 文件。

  • 使用 导入板材，选择修改后的 .csv 文件。

  • 新添加的材料和板材将出现在 TruSmartCAM 中。

材料导入需要是一个特殊构造的基于文本的文件，其格式如下。

Material-Name=[BendMaterial],[Density],[FName];
RawMaterial-Thickness,[Name]=[Film],[Treatment],[Surface];
Sheet-Width,Height,Grain,Desc,HeatNumber=Qty,[Factory],[Shelf],[Cost],[Comments]

示例： Aluminum0.5,Aluminum-051500,1500,None,=112,,,22,2500,1250,Horizontal,=105,,,7,2500,1500,None,=100

要轻松创建导入文件，请首先导出材料，然后对其进行修改或查看结构。
