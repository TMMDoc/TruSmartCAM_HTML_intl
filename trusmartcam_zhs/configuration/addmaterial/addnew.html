添加新材料和板材

可以通过点击 出厂 图标并选择 板材 来访问板材管理。这提供了已设置材料的概览。选择左侧列出的任何材料将显示可用的板材、厚度、尺寸和数量。

要添加新材料，请在 出厂 中的 板材 页面点击 添加新物品 → 新材料…。

输入 材料、原材料、 厚度 和材料参数。点击 OK。

查看材料

出厂 → 板材 中的 *查看… 选项具有以下选项：

  • *展开全部：展开所有材料类别。

  • *收起全部：折叠所有材料类别。

  • *显示所有材料：列出所有原材料，无论板材库存如何。

  • *显示库存材料：仅列出具有板材库存的原材料。

以粗体显示的原材料厚度表示有板材可用。

添加板材

要添加新板材，请在 出厂 中的 Sheets 页面点击 添加新物品 → *新板材。修改板材参数。

  • 搜索… (Ctrl+F) (Ctrl + F)…​：根据尺寸、纹理、注释、热号、余料和蒸发过滤板材。

  • 滑块 (+ / –)：调整板材预览的缩放级别。

  • Filter by：根据所选选项（宽度、高度、 区域、数量、 Grain 或 类型）控制板材预览。

编辑或删除材料或板材只能在将零件分配给该材料或在该板材上排版之前完成。

在此阶段无法更改材料名称。因此，必须创建新材料。
