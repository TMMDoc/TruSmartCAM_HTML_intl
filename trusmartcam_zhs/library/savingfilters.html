保存筛选器

已保存的筛选器允许您在零件、作业和排版中重复使用常用条件，从而简化工作流程。每个模块都带有默认筛选器，但站点管理员或编程人员可以根据需要创建其他筛选器列表。重复使用已保存的筛选器可节省时间并有助于保持搜索的一致性。

  • 点击 图标查看默认筛选器。

保存筛选器的步骤

  • 点击 图标，选择任意筛选器项。

  • 从所选筛选器项的下拉菜单中选择一个选项。选中的条件旁边会出现一个勾选标记。

  • 要应用该条件，请点击 图标，然后选择 添加条件 (Ctrl+PLUS)]。您可以重复此步骤以添加多个条件。

  • 要保存这些带有条件的筛选器，请点击 保存过滤器… (Ctrl+S)]。

  • 输入名称，然后点击 保存。

  • 在 保存过滤器 对话框中，您还可以删除任何先前保存的筛选器。

  • 这些筛选器将保存在默认筛选器列表中，此处 材料 被保存到默认列表中。

由 SiteAdmin 和 Admin 创建的筛选器在所有用户之间共享。由 Programmer 和 Operator 创建的筛选器仅在 Programmer 和 Operator 用户角色之间共享。

筛选器命令

命令 图标 快捷键 说明

Clear Selection

Ctrl+X

取消选择所有已选择的选项并刷新筛选器项视图。

Select Highlighted

Ctrl+A

选择所有高亮显示的选项。如果项目高亮显示不适用于所选字段，则此命令保持禁用状态。

Match All

&

仅显示满足所有选定条件的项。

Pin In/Out Menu

Ctrl+Down/Up

固定后，即使当前焦点移动到另一个窗口，下拉菜单也会保持打开状态。

Close Menu

Esc

关闭下拉菜单。
