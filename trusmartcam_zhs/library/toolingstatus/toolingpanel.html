工装面板

本节说明与工装相关的操作，如 导出输出、Check Tooling、隐藏装备、设置为基面平面…​、复制装备… 以及用于管理工装实例的其他选项。

以只读模式打开查看
以只读模式打开所选工装。

Check-out the document for editing
启用所选工装的编辑。

导出输出
将所选工装输出导出到配置的目标。

Check Tooling
验证现有工装而不进行任何更改，如果验证失败则更新其状态。

复制装备…
从所选参考工装复制冲头、顺序和后挡料设置，并在可行的情况下将其应用于其他解决方案。

设置为基面平面…​
将所选折弯或折边工装标记为用于生成切割解决方案的参考平面尺寸。

Re-tool…​
重新处理零件的所选工装，而不影响其他工装。

隐藏装备
抑制所选工装，使其从进一步处理和输出生成中排除。

上一个 / 下一页 / 零件
导航到上一个或下一个工装，或返回到父零件。

Sort and filter
显示与所选工装具有相似特征的其他工装以供快速参考。

导出输出

此选项将零件输出导出到 出厂设置 中配置的目标。（请参阅 Bend Settings 和 Cut Settings）

仅导出与所选工装操作关联的工装，而不是整个零件。

导出的文件取决于操作类型。对于折弯和折边操作，会生成 NC 文件和报告。对于切割和冲压操作，仅导出零件文件。

Check Tooling

  • 如果 CAM 设置发生更改或折弯工具丢失，您不需要重新工装零件。

  • 相反，使用检查工装来验证现有工装而不进行任何更改。

  • 如果验证失败，该实例的工装状态会更新，并删除相关的输出文件。

  • 检查工装还会检测先前使用 忽略错误 手动覆盖错误的零件，并在需要时将其状态更新为错误。

复制装备…

复制工装允许您快速将一个解决方案的折弯工装应用于其他解决方案，从而节省时间和精力。

这会从所选参考工装（解决方案）复制冲头、顺序和后挡料设置，并尝试将相同的设置应用于所有其他解决方案。

  • 如果复制的设置对目标机床可行，则保存并使用。

  • 如果设置不可行，该机床的原始工装保持不变。

例如，您可能对不同的机床有不同的工装：

  • 此处，机床 5230SX 6A 使用鹅颈冲头，机床 5085X 6A B23 使用尖头冲头。

当您在机床 5230SX 6A 上使用 复制装备… 时，它会尝试将鹅颈设置复制到所有其他机床。

相同的冲头、顺序和后挡料设置会尽可能应用。如果无法使用复制的设置，原始工装保持不变。

应用复制工装后，鹅颈设置已成功应用于所有其他可行的机床。

设置为基面平面…​

设置为基面平面…​ 功能定义哪个折弯或折边工装用作生成切割解决方案的参考平面尺寸。当工装标记为 Base-Flat 时，TruSmartCAM 会根据该机床更新零件的平面尺寸，并相应地重新生成切割解决方案。Base-Flat 工装在 UI 中显示为蓝色边框。

不同的折弯机床可能会为同一零件产生平面尺寸的细微变化。当您将工装设置为 Base-Flat 时，TruSmartCAM 使用该机床的平面尺寸作为主参考并自动重新计算切割解决方案。

自动 base flat 选择

当工厂设置 在折弯过程中进行平面修正 启用时：

  • TruSmartCAM 会在零件导入期间自动将折弯工装（包含最大折弯长度表）设置为 Base-Flat。

  • 如果所选折弯工装遇到错误或不可行，TruSmartCAM 会智能切换并将另一个有效的折弯工装设置为 Base-Flat。这确保平面尺寸参考和切割解决方案始终保持一致和有效，无需人工干预。

增强的 base flat 管理

TruSmartCAM 在保存折弯或折边工装编辑时在 保存到 Praxis 对话框中提供 Base-Flat 选项。这些选项仅针对处于正常状态的工装显示，并确保平面尺寸和切割解决方案保持准确。

设置为基面平面并计算切割程序

此选项用于首次将工装标记为 Base-Flat 并自动生成相应的切割解决方案。

出现时机：

  • 工装当前未标记为 Base-Flat。

  • 用户已签出工装进行编辑并正在保存更改。

要使用此选项，请签出折弯或折边工装并进行所需的修改，例如更改折弯参数或平面尺寸。按 Ctrl+S 保存更改。在 保存到 Praxis 对话框中，选择 设置为基面平面并计算切割程序 并点击 确定。TruSmartCAM 将设置工装为 Base-Flat 并使用更新的平面尺寸生成切割解决方案。

更新基面平面并重新计算切割程序

此选项用于工装已标记为 Base-Flat 并已编辑，需要重新生成平面尺寸和切割解决方案时。

出现时机：

  • 工装已标记为 Base-Flat。

  • 用户已签出、编辑并正在保存更改。

编辑现有 Base-Flat 工装时，根据需要更新折弯参数或平面尺寸几何形状。在 保存到 Praxis 对话框中，选择 更新基面平面并重新计算切割程序。然后 TruSmartCAM 将更新 Base-Flat 工装并重新生成所有相关切割解决方案以反映最新更改。

一次只能将一个折弯工装设置为 Base-Flat。

当 Base-Flat 更改时，现有切割解决方案会自动重新处理。

此功能适用于 TruSmartCAM 中的折弯和折边工装。

重新装备…

此选项重新处理零件的所选工装。

在工装面板中右键单击所需的工装并点击 重新装备…

继续之前会显示确认对话框。

点击 是 继续重新工装，或点击 否 取消操作。

仅重新工装所选工装。不影响与零件关联的其他工装。在重新工装过程中删除与所选工装相关的现有输出并重新生成。

当工装数据过时、不正确或对机床或工装配置进行更改时使用此选项。

忽略错误

  • 如果工装实例显示警告，但您仍想继续使用它，可以通过点击 忽略错误 来覆盖警告。

  • 这将允许系统为该实例导出输出文件，即使存在工装问题。

仅在确定可以安全忽略警告时使用 Ignore Errors。覆盖实际错误可能导致机床损坏或安全风险。

抑制/取消抑制工装

  • 当机床不活动或您不希望特定零件由特定机床处理时，使用 隐藏装备 将该零件从该机床的排样中排除。

  • 抑制后，将鼠标悬停在工装实例上会显示类似以下消息："Tooling suppressed by <User name> on <Date Time>"。

  • 抑制工装会删除该机床和零件的输出文件。

  • 使用 显示装备 恢复该机床上零件的工装状态。

  • 取消抑制后，会再次生成相应的输出文件。
