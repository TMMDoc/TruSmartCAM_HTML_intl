零件筛选器

零件库筛选器允许您通过应用特定条件快速搜索和定位库中的零件。通过将结果缩小到仅满足您条件的零件，这些筛选器使管理和访问相关零件变得更加容易高效。

状态

  • 使用 状态 筛选器根据工装状态搜索零件。

  • 下拉列表将工装状态显示为彩色框以及所有可用机床及其适用的错误。

  • 彩色框表示以下状态：

    • 白色：工装正常

    • 橙色：工装有错误

    • 黄色：工装有警告

    • 灰色：工装已抑制

  • 如果零件有窄凸缘或未分配工装错误，选择这些将显示相应的零件。

  • 通过点击灰色框将显示在所选机床上具有抑制工装的零件。

折弯安装

  • 使用 折弯安装 筛选器根据使用的折弯工具搜索零件。

  • 下拉列表显示所有可用的折弯工具，可以按工具名称、类型（冲头、鹅颈、半径等）或描述进行筛选。

  • 选择多个工具并点击 匹配所有|&** 以对使用所选两种工具的工具进行排序。

使用过的模具

  • 按 使用过的模具 筛选的工作方式与按 折弯安装 筛选类似。

  • 工具图标以比例大小显示（不精确，但相对）。

  • 当工装预览打开时，将鼠标悬停在筛选列表中的工具上会在预览窗格中突出显示该工具。

  • 可以选择多个工具并使用 匹配所有|&** 应用。

状态

  • 使用 状态 筛选器根据零件的当前状态搜索零件。

  • 下拉列表显示所有适用的状态，具体取决于可用的库零件。

  • 状态列表根据所选零件及其状态动态变化。

  • 例如，选择 Material Missing 以查看处于未解决材料状态的零件。
