装配体

从装配体创建静态排样

可以使用交互式排样向导为装配体生成静态排样。这些排样是仅与所选装配体关联的固定布局。

  • 切换到 组件图纸 视图。

  • 右键单击所需的装配体并选择 创建排样…。

  • 交互式排样向导打开。

  • 默认情况下，交互式排样向导切换到 选择机床和材料 页面。

  • 机床显示为带有颜色条的形式，指示可用的生产负载：

    • 绿色条：已计划小时数

    • 灰色条：可用小时数（8 小时班次内）

  • 在每台机床下，所选零件按其原材料分组（零件/总数量）。

  • 如果零件对于某台机床没有有效的工装解决方案，则该零件和材料将被忽略，不会在该机床上进行处理。

  • 如果为一台机床选择了原材料，它将不再可用于其他机床。

  • 设置好机床排样计划后，选择首选板材并覆盖板材参数，如 优先级、数量、板材边缘 等。

  • 您可以筛选板材以显示 所有板材、标准板材 或 残留板材。点击 下一页。

在此阶段无法增加数量。您可以保持与板材库存数量相同，或减少数量并检查排样结果。

  • 更新数量 出现在交互式排样计划窗口中。

  • 默认情况下，全部选中 (F5) 包含所有零件进行排样，或者您可以手动选择优先零件并调整零件数量。点击 下一页。

  • 排样开始，排样到板材中的零件状态更新为蓝色。

  • 机床的生产负载根据排样结果更新。

  • 如果有些零件仍未排样，会出现消息 一些零件仍未排样 - 这些零件保留在原始作业中。

    • 如果没有合适的板材可用，零件可能仍未排样。

  • 静态排样可以编辑以包含未排样的零件：

    • 选择要插入未排样零件的布局并将其签出进行编辑。

  • TecZone 打开。点击 选中 或 全部选中 从父装配体中选择未排样的零件。

  • 根据需要在布局中安排零件。

  • 生成的排样会自动以父装配体命名以便于识别。

静态排样不能包含来自其他装配体的零件。允许混合装配体排样会阻止在不释放其他装配体的情况下释放一个装配体，因此设计上不允许混合。

非钣金组件

  • 导入装配体时，会提取钣金零件并将其添加到零件库中的装配体中。

  • 非钣金组件会被分离并添加到 组件 中。

  • 装配体会添加到装配体库中，其 BOM（物料清单）同时包含零件和组件。

  • 如果零件已作为钣金零件导入，系统会更新现有条目并从零件库中删除重复项。

  • 选择装配体会同时显示零件和组件，组件数量显示在组件图像下方（所选装配体组件数和总组件数）。

  • 组件工具提示显示名称、材料和尺寸。组件在生产期间被排除 - 它们不会被排样，并且不会为组件生成板材。

  • 示例装配体和组件位于：C:\Program Files\Metamation\Praxis\Samples\Assys\Components，可用于测试组件导入和装配体工作流程。

从装配体创建作业

  • 右键单击装配体，点击 新…，然后点击 套料。

  • 在新建作业对话框中，设置装配体数量，点击 确定，然后根据需要更新作业详细信息（如到期日期和优先级）。

  • 系统从装配体创建作业，并将静态排样复制为作业排样，更新板材数量。复制的排样以作业 ID 为前缀。

  • 作业排样可以编辑或删除，而不会影响原始静态排样。

  • 作业排样可以使用 调整/紧凑… 或 添加零件… 重新打包。

  • 有关详细步骤，请参阅 Adapt/Compact 和 Add parts。

如果零件没有出现在排样中，请检查排样编辑对话框 - 您可以手动插入未排样的零件。

使用已保存的搜索和筛选器快速查找多个装配体中的零件。

使用清晰、描述性的装配体和排样名称，以避免在将排样复制到作业时出现混乱。
