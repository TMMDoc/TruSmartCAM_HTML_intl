安装

强制要求

首次启动 SolidWorks 时必须登录 TruSmartCAM 桌面应用程序。

确保 TruSmartCAM 监视器在 Windows 通知区域中运行。

如果 TruSmartCAM 桌面应用程序未打开，请验证 TruSmartCAM 服务器服务是否正在运行，以允许 SolidWorks 插件正常工作。如果两者均未激活，SolidWorks 将在启动时显示错误消息。

请按照以下步骤安装和设置 TruSmartCAM 的 SolidWorks 插件：

  1. 运行 TruSmartCAM 安装程序，在组件选择页面上启用 SolidWorks Extension。（如果 SolidWorks 安装在 TruSmartCAM 客户端上，请跳过服务器选项。）

  2. 完成安装，然后启动 TruSmartCAM 桌面应用程序，并使用有效的用户凭据登录。

  3. 启动 SolidWorks。

  4. 在 SolidWorks 中，转到 Tools → TruSmartCAM → About TruSmartCAM，以验证插件版本和详细信息。

  5. 要在启动时启用或禁用插件，请转到 Tools → Add-ins，并在 Other Add-ins 下找到 TruSmartCAM。
